## 🎓 毕业项目要求

### 项目目标

**做一个功能完整、能实际使用、能拿得出手的聊天机器人。**

这不只是一个作业，而是你学完这门课的作品集。以后你可以：
- 发给朋友看："看，我自己做的"
- 展示给客户："这是我做的案例"
- 放在简历里："独立完成AI客服机器人搭建"

### 必须完成的内容

```
✅ 必须完成：

1️⃣ 机器人搭建
   → 用扣子创建机器人
   → 配置完整的人设
   → 上传知识库（至少 30 个问答对）

2️⃣ 功能完整
   → 能回答常见问题
   → 能处理未知问题（转人工或说不知道）
   → 有明确的引导

3️⃣ 实际发布
   → 发布到至少 1 个渠道
   → 抖音 / 飞书 / 微信 / 网站
   → 能正常访问和使用

4️⃣ 测试验证
   → 测试 20 个以上问题
   → 记录准确率
   → 截图保存测试结果

5️⃣ 项目文档
   → 机器人定位（做什么的）
   → 知识库结构（包含哪些内容）
   → 使用场景（在哪里用）
```

### 毕业项目选题建议

```
🎯 入门级（新手推荐）：

项目1：个人IP机器人
→ 基于你自己的知识/技能
→ 回答粉丝相关问题
→ 发布到抖音/微信

项目2：店铺客服机器人
→ 基于你的产品/服务
→ 回答商品/价格/售后问题
→ 发布到微信/抖音

项目3：个人助理机器人
→ 日程提醒/待办事项/知识整理
→ 自己用，提高效率
→ 发布到飞书/微信

---

🚀 进阶级（有一定经验）：

项目4：企业知识库机器人
→ 基于公司文档
→ 员工自助查询
→ 发布到飞书/钉钉

项目5：培训考试机器人
→ 产品知识/销售培训
→ 在线考试+记录
→ 发布到飞书/网站

项目6：预约登记机器人
→ 服务预约/信息收集
→ 连接外部系统
→ 发布到微信/网站

---

💰 变现级（想赚钱的）：

项目7：完整电商客服系统
→ 商品+物流+售后全覆盖
→ 多渠道发布
→ 展示给潜在客户

项目8：垂直领域问答机器人
→ 法律/医疗/教育/金融
→ 专业资料库
→ 付费问答 or 获客引流

项目9：机器人SaaS服务
→ 标准化产品
→ 多客户使用
→ 按月收费
```
