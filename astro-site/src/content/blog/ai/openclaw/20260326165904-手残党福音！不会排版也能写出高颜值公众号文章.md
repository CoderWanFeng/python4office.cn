## 📝 零基础教程

### 第一步：打开工具

访问 https://python4office.cn/md2mp/

界面长这样：

```
┌─────────────────────────────────────┐
│  左边写文章    │    右边看效果      │
│               │                    │
│  # 标题        │    【标题】        │
│               │                    │
│  正文内容...   │    正文内容...     │
│               │                    │
└─────────────────────────────────────┘
```

**就这么简单。**

### 第二步：写你的文章

你只需要会这 5 个 Markdown 语法：

#### 1. 标题

```markdown
# 一级标题（文章大标题）

## 二级标题（章节标题）

### 三级标题（小节标题）
```

#### 2. 正文

直接写就行：

```markdown
这是一段普通的文字。

**这是加粗的文字**（重点内容）

*这是斜体的文字*（强调一下）
```

#### 3. 列表

```markdown
- 第一项
- 第二项
- 第三项
```

或者：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

#### 4. 引用

```markdown
> 这是一段引用文字
> 适合放金句或重点提示
```

#### 5. 代码

```markdown
```python
def hello():
    print("Hello, World!")
```
```

**就这 5 个！** 5 分钟就能学会。

### 第三步：复制发布

1. 点击"复制"按钮
2. 打开公众号后台
3. 新建图文
4. 粘贴（Ctrl+V）
5. 发布

**搞定！**

## 💡 常见问题

### Q1: Markdown 是什么？难学吗？

**答**：Markdown 是一种轻量级标记语言。

**难度**：比学 Excel 简单 10 倍。

只需要记住 5 个语法（上面已经教你了）。

**学习时间**：5-10 分钟。

### Q2: 我习惯了 Word，能直接用吗？

**答**：可以，但不建议。

Word 排版到公众号：
- 格式会乱
- 需要重新调整
- 效率低

建议直接学 Markdown，**一劳永逸**。

### Q3: 图片怎么插入？

**答**：

```markdown
![图片描述](图片链接)
```

图片链接可以是：
- 图床链接（推荐）
- 公众号后台上传后复制链接

### Q4: 写错了怎么办？

**答**：直接改！

Markdown 是纯文本，随时可以修改。

改完预览一下，没问题就复制。
