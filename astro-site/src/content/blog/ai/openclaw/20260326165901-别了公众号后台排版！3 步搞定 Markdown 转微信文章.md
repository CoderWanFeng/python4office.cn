## 📝 保姆级教程

### 步骤 1：打开编辑器

访问 https://python4office.cn/md2mp/

你会看到这样的界面：

```
┌─────────────────┬─────────────────┐
│   Markdown 编辑  │   实时预览      │
│                 │                 │
│   左边写代码     │   右边看效果    │
│                 │                 │
└─────────────────┴─────────────────┘
```

### 步骤 2：写你的文章

直接写 Markdown，和你平时一样：

```markdown
# 这是一级标题

## 这是二级标题

### 这是三级标题

这是一段普通文本，**加粗**和*斜体*都支持。

### 列表

- 项目 1
- 项目 2
- 项目 3

### 引用

> 这是一段引用文字

### 代码块

```python
def hello():
    print("Hello, World!")
```
```

**提示**：
- 标题用 `#` 表示，几个#就是几级标题
- 加粗用 `**文字**`
- 斜体用 `*文字*`
- 代码块用三个反引号 ```

### 步骤 3：配置样式

右侧面板可以调整：

| 配置项 | 推荐设置 |
|--------|----------|
| 字体大小 | 15px（阅读最舒适） |
| 代码主题 | 默认或暗黑（看你的文章风格） |
| 显示行号 | 勾选（技术文章必备） |
| 代码折叠 | 勾选（长代码不占空间） |

### 步骤 4：复制发布

1. 点击"复制"按钮
2. 打开公众号后台
3. 新建图文
4. **直接粘贴**（Ctrl+V）
5. 预览一下，没问题就发布

**搞定！** 整个过程不超过 5 分钟。

## 🔧 常见问题

### Q1: 图片怎么插入？

```markdown
![图片描述](图片 URL)
```

图片需要先上传到图床，或者用公众号后台的图片上传功能。

### Q2: 代码块没有高亮怎么办？

确保你在代码块开始处指定了语言：

```markdown
```python  # 这里指定语言
def hello():
    print("Hello")
```
```

支持的语言：python、javascript、java、cpp、go 等主流语言。

### Q3: 粘贴到公众号后格式乱了？

- 确保是**直接粘贴**，不要先粘贴到 Word 再复制
- 如果还是有问题，尝试用浏览器的"复制"功能

### Q4: 手机预览和电脑不一样？

这是正常的，公众号会自动适配手机屏幕。

只要电脑预览没问题，手机一般也没问题。
