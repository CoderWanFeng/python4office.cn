From Europe and America to Asia, multiple countries have closed their doors to it for security and privacy concerns. Here's the country list:

## List

| Country | Ban Status |
| ------- | ---------- |
| USA | US Congress banned employees from installing DeepSeek on official devices; US Navy banned members from using DeepSeek; NASA banned employees from using DeepSeek; Texas banned DeepSeek on government devices |
| Italy | Italy's data protection authority ordered DeepSeek to be blocked, citing concerns about how it handles personal data |
| Australia | The Australian government, for security reasons, banned government staff from installing and using DeepSeek's AI application, requiring the product to be deleted from all government systems and devices |
| South Korea | South Korea's Ministry of Defense banned DeepSeek from accessing its networked military computers; various departments and police also blocked work computers from accessing the application because DeepSeek didn't respond to data management requests |
| India | India's Ministry of Finance warned employees not to use AI tools like ChatGPT and DeepSeek, believing these tools could pose data protection risks to government confidential information |
| UK | The UK government, for security reasons, banned government staff from using DeepSeek |
| France | French regulators questioned DeepSeek, focusing on its data privacy issues |
| Germany | The German government, for security reasons, banned government staff from using DeepSeek |
| Netherlands | The Dutch government, for security reasons, banned government staff from using DeepSeek |
| Ireland | Ireland's Data Protection Commission required DeepSeek to provide more information, focusing on its data privacy issues |
| Belgium | The Belgian government, for security reasons, banned government staff from using DeepSeek |
| Japan | The Japanese government, for security reasons, banned government staff from using DeepSeek |
