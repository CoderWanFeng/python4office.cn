## 🎨 Gamma 详细教程（推荐！）

Gamma 是我目前最推荐的工具，设计感真的是 yyds。它不是传统 PPT 的那种"格子填文字"风格，而是更像做网页的感觉，看起来特别高级。

### 第一步：注册账号

打开 [gamma.app](https://gamma.app)，用 Google 账号或者邮箱注册。免费账号每个月有 10 次 AI 生成额度，用完就得等下个月或者付费。

> 💡 小技巧：注册多个免费邮箱可以白嫖更多次数（不推荐用于商业用途哦）

### 第二步：创建新 PPT

登录后点击 **"Create new"** → 选择 **"Generate"** 模式。

这里有三种创建方式：
1. **Generate（AI 生成）**：输入主题，AI 帮你从大纲到内容全包
2. **Import（导入生成）**：上传 Word 文档/文章链接，AI 自动提取内容生成 PPT
3. **Blank（空白创建）**：自己手动添加页面，AI 只负责辅助

**我推荐用 Generate 模式**，最省事。

### 第三步：写好提示词，这是关键！

提示词写得好不好，直接决定 PPT 的质量。下面给几个我自己常用的模板：


### 第四步：AI 生成大纲，先审再生成

Gamma 会先给你出一个大纲（Outline），这一步非常重要！**一定要先看大纲哪里不对，在这一步修改，比生成后改要方便 10 倍。**

常见需要调整的地方：
- 页面数量（太多或太少）
- 某些页面内容不准确
- 需要合并或拆分的页面

没问题了再点 **"Continue"**，AI 才会正式生成完整 PPT。

### 第五步：在线编辑，个性化调整

生成后的 PPT 可以直接在线编辑，支持：
- 增删页面
- 修改文字
- 换主题配色
- 调整布局
- 插入图片/图表/视频

> 💡 小技巧：点击左侧 **"Edit"** 进入编辑模式，点击右侧 **"Present"** 进入演示模式，两者不要搞混！

### 第六步：导出和分享

支持导出为：
- **PDF**：最通用，适合打印和邮件发送
- **PPTX**：可以在本地用 PowerPoint 打开编辑
- **分享链接**：一键生成分享 URL，对方无需登录即可查看

## 🎭 用 AI 美化现有 PPT

有时候你已经有了 PPT 内容，只是做得很丑或者排版很乱。AI 可以帮你重新设计。

### 场景 1：整体重新设计

**提示词**：
```
我有以下 PPT 内容：
[粘贴 PPT 全部文字内容]

请帮我重新设计这份 PPT，要求：
1. 现代商务风格，专业大气
2. 统一配色（建议蓝色系或深灰色系）
3. 每页布局优化，避免文字堆砌
4. 添加合适的图标和图示
5. 封面和结尾页要设计感强
6. 控制在 10-12 页

请给出详细的设计方案，包括：
- 每页的布局建议
- 推荐的图表类型
- 需要补充的图标/配图
- 配色方案
```

### 场景 2：生成配套图表

做 PPT 最烦的就是做图表。数据在那儿，但还要在 Excel 里画一遍。AI 可以帮你直接出图表建议：

**提示词 1：柱状图**
```
我有以下月度销售数据（单位：万元）：
1 月：100 / 2 月：120 / 3 月：90 / 4 月：150 / 5 月：180 / 6 月：200

请帮我设计一个柱状图：
- 配色：商务蓝色系
- 柱子顶部显示数字
- Y 轴从 0 开始
- 添加趋势线
- 标题：2026 上半年月度销售额走势
```

**提示词 2：饼图**
```
我有以下客户来源数据：
- 自然流量：35%
- 老客户转介绍：30%
- 百度竞价：20%
- 行业展会：10%
- 其他：5%

请设计一个饼图：
- 配色：渐变色系（从深到浅）
- 显示百分比
- 添加图例
- 标题：客户来源分布
```

**提示词 3：流程图**
```
请帮我生成一个销售流程图，展示以下步骤：
客户咨询 → 需求调研 → 方案制定 → 报价谈判 → 合同签订 → 项目启动 → 交付验收 → 售后跟进

要求：
- 横向流程图
- 每个步骤用图标表示
- 配色专业，适合商务汇报
- 箭头连接每一步
```

**提示词 4：对比表格**
```
请帮我设计一个竞品对比表格，对比维度如下：
- 功能完整性
- 价格
- 易用性
- 售后服务
- 客户口碑

对比产品：我们的产品 vs 竞品 A vs 竞品 B

要求：
- 用表格形式呈现
- 优势项标绿，劣势项标红
- 底部有综合推荐
```

### 场景 3：AI 生成演讲备注（超实用！）

很多 PPT 做好了，但演讲的时候不知道该说什么。AI 可以帮你生成演讲备注：

**提示词**：
```
我有以下 PPT 页面的内容：
[粘贴某一页 PPT 的全部文字]

请帮我生成这一页的演讲备注（Speaker Notes）：
- 字数：100-150 字
- 语气：口语化，像在跟观众聊天
```

```
当然，增长背后也有一些问题需要注意……（过渡到下一页）
```

## ✍️ PPT 内容优化

PPT 文字太啰嗦？AI 帮你精简。

### 场景 1：把长文字压缩成 PPT 要点

**提示词**：
```
请帮我优化以下 PPT 文字，精简到核心要点，适合演讲展示：

原文字：
"我们公司在今年第一季度取得了非常不错的销售业绩，总共完成了 500 万元的销售额，相比去年同期增长了 20% 左右，这是一个非常令人鼓舞的成绩。特别值得一提的是，我们的华东区表现特别突出，贡献了 200 万的销售额，占总体的 40%，同比增长高达 35%，远超其他区域。"

要求：
- 精简到 50 字以内
- 突出关键数据（500万、20%、35%）
- 适合大字体 PPT 展示
- 语言简洁有力
```

**AI 返回示例**：
```
2026 Q1 · 开门红
• 销售额：500 万（↑20%）
• 华东区：200 万（↑35%），占整体 40%
• 团队执行力显著提升
```

### 场景 2：写有感染力的 PPT 开头

**提示词**：
```
请帮我写一个 PPT 开头页的文案，要求：

主题：数字化转型是中小企业的必答题
风格：引起紧迫感，但不制造焦虑
受众：中小企业老板
要求：
- 一句话点明为什么要听这页
- 留一个悬念，引出正文
- 30 字以内，掷地有声
```

**AI 返回示例**：
```
"不是要不要转型，是再不转，你的竞争对手就转完了。"
```

### 场景 3：写有号召力的 PPT 结尾

**提示词**：
```
请帮我写一个 PPT 结尾页的文案，要求：

主题：感谢聆听 / 欢迎交流
风格：温暖有力，不俗套
受众：同事/领导/客户
要求：
- 表达感谢 + 期待下一步
- 有明确的行动号召
- 50 字以内
```

## 🔥 进阶技巧：如何用 AI 做出让人眼前一亮的 PPT？

### 技巧 1：先写内容大纲，再让 AI 匹配设计

很多同学的做法是：把提示词往 AI 里一扔，出来什么就用什么。

我的做法是：**先用 AI 生成大纲，自己过一遍，改好之后，再生成 PPT。**

这样做的好处是：内容的逻辑顺序、重点的取舍，都在你的掌控之中。AI 生成的内容不一定完全符合你的业务实际，需要你来做判断。

### 技巧 2：Gamma 的 "Import" 功能超好用

如果你已经有一篇写好的文章或者文档，不要自己手动复制粘贴到 PPT 里。直接把文档/链接扔给 Gamma 的 Import 功能，AI 会自动：
- 理解文档结构
- 提取关键内容
- 重新组织成 PPT 大纲
- 生成配套的页面设计

这比你"复制粘贴 → 调格式"能省至少 30 分钟。

### 技巧 3：WPS AI 的"演讲者模式"

WPS 有个很实用的功能：演讲者模式。在这个模式下，你可以在演讲时看到当前页的备注、下一页的预览、以及计时器。AI 还可以根据你的演讲时长，智能建议每页的分配时间。

### 技巧 4：配色不要超过 3 种

AI 生成 PPT 有一个常见问题：颜色太多太花。**建议整个 PPT 只用 2-3 种颜色：**
- 主色（标题、重要数据）：1 种
- 辅色（图标、次级文字）：1 种
- 背景色（白/浅灰）：1 种

在 Gamma 和 WPS 里都可以一键更换主题（Theme），换到满意为止。

## ⚠️ 避坑指南

### 坑 1：提示词太模糊，AI 生成跑偏

❌ 错误示范：
```
帮我做个 PPT
```

✅ 正确做法：
```
帮我生成一个 10 页的季度工作总结 PPT，包含业绩数据、问题分析、下季度计划，用商务风格
```

**原则：提示词越具体，输出越精准。**

### 坑 2：AI 生成后直接用，不检查内容

AI 有时候会"一本正经地胡说八道"，特别是数据。**一定要逐页核对数据准确性！** 我有一次就差点闹笑话——AI 给我生成的数据中，增长比例居然超过了 100%，我核对才发现算错了。

### 坑 3：页数太多，听众记不住

PPT 不是越长越好。**10-15 页是黄金长度。** 如果内容很多，优先保重点，不要什么都往上放。听众在会议结束后能记住 3 个要点，这页 PPT 就是成功的。

### 坑 4：全文字，没有图表

PPT 上全是字是大忌。再好的内容，如果全是文字，读的人也会睡着。**每页至少有一个图表/图标/图片。** 如果某页真的只能用文字，那文字不要超过 5 行，每行不要超过 10 个字。

### 坑 5：导出格式选错

分享 PPT 时，默认选 PPTX 格式（对方可以用 PowerPoint/WPS 打开编辑）。如果只是给对方看，不希望对方修改，选 PDF 格式。

## 💪 实战案例：5 分钟做完一份会议汇报 PPT

我来演示一个真实场景：公司月度经营会议，需要做一份 10 页的汇报 PPT。

**原始数据**（你只有这些）：

- 3 月净利润：36 万，净利率 20%
- 3 个大客户签约
- 团队 8 人
- 主要问题：库存不足

**用 AI 的实操步骤：**

```
提示词（完整版）：
帮我生成一份月度经营会议汇报 PPT，包含：
1. 封面：2026 年 3 月经营月报
2. 核心指标：销售额 180 万（↑15%）、净利润 36 万（↑20%）
3. 区域表现：华东/华南/华北三个大区数据
4. 重点客户：TOP5 客户贡献 45%
5. 团队情况：8 人，招聘进度
6. 问题与挑战：库存不足导致部分订单延迟
7. 应对措施：已启动紧急采购，预计 4 月第一周解决
8. 4 月目标：销售额 200 万
9. 需要的支持：增加库存预算
10. Q&A 结尾页

风格：商务简洁，数据图表突出，10-12 页
```

生成后我在 Gamma 里：
1. 换了一个深蓝色主题
2. 把数字放大、加粗、标色
3. 检查了每页的数据（果然有一处百分比 AI 算错了，我自己改了）
4. 导出 PDF，发送给行政排版

**总耗时：11 分钟。** 自己手工做的话，至少 2 小时起步。
