## 四、参赛规则

### 技术要求

- 使用本地 OpenClaw 或者 AtomGit 龙虾环境开发均可
- 优先支持 Python 技术栈，使用 Python 可获加分

### 作品方向（无强制限定，实用/创意优先）

- AI 代码生成/辅助开发工具
- AI 文档处理、智能总结、知识库助手
- Python + AI 自动化脚本、效率工具
- AI 数据可视化、小工具、交互应用等创新项目

### 原创要求

- 禁止抄袭、盗用、多次获奖项目参赛
- 一经发现作弊，立即取消资格与成绩
- 最终解释权归大赛组委会所有

## 五、评分标准（总分 100 分 + 特别加分 10 分）

- AI 工具使用深度（20 分）：龙虾环境、大模型、OpenClaw 使用程度与创新
- 创意创新（20 分）：项目新颖度、技术突破、解决问题的思路
- 实用价值（20 分）：落地可行性、场景刚需、用户体验
- 项目完成度（20 分）：功能完整、运行稳定、可演示
- 技术难度（10 分）：技术门槛、架构合理性、代码规范
- 特别加分（10 分）：使用 Python 技术栈 + 贴合 AtomGit 生态
