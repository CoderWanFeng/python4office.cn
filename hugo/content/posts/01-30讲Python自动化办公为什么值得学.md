## Python自动化办公解决什么

### 1行代码 = 你2小时的工作

比如合并100个Excel文件：
```python
import office
office.excel.concat(r'./文件夹')
```

一行代码，10秒搞定。

比如批量重命名1000个文件：
```python
import office
office.file.replace4filename(r'./文件夹', '旧名字', '新名字')
```

还是一行代码。
