## 🛠️ 配置教程（真·5 分钟）

### 第一步：下载打包好的工具包

**下载地址**：（私信我获取，或参加培训课程）

**包含内容**：
- 可执行文件（.exe）
- 配置文件
- 示例发票
- 使用说明

**不用安装 Python**，不用配置环境，下载就能用。

### 第二步：配置密钥

**打开 `config.txt`**：

```
# 腾讯云密钥配置
SecretId=你的 SecretId
SecretKey=你的 SecretKey
```

**密钥怎么来**？
1. 访问 https://curl.qcloud.com/9ExTmaya
2. 注册腾讯云账号
3. 搜索"文字识别"
4. 控制台 → 密钥管理 → 新建密钥
5. 复制粘贴到 config.txt

**保存**。

### 第三步：准备发票

**创建一个文件夹**，比如：

```
C:/发票识别/待识别/
```

**把发票放进去**：
- PDF 可以
- JPG 可以
- PNG 可以
- 混着放也可以

### 第四步：双击运行

**双击 `运行.bat`**：

```
正在识别发票...
已处理：1/50
已处理：2/50
已处理：3/50
...
✅ 识别完成！
结果保存在：C:/发票识别/结果.xlsx
```

**打开 Excel**，所有发票信息整整齐齐。

**完成**。

## ⚠️ 常见问题（FAQ）

### Q1：真的不用安装任何东西吗？

**A**：不用。

工具包已经打包好了所有依赖，下载解压就能用。

**唯一需要**：Windows 系统（Mac 版在开发中）。

### Q2：安全吗？会不会有病毒？

**A**：安全。

- 代码开源，可以审查
- 只是调用腾讯云 API
- 不收集任何数据

**如果你担心**：可以找懂技术的朋友帮忙看看代码。

### Q3：密钥安全吗？

**A**：密钥存在本地 config.txt 里。

**安全建议**：
- 不要发给别人
- 不要上传到网上
- 定期更换

### Q4：识别错了怎么办？

**A**：有 3 种情况：

1. **图片模糊**：重新拍照，确保清晰
2. **发票特殊**：手动修正（Excel 里直接改）
3. **系统问题**：群里提问，我们帮你解决

**准确率**：98%+，2% 的错误可以手动修正。

### Q5：一个月能识别多少张？

**A**：腾讯云免费额度 1000 次/月。

**超出后**：0.035 元/次。

**大部分用户**：免费额度够用。
