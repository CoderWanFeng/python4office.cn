🔍 风险识别

1. 代码审查

检查点：

  • 敏感操作（文件删除、网络请求）
  • 权限请求（是否过度）
  • 第三方依赖（是否可信）

2. 行为监控

监控点：

  • 网络请求目标
  • 文件访问范围
  • API 调用频率

3. 权限分析

分析点：

  • 必要权限 vs 过度权限
  • 权限使用记录
  • 权限变更历史

🛡️ 审计工具

  1. Snyk Skill Security Scanner
  2. Agent Trust Hub
  3. VirusTotal

✅ 安全检查清单

  • 代码审查通过
  • 无恶意行为
  • 权限合理
  • 依赖安全
  • 有维护记录
