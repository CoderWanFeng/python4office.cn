1. 功能介绍

今天我们介绍这个库的功能之一：

pdf转word文档: 只需要一行代码，实现用url进行图片下载。

2. 使用说明

下载python-office

只需要下面这一条命令，就可以自动下载和安装python-office

pip install python-office

调用功能

照抄下面代码，修改文件存放位置，右键选择运行

import office # 导入python-office

office.image.down4img(url='https://profile-avatar.csdnimg.cn/c7ea241d22e84bb1a9b8e929f8e35c6c_qq_41813454.jpg!1',
                      output_path=r'.\test_files\30-09-down4img')
# 参数解释：
# url：需要下载的图片的链接
# output_path: 下载后的图片存放位置
