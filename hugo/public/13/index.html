每天要发几十封重复的邮件？

每周要给客户发周报邮件？

今天教你怎么用 python-office 自动发送邮件。

1、发送第一封邮件

import office

# 1行代码发送邮件
office.email.send(
    email='收件人@example.com',
    title='测试邮件',
    content='你好，这是程序员晚枫发的测试邮件。'
)

运行后，收件人会收到一封邮件。

🎉 Done！自动发邮件成功！

2、发送带附件的邮件

import office

# 发送带附件的邮件
office.email.send(
    email='收件人@example.com',
    title='周报',
    content='这是本周的周报，请查收。',
    attachment='周报.xlsx'
)

3、发送HTML格式的邮件

想让邮件更美观？用 HTML 格式：

import office

html_content = '''
<html>
<body>
<h2>程序员晚枫的邮件</h2>
<p>这是一封<strong>格式化</strong>的邮件。</p>
<p>你可以添加图片、超链接等。</p>
</body>
</html>
'''

office.email.send(
    email='收件人@example.com',
    title='HTML邮件',
    content=html_content,
    type='html'  # 指定为HTML格式
)

4、群发邮件

一次发给多个人：

import office

# 收件人列表
emails = [
    '客户1@example.com',
    '客户2@example.com',
    '客户3@example.com'
]

for email in emails:
    office.email.send(
        email=email,
        title='产品更新通知',
        content='尊敬的用户，我们的产品已更新，欢迎体验。'
    )

print(f'已发送 {len(emails)} 封邮件！')

5、实战案例：每日自动发送数据报表

每天早上9点自动发送数据报表：

import office
from datetime import date

def send_daily_report():
    """发送每日报告"""
    today = date.today().strftime('%Y-%m-%d')
    
    # 生成或读取当日报表
    report_file = f'报表_{today}.xlsx'
    
    # 发送邮件
    office.email.send(
        email='领导@example.com',
        title=f'【日报】{today} 数据报告',
        content=f'''领导好，

这是 {today} 的数据报告，请查收附件。

祝工作顺利！
程序员晚枫''',
        attachment=report_file
    )
    
    print(f'{today} 日报已发送！')

# 运行
send_daily_report()

7、常见问题

Q：发送失败，提示"认证失败"？

A：需要在邮箱设置里开启 SMTP 服务，获取授权码。QQ邮箱、163邮箱、企业邮箱都支持。

Q：想用企业邮箱发？

A：加上服务器配置：

office.email.send(
    email='收件人@example.com',
    title='标题',
    content='内容',
    smtp_server='smtp.你的企业.com',
    smtp_port=465,
    smtp_user='your@email.com',
    smtp_password='your_password'
)

Q：发送太慢？

A：用多线程并发发送：

from concurrent.futures import ThreadPoolExecutor

with ThreadPoolExecutor(max_workers=5) as executor:
    for customer in customers:
        executor.submit(office.email.send, customer['email'], '标题', '内容')
