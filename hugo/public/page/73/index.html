🎬 开篇：让类的实例像函数一样调用

你有没有想过，为什么这样写是合法的？

class Counter:
    def __init__(self):
        self.count = 0
    
    def __call__(self):
        self.count += 1
        return self.count

counter = Counter()
print(counter())  # 1 - 像"函数"一样调用！
print(counter())  # 2
print(counter())  # 3

这就是 __call__ 方法的魔力：让对象实例可以像函数一样被调用。
