安装很简单，在有python环境的电脑上，只需要执行下面这一行命令。

2. PDF合并

直接上代码！

代码

# 导入这个库：python-office，简写为office
import office

office.image.pencil4img(input_img=r'D:\workplace\code\test\down4img\girl.jpg')


#参数作用：
# input_img = 原图片的名字
