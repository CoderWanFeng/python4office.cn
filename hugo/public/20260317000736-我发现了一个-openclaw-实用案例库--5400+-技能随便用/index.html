🔧 如何使用

方式一：ClawHub 安装

# 安装技能
clawhub install <skill-slug>

方式二：手动安装

全局安装：

# 复制到全局技能目录
cp -r skill_folder ~/.openclaw/skills/

工作区安装：

# 复制到项目技能目录
cp -r skill_folder <project>/skills/

优先级：工作区 > 本地 > 内置

方式三：直接使用

最简单的方式：

把技能的 GitHub 链接发给你的 AI 助手，它会自动处理安装。
