白开水 AI AI 导航，使用教程，新手入门

大家好，我是正在实战各种 AI 项目的程序员晚枫。

白开水 AI 导航站发布了（https://www.python4office.cn/ai-nav/），但很多人不知道怎么用。

今天这篇保姆级教程，带你从零开始，学会使用导航站。

📋 准备工作

1. 设备要求

项目 要求
设备 电脑/手机/平板
浏览器 任意现代浏览器
网络 可访问外网（部分工具需要）

2. 账号准备

部分工具需要注册账号：

  • 邮箱（推荐 Gmail、Outlook）
  • 手机号（部分国内工具）
  • 支付方式（付费工具）

🌐 访问导航站

步骤 1：打开官网

浏览器访问：https://www.python4office.cn/ai-nav/

步骤 2：浏览首页

首页展示：

  • 8 大分类入口
  • 热门推荐工具
  • 最新收录工具
  • 搜索框

步骤 3：使用搜索

在搜索框输入关键词：

  • 工具名称（如"ChatGPT"）
  • 功能（如"图像生成"）
  • 场景（如"编程"）

📖 查看工具详情

每个工具包含以下信息：

1. 基本信息

  • 工具名称
  • 官方 Logo
  • 简短介绍

2. 详细信息

  • 完整功能描述
  • 适用场景
  • 技术特点

3. 使用信息

  • 官方网站链接
  • 价格信息
  • 使用难度
  • 推荐指数

4. 补充信息

  • 同类工具推荐
  • 使用技巧
  • 注意事项

🔧 使用导航站找工具

场景 1：想找 AI 聊天工具

步骤：

  1. 点击"大模型"分类
  2. 浏览列表
  3. 对比 ChatGPT、Claude、文心一言
  4. 选择适合的点击官网链接

场景 2：想找编程辅助工具

步骤：

  1. 点击"AI 编程"分类
  2. 查看 OpenClaw、Copilot、Cursor
  3. 对比功能和价格
  4. 选择并前往官网

场景 3：想做图片生成

步骤：

  1. 点击"图像处理"分类
  2. 查看 Midjourney、SD、DALL-E 3
  3. 对比风格和价格
  4. 注册并开始使用

场景 4：想提升办公效率

步骤：

  1. 点击"办公效率"分类
  2. 查看 Notion AI、Grammarly 等
  3. 选择符合需求的
  4. 开始使用

📚 进阶使用技巧

1. 善用筛选

根据以下条件筛选：

  • 价格（免费/付费）
  • 难度（入门/进阶）
  • 推荐指数（4 星以上）

2. 关注更新

  • 每周新增工具
  • 价格变动提醒
  • 新功能上线

3. 对比选择

同类工具对比维度：

  • 功能完整性
  • 价格合理性
  • 使用难度
  • 用户评价

4. 收藏工具

遇到好工具可以：

  • 浏览器书签
  • 记录到笔记
  • 分享给朋友

🎯 不同人群的使用建议

完全新手

推荐分类：大模型、办公效率

学习路径：

  1. 从 ChatGPT 开始
  2. 学习基本对话
  3. 尝试办公工具
  4. 逐步探索其他

时间投入：每天 30 分钟，1 周入门

办公人士

推荐分类：办公效率、数据分析

学习路径：

  1. Notion AI 整理笔记
  2. Grammarly 检查文档
  3. 数据分析工具处理报表
  4. 提升工作效率

时间投入：按需使用，立竿见影

开发者

推荐分类：AI 编程、大模型

学习路径：

  1. OpenClaw 开发 Agent
  2. Copilot 辅助编程
  3. 大模型 API 集成
  4. 构建 AI 应用

时间投入：持续学习，长期投入

设计师

推荐分类：图像处理、视频生成

学习路径：

  1. Midjourney 生成素材
  2. Runway 制作视频
  3. 学习提示词工程
  4. 提升创作效率

时间投入：每周 5 小时，1 月入门

❓ 常见问题

Q1: 导航站是免费的吗？

A: 导航站本身完全免费，但部分工具需要付费。

Q2: 工具链接安全吗？

A: 所有链接都是官方网站，可以放心点击。

Q3: 如何推荐新工具？

A: 联系白开水 AI 社区，提交工具信息。

Q4: 工具价格准确吗？

A: 尽力保持更新，但请以官网为准。

Q5: 有手机版吗？

A: 导航站支持响应式，手机可以正常访问。

📈 学习计划

第 1 天：熟悉导航站

  • 浏览所有分类
  • 了解工具分布
  • 收藏感兴趣的

第 2-3 天：试用工具

  • 选择 2-3 个工具
  • 注册并试用
  • 记录使用感受

第 4-7 天：深入学习

  • 选择 1 个主力工具
  • 系统学习功能
  • 应用到实际工作

第 2 周：拓展探索

  • 尝试其他分类
  • 发现新工具
  • 建立工具组合

第 3-4 周：分享反馈

  • 总结使用经验
  • 推荐给有需要的人
  • 反馈给导航站
