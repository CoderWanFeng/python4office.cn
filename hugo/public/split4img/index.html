代码说明

实现这个功能的是第三方库：poimage，下载命令如下：

pip install poimage

下载以后，切割图片只需要下面这1行代码：


import poimage

poimage.split4img(img_path=r"./imgs/icon2.jpg", output_path=r'D:\程序员晚枫的文件夹\output')

参数说明：

  • img_path：需要切割的图片，存放路径；
  • output_path：切割后的图片，存放在哪里。
