第 8 讲：SEO 基础入门 - 让 Google 主动给你送流量

2026-03-02 14:35
个人网站，SEO 搜索引擎优化，关键词

大家好，我是正在实战各种AI 项目的程序员晚枫。

🎯 一个让我震惊的数据

我有个学员，网站上线6个月后，每月有3000+访客。我问他最大的流量来源是什么？

Google搜索，占比70%。

而且这些搜索流量是"被动"的——他每天睡觉的时候，都有人在Google搜到他的文章，然后访问他的网站。

SEO是个人网站最被低估的流量来源。 它不需要你每天发朋友圈、不需要投广告、不需要任何社交运营——你只需要把内容写好，Google就会持续给你送流量。

但SEO需要时间。今天开始做SEO，3-6个月后你才会看到效果。 所以，越早开始越好。

什么是SEO？

SEO（Search Engine Optimization）：搜索引擎优化

简单说：让你的网站在Google搜索结果中排名更靠前。

为什么SEO最重要？

搜索流量的价值：

维度 SEO流量 社交媒体流量 付费广告
成本 免费 免费（但花时间） 付费
精准度 高（主动搜索） 低（被动刷到） 高（定向投放）
持续性 长期（几年） 短期（几天） 停止即停
被动性 睡着也有 需要运营 需要投钱

数据说话：

  • Google搜索结果第1名：点击率31.7%
  • 第2名：24.7%
  • 第3名：18.7%
  • 第10名：只有3.1%

结论：排名越靠前，流量越多。SEO就是让你排名靠前的技术。

SEO核心概念

1. 关键词（Keyword）

用户搜索时输入的词，比如：

  • “心理咨询师”
  • “焦虑症症状”
  • “如何缓解压力”

关键词分类：

  • 核心关键词：2-3个字，搜索量大，竞争大（如"心理咨询"）
  • 长尾关键词：4+个字，搜索量小，竞争小（如"重庆心理咨询师推荐"）
  • 问题型关键词：“如何…““为什么…““怎么…"（如"焦虑症怎么自测”）

2. 搜索意图（Search Intent）

用户搜索背后的目的：

  • 信息型：想了解某事（“什么是焦虑症”）→ 写科普文章
  • 导航型：想找特定网站（“简单心理官网”）→ 你无法优化这类词
  • 交易型：想购买（“心理咨询预约”）→ 写服务介绍页

3. 页面权重（Page Authority）

Google认为你的页面有多重要，取决于：

  • 内容质量（深度、原创性）
  • 外部链接数量（其他网站链接到你）
  • 用户停留时间
  • 网站整体权威性

关键词研究

用AI找关键词

我是 [职业/行业]，目标用户是 [用户画像]。

请帮我列出50个他们可能会搜索的关键词，包括：
1. 核心关键词（2-3个字）
2. 长尾关键词（4个字以上）
3. 问题型关键词（"如何…""为什么…"）
4. 地域型关键词（"[城市]+[服务]"）

按搜索量从高到低排序，并标注竞争等级（高/中/低）

关键词工具推荐

免费工具：

  • Google关键词规划师（需要广告账户）
  • Ubersuggest（免费版每天3次）
  • AnswerThePublic（问题型关键词）
  • Google搜索建议（直接看下拉词）

付费工具：

  • Ahrefs（$99/月，最强大）
  • SEMrush（$119/月）
  • Moz Pro（$99/月）

国内工具：

  • 5118.com
  • chinaz.com
  • aizhan.com

如何选择关键词？

好关键词的标准：

  • ✅ 有搜索量（至少100次/月）
  • ✅ 竞争度低（新手能做）
  • ✅ 与你的业务相关
  • ✅ 有商业价值（能转化）

新手策略：

  1. 先做长尾关键词（竞争小，容易排名）
  2. 积累权重后再做核心词
  3. 每个页面聚焦1-2个关键词

页面SEO优化

1. 标题优化（Title Tag）⭐⭐⭐⭐⭐

标题是SEO最重要的因素之一。

优化技巧：

  • 包含目标关键词
  • 控制在60个字符以内
  • 吸引点击（加入数字、疑问、承诺）

示例：

❌ 差：心理咨询
✅ 好：心理咨询师推荐：10年经验的真心建议
✅ 好：焦虑症怎么办？专业心理咨询师告诉你5个方法

2. 描述优化（Meta Description）⭐⭐⭐

描述不影响排名，但影响点击率。

示例：

❌ 差：本网站提供心理咨询服务
✅ 好：有焦虑情绪怎么办？10年资深心理咨询师分享5个实用方法，
帮你快速缓解焦虑。免费自测量表+预约入口。

3. 内容优化⭐⭐⭐⭐⭐

  • 关键词自然出现（密度1-2%，不要堆砌）
  • 小标题包含关键词（H2/H3标签）
  • 内容要有深度（1500字以上）
  • 加入图片、视频等多媒体
  • 内部链接（链接到其他相关文章）
  • 外部链接（链接到权威来源）

4. URL优化⭐⭐

❌ 差：website.com/?p=123
✅ 好：website.com/anxiety-symptoms

5. 图片优化⭐⭐

<!-- 差 -->
<img src="IMG_1234.jpg">

<!-- 好 -->
<img src="anxiety-symptoms-chart.jpg" alt="焦虑症症状图表：5个常见症状对比">

用AI优化SEO

优化文章标题

我写了一篇关于"[主题]"的文章。

请帮我生成10个SEO友好的标题，要求：
1. 包含关键词"[关键词]"
2. 控制在60个字符以内
3. 吸引点击（加入数字、疑问、承诺等）
4. 每个标题说明使用了什么心理学技巧

优化文章内容

请帮我优化这篇文章的SEO：
1. 检查关键词密度（目标1-2%）
2. 建议在哪里加入关键词
3. 优化小标题（加入关键词）
4. 建议内部链接位置（3-5个）
5. 建议外部链接来源（2-3个权威网站）
6. 生成Meta Description

技术SEO基础

1. 网站速度

检查工具：pagespeed.web.dev

优化建议：

  • 压缩图片（用TinyPNG）
  • 使用CDN（Cloudflare免费）
  • 减少插件（WordPress用户注意）
  • 启用缓存

2. 移动端适配

Google优先索引移动版网站。用手机访问你的网站，确保正常显示。

3. HTTPS

确保网站使用HTTPS。大部分主机商免费提供SSL证书。

4. 网站地图（Sitemap）

帮助Google发现你网站的所有页面。

生成工具：

  • XML-Sitemaps.com（免费）
  • WordPress插件：Yoast SEO（自动生成）

5. robots.txt

告诉Google哪些页面可以抓取，哪些不可以。

User-agent: *
Allow: /
Sitemap: https://yourdomain.com/sitemap.xml

提交Google

Google Search Console设置

  1. 打开 search.google.com/search-console
  2. 添加你的网站
  3. 验证网站所有权（DNS验证或HTML标签）
  4. 提交Sitemap
  5. 等待Google抓取（1-7天）

验证是否被收录

在Google搜索框输入：site:yourdomain.com

如果看到结果，说明已经被收录了。

避坑指南：SEO的5个误区

误区1：关键词堆砌

❌ “心理咨询_心理咨询师_心理咨询价格_心理咨询推荐”

✅ 关键词自然融入内容，密度1-2%就够了

误区2：买外链

❌ 花钱买1000个外链

✅ Google惩罚买外链的行为。做有价值的内容，自然获得外链

误区3：只优化首页

❌ 所有精力都放在首页

✅ 每个页面都优化，长尾流量才是大头

误区4：抄袭内容

❌ 复制别人的文章，稍微改改

✅ Google会惩罚重复内容。必须原创或深度改写

误区5：期望快速见效

❌ “我做了1周SEO，为什么没流量？”

✅ SEO是长期工作，3-6个月才见效。但一旦见效，流量稳定且免费

实战：一篇SEO优化文章的完整流程

  1. 关键词研究：用AI或工具找到目标关键词
  2. 分析竞品：搜索关键词，看前3名怎么写的
  3. 写更长的内容：如果竞品写了1500字，你就写2000字
  4. 加入独特价值：竞品没有的案例、数据、观点
  5. 优化标题和描述：包含关键词，吸引点击
  6. 加入内部链接：3-5个相关文章链接
  7. 优化图片：Alt标签包含关键词
  8. 发布后提交：在Search Console提交URL
