你有没有遇到过这种尴尬：

“我写了个Python脚本，发给同事用。” “同事：我没有Python环境啊……”

PyInstaller帮你解决这个问题。

一行命令打包成exe

pip install pyinstaller
pyinstaller --onefile your_script.py

生成的exe文件，双击就能运行，不需要安装Python。
