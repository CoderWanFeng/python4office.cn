每天要下载邮件里的附件？

客户发来的文件要一个个保存？

今天教你怎么用 python-office 自动收取邮件并下载附件。

1、读取最新邮件

import office

# 读取最新邮件
emails = office.email.read(lastest=5)

for email in emails:
    print(f'发件人: {email["from"]}')
    print(f'标题: {email["subject"]}')
    print(f'时间: {email["date"]}')
    print(f'内容: {email["content"]}')
    print('---')

运行后，会打印出最近 5 封邮件的内容。

2、自动下载附件

import office

# 读取邮件并下载附件
office.email.download(
    save_path='C:\\邮件附件',  # 保存路径
    lastest=10  # 最近10封
)

运行后，所有邮件的附件都会下载到指定文件夹。

5、实战案例：自动归档客户邮件

把每个客户的邮件归档到对应文件夹：

import office
import os
import re

def archive_emails():
    """归档邮件到对应文件夹"""
    emails = office.email.read(lastest=50)
    
    for email in emails:
        # 提取发件人邮箱
        sender = email['from']
        
        # 创建文件夹
        folder = os.path.join('邮件归档', sender.replace('@', '_at_'))
        os.makedirs(folder, exist_ok=True)
        
        # 保存邮件内容
        filename = f"{email['date'].replace(':', '-')}_{email['subject']}.txt"
        filepath = os.path.join(folder, filename)
        
        with open(filepath, 'w', encoding='utf-8') as f:
            f.write(f"发件人: {email['from']}\n")
            f.write(f"标题: {email['subject']}\n")
            f.write(f"时间: {email['date']}\n")
            f.write(f"\n内容:\n{email['content']}")
        
        print(f'已归档: {email["subject"]}')

archive_emails()

6、实战案例：自动下载客户订单附件

每天自动下载订单邮件的附件：

import office
import os
from datetime import date

def download_orders():
    """下载订单邮件附件"""
    today = date.today().strftime('%Y-%m-%d')
    save_path = f'订单_{today}'
    os.makedirs(save_path, exist_ok=True)
    
    # 读取今天的订单邮件
    emails = office.email.read(
        subject='订单',  # 按标题筛选
        lastest=20
    )
    
    count = 0
    for email in emails:
        # 下载附件
        office.email.download(
            save_path=save_path,
            email_id=email['id'],
            file_type='.xlsx'  # 只下载Excel
        )
        count += 1
    
    print(f'今日共下载 {count} 个订单附件')

download_orders()

7、常见问题

Q：读取邮件报错？

A：需要在邮箱设置里开启 IMAP 服务，获取授权码。

Q：下载的附件名是乱码？

A：用 decode_filename=True 参数可以解决中文文件名乱码问题。

Q：想读取指定日期的邮件？

A：

emails = office.email.read(
    start_date='2024-01-01',
    end_date='2024-01-31'
)
