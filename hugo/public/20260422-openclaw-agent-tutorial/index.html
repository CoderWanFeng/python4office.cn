OpenClaw怎么用？

今天我给你手把手讲清楚。

我是程序员晚枫，全网30万+粉丝，python-office开源作者。

OpenClaw是腾讯出的开源AI Agent平台，免费使用，支持国内模型，适合从0开始学。

📦 第一步：安装OpenClaw

先安装OpenClaw CLI工具。

步骤：

  1. 确保你的电脑已经安装了Node.js
  2. 打开终端，运行安装命令：
npm install -g @openclaw/cli@latest
  1. 验证安装：
openclaw --version

如果输出版本号，说明安装成功了。

🔧 第二步：配置模型

OpenClaw支持很多模型，推荐用DeepSeek或智谱AI。

步骤：

  1. 初始化配置：
openclaw init

  1. 按照提示配置你的API Key

  2. 也可以手动编辑配置文件

🚀 第三步：创建你的第一个Agent

现在创建你的第一个AI Agent。

步骤：

  1. 创建一个新项目：
openclaw create my-first-agent
  1. 进入项目目录：
cd my-first-agent
  1. 编辑Agent配置文件，设置你的指令和工具

💡 第四步：测试Agent

创建好了，测试一下。

步骤：

  1. 运行Agent：
openclaw run

  1. 输入你的指令，比如：帮我整理这个文件夹里的图片，按日期分类

  2. 看Agent的执行过程和结果

🎯 第五步：优化Agent

测试通过了，现在优化一下。

优化方向：

  1. 添加更多工具
  2. 优化指令
  3. 添加记忆功能
  4. 优化输出格式
