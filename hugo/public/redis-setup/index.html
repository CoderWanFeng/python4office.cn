CentOS8下Redis的安装

2022-02-07 14:11
CentOS Linux Redis

redis可以使用yum安装，也可以使用源码的方式安装 本次使用yum安装。

源码安装教程：https://www.runoob.com/redis/redis-install.html

  1. 首先修复yum 最近项目使用的服务器系统是CentOS8，在使用yum安装redis时发现yum已经不能用了。

修复方式，来自阿里云 · 开发者社区：https://developer.aliyun.com/mirror/centos?spm=a2c6h.13651102.0.0.2ab21b11a5xMh1

  1. 然后安装redis
yum install redis
  1. 启动
cd /usr/bin #进入redis的安装目录
./redis-server /etc/redis.conf #启动redis
./redis-cli #进入redis的命令行界面
  1. 配置redis的密码 因为项目安全检查的需要，所以这里给redis配置一个密码。
CONFIG get requirepass #查看密码
CONFIG set requirepass "runoob" #设置密码
AUTH password #使用密码登录
  1. redis链接方式
redis://[:password]@host:port/db
  1. 远程连接redis的几个问题
bind 0.0.0.0 #设置host为所有ip
requirepass yourpassword #这里的注释去掉，否则redis虽然正常可用，但是pycharm链接不上。
