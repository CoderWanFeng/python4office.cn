📚 课程核心：为什么叫"龙虾"？

名字由来

龙虾 = L.O.X.P. = Learn Open-source X Python

寓意：
- L：Learning（持续学习）
- O：Open-source（开源精神）
- X：Expert（专家级）
- P：Python（Python 生态）

课程定位

维度 定位
目标人群 1-3 年 Python 开发者
前置要求 掌握 Python 基础，有项目经验
课程目标 从"会用"到"精通"
学习周期 8 周，每周 10 小时
课程形式 直播 + 录播 + 实战 + 答疑
