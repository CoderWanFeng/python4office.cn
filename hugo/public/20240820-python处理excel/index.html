学习计划

1、基础

Python基础学完有2个作用：

  • 能自己写出适合自己业务场景的代码。
  • 代码报错后，自己有解决的思路。没有基础，遇到问题百度都不知道问啥。

如果只想复制粘贴就行，那这部分的学习可以跳过。
