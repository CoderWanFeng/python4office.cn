第31讲：品牌建设与推广

学习如何打造 Skill 品牌，通过有效推广获取更多用户。

一、品牌定位

1.1 品牌定位框架

品牌定位画布

1. 目标用户
   - 谁是我的核心用户？
   - 他们有什么痛点？
   - 他们在哪里？

2. 价值主张
   - 我的 Skill 解决什么问题？
   - 与竞品有什么不同？
   - 为什么用户选择我？

3. 品牌个性
   - 专业严谨 vs 轻松活泼
   - 高端精致 vs 亲民实用
   - 创新前沿 vs 稳定可靠

4. 品牌承诺
   - 用户能期待什么？
   - 我保证交付什么？

场景型 财务管家、HR小助手 垂直类
