配置步骤

第一步：订阅火山方舟Coding Plan

第二步：获取API Key

  1. 登录火山引擎控制台
  2. 进入方舟平台 → API Key管理
  3. 创建新的API Key
  4. 复制保存（只显示一次）

第三步：配置Cursor

  1. 打开Cursor设置（Cmd/Ctrl + ,）
  2. 找到 Models 选项
  3. 点击 “Add Model”
  4. 填写配置：
    • Model Name: doubao-pro-32k（或其他模型）
    • API Key: 你的火山方舟API Key
    • Base URL: https://ark.cn-beijing.volces.com/api/v3
  5. 保存并测试

第四步：开始使用

现在你可以在Cursor里：

  • 按 Cmd/Ctrl + K 唤起AI对话
  • 选择火山方舟模型
  • 开始AI编程
