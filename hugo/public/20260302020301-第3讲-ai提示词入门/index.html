AI 自动化，提示词

大家好，我是正在实战各种 AI 项目的程序员晚枫。

🎭 场景导入：你遇到过这些情况吗？

场景1：AI回答的完全不是你想要的

AI：亲爱的[收件人]，[正文内容]，祝好。[你的名字]

你：处理一下这个表格

场景3：AI输出的格式乱七八糟

你：分析一下这些数据
AI：数据整体呈现上升趋势其中A项增长最快达到45%
    B项略有下降需要注意C项保持稳定...
你：能排版一下吗？看着好累...

问题出在哪里？不是AI不够聪明，是你的提示词写得不够好！

这一讲，我教你一套经过验证的提示词方法论，让AI输出你真正想要的内容。

📚 一、提示词的核心原理

为什么提示词很重要？

AI就像一个能力很强但不懂你心思的助手：

  • ✅ 它能写文章、写代码、做分析
  • ❌ 但它不知道你具体要什么、什么风格、什么格式

提示词 = 你和AI的沟通桥梁

质量对比：

提示词质量 AI输出质量 你的满意度
随意输入 5分（勉强能用） 20%
有基本结构 7分（能用） 60%
优化过的提示词 9分（惊喜） 95%

提示词万能公式

公式：角色 + 任务 + 要求 + 格式

┌─────────────────────────────────────┐
│ 提示词万能公式                        │
├─────────────────────────────────────┤
│ 1. 角色：让AI知道它的身份              │
│    → "你是一名资深的财务分析师"         │
│                                       │
│ 2. 任务：明确要做什么                  │
│    → "分析这份销售数据"                │
│                                       │
│ 3. 要求：具体的要求和约束              │
│    → "找出增长最快的产品，分析原因"     │
│                                       │
│ 4. 格式：输出的格式                    │
│    → "用表格呈现，包含产品名称、增长率、原因"│
└─────────────────────────────────────┘

示例2：生成Excel公式

❌ 差的提示词：

帮我写个VLOOKUP公式

AI输出：

=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])

问题： 只有语法，没有实际应用。

✅ 好的提示词：

你是一名Excel专家。
帮我写一个VLOOKUP公式，需求如下：
1. 在Sheet2的A列查找"张三"
2. 找到后返回该行B列的工资数据
3. 如果找不到，显示"无数据"
4. 数据表从Sheet2的A1开始

请：
1. 直接给出完整公式
2. 解释每个参数的含义
3. 提供一个实际应用的例子

AI输出：

✅ 完整公式：

=IFERROR(VLOOKUP("张三", Sheet2!A:B, 2, FALSE), "无数据")

📝 参数解释：

- "张三"：要查找的值
- Sheet2!A:B：查找范围（A列到B列）
- 2：返回第2列（B列）的数据
- FALSE：精确匹配
- IFERROR：如果找不到，显示"无数据"

💡 实际应用示例：

假设Sheet2数据如下：
A列(姓名)  | B列(工资)
张三       | 8000
李四       | 9000
王五       | 7500

在Sheet1的A1单元格输入公式，会返回：8000

如果查找"赵六"（不存在），会返回：无数据

模板4：Excel公式/函数

你是一名Excel专家。

[具体描述要解决的问题]

- 列A：[说明A列内容]
- 列B：[说明B列内容]
- ...

[描述你想要的结果]

要求：
1. 给出完整公式
2. 解释每个参数
3. 提供示例

使用示例：

你是一名Excel专家。
帮我解决以下Excel问题：

问题描述：
根据员工的销售额计算提成，规则如下：
- 销售额<5万：提成3%
- 销售额5-10万：提成5%
- 销售额>10万：提成8%

数据结构：
- 列A：员工姓名
- 列B：销售额
- 列C：提成（需要计算）

期望结果：
在C列自动计算每个员工的提成金额

要求：
1. 给出完整公式
2. 解释公式逻辑
3. 提供示例

AI输出：

✅ 完整公式：

=IF(B2<50000, B2*0.03, IF(B2<100000, B2*0.05, B2*0.08))

📝 公式解释：

这是嵌套IF函数，逻辑如下：
1. 如果B2<50000，提成 = B2 × 3%
2. 如果50000≤B2<100000，提成 = B2 × 5%
3. 如果B2≥100000，提成 = B2 × 8%

💡 示例：

| 员工 | 销售额 | 提成公式 | 提成金额 |
|------|--------|---------|---------|
| 张三 | 30000  | 30000×3% | 900 |
| 李四 | 80000  | 80000×5% | 4000 |
| 王五 | 120000 | 120000×8%| 9600 |

🔧 四、提示词优化技巧

技巧1：给AI明确的角色

原理： 角色设定让AI知道"我是谁"，从而调用对应的知识库和表达风格。

对比：

❌ 差：写一份报告
→ AI不知道用什么风格

✅ 好：你是一名资深财务分析师，写一份财务分析报告
→ AI会用财务分析师的专业术语和分析框架

✅ 更好：你是一名有10年经验的互联网公司财务分析师，
        擅长用数据讲故事，写一份面向管理层的财务分析报告
→ AI会用更精准的角色设定

常用角色设定：

场景 角色设定
写方案 “你是一名资深的项目经理”
写报告 “你是一名专业的数据分析师”
写邮件 “你是一名资深的商务助理”
写文案 “你是一名有10年经验的文案策划”
写代码 “你是一名全栈工程师”

技巧2：提供充分的背景信息

原理： 背景信息越多，AI理解越准确。

对比：

❌ 差：分析这些数据
→ AI不知道数据的背景、目的

✅ 好：这是某电商平台2026年Q1的销售数据，
     包含500条订单记录，需要向CEO汇报，
     找出增长点和潜在风险
→ AI知道：数据来源、数据量、汇报对象、分析目标

✅ 更好：这是某电商平台2026年Q1的销售数据，
        数据来自ERP系统，包含订单日期、客户、产品、
        金额、区域等字段。公司目标是Q1增长20%，
        目前已完成102%。需要向CEO汇报，
        重点分析：增长原因、区域差异、Q2策略建议
→ AI拥有完整的上下文

技巧3：指定输出格式

原理： 明确的格式要求让输出更易用。

常用格式：

1. 表格格式
"用表格呈现，包含列：[列1]、[列2]、[列3]"

2. 列表格式
"用无序列表输出，每条不超过20字"

3. 分步格式
"分步骤说明，每步包含：步骤名、操作、预期结果"

4. Markdown格式
"用Markdown格式输出，包含标题、列表、代码块"

5. JSON格式
"输出JSON格式，包含字段：[字段1]、[字段2]"

6. 代码格式
"用Python代码实现，包含注释说明"

技巧4：分步骤提问

原理： 复杂任务拆解，质量更高。

示例：

❌ 差：帮我做一个完整的营销方案
→ 任务太大，AI输出质量不稳定

✅ 好：分步骤完成

步骤1：帮我生成10个营销活动主题
步骤2：从中选择3个最好的，为每个主题生成详细方案
步骤3：为每个方案设计执行时间表
步骤4：计算每个方案的预算和预期ROI
步骤5：生成最终汇报PPT大纲

技巧5：提供示例

原理： 给AI一个参照物，输出更精准。

示例：

你是一名文案策划。
写3个产品推广文案。

参考风格：
"别让你的时间在无意义的加班中流失，
Python自动化办公，让你每天提前2小时下班！"

要求：
- 句式：[痛点] + [解决方案] + [结果]
- 字数：30字左右
- 语气：有冲击力

技巧6：设置约束条件

原理： 约束让AI输出更聚焦。

常用约束：

1. 字数约束
"控制在500字以内"
"每个要点不超过50字"

2. 内容约束
"不要使用专业术语"
"避免空话套话"

3. 风格约束
"用口语化表达"
"用正式公文风格"

4. 格式约束
"只用表格输出"
"不使用Markdown格式"

5. 逻辑约束
"按照重要性排序"
"先结论后原因"

⚠️ 五、常见错误与避坑指南

错误1：提示词太短

错误示例：

写个邮件

问题： AI不知道写给谁、什么目的、什么风格

正确做法：

你是一名HR专员，写一封面试邀请邮件，
发给应聘产品经理职位的候选人，
面试时间是下周三下午2点，
地点在公司18楼会议室，
语气专业友好，300字左右。

错误2：指令模糊

错误示例：

帮我处理一下这个表格

问题： “处理"是什么意思？筛选？排序？计算？

正确做法：

帮我处理这个Excel表格：
1. 筛选出销售额>10000的记录
2. 按销售额降序排列
3. 计算总销售额和平均销售额
4. 用表格输出结果

错误3：没有格式要求

错误示例：

分析一下这份数据

问题： AI可能输出大段文字，难以阅读

正确做法：

分析这份数据，用以下格式输出：

1. 总体概览（3句话）
2. 关键发现（用列表，5条）
3. 数据明细（用表格）
4. 行动建议（用列表，3条）

错误4：一次性要求太多

错误示例：

帮我写一份完整的营销方案，
包括市场分析、竞品分析、用户画像、
营销策略、执行计划、预算、KPI...

问题： 任务太大，AI输出质量不稳定

正确做法：

分步完成：

第1步：先做市场分析
第2步：基于市场分析，做竞品分析
第3步：基于前两步，制定营销策略
...

每步完成后，我会确认和修改，再进行下一步。

错误5：不提供反馈

错误示例：

你：写一个方案
AI：[输出方案]
你：[复制走人，不满意也不说]

问题： AI不知道你的偏好，下次还会输出一样的内容

正确做法：

你：写一个方案
AI：[输出方案]
你：整体可以，但太长了，精简到500字以内，
   另外把第三部分换成案例分析
AI：[按要求修改]
你：好了，这次完美！

案例3：Excel复杂公式生成

场景： 要写一个复杂的多条件判断公式

提示词模板：

你是一名Excel高级用户。
帮我写一个复杂的计算公式。

需求：
根据销售额和客户等级计算折扣：
- 销售额≥10万 且 VIP客户：8折
- 销售额≥10万 且 普通客户：85折
- 销售额<10万 且 VIP客户：9折
- 销售额<10万 且 普通客户：95折

数据结构：
- A列：客户名称
- B列：销售额
- C列：客户等级（VIP/普通）
- D列：折扣（需要计算）

要求：
1. 给出完整公式
2. 解释公式逻辑
3. 提供测试用例

AI输出：

✅ 完整公式：

=IF(AND(B2>=100000, C2="VIP"), 0.8,
 IF(AND(B2>=100000, C2="普通"), 0.85,
 IF(AND(B2<100000, C2="VIP"), 0.9, 0.95)))

📝 公式解释：

这是嵌套IF + AND函数，逻辑如下：
1. 如果销售额≥10万 且 等级=VIP → 0.8（8折）
2. 如果销售额≥10万 且 等级=普通 → 0.85（85折）
3. 如果销售额<10万 且 等级=VIP → 0.9（9折）
4. 其他情况（销售额<10万 且 等级=普通）→ 0.95（95折）

💡 测试用例：

| 客户 | 销售额 | 等级 | 折扣 |
|------|--------|------|------|
| 客户A | 120000 | VIP | 0.8 |
| 客户B | 120000 | 普通 | 0.85 |
| 客户C | 80000 | VIP | 0.9 |
| 客户D | 80000 | 普通 | 0.95 |
