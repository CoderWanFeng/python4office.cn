行政文员学Python，第1个月就涨薪了，她的秘诀是什么

小林是公司行政，每天处理Excel报表。学Python1个月后，她的工作效率提升了10倍，第2个月就涨薪了。

大家好，我是程序员晚枫。

今天分享一个小林的故事。

小林的困境

小林是公司的行政专员，每天的工作是：

  • 处理Excel报表
  • 汇总各部门数据
  • 发送批量邮件
  • 整理合同文件

听起来不难，但每天要花6-7个小时在这些重复工作上。

加班是常态。

小林的转折点

有一天，她在网上刷到我的视频：“Python自动化办公，1行代码搞定Excel”。

她抱着试试看的心态，买了一门课。

学了1个月后，她用Python写了几个脚本：

  • 自动汇总报表
  • 批量发送邮件
  • 定时生成数据报告

每天的工作从6小时缩短到30分钟。

老板的反应

月底汇报工作，老板问：“这个月效率怎么这么高？”

小林说：“我学了点Python。”

老板当场说：“给你涨500块工资，继续研究。”

小林做对了什么

1. 选对了方向

不是学通用的Python，而是专门学"自动化办公"这个方向。

2. 边学边用

学一个知识点，马上用到工作里。

3. 解决了实际问题

不是为学而学，是为了让工作更轻松。
