📅 第 1 个月：入门学习

  • Python 安装配置
  • 基础语法了解

第一个程序：

import pandas as pd

df = pd.read_excel('data.xlsx')

“原来编程没那么难！” “代码跑起来的那一刻，太有成就感了！”

第 2 周：理解代码

目标：读懂案例代码

学习内容：

  • 理解变量和函数

  • 修改案例参数

第一次修改：

“能修改代码为我所用，感觉太棒了！”

第 3 周：组合案例

目标：组合多个案例

学习内容：

感受：

“学编程真的有用！” “工作效率提升了，还有时间搞副业！”
