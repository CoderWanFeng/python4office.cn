日报自动生成Skill配置教程：每天节省30分钟

2026-04-06 10:44 · AI Skills Skill使用教程
日报 自动化 Skill AI办公 工作汇报 教程

大家好，我是正在实战各种 AI 项目的程序员晚枫。

每天下班前最痛苦的事是什么？写日报——回忆今天做了什么、整理数据、写总结，至少 30 分钟。

今天教你配置 日报自动生成 Skill，让 AI 自动帮你汇总数据、生成日报。

一、Skill 简介

功能概述

日报自动生成 Skill 可以：

  • ✅ 自动汇总任务管理工具数据（飞书任务/钉钉/Notion 等）
  • ✅ 自动读取日历事件
  • ✅ 自动统计代码提交/Git 记录
  • ✅ 自动读取邮件/消息摘要
  • ✅ 生成结构化的日报文档
  • ✅ 定时自动发送

适用人群

人群 收益
程序员 自动汇总代码提交、Bug 修复
产品经理 自动汇总需求进度、会议记录
销售人员 自动汇总客户跟进、订单数据
运营人员 自动汇总数据报表、活动进展
项目经理 自动汇总项目进度、风险预警

二、安装 Skill

Coze 扣子安装

  1. 打开 Coze 官网
  2. 进入 “技能商店”
  3. 搜索 “日报生成” 或 “工作汇报”
  4. 点击 “安装”

OpenClaw 安装

openclaw skills install daily-report
openclaw skills install calendar-tools
openclaw skills install git-tools

飞书 CLI

飞书 CLI 原生支持日历、任务、文档操作，可直接用于日报生成。

七、常见问题

Q1：数据源连接失败怎么办？

  • 检查 API 权限是否开启
  • 确认账号有访问权限
  • 重新授权连接

Q2：生成的日报不准确？

你：调整日报生成规则，只包含标记为"重要"的任务

AI：已调整规则，将只汇总高优先级任务。

Q3：如何添加自定义字段？

你：在日报中添加"心情指数"字段，让我每天手动填写

AI：已添加自定义字段：
   - 字段名: 心情指数
   - 类型: 1-5 星评分
   - 填写方式: 手动

Q4：如何导出历史日报？

你：导出 3 月份的所有日报为 Word 文档

AI：导出完成！
   共 22 篇日报
   已合并为: 3月工作月报.docx
