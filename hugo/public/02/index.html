02-Python下载和安装：3分钟搞定（Mac版）

今天这讲，我们来搞定 Mac 上安装 Python。

1、为什么要单独讲Mac版？

因为 Mac 和 Windows 的安装方式完全不一样。

Windows 可以直接去官网下载安装包，一路点"下一步"就搞定了。

但 Mac 自带 Python（Apple 写的版本），如果你直接去官网安装，会遇到版本冲突的问题。

所以 Mac 用户请认真看这讲，Windows 用户可以直接跳到下一讲。

2、第一步：打开终端

Mac 上安装 Python，需要用到终端（Terminal）。

打开方式：

  • 按 Command + 空格，搜索"终端"，回车
  • 或者在应用程序 > 实用工具 > 终端

你会看到一个黑底白字的界面，长这样：

用户名deMacBook-Pro ~ %

这就是终端，我们的所有操作都在这里完成。

3、第二步：安装Homebrew

Homebrew 是 Mac 上的一个包管理工具，类似于手机上的"应用商店"，可以帮我们轻松安装各种工具。

在终端里粘贴这段代码，回车：

/bin/bash -c "$(curl -fsSL https://gitee.com/ineo6/homebrew-install/raw/master/install.sh)"

安装过程中，会提示你输入电脑密码（输入时屏幕不显示内容是正常的，输完回车就行）。

安装完成会看到 Installation successful! 的提示。

💡 如果你之前已经安装过 Homebrew，可以跳过这步。

4、第三步：用Homebrew安装Python

在终端里依次执行：

# 更新Homebrew
brew update

# 安装Python
brew install python

等待几分钟，看到 🍺 /usr/local/Cellar/python/3.x.x 这样的提示，说明安装成功。

5、第四步：验证安装

在终端输入：

python3 --version

看到类似 Python 3.12.0 这样的版本号，说明安装成功！

6、第五步：安装python-office

现在 Python 装好了，接下来安装我们的主角——python-office。

在终端执行：

pip3 install python-office

看到 Successfully installed python-office-1.x.x 这样的提示，说明安装成功。

7、第一个程序：验证环境

打开终端，输入 python3，进入 Python 交互模式。

然后输入：

import office
print("Hello，程序员晚枫！")

如果看到输出 Hello，程序员晚枫！，说明环境完全OK！

按 exit() 可以退出交互模式。

8、常见问题

Q：终端里粘贴命令没反应？

A：Mac 用 Command + V 粘贴，或者右键选择"粘贴"。

Q：提示"Permission denied"（权限被拒绝）？

A：命令前面加 sudo，例如 sudo brew install python，然后输入密码。

Q：安装太慢了怎么办？

A：用国内镜像源：

pip3 install python-office -i https://mirrors.aliyun.com/pypi/simple/
