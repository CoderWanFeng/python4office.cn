✓ 工具：文心一言、通义千问

学习目标：

✓ 掌握主流 AI 工具

✓ 写出好的提示词
✓ 提高工作效率

学习内容：

1. AI 写作工具

✓ 用 AI 写一段代码

🎯 新手常见问题

问题 1：需要编程基础吗？

答案：不需要！

AI 工具分两类：
1. 无需编程：90% 的工具
   - AI 写作
   - AI 绘画
   - AI 办公

2. 需要编程：10% 的工具
   - API 调用
   - 本地部署
   - 定制开发

新手从无需编程的开始！

问题 2：需要买课吗？

答案：初期不需要！

✓ 导航站学习专区

- 第 3 个月：开始变现

变现方式：
✓ AI 代写
