一、准备工作

注册阿里云账号

  1. 访问阿里云官网
  2. 点击「免费注册」
  3. 使用手机号或邮箱注册
  4. 完成实名认证（必需）

需要的工具

  • 电脑（Windows/Mac/Linux 都可以）
  • 稳定的网络

二、开通百炼 Coding Plan

第一步：访问开通页面

👉 点击访问阿里云百炼 Coding Plan 开通页面

第二步：选择套餐

阿里云提供三种套餐：

套餐 价格 tokens/月
基础版 29 元 80 万
进阶版 59 元 300 万
企业版 199 元 无限

根据你的需求选择合适的套餐。

第三步：完成支付

支持支付宝、银行卡等多种支付方式。

三、获取 API 密钥

第一步：进入控制台

开通成功后，进入阿里云百炼控制台。

第二步：创建密钥

  1. 找到「API-KEY」入口
  2. 点击「创建 API-KEY」
  3. 保存好 API-KEY

⚠️ 注意：API-KEY 要妥善保管，不要泄露给他人。

四、接入开发工具

支持的 IDE

  • VS Code（推荐）
  • JetBrains 全家桶
  • 阿里云 IDE 插件

VS Code 接入教程

  1. 安装阿里云 AI 编程插件
  2. 输入 API-KEY
  3. 开始使用

不需要！ 阿里云是国产服务，国内访问速度快。
