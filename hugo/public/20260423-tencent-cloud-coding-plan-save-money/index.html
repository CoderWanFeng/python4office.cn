二、 Tokens 节省技巧

1. 控制上下文长度

不要让每次对话都带上所有历史记录。在合适的时机开启新对话。

# 不好：每次都带全部历史
messages = conversation_history  # 越来越长

# 好：定期开启新对话
if len(messages) > 20:
    messages = []  # 重置

2. 使用简洁的 prompt

啰嗦写法 简洁写法
“请你帮我写一个 Python 的快速排序算法好吗？非常感谢！” “Python 快速排序”

节省 tokens = 节省钱。

3. 批量处理

多个相似任务合并一次问，减少 API 调用次数。
