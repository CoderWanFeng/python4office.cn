📋 对比维度

我选了 6 个核心维度：

1. 课程内容（深度、广度、实用性）
2. 教学方式（直播/录播、互动性）
3. 师资力量（讲师背景、实战经验）
4. 学习服务（答疑、作业批改、社群）
5. 价格性价比（投入产出比）
6. 学员口碑（真实评价、就业率）
