🎯 这个技能能帮你什么？

一句话：把发票丢进去，Excel 吐出来。

详细说：

  • 支持 PDF、JPG、PNG 格式发票
  • 自动识别发票上所有字段
  • 自动整理成 Excel 表格
  • 单张、批量都能处理

🛠️ 上手教程（5 分钟）

第一步：安装

pip install poocr

第二步：获取密钥

访问：https://curl.qcloud.com/9ExTmaya

注册 → 登录 → 文字识别 → 密钥管理 → 新建密钥

保存好 SecretId 和 SecretKey。

第三步：运行

创建文件 auto_invoice.py：

import poocr

poocr.ocr2excel.VatInvoiceOCR2Excel(
    input_path='C:/invoices/today',     # 今天的发票文件夹
    output_path='C:/invoices/output',   # 输出目录
    id='你的 SecretId',
    key='你的 SecretKey'
)

print("✅ 发票识别完成！可以下班了！")

运行：

python auto_invoice.py

输出：

✅ 发票识别完成！可以下班了！

打开 Excel，所有发票信息整整齐齐。

收拾东西，准时下班。

Q1：我没有编程基础，能用吗？

Q2：安全吗？发票信息会不会泄露？

安全。

  • 代码开源，可以审查
  • 数据本地处理，不上传
  • 腾讯云是大厂，有安全保障

唯一要做的：保管好你的腾讯云密钥。

Q3：识别准确率怎么样？

电子版 PDF：99%+

拍照图片：98% 左右（清晰的）

模糊/反光/折叠：可能识别失败，需要重新拍照。

建议：优先用电子版，拍照要用扫描 APP。

Q4：一个月能识别多少张？

免费额度：1000 次/月

超出后：0.035 元/次

大部分中小企业：免费额度够用。

发票多的企业：10000 张也才 350 元，很便宜。
