📚 阶段1：基础准备（2-3周）

推荐练习

  • 写一些简单的数据处理脚本
  • 练习文件读写操作

避坑指南

❌ 不要在这阶段花太多时间，会基础语法就行，后面边做边学

📚 阶段2：数据处理核心（4-5周）

这是最重要的阶段，80%的数据分析工作都在这。

NumPy（1周）

import numpy as np

# 数组操作
arr = np.array([1, 2, 3, 4, 5])
print(arr.mean())  # 平均值
print(arr.std())   # 标准差

# 矩阵运算
matrix = np.array([[1, 2], [3, 4]])
print(matrix.T)    # 转置

Pandas（3-4周）

import pandas as pd

# 读取数据
df = pd.read_csv('data.csv')

# 数据探索
print(df.head())      # 前5行
print(df.info())      # 数据信息
print(df.describe())  # 统计描述

# 数据筛选
df[df['age'] > 25]  # 筛选年龄大于25的

# 分组聚合
df.groupby('department')['salary'].mean()

数据清洗技巧

  • 处理缺失值（删除/填充）
  • 处理重复值
  • 数据类型转换
  • 异常值检测

📚 阶段3：数据查询（2周）

企业里大部分数据都存在数据库里，SQL是必学技能。

基础查询

-- 查询特定列
SELECT name, age FROM employees;

-- 条件筛选
SELECT * FROM employees WHERE salary > 5000;

-- 排序
SELECT * FROM employees ORDER BY salary DESC;

多表关联

-- 内连接
SELECT e.name, d.department_name
FROM employees e
INNER JOIN departments d ON e.dept_id = d.id;

聚合与分组

-- 按部门统计平均工资
SELECT department, AVG(salary) as avg_salary
FROM employees
GROUP BY department;

窗口函数（进阶）

-- 计算每个员工的工资排名
SELECT name, salary,
       RANK() OVER (ORDER BY salary DESC) as rank
FROM employees;

📚 阶段4：数据可视化（2-3周）

Matplotlib基础

import matplotlib.pyplot as plt

# 折线图
plt.plot([1, 2, 3, 4], [1, 4, 9, 16])
plt.xlabel('X轴')
plt.ylabel('Y轴')
plt.title('平方函数')
plt.show()

Seaborn统计可视化

import seaborn as sns

# 散点图
sns.scatterplot(data=df, x='age', y='salary')

# 箱线图
sns.boxplot(data=df, x='department', y='salary')

# 热力图
corr = df.corr()
sns.heatmap(corr, annot=True)

可视化原则

  • 选择合适的图表类型
  • 注意颜色搭配
  • 添加清晰的标题和标签
  • 一张图讲清楚一个故事

🎯 学习建议

1. 不要追求完美

不要等"完全学会"再开始下一个阶段。数据分析是实践性很强的技能，边做边学效果最好。

2. 用真实数据练习

不要只用教程里的示例数据，去Kaggle、UCI下载真实数据集练习。

3. 输出学习笔记

写博客、做笔记、发知乎。输出是最好的学习方式。

4. 加入学习社群

一个人走得快，一群人走得远。加入学习社群，有问题可以互相讨论。
