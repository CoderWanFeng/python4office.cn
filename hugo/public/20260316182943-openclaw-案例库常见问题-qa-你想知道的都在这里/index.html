🔧 使用问题

Q5: 代码复制后运行报错怎么办？

A: 按以下步骤排查：

  1. 检查 Python 版本

    python --version
# 需要 3.9+

  2. 检查依赖包

    pip install -r requirements.txt

  3. 查看错误信息

    • ModuleNotFoundError → 缺包
    • FileNotFoundError → 路径错
    • 其他 → 搜索错误信息

  4. 查看案例说明

    • 是否有特殊要求
    • 是否需要配置 API 密钥

Q6: 如何修改案例适配自己的场景？

A: 3 步走：

  1. 理解代码逻辑

    • 看懂每个函数的作用
    • 理解数据流向

  2. 找到关键参数

    • 输入路径
    • 输出路径
    • 配置选项

  3. 修改并测试

    • 小步修改
    • 每步测试
    • 有问题回退

Q11: 案例运行慢怎么办？

A: 可能原因：

  1. 网络问题

    • API 调用慢
    • 解决：检查网络

  2. 数据量大

    • 处理时间长
    • 解决：分批处理

  3. 性能问题

    • 代码效率低
    • 解决：优化代码
