恭喜你！Python 已经装好了，现在我们开始写代码。

1、编程界的"Hello World"

每个程序员学编程的第一件事，就是写一个 Hello World 程序。

这是编程界的传统，类似于武术里的"站桩"——看似简单，却是基本功。

今天我们就来写一个属于我们自己的 Hello World。

2、打开记事本（或其他编辑器）

Windows用户：

按 Win + R，输入 notepad，回车，打开记事本。

Mac用户：

按 Command + 空格，搜索"文本编辑"，回车。

💡 推荐下载 VS Code 或 PyCharm 作为专业的 Python 编辑器，功能更强。

3、写下你的第一行代码

在记事本里输入：

print("Hello World！我是程序员晚枫！")

注意：

  • 括号要用英文括号，不能用中文括号 （
  • 引号也要用英文引号，不能用中文引号 ""

4、保存文件

把文件保存为 hello.py：

  • Windows：文件 → 另存为 → 文件名填 hello.py → 编码选 UTF-8
  • Mac：文件 → 存储 → 文件名填 hello.py

⚠️ 文件名不要用中文，后缀必须是 .py

5、运行程序

打开终端/命令提示符，进入文件所在目录：

cd 你存放文件的路径
python hello.py

比如我的文件放在桌面，就输入：

cd Desktop
python hello.py

看到输出：

Hello World！我是程序员晚枫！

🎉 恭喜你！你的第一个程序跑起来了！

6、print是什么？

print 是 Python 的一个"函数"，意思是"打印到屏幕"。

简单理解：print() 就是把括号里的内容显示出来。

你试试这些：

print("今天天气真好")
print(520)
print(3 + 5)

7、中文显示：第1行加注释

如果你想让程序显示中文，需要在第一行加一行注释：

# -*- coding: utf-8 -*-
print("你好，程序员晚枫！")

这样就能正常显示中文了。

8、完整代码模板

以后写 Python 程序，直接用这个模板：

# -*- coding: utf-8 -*-

# 作者：程序员晚枫
# 功能：Hello World

print("Hello World！")
print("我是程序员晚枫！")
