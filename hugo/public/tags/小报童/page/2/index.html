16-微信机器人：自动回复和管理群聊

今天教你怎么用 python-office 开发微信机器人——自动回复消息、管理群聊。

1、自动回复消息

import office

# 设置自动回复
@office.robot.reply()
def auto_reply(msg):
    if '你好' in msg:
        return '你好！我是程序员晚枫的微信机器人~'
    elif '课程' in msg:
        return '想了解课程可以加微信：aiwf365'
    else:
        return '收到，稍后回复你！'

# 启动机器人
office.robot.start()

运行后，有人给你发消息，会自动回复！

18-图片处理：批量压缩、裁剪、加水印

2024-09-17

大家好，这里是程序员晚枫，正在all in AI编程实战。

每次发朋友圈要手动压缩图片？

水印要一个个加？

今天教你怎么用 python-office 批量处理图片。

1、图片压缩

import office

# 压缩图片
office.image.compress(
    path='原图.jpg',
    output_file='压缩后.jpg',
    quality=50  # 质量0-100，越小越小
)

2、批量压缩文件夹里的图片

import office
import os

folder = '图片文件夹'

for file in os.listdir(folder):
    if file.endswith(('.jpg', '.png')):
        src = os.path.join(folder, file)
        dst = os.path.join(folder, '压缩_' + file)
        office.image.compress(path=src, output_file=dst, quality=30)

print('批量压缩完成！')

3、图片裁剪

import office

# 裁剪图片（左上角为原点）
office.image.cut(
    path='原图.jpg',
    output_file='裁剪后.jpg',
    x=100,    # 左上角X坐标
    y=100,    # 左上角Y坐标
    width=500,  # 裁剪宽度
    height=500  # 裁剪高度
)

4、加水印

import office

# 添加文字水印
office.image.add_text(
    path='原图.jpg',
    output_file='水印图.jpg',
    text='程序员晚枫',  # 水印文字
    color='white',       # 文字颜色
    size=50,             # 字号
    opacity=0.5          # 透明度
)

5、加图片水印

import office

# 添加图片水印
office.image.add_image(
    path='原图.jpg',
    output_file='水印图.jpg',
    logo='logo.png',      # 水印图片
    opacity=0.3,          # 透明度
    position='right-bottom'  # 位置：center/left-top/right-bottom等
)
