🛠️ python-office 常用命令速查

# Excel
office.excel.read(path='file.xlsx')           # 读取
office.excel.write(path='file.xlsx', data=data)  # 写入
office.excel.merge(folder_path='folder')      # 合并
office.excel.chart(path='chart.xlsx', data=data, chart_type='bar')  # 图表

# Word
office.word.read(path='file.docx')            # 读取
office.word.write(path='file.docx', content='text')  # 写入
office.word.docx2pdf(path='file.docx')        # 转PDF

# PDF
office.pdf.read(path='file.pdf')              # 读取
office.pdf.merge(file_list=[...])             # 合并
office.pdf.add_watermark(path='file.pdf')     # 加水印

# 邮件
office.email.send(email='to@example.com', title='title', content='text')  # 发送

# 文件
office.file.search(folder='folder', keyword='keyword')  # 搜索
office.file.classify(folder='folder', type_rule=True)    # 分类

# OCR
office.ocr.text(path='image.jpg')            # 文字识别

💡 学习心得

我的经验

  1. 动手比看懂重要——一定要跟着敲代码
  2. 从小项目开始——不要一开始就想着做大系统
  3. 遇到问题先搜索——Stack Overflow 是你的好朋友
  4. 记录和总结——写博客、教别人是最好的学习方式

常见误区

  • ❌ 只看不练——代码需要动手才能学会
  • ❌ 追求完美——先跑起来，再优化
  • ❌ 什么都学——选择自己需要的深入学
  • ❌ 闭门造车——多交流，多看别人的代码
