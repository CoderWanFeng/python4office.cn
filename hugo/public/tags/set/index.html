🎬 开篇：一个去重问题引发的思考

你有没有写过这样的代码？

# 需求：统计文章中出现过的所有单词
text = "the quick brown fox jumps over the lazy dog the fox"
words = text.split()

# ❌ 传统写法
unique_words = []
for word in words:
    if word not in unique_words:
        unique_words.append(word)
print(unique_words)
# ['the', 'quick', 'brown', 'fox', 'jumps', 'over', 'lazy', 'dog']

# ✅ set 写法：一行搞定
unique_words = list(set(words))
print(unique_words)
