🎬 开篇：装饰器的本质

你有没有见过这样的代码？

@timer
def slow_function():
    time.sleep(1)
    return "Done"

@cache
def fibonacci(n):
    if n < 2:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

@retry(times=3)
def fetch_data():
    return requests.get("https://api.example.com")

这些 @xxx 就是装饰器，它们是 Python 最优雅的特性之一。
