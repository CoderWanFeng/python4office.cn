📊 数据分析技能汇总

1. 数据清洗

  • data-cleaner：数据清洗
  • missing-value-handler：缺失值处理
  • outlier-detector：异常值检测

2. 数据处理

  • data-transformer：数据转换
  • data-merger：数据合并
  • data-splitter：数据分割

3. 数据可视化

  • chart-generator：图表生成
  • dashboard-creator：大屏制作
  • report-visualizer：报表可视化

4. 统计分析

  • statistic-analyzer：统计分析
  • correlation-finder：相关性分析
  • regression-analyzer：回归分析

5. 机器学习

  • model-trainer：模型训练
  • feature-engineer：特征工程
  • model-evaluator：模型评估

💡 典型工作流

数据清洗 → 数据处理 → 统计分析 → 可视化 → 报告生成
