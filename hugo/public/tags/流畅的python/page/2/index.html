🎬 开篇：让类的实例像函数一样调用

你有没有想过，为什么这样写是合法的？

class Counter:
    def __init__(self):
        self.count = 0
    
    def __call__(self):
        self.count += 1
        return self.count

counter = Counter()
print(counter())  # 1 - 像"函数"一样调用！
print(counter())  # 2
print(counter())  # 3

这就是 __call__ 方法的魔力：让对象实例可以像函数一样被调用。

🎬 开篇：装饰器的本质

你有没有见过这样的代码？

@timer
def slow_function():
    time.sleep(1)
    return "Done"

@cache
def fibonacci(n):
    if n < 2:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

@retry(times=3)
def fetch_data():
    return requests.get("https://api.example.com")

这些 @xxx 就是装饰器，它们是 Python 最优雅的特性之一。
