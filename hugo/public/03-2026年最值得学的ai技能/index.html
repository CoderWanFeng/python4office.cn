想做一个好用的对话机器人，需要掌握这些技能：

1. 大模型API使用

import openai

2. 提示词工程（Prompt Engineering）

好的提示词能让机器人更聪明：

system_prompt = """你是一个专业的电商客服助手。

你的职责：
1. 回答用户关于产品的问题
2. 帮助用户解决订单问题
3. 推荐适合用户的产品

回答原则：
- 语气友好、耐心
- 回答简洁、准确
- 不知道的问题不要瞎编
"""

3. RAG（检索增强生成）

让机器人能回答企业专属问题：

# 1. 用户提问
user_question = "你们的退款政策是什么？"

# 2. 从知识库检索相关内容
relevant_docs = search_knowledge_base(user_question)

# 3. 把检索结果加入提示词
enhanced_prompt = f"""基于以下信息回答问题：
{relevant_docs}

用户问题：{user_question}
"""

# 4. 调用大模型生成回答
response = llm.generate(enhanced_prompt)

4. 多轮对话管理

记住对话上下文，让交流更自然：

class ConversationManager:
    def __init__(self):
        self.history = []
    
    def add_message(self, role, content):
        self.history.append({"role": role, "content": content})
        # 只保留最近10轮对话
        if len(self.history) > 20:
            self.history = self.history[-20:]
    
    def get_context(self):
        return self.history
