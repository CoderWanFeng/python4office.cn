🛠️ 学习路线（财务专用）

第 1 步：安装 Python（30 分钟）

下载地址：https://www.python.org/downloads/

安装步骤：

  1. 下载安装包
  2. 双击运行
  3. ✅ 勾选"Add Python to PATH"
  4. 点击"Install Now"

验证：

python --version

输出 Python 3.x.x 就成功了。

第 2 步：安装 poocr 库（5 分钟）

打开命令行（Windows 按 Win+R，输入 cmd）：

pip install poocr

等待安装完成，看到"Successfully installed"就 OK 了。

第 3 步：获取腾讯云密钥（15 分钟）

访问：https://curl.qcloud.com/9ExTmaya

步骤：

  1. 注册腾讯云账号（手机号就行）
  2. 登录
  3. 搜索"文字识别"
  4. 进入控制台 → 密钥管理
  5. 新建密钥
  6. 保存 SecretId 和 SecretKey

免费额度：1000 次/月，够用好久。

第 4 步：写第一行代码（30 分钟）

新建一个文件：invoice.py

输入代码：

import poocr

poocr.ocr2excel.VatInvoiceOCR2Excel(
    input_path='C:/invoices/test.pdf',  # 改成你的发票路径
    output_path='C:/invoices/output',   # 改成你的输出目录
    id='你的 SecretId',                  # 改成你的 SecretId
    key='你的 SecretKey'                 # 改成你的 SecretKey
)

print("识别完成！")

运行：

python invoice.py

看到"识别完成！"，恭喜你，已经入门了！

第 5 步：批量处理（1 小时）

修改代码：

import poocr

poocr.ocr2excel.VatInvoiceOCR2Excel(
    input_path='C:/invoices/2026-03',   # 放 300 张发票的文件夹
    output_path='C:/invoices/output',
    id='你的 SecretId',
    key='你的 SecretKey'
)

print("批量识别完成！")

把 300 张发票放进文件夹，运行脚本。

3 分钟后，打开输出的 Excel 文件——所有发票信息整整齐齐。

那一刻的成就感，谁用谁知道。

🎯 财务人为什么要学 Python？

原因 1：重复工作太多

财务日常：

  • 录入发票
  • 对账
  • 做报表
  • 整理数据

这些工作：

  • 重复
  • 耗时
  • 容易出错
  • 价值低

Python 能做的：

  • 自动录入
  • 自动对账
  • 自动生成报表
  • 自动整理数据

把你从重复劳动中解放出来，去做更有价值的分析和决策。

原因 2：职业天花板低

传统财务路径：

出纳 → 会计 → 主管 → 经理

问题：

  • 晋升慢
  • 竞争激烈
  • 可替代性强

会 Python 的财务路径：

财务 → 财务自动化专家 → 财务数据分析师 → 财务总监

优势：

  • 差异化竞争
  • 不可替代
  • 薪资更高

原因 3：行业在变化

趋势：

  • 电子发票普及
  • 财务共享中心兴起
  • RPA（机器人流程自动化）应用
  • AI 在财务领域渗透

不变的是：

  • 业务理解
  • 数据分析
  • 决策支持

会变的是：

  • 工具（从 Excel 到 Python）
  • 效率（从手动到自动）
  • 价值（从录入到分析）

适应变化的人，才能生存。

💡 给财务人的学习建议

建议 1：从痛点开始

不要为了学 Python 而学 Python。

找一个你每天最痛的工作：

  • 发票录入
  • 银行对账
  • 报表合并

然后用 Python 解决它。

解决了痛点，你就有动力继续学。

建议 2：边学边用

不要等学完了再用。

学一点，用一点：

  • 今天学了变量，就用变量存储发票金额
  • 明天学了循环，就用循环处理多张发票
  • 后天学了文件操作，就自动读取发票 PDF

在实践中学习，进步最快。

建议 3：找组织

一个人学容易放弃。

找组织：

  • 加入 Python 学习群

  • 参加线下 meetup

我们也有交流群（文末有二维码），里面有很多财务朋友在学 Python。
