## 1. Huawei Cloud's AI Service Matrix

Although Huawei doesn't have Coding Plan, Huawei Cloud's AI product line is rich:

### Pangu Large Model Series

Huawei's **Pangu Large Model** is its core competitiveness:

| Model | Features | Suitable Scenarios |
|-------|----------|-------------------|
| Pangu NLP | Strong Chinese understanding | Text generation, dialogue |
| Pangu CV | Vision capability | Image recognition, video analysis |
| Pangu Weather | High precision | Weather forecast |
| Pangu Medicine | Professional field | Drug development |

### Ascend AI Cloud Services

Huawei's **self-developed Ascend chips** are the hardware foundation of its AI capability:

- **Ascend 910**: Training chip, powerful computing
- **Ascend 310**: Inference chip, low power consumption

Ascend cloud services let you use Ascend computing without purchasing hardware.

## 2. Who is Huawei Cloud Suitable For?

### Suitable Scenarios

| Audience | Reasons |
|---------|---------|
| **Enterprise users** | Complete enterprise-level service, SLA guarantee |
| **AI training needs** | Ascend computing good value for money |
| **Industry AI applications** | Pangu industry models targeted |
| **HarmonyOS developers** | Convenient within Huawei ecosystem |
| **Government/SOE** | Localization requirements, data security |

### Not Suitable Scenarios

| Audience | Reasons |
|---------|---------|
| **Personal developers** | High price, high threshold |
| **Simple code completion** | Coding Plan model more cost-effective |
| **Students** | Price not friendly enough |
| **Rapid prototype development** | Enterprise process slower |
