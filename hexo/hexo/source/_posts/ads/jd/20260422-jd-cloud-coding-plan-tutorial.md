## 一、准备工作

### 需要准备的东西

1. 一个京东账号（没有的话去 jd.com 注册）
2. 实名认证（国内云服务都需要）
3. 一台电脑（Windows/Mac/Linux 都行）
4. 稳定的网络

### 推荐浏览器

Chrome、Edge、Firefox 都行，推荐 Chrome。

## 二、注册与开通

### 第一步：访问京东云

**👉 [点击查看京东云 JD Cloud Coding Plan 详情](https://www.python-office.com/openclaw/coding-plan/)**

找到页面里的京东云入口，进入注册页面。

### 第二步：完成实名认证

- 个人开发者：身份证实名即可
- 企业用户：需要营业执照等企业资质

### 第三步：订阅 Coding Plan

1. 选择「AI 服务」→「Coding Plan」
2. 选择套餐（月付/年付）
3. 完成支付

## 三、配置开发工具

### 方法一：VS Code 插件

1. 打开 VS Code
2. 按 `Ctrl+Shift+X` 打开扩展市场
3. 搜索「JD Cloud」或「京点点」
4. 安装插件
5. 登录你的京东云账号
6. 开始使用

### 方法二：JetBrains 系列

支持 IntelliJ IDEA、PyCharm、WebStorm 等：

1. 打开 Settings → Plugins
2. 搜索「JD Cloud」
3. 安装并登录

### 方法三：命令行工具

适合喜欢终端操作的开发者：

```bash
# 安装命令行工具（以 pip 为例）
pip install jdcloud-sdk-python

# 登录
jdcloud configure

# 开始使用
jdcloud ai-code --help
```

## 四、常见问题

### Q1：免费额度有多少？

新用户通常有试用额度，具体以官方为准。订阅后额度会大幅增加。

### Q2：支持哪些模型？

京东云 Coding Plan 主打：
- 京东自研模型
- 主流开源模型（如 Qwen、DeepSeek 等）

具体模型列表可在控制台查看。

### Q3：国内访问速度如何？

京东云在国内有多处节点，访问速度快，延迟低。比海外厂商稳定很多。

## 五、进阶使用技巧

### 1. 快捷键设置

在 IDE 设置里找到 Coding Plan 相关设置，开启你喜欢的快捷键。

### 2. 上下文优化

给 AI 更多上下文（代码注释、项目结构），生成结果会更准确。

### 3. 多模型切换

如果结果不满意，可以切换到其他模型试试，找到最适合你的那个。
