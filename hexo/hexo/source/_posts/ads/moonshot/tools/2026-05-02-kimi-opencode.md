## 二、配置步骤

### Step 1：开通 Kimi 服务

👉 **专属优惠通道**：[https://platform.moonshot.cn/](https://platform.moonshot.cn/)

注册并开通 Kimi 服务，获取你的 API Key。

---

### Step 2：在 OpenCode 中配置 Kimi

1. 打开 OpenCode 设置 → AI Providers
2. 选择"Custom"或"OpenAI Compatible"
3. 填写以下信息：

| 字段 | 内容 |
|------|------|
| API URL | `https://api.moonshot.cn/v1` |
| API Key | 你的 Kimi API Key |
| 模型名称 | `moonshot-v1-8k` |

4. 保存并设置为默认模型

---

### Step 3：开始使用

配置成功后，在 OpenCode 中直接使用 Kimi 进行代码生成和优化。
