最近几年，国产大模型赛道杀出了一匹黑马——**月之暗面（Moonshot AI）**，旗下的 Kimi 以「长上下文」闻名，现在也推出了 Coding Plan。

## 一、什么是 Moonshot AI / Kimi Coding Plan？

月之暗面是国内 AI 领域的新锐公司，Kimi 是其核心产品，主打超长上下文处理能力。

### 核心特点

| 特性 | 说明 |
|------|------|
| **超长上下文** | 支持 128K token 超长上下文 | 适合大项目 |
| **Kimi 模型** | 月之暗面自研，效果稳定 |
| **中文优化** | 中文理解能力强 |
| **长文本处理** | 特别适合处理大型代码文件 |

### Kimi 的优势场景

- 大型代码库分析（几十个文件一起喂进去）
- 长文档处理与总结
- 需要记住大量上下文的复杂任务
- 中文编程需求

## 二、为什么选 Kimi Coding Plan？

### 1. 超长上下文是杀手锏

其他 AI 模型通常只能处理几K的上下文，Kimi 支持 128K token。

这意味着你可以：
- 一次喂进去几十个源代码文件
- 让 AI 一次性理解整个项目结构
- 做大型代码库的全局分析

### 2. 中文理解出色

Kimi 对中文的理解能力在业内属于第一梯队。写中文注释、描述中文需求，效果很好。

### 3. 适合大项目

如果你做的是大型项目（代码量多、文件多），Kimi 的长上下文优势会很明显。
