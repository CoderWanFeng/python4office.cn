## 1. Who Is It For?

| Audience | Reason |
|------|------|
| **Large Project Developers** | 128K context can analyze entire projects at once |
| **Code Review Needs** | See all code at once for more accurate reviews |
| **Large Codebase Analysis** | Process dozens to hundreds of files together |
| **Learning New Projects** | Quickly understand entire project architecture |
