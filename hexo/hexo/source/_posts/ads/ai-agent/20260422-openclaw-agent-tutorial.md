## 📦 第一步：安装OpenClaw

先安装OpenClaw CLI工具。

**步骤：**

1. 确保你的电脑已经安装了Node.js
2. 打开终端，运行安装命令：

```bash
npm install -g @openclaw/cli@latest
```

3. 验证安装：

```bash
openclaw --version
```

如果输出版本号，说明安装成功了。

---

## 🔧 第二步：配置模型

OpenClaw支持很多模型，推荐用DeepSeek或智谱AI。

**步骤：**

1. 初始化配置：
```bash
openclaw init
```

2. 按照提示配置你的API Key

3. 也可以手动编辑配置文件

---

## 🚀 第三步：创建你的第一个Agent

现在创建你的第一个AI Agent。

**步骤：**

1. 创建一个新项目：
```bash
openclaw create my-first-agent
```

2. 进入项目目录：
```bash
cd my-first-agent
```

3. 编辑Agent配置文件，设置你的指令和工具

---

## 💡 第四步：测试Agent

创建好了，测试一下。

**步骤：**

1. 运行Agent：
```bash
openclaw run
```

2. 输入你的指令，比如：帮我整理这个文件夹里的图片，按日期分类

3. 看Agent的执行过程和结果
