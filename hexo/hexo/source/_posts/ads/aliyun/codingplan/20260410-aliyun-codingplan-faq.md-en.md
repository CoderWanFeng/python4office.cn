## 1. Basic Questions

### Q1: What is Alibaba Cloud Coding Plan?

**AI programming subscription service** launched by Alibaba Cloud Bailian, providing:
- Fixed monthly fee, no Token anxiety
- Multi-model free switching (Qwen + Kimi + GLM, etc.)
- Support mainstream AI programming tools

### Q2: What's the difference from Tongyi Lingma?

| | Tongyi Lingma | Alibaba Cloud Coding Plan |
|:---|:---|:---|
| **Positioning** | Intelligent coding assistant | AI programming package |
| **Usage** | IDE plugin | API call |
| **Billing** | Free/pay per person-month | Fixed monthly fee |
| **Suitable for** | Personal development | Tool integration/team |

**Simple understanding**: Tongyi Lingma is a "partner", Coding Plan is "fuel".

### Q3: Is 7.9 yuan for first month real?

**Yes**. Lite version first month special price is ¥7.9, but only for the first month.

Starting from the second month, original price applies, check official website for details.

## 2. Package and Billing

### Q4: What's the difference between Lite and Pro versions?

| Comparison | Lite Version | Pro Version |
|:---|:---|:---|
| **First Month Price** | ¥7.9 | ¥39.9 |
| **Monthly Request Quota** | 18,000 times | 90,000 times |
| **Model Support** | All | All |
| **Suitable For** | Personal/light use | Heavy/team use |

### Q5: What to do when quota is used up?

- Wait for next month reset
- Upgrade to Pro version
- Purchase extra quota (if available)

### Q6: Can I unsubscribe?

Unsubscribe policy based on Alibaba Cloud official website. Generally:
- Monthly subscription can unsubscribe anytime
- Remaining time still usable after unsubscribe

### Q7: Can I switch between Lite and Pro?

Yes. Can upgrade or downgrade packages in console.
