## Detailed Comparison

### 1. Price Comparison

| Product | Lite First Month | Lite Original | Pro First Month | Pro Original |
|:---|:---|:---|:---|:---|
| Alibaba Cloud Coding Plan | **¥7.9** | - | **¥39.9** | - |
| Volcano Ark Coding Plan | ¥8.91 | ¥40 | ¥44.91 | ¥200 |

**Tencent Cloud TOKEN Plan**: No fixed monthly fee, pay as you go

**Conclusion**: Alibaba Cloud has lowest first month price, smallest trial cost.

### 2. Model Comparison

| Product | Self-developed Models | Third-party Models | Model Count |
|:---|:---|:---|:---|
| Alibaba Cloud Coding Plan | **Qianwen series (Qwen3.5, etc.)** | Kimi, GLM, MiniMax | 8+ |
| Volcano Ark Coding Plan | Doubao series | Kimi, GLM, DeepSeek | 6 |
| Tencent Cloud TOKEN Plan | Tencent Hunyuan | Multiple options | Flexible |

**Conclusion**: Alibaba Cloud Qianwen code models perform excellently in Chinese scenarios.

### 3. Tool Support Comparison

All three products support mainstream tools:
- Cursor
- Claude Code
- Cline
- Each vendor's own tools (Qwen Code, Doubao, etc.)

**Conclusion**: Not much difference in tool support.

### 4. Stability Comparison

| Product | Infrastructure | Network Advantage |
|:---|:---|:---|
| Alibaba Cloud Coding Plan | Alibaba Cloud global infrastructure | Many domestic nodes, fast speed |
| Volcano Ark Coding Plan | ByteDance infrastructure | Stable domestic service |
| Tencent Cloud TOKEN Plan | Tencent Cloud infrastructure | Enterprise-level stability |

**Conclusion**: All are from major companies, all have stability guarantees.

## Selection Advice

### Choose Alibaba Cloud Coding Plan if:

✅ Want to experience **powerful capability of Qianwen code models**
✅ Already using **other Alibaba Cloud services**
✅ Pursuing **lowest trial cost** (7.9 yuan first month)
✅ Need **enterprise-level service** and ecosystem integration
✅ Mainly doing **Chinese project development**

### Choose Volcano Ark Coding Plan if:

✅ Pursuing **ultimate value for money**
✅ Like **Doubao model's style**
✅ Stable usage, suitable for **fixed packages**

### Choose Tencent Cloud TOKEN Plan if:

✅ Usage **fluctuates greatly**, don't want to be bound by packages
