## 四、工具配置问题

### Q12：支持哪些编程工具？

支持主流AI编程工具：
- Qwen Code（阿里官方）
- Claude Code
- Cursor
- Cline
- Codex
- Kilo CLI

### Q13：Qwen Code怎么配置？

1. 下载安装Qwen Code
2. 运行 `/auth` 选择Api-Key方式
3. 选择Coding Plan
4. 填入API Key

### Q14：Cursor可以用吗？

可以。在Cursor中配置阿里云Coding Plan的API Key和Base URL即可。

### Q15：VS Code怎么接入？

通过Cline插件，配置阿里云Coding Plan的API信息。
