## I. Basic Questions

### Q1: What is Alibaba Cloud Coding Plan?

An **AI programming subscription service** launched by Alibaba Cloud Bailian, providing:
- Fixed monthly fee, no token anxiety
- Multiple models freely switchable (Qianwen + Kimi + GLM, etc.)
- Support for mainstream AI programming tools

### Q2: What's the difference between Coding Plan and Tongyi Lingma?

| | Tongyi Lingma | Alibaba Cloud Coding Plan |
|:---|:---|:---|
| **Positioning** | Intelligent coding assistant | AI programming package |
| **Usage method** | IDE plugin | API call |
| **Billing** | Free/per person per month | Fixed monthly fee |
| **Suitable for** | Individual development | Tool integration/teams |

**Simple understanding**: Tongyi Lingma is a "partner", Coding Plan is "fuel".

### Q3: Is the first month ¥7.9 real?

**Yes, it's real**. The Lite version has a special first-month price of ¥7.9, but only for the first month.

From the second month onwards, it returns to the original price, subject to the official website.

### Q5: What if I use up my request quota?

Unsubscription policy is subject to Alibaba Cloud's official website. Generally:

Yes. You can upgrade or downgrade the package in the console.

## IV. Tool Configuration Questions

### Q12: Which programming tools are supported?

Supports mainstream AI programming tools:
- Qwen Code (Alibaba official)
- Claude Code
- Cursor
- Cline
- Codex
- Kilo CLI

### Q13: How to configure Qwen Code?

1. Download and install Qwen Code
2. Run `/auth` and select Api-Key method
3. Select Coding Plan
4. Enter API Key

### Q14: Can Cursor be used?

Yes. Configure Alibaba Cloud Coding Plan's API Key and Base URL in Cursor.

### Q15: How to connect to VS Code?

Through the Cline plugin, configure Alibaba Cloud Coding Plan's API information.
