## AtomCode 是什么？

一句话：**Claude Code 的国产开源替代方案**。

Claude Code 谁都知道好用，但对国内开发者来说，体验一言难尽——封号、限速、网络问题……用起来心累。

AtomCode 就是来填这个空白的。

| 特性 | Claude Code | AtomCode |
|------|------------|----------|
| 开源 | ❌ 闭源 | ✅ MIT 开源 |
| 国产模型支持 | ❌ | ✅ DeepSeek/Qwen/智谱GLM |
| Token 费用 | 💰 按量付费 | 🆓 长期限免 |
| 网络要求 | ⚠️ 需翻墙 | ✅ 国内直连 |
| 包体大小 | 较大 | < 50MB |
| 启动速度 | 一般 | 秒级启动 |

## 核心特点：5 个理由让你选 AtomCode

### 🔥 1. 纯 Rust 打造，性能拉满

AtomCode 是**纯 Rust 自研**，没有 fork 任何项目。

Rust 是什么？被公认是"性能之王"的编程语言，以安全和高效著称。

AtomCode 团队用 **28 天**完成了核心开发，**3.7 万行 Rust 代码，1096 次提交**。

结果就是：**包体小于 50MB，秒级启动，资源消耗极低**。普通笔记本就能流畅运行。

### 🔥 2. 原生支持国产大模型

这是 AtomCode 最核心的差异化优势。

原生深度支持：
- **DeepSeek** — 性价比之王
- **Qwen（通义千问）** — 阿里旗舰
- **智谱 GLM** — 清华系代表
- **OpenAI 接口** — 国际模型也能接

你可以根据任务类型**自由切换模型**，真正实现"模型自由"。

做复杂推理用 DeepSeek，做代码补全用 Qwen，做创意生成用 GLM——**哪个好用用哪个**。

### 🔥 3. Token 长期限免

这是最实在的一点。

很多 AI 编码工具，用着用着就要充钱。Token 消耗是个无底洞。

AtomCode 承诺：**Token 长期限免**。

对个人开发者来说，这意味着零成本就能用上顶级的 AI 编码能力。

### 🔥 4. 智能机制加持，复杂任务也稳

针对 AI 编码过程中最头疼的几个问题，AtomCode 内置了三大机制：

- **📌 强制规划** — 在动手之前先拆解任务逻辑，避免 AI "跑偏"
- **📌 诊断压缩** — 精准定位代码和逻辑问题，自动优化
- **📌 JSON 修复** — 自动修正格式和语法错误，保证任务闭环

简单任务直接搞定，复杂任务也能稳稳交付。

### 🔥 5. 安全可控，隐私无忧

AtomCode 运行在你的**本地机器**上。

代码不会上传到任何服务器，**数据安全完全可控**。

对企业开发者来说，这一点尤为重要——不用担心理代码泄露的问题。

## 怎么用？

👉 官网：https://atomcode.atomgit.com/

安装超简单，一行命令搞定。支持 macOS、Linux、Windows 全平台。

装好之后，终端输入 `atomcode` 就能用了。

```bash
# 示例：让 AtomCode 帮你写一个文件批量重命名工具
❯ 帮我写一个Python脚本，把Downloads文件夹里所有PDF文件按日期重命名
```

 AtomCode 会自动完成：分析需求 → 写代码 → 运行测试 → 交付结果。

**全程你不需要写一行代码。**
