## 📝 第一步：注册账号

3分钟搞定：

**步骤：**
1. 打开AtomGit官网
2. 点击注册
3. 用手机号或邮箱注册
4. 登录

---

## 🚀 第二步：创建仓库

登录后，创建你的第一个仓库：

**步骤：**
1. 点击右上角的 **+** 号
2. 点击 **新建仓库**
3. 填写信息：
   - 仓库名称
   - 描述（可选）
   - 公开/私有
4. 点击创建

---

## 💻 第三步：上传代码

仓库创建好了，上传你的代码：

**方法1：Git命令行**
```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://atomgit.com/你的用户名/你的仓库名.git
git push -u origin main
```

**方法2：用AtomCode**
直接在AtomCode里操作，更简单！

---

## 👥 第四步：协作开发

如果是团队项目：

**怎么做：**
1. 进入仓库 **设置**
2. 点击 **协作者管理**
3. 添加成员的AtomGit用户名
4. 成员接受邀请后，就可以一起开发了
