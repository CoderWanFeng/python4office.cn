## 一、什么是移动云 Coding Plan？

移动云的 Coding Plan 基于中国移动自研的**九天大模型**系列，提供 AI 编程辅助服务。

### 核心特点

| 特性 | 说明 |
|------|------|
| **九天大模型** | 中国移动自研，国产大模型 |
| **运营商优势** | 网络资源丰富，访问稳定 |
| **央企背景** | 数据安全有保障 |
| **价格友好** | 运营商定价通常比较亲民 |

### 九天系列模型

- **九天·乾元**：最新旗舰版
- **九天·坤舆**：通用能力版
- **九天·讯飞**：语音+AI 融合版

## 二、为什么选移动云 Coding Plan？

### 1. 运营商网络优势

中国移动作为运营商，在网络上有着天然优势：
- 国内访问速度快
- 网络覆盖广
- 稳定性有保障

### 2. 央企背景加持

中国移动是央企，在数据安全方面：
- 符合国产化要求
- 数据安全有保障
- 适合政企客户

### 3. 价格亲民

运营商做 AI 服务，通常定价比较友好，适合个人开发者和中小企业。
