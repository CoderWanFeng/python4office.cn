**秒回：**

```
装饰器（Decorator）是 Python 的高级语法。

作用：在不修改原函数的情况下，给函数增加新功能。

例子：
```python
def log(func):
    def wrapper(*args, **kwargs):
        print(f"调用 {func.__name__}")
        return func(*args, **kwargs)
    return wrapper

@log
def say_hello():
    print("你好")

say_hello()
# 输出：
# 调用 say_hello
# 你好
```

常见用法：
- 日志记录
- 权限验证
- 性能测试
