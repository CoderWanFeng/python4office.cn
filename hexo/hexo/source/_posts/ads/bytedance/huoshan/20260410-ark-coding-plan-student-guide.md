## 学生党使用场景

### 场景1：课程作业

**问题：** 数据结构的算法作业不会写

**解决方案：**
```

要求有详细注释，并解释每一步的原理。
```

**推荐模型：** DeepSeek-V3（算法强）

### 场景2：毕业设计

**问题：** 需要快速搭建项目原型

**解决方案：**
用AI辅助生成基础代码框架，自己专注于业务逻辑。

**推荐模型：** Doubao（速度快）

**问题：** 想学Docker，但文档看不懂

```
Prompt: 请用通俗易懂的方式解释Docker的工作原理，
并给出一个Python项目的Docker化示例。
```

## 学生党专属Prompt技巧

### 技巧1：让AI当助教

```
你是一位耐心的编程老师，我正在学习[知识点]。
请用简单易懂的方式解释这个概念，并给出一个实际例子。
```

### 技巧2：让AI批改作业

```
请检查以下代码是否有问题，如果有请指出并解释：
[粘贴代码]
```

```

并给出答案和解析。
```
