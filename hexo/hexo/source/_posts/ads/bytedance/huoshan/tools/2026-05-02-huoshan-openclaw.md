## 二、配置步骤

### Step 1：开通火山方舟 Coding Plan

如果你还没有火山方舟 Coding Plan，先去开通：

👉 **专属优惠通道**：[https://www.volcengine.com/product/ark](https://www.volcengine.com/product/ark)

开通后，在火山方舟控制台获取你的 AppID 和 SecretKey。

---

### Step 2：在 OpenClaw 中配置火山方舟

1. 打开 OpenClaw，找到设置/配置页面
2. 进入模型配置，选择"自定义API"或"OpenAI兼容接口"
3. 填写以下信息：

| 字段 | 内容 |
|------|------|
| API URL | `https://ark.cn-beijing.volces.com/api/v3` |
| API Key | 你的 AppID:SecretKey |
| 模型名称 | `doubao-20250526` |

4. 保存并测试

---

### Step 3：开始使用豆包大模型

配置成功后，在 OpenClaw 中直接使用豆包大模型：

```
帮我写一个Python脚本，自动整理桌面文件
```

豆包对中文语境理解更准确，输出的代码更符合国内开发者的习惯。
