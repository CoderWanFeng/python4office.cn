## 二、配置步骤

### Step 1：开通火山方舟 Coding Plan

👉 **专属优惠通道**：[https://www.volcengine.com/product/ark](https://www.volcengine.com/product/ark)

开通后，获取你的 AppID 和 SecretKey。

---

### Step 2：在 Cline 中配置火山方舟

1. 打开 Cline 设置 → Model Provider
2. 选择"Custom"或"OpenAI Compatible"
3. 填写以下信息：

| 字段 | 内容 |
|------|------|
| API URL | `https://ark.cn-beijing.volces.com/api/v3` |
| API Key | 你的 AppID:SecretKey |
| 模型名称 | `doubao-20250526` |

4. 保存并测试连接

---

### Step 3：开始使用

配置成功后，在 VS Code 中使用 Cline，响应速度比海外模型快很多。

## 三、豆包在 Cline 里能做什么？

### 💻 代码生成

```
帮我写一个Python脚本，处理CSV文件并生成报表
```

豆包中文理解强，输出的代码更符合国内开发者的习惯。

---

### 🔍 代码审查

```
帮我审查这段代码，找出潜在问题
```

豆包可以快速发现代码中的问题并给出修复建议。
