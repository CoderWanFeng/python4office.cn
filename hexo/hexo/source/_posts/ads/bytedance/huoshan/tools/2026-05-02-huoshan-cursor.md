### Step 1：开通火山方舟 Coding Plan

开通后，获取你的 AppID 和 SecretKey。

---

### Step 2：在 Cursor 中配置火山方舟

1. 打开 Cursor 设置 → Models
2. 选择"Add Custom Model"或"Custom Provider"
3. 填写以下信息：

| 字段 | 内容 |
|------|------|
| API URL | `https://ark.cn-beijing.volces.com/api/v3` |
| API Key | 你的 AppID:SecretKey |
| 模型名称 | `doubao-20250526` |

4. 保存并设置为默认模型

---

### Step 3：开始使用

配置成功后，在 Cursor 中直接使用 AI 功能，响应速度比海外模型快很多。
