### 第二步：获取API Key

1. 登录火山引擎控制台
2. 进入方舟平台 → API Key管理
3. 创建新的API Key
4. 复制保存（只显示一次）

### 第三步：配置Cursor

1. 打开Cursor设置（Cmd/Ctrl + ,）
2. 找到 Models 选项
3. 点击 "Add Model"
4. 填写配置：
   - **Model Name**: `doubao-pro-32k`（或其他模型）
   - **API Key**: 你的火山方舟API Key
   - **Base URL**: `https://ark.cn-beijing.volces.com/api/v3`
5. 保存并测试

### 第四步：开始使用

现在你可以在Cursor里：
- 按 `Cmd/Ctrl + K` 唤起AI对话
- 选择火山方舟模型
- 开始AI编程

## 常见问题

**Q: 配置后没反应？**
A: 检查API Key是否正确，Base URL是否填写完整。

**Q: 响应慢？**
A: 尝试切换不同模型，或检查网络连接。

**Q: 和Cursor Pro有什么区别？**
A: Cursor Pro是官方模型，Coding Plan是自配置模型。后者更便宜、更灵活。
