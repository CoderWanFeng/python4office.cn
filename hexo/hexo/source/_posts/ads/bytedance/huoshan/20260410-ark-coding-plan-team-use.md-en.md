## Team Configuration Plan

### Plan A: Individual Subscriptions (Recommended)

- One account per person
- Each manages their own API Key
- Cost: 36 yuan × 5 people = 180 yuan/month

**Advantages:**
- No interference between each other
- Independent usage statistics
- Simple handover when leaving

### Plan B: Shared API Key

- One person subscribes, generates multiple Keys
- Team members share

**Advantages:**
- Lower initial cost
- Unified management

**Disadvantages:**
- Usage easily exceeds
- Hard to track individual usage
