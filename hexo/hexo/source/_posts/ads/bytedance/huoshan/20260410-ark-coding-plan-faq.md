## 使用相关问题

### Q5: 支持哪些IDE？

理论上支持所有支持OpenAI API格式的IDE：
- ✅ Cursor
- ✅ VS Code (Continue插件)
- ✅ JetBrains系列 (CodeGPT插件)
- ✅ Vim/Neovim
- ✅ 任何支持自定义API的工具

### Q6: 响应速度怎么样？

国内服务器，访问速度很快：
- 一般请求：1-3秒
- 复杂请求：3-10秒

比国外工具快很多。

### Q7: 代码生成质量如何？

取决于选择的模型：
- DeepSeek-V3：代码能力最强
- Doubao：速度快，日常够用
- GLM-4：中文注释好

### Q8: 可以同时用多个模型吗？

可以！一个订阅，所有模型都能用。

可以在工具里配置多个模型，随时切换。

### Q9: 有使用限制吗？

一般没有硬性限制，但：
- 极端高频使用可能触发限流
- 具体限制以官方说明为准

### Q10: API Key泄露了怎么办？

立即：
1. 登录控制台删除旧Key
2. 创建新Key
3. 更新所有配置
