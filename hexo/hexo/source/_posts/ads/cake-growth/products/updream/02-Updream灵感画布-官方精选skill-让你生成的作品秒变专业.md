## 一、Updream 灵感画布是什么？

**Updream** 是一款"对话式 AI 创作画布"——你不需要懂 prompt，直接说"在画布上为我创建……"，它就开始画。

但我今天重点不是画布本身，而是它的**官方精选 skill 商店**——每个 skill 都是一个**专业级的提示词模板**。

我用了 2 周，最大的感受是：**省的不是时间，是 prompt 写崩时那种挫败感**。你吭哧吭哧写了 3 小时 prompt，AI 出来一张"廉价感满满"的图，那种感觉你懂的。

## 二、6 个最有用的官方精选 skill

### 1. AI 短剧资产设计师 ⭐ （使用 5.0k 次）

**作者**：安提米苏

**核心能力**：

输入任意角色、道具、场景，自动输出**带版式规范的中文正向 prompt**。

实战示例：

```
✅ 你：「古风女主，24 岁，腰挂玉佩，手持长剑，白衣飘飘」
✅ Skill：（自动生成完整 prompt）

>>> 中文版（适合国内 AI 模型）：
古风女主，全身立绘，24 岁华族少女，腰挂羊脂玉佩，
手持银色长剑，白色轻纱长裙飘动；蝴蝶忍风，水墨笔触，
背景中式庭院，月光下，发丝飘动；Q 版风格，可爱化，
专注表情，浅色调，电影分镜构图，...

>>> 英文版（适合 Midjourney / SD）：
A 24-year-old Chinese heroine in traditional Hanfu costume...
```

**价值**：跳过 90% 的 prompt 学习时间，**直接生成专业级中文 prompt**。

### 2. 大圆镜科普文案写手 （使用 376 次）

**作者**：大圆镜科普

**核心能力**：专为生命科学类科普视频定制。

输出包含：

- 完整叙事框架（开场 - 转折 - 收尾）
- 语言风格（科普 + 通俗）
- 37 个选题的资料库
- **BFS 图片索引**（直接配图）
- 口播稿 + 分镜建议

**价值**：**文科生也能写科普视频脚本**。

### 3. 告别廉价感｜SD2.0 电影级提示词优化大师 ⭐ （3.0k）

**作者**：么鸣堂

**核心能力**：把你写的分镜稿，翻译成**电影摄影语言**。

内置 5 大视觉风格：

| 风格 | 适用 |
|------|------|
| **新中式美学** | 国风短剧 |
| **赛博朋克** | 科技感短剧 |
| **胶片质感** | 文艺短片 |
| **写实纪录** | 纪实类 |
| **动画分镜** | 动画短剧 |

**价值**：**让你的 AI 分镜图有"电影感"，而不是"廉价感"**。

### 4. 角色多视图+细节展示 （3.0k）

**作者**：hdxfz

**核心能力**：一个角色生成**多视图 + 细节展示 + 不同情绪**。

实战示例：

```
✅ 你：「画一个穿越男主，正面 + 侧面 + 背面，附腰带和腰包细节」
✅ Skill：（自动生成 5 张图）
  - 正面全身
  - 侧面立绘
  - 背面立绘
  - 腰带细节特写
  - 腰包细节特写
```

**价值**：角色形象统一，**短剧角色一致性大问题轻松解决**。

### 5. 二创助手：角色图微调器 ★ 提高过审概率 （840）

**作者**：高坂大芥末

**核心能力**：给一张角色图 → **微调 + 调整光影和材质** → 输出版权清洁的角色三视图。

特别适合做**二创**（用现有 IP 角色）：

- 全身细节微调
- 光影和材质重新打光
- 输出三视图（正视、侧视、背视）
- 背景纯灰，横版 3:2
- 使用 `banana-pro` 模型

**价值**：**过审率大幅提高**（已经是卖点"提高过审概率"）。

### 6. 第一人称视角萌宠视频提示词生成器 （2.3k）

**作者**：小陈的脚凑凑的

**核心能力**：生成 **第一人称视角 (POV) 视频提示词**，带：

- 真实画风
- 一镜到底
- 手持呼吸感

输出格式：

```json
{
  "shot_type": "POV",
  "movement": "一镜到底",
  "style": "手持 + 呼吸感",
  "scene": "客厅，猫咪走来走去",
  "camera": "跟随猫咪移动"
}
```

**价值**：**直接用于视频生成模型（可灵 / Sora / Hailuo）**，不用自己转换。
