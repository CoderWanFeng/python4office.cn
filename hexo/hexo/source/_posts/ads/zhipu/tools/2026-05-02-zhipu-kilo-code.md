## 二、配置步骤

### Step 1：开通智谱 GLM 服务

👉 **专属优惠通道**：[https://open.bigmodel.cn/](https://open.bigmodel.cn/)

注册并开通智谱 GLM 服务，获取你的 API Key。

---

### Step 2：在 Kilo Code 中配置智谱 GLM

1. 打开 Kilo Code 设置 → AI Providers
2. 选择"Custom"或"OpenAI Compatible"
3. 填写以下信息：

| 字段 | 内容 |
|------|------|
| API URL | `https://open.bigmodel.cn/api/paas/v4` |
| API Key | 你的智谱 API Key |
| 模型名称 | `glm-4-flash` |

4. 保存并设置为默认模型

---

### Step 3：开始使用

配置成功后，在 Kilo Code 中直接使用 GLM 进行代码生成和优化。
