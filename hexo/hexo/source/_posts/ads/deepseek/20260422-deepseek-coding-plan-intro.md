## 💡 DeepSeek Coding Plan 是什么

**官方定义**：基于 DeepSeek 开源大模型的编程辅助订阅服务。

**它和你自己调 API 有什么区别**：

| 维度 | 自己调 API | Coding Plan |
|------|-----------|-------------|
| 计费方式 | 按 token | 月度订阅 |
| 适合场景 | 偶发 / 调试 | 每天高强度编程 |
| 模型切换 | 自由 | 套餐内可切换 |
| 价格 | 1-3 元/M tokens | 约等于 50-200 次完整对话/月 |
| 配套 IDE | 自己接 | Cursor / Copilot / VS Code 一键接 |

**核心特点**：

- ✅ 模型开源：V3 / Coder / V4 全开源，可本地部署
- ✅ 性价比极高：API 1 元起，Coding Plan 月费对标 Cursor $20 但功能不输
- ✅ 代码能力强：HumanEval 90%+，对标 GPT-4
- ✅ 社区活跃：全球开发者 Top 3

## 🚀 选 DeepSeek Coding Plan 的 3 个核心理由

### 1. 开源免费，本地部署零 API 费用

DeepSeek 的模型 **完全开源**（MIT 协议），意味着：

- ✅ 可以本地部署（24G 显存跑 V3 量化版）
- ✅ 可以二次开发（微调 / 蒸馏）
- ✅ 可以私有化使用（数据不出公司）
- ✅ 商用免费（协议允许）

**有技术能力的团队，本地部署 = 0 API 费用**。

### 2. 代码能力对标 GPT-4

DeepSeek-Coder 在主流代码 benchmark 上的表现：

- **HumanEval**：90%+ pass@1
- **MBPP**：85%+
- **LiveCodeBench**：对标 GPT-4 Turbo
- **中文代码**：显著优于 GPT-4 / Claude

**真实体验**：
- 代码补全：准确率对标 Copilot
- 代码审查：能找出潜在 bug 和性能问题
- Agent / 多文件改动：V4 Pro 表现稳定

### 3. 性价比之王

**价格对比**（月度订阅）：

| 工具 | 月费 | 包含额度 | 单次对话成本 |
|------|------|----------|--------------|
| Cursor Pro | $20 (~145 元) | 无限 GPT-4 | ~0.5 元 |
| GitHub Copilot | $10 (~72 元) | 2000 次补全 | ~0.04 元 |
| Claude Pro | $17 (~123 元) | 有限 Claude | ~0.4 元 |
| **DeepSeek Coding Plan** | **~50 元起** | **1M tokens 起** | **~0.05 元** |

**省钱 = 硬道理**。
