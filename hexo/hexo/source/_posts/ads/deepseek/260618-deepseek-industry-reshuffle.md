## 一、2025 年中：洗牌前的"三足鼎立"

那时候 AI 编程工具市场，**3 家美企瓜分 90% 份额**：

| 工具 | 月费 | 底层模型 | 国内访问 |
|------|------|----------|----------|
| Cursor | **$20** | GPT-4 / Claude（独家绑定）| ❌ 经常断 |
| GitHub Copilot | **$10** | OpenAI 系列 | ❌ 经常断 |
| Claude Code | **$17** | Claude 独家 | ❌ 经常断 |

**痛点 3 个**：
- 💸 贵：月付 $20 起步，团队用一年 2-3 万
- 🐌 慢：国内访问经常 504、超时
- 🔒 锁：模型独家绑定，想换？没门

**那时候"AI 编程"是有钱人的玩具**。

## 二、2025-2026：DeepSeek 一己之力打穿地板

DeepSeek V3 → V4 → V4 Pro 2.5 折，**把 API 价格砍到 1 元/百万 tokens**。

| 模型 | 输入价格（每百万 tokens）| 价格屠夫指数 |
|------|------------------------|--------------|
| GPT-4 | ~200 元 | 基准 1x |
| Claude Sonnet 4.6 | ~21 元 | 10x 便宜 |
| **DeepSeek V4 Pro（2.5 折）**| **3 元** | **66x 便宜** |
| DeepSeek V3.2 | ~1 元 | 200x 便宜 |

**关键变化**：
- ✅ 价格砍掉 **2 个数量级**（不只是 50%）
- ✅ OpenAI 兼容接口，**任何工具都能接**
- ✅ 中文 + 代码能力对标 GPT-4
- ✅ 国内节点直连，**秒级响应**

**这把所有 AI 编程工具的定价模型逼到了墙角**。

### 红利 1：模型可换 → 个人开发者

## 五、给程序员的 3 条建议

### 建议 1：不要订阅年度 AI 工具

**为什么**：现在大洗牌期，**月度订阅能保留"随时换"的灵活性**。
年度订阅像是"赌某家不会降价"——目前看，多数家都在降。

### 建议 2：学会"模型可换"配置

**5 分钟投资，永久受益**：
- Cursor：Settings → Models → OpenAI Compatible
- VS Code + Cline：装插件 → 配 Custom Provider
- Claude Code：环境变量切 base_url
- 命令行：Ollama + DeepSeek 本地部署

**这套技能未来 5 年都会值钱**。

### 建议 3：关注"模型 + 工具"的产品组合

**单独看一家意义不大**，要看组合：
- **日常**：Cursor + DeepSeek V3.2
- **审查**：Roo Code + DeepSeek-V4 Pro
- **本地**：VS Code + Ollama + DeepSeek Coder
- **企业**：JetBrains + 自部署 DeepSeek V3

**会组合的人，是这波红利的最大受益者**。
