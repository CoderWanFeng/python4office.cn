## 一、Django Girls 是什么？

**Django Girls**：

- 创立于 **2014 年**

## 四、Django Girls 一天的流程

**一个 Django Girls 工作坊是这样的**：

### 上午（9:00-12:00）

- 9:00 - 欢迎 + 安装环境
- 9:30 - Python 基础（变量、if、循环）
- 10:30 - 休息
- 10:45 - Python 进阶（列表、字典、函数）
- 12:00 - 午餐

### 下午（13:00-17:00）

- 13:00 - Django 基础（MTV 模式）
- 14:30 - 休息
- 14:45 - 写第一个博客
- 16:30 - 部署到互联网
- 17:00 - 结束 + 合影

**1 天 = 写出一个真实的网站**。

**网站包括**：

- 首页
- 博客列表
- 文章详情
- **部署上线**

**很多学员说："这是我人生第一个网站！"**

## 八、为什么是 Django？

**为什么 Django Girls 教 Django 而不是 Flask、FastAPI**？

### 原因 1：Django 全能

- Web 框架一站式
- 不用选库
- **新手友好**

### 原因 2：MTV 模式清晰

- Model（数据）
- Template（页面）
- View（逻辑）
- **结构清晰**

### 原因 3：Admin 后台自动生成

- 不用自己写后台
- 学员有"成就感"
- **写完就能用**

### 原因 4：文档友好

- 官方文档写得好
- 中文翻译完整
- **新手能看**

### 原因 5：Django 社区支持

- Django Software Foundation
- **Django Girls 是 Django 社区的"亲儿子"**
- 长期支持
