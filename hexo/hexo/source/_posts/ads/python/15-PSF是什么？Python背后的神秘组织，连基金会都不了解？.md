## 一、PSF 是什么？

**PSF 全称：Python Software Foundation（Python 软件基金会）**。

**性质**：美国 501(c)(3) 非营利组织

**成立时间**：2001 年

**使命**：

> The mission of the Python Software Foundation is to promote, protect, and advance the Python programming language, and to support and facilitate the growth of a diverse and international community of Python programmers.

**翻译**：促进、保护、推进 Python 编程语言，支持和促进多元化、国际化 Python 程序员社区的发展。

**简单说**：

- Python 是开源的
- **但需要有人维护**
- **PSF 就是干这个的**

---

## 二、PSF 做的 4 件事

### 1️⃣ 资助（Grants）

**2024 年 PSF 发放资助**：

- **$655,000 美元**
- 资助 257 个项目/个人
- 覆盖 61 个国家

**资助方向**：

- Python 核心开发
- Python 教育资源
- Python 社区活动
- Python 多元化项目

**中国也受益**——多个中国 Python 社区曾获 PSF 资助。

### 2️⃣ 基础设施（Infrastructure）

**PSF 维护这些关键服务**：

| 服务 | 网址 | 作用 |
|------|------|------|
| **python.org** | https://python.org/ | 官方网站 |
| **PyPI** | https://pypi.org/ | 包索引（所有 pip 包都在这）|
| **Python 文档** | https://docs.python.org/ | 官方文档 |
| **PEP 系统** | https://peps.python.org/ | 增强提案 |
| **GitHub python 组织** | https://github.com/python | 源代码 |
| **邮件列表** | https://www.python.org/community/lists/ | 社区讨论 |
| **Discord/Slack 桥接** | - | 实时聊天 |

**这些服务一年要花**数百万美元**。**

**PSF 来买单。**

### 3️⃣ PyCon US 大会

**PSF 是 PyCon US 的主办方**：

- **全球最大 Python 大会**
- 每年 5 月举办
- 2025 年发 38.4 万美元**差旅资助**给 254 位与会者
- **让没钱的人也能参加**

**PSF 让"没钱也能学 Python、也能用 Python、也能贡献 Python"**。

### 4️⃣ 多样性、多元化

**PSF 的 D&I（多元与包容）工作组**：

- 鼓励女性进入 Python
- 支持少数族裔开发者
- **Django Girls、PyLadies** 等组织受 PSF 支持

## 三、PSF 怎么赚钱？

**PSF 是非营利组织，但需要钱**。

**钱从哪里来**：

### 来源 1：会员费

- **Fellow 会员**：年费（具体金额 https://www.python.org/psf/membership/）
- **管理会员**（Managing Member）
- **赞助会员**（Contributing Member）

**会员有什么权益**：

- PyCon US 折扣
- 投票权
- 选举 PSF 董事会
- 特殊周边

### 来源 2：赞助商

**PSF 官网赞助商分 4 个等级**：

- **Visionary Sponsor**：顶层赞助商
- **Sustainability Sponsor**
- **Contributing Sponsor**
- **Supporting Sponsor**

**2026 年新晋 Visionary Sponsor**：

- **Hudson River Trading (HRT)**（2026 年 5 月）

**赞助商给钱，PSF 帮他们在 PyCon 等活动上曝光。**

### 来源 3：捐款

- 个人捐款
- 企业捐款
- **PSF 接受 501(c)(3) 抵税**

**捐款页面**：https://www.python.org/psf/donations/

### 来源 4：PyCon 收入

- PyCon US 门票
- PyCon US 赞助
- **这是 PSF 最大的收入来源**

---

## 四、PSF 的治理结构

### 董事会（Board of Directors）

**PSF 由董事会管理**：

- 11 位董事
- **由会员选举**
- 任期 3 年
- 每年改选一部分

**2026 年 PSF 董事会选举**：

- 2026 年 6 月 18 日开始（已过）
- 即将宣布结果
- **新董事将决定 PSF 未来方向**

**董事会决定**：

- 资助分配
- 战略方向
- 财务预算

### 工作人员

**PSF 有少量全职员工**：

- **Executive Director（执行董事）**
- **CPython Developer in Residence**（CPython 驻场开发者）
- 行政、运营人员

**第一位 CPython 驻场开发者**：Łukasz Langa（2018-2023，5 年任期）

**2023 年后由 Pablo Galindo 接管。**

### 志愿者

**PSF 大量工作由志愿者完成**：

- 文档翻译
- 会议组织
- 社区管理
- **这些都是免费的**

## 五、PSF 和 Python 社区的关系

### PSF = 基础设施提供者

**PSF 不"控制"Python**，**PSF 提供资源让社区发展**。

### 关系 1：与 CPython 核心团队

- **核心团队独立工作**
- PSF 提供**差旅资助、基础设施**
- **核心团队不受 PSF 行政干预**

### 关系 2：与 Python 软件包（PyPI）

- **PSF 拥有 PyPI**
- **由志愿者和 PSF 员工共同维护**
- **任何人都可以上传包**

### 关系 3：与 PyCon 各地组织

- **PyCon US 由 PSF 主办**
- **PyCon EU、APAC 等由各地社区组织**
- **PSF 提供指导、支持**

### 关系 4：与 Guido（Python 之父）

- **Guido 是 PSF 创始成员**
- 卸任 BDFL（终身仁慈独裁者）后
- **仍然是核心社区成员**

## 七、PSF 2026 战略计划（草案）

**2026 年 6 月，PSF 开放战略计划草案供社区反馈**：

**草案要点**：

- **更聚焦核心**：减少分散项目
- **加强多元化**：支持更多发展中国家
- **扩展资助**：增加小额资助
- **PyCon 创新**：线上+线下混合
- **社区健康**：解决社区治理问题

**这是 PSF 未来 3 年的方向。**

**反馈截止日期：2026 年 6 月底**。

## 八、PSF 成员怎么加入？

### 5 个步骤成为 PSF 会员：

1. **访问** https://www.python.org/psf/membership/
2. **注册账号**
3. **选择会员类型**
4. **支付年费**
5. **成为 Fellow/Managing/Contributing 会员**

**会员权益**：

- ✅ 投票权
- ✅ 选举董事会
- ✅ PyCon US 折扣
- ✅ 社区认可
- ✅ 特殊周边

**普通开发者成为 PSF 会员**，**最直接的支持 Python 方式**。

## 十、PSF 的 5 个常见误解

### 误解 1：PSF 拥有 Python

- ❌ 错
- ✅ **Python 是开源的，没人"拥有"**
- PSF 拥有**商标、域名、PyPI**

### 误解 2：PSF 给 Python 核心开发者发工资

- ❌ 错
- ✅ **核心开发者大部分是志愿者**
- 少数 PSF 资助**驻场开发者**

### 误解 3：PSF 决定 Python 方向

- ❌ 错
- ✅ **Python 方向由 PEP 流程决定**
- PSF 提供基础设施

### 误解 4：只有美国人能加入 PSF

- ❌ 错
- ✅ **PSF 会员全球开放**
- **中国 Python 开发者也能加入**

### 误解 5：PSF 很赚钱

- ❌ 错
- ✅ **PSF 收入刚好覆盖支出**
- 每年还要想办法筹款

## 十一、最后的最后

**PSF 这事，3 句话总结**：

1. **PSF 是什么**：501(c)(3) 非营利组织，维护 Python 生态
2. **PSF 干什么**：资助 + 基础设施 + PyCon + 多元化
3. **怎么支持**：加入会员 / 捐款 / 参加 PyCon

**PSF 是 Python 圈最重要的"幕后英雄"。**

**没有 PSF，python.org 早就消失了，PyPI 也不存在。**

**我们用 Python 写代码时，**每一次 pip install 都在用 PSF 的服务**。**

**如果你是 Python 受益者，**考虑支持 PSF**。**

**捐钱也好，加入会员也好，参加 PyCon 也好。**

**让 Python 走得更远，**需要每个人的支持**。**
