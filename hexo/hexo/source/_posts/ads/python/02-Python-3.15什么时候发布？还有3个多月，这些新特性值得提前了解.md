## 三、3.15 值得关注的 5 大新特性

**根据 Python 官方 PEP 和社区讨论，3.15 主要有这些新东西**：

### 特性 1：JIT 编译器（实验性）

**这是 3.15 最大的亮点！**

- Python 终于有了**官方 JIT 编译器**
- 代码执行速度预计提升 **5-15%**
- 默认关闭，需要手动开启
- 未来 3.16、3.17 会持续优化

**对普通开发者的影响**：
- 跑数据分析、AI 模型训练，**会快一点**
- Web 开发、脚本编写，**基本无感**
- 但这是 Python 性能追赶的关键一步

### 特性 2：类型注解继续增强

- **PEP 747**：TypeExpr 类型表达式
- **PEP 749**：注解元数据
- 类型系统更强大
- **写大型项目的福音**

### 特性 3：REPL 大升级

- 之前要装 `IPython` 才能用的功能
- 3.15 把它**内置**到官方 REPL
- 语法高亮、自动补全、多行编辑
- **官方 REPL 终于不反人类了**

### 特性 4：错误信息更友好

- 新增 **100+ 错误信息的解释**
- 比如 `IndentationError` 会直接告诉你**哪一行、为什么错**
- **新手友好度大幅提升**

### 特性 5：标准库更新

- `pathlib` 继续完善
- `asyncio` 性能优化
- `typing` 增强
- 一些过时的模块被标记为弃用

## 五、现在该做什么准备？

**距离 3.15 发布还有 103 天，你可以做这些**：

### 第 1 件事：检查你现在的版本

```bash
python --version
```

**如果还在用 3.9 或更早**，**别等 3.15 了**，先升到 3.12 或 3.13 才是正经事。

### 第 2 件事：检查依赖库

```bash
pip list --outdated
```

**看看你最常用的库，最新版本支持到 Python 多少**。

### 第 3 件事：本地试装 3.15 beta

```bash
# 下载 3.15 beta
# 不要在生产环境装
python3.15 -c "print('Hello 3.15')"
```

### 第 4 件事：等正式版

**别急，让子弹飞一会儿。**

**正式版发布后等 1-2 个小版本再上，更稳。**
