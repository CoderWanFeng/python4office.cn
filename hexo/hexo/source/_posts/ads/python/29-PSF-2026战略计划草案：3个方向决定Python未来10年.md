## 一、为什么有这份草案？

**PSF 是 Python 的"管家"**。

**2026 年，Python 面临 3 大新挑战**：

### 挑战 1：AI 时代

- LLM 改变编程方式
- Python 是 AI 的"母语"
- **PSF 需要调整方向**

### 挑战 2：全球扩张

- 非洲、拉美、东欧 Python 大发展
- 多元化、多样性压力
- **PSF 需要资源支持**

### 挑战 3：治理挑战

- 社区越来越大
- 决策越来越难
- **PSF 需要新治理模型**

**这份草案就是 PSF 对 3 大挑战的回应。**

---

## 二、3 大核心方向

### 方向 1：聚焦核心，AI 优先 ⭐⭐⭐⭐⭐

**草案要点**：

- PSF 将**优先支持 AI 相关项目**
- 资助 PyTorch、Transformers、LangChain 生态
- **支持 LLM 应用开发**
- 加强 AI 安全研究

**为什么**：

- Python 是 AI 时代的关键语言
- **AI 是 Python 增长的"第二曲线"**
- 错过 AI = 错过下一个 10 年

**具体措施**：

- 设立 **AI 专项基金**（100 万美元/年）
- 资助 **AI 开发者驻场**（与 OpenAI、Anthropic 合作）
- 支持 **AI 工具链**（PyTorch、Transformers）
- 推动 **AI 安全标准**

### 方向 2：全球扩张，多元化 ⭐⭐⭐⭐⭐

**草案要点**：

- **支持非洲、拉美、东欧** Python 社区
- 设立**多语言翻译基金**
- 资助 **女性、少数族裔** 开发者
- 加强 **发展中国家** Python 教育

**为什么**：

- Python 社区应该代表全球
- 多元化带来创新
- **发展中国家是 Python 未来用户**

**具体措施**：

- 设立 **Global PyCon Fund**（50 万美元/年）
- 资助 **本地化翻译**（中文、阿拉伯语、斯瓦希里语等）
- 支持 **Django Girls** 等多元化项目
- 推动 **包容性代码风格**（PEP 8 改进）

### 方向 3：治理现代化 ⭐⭐⭐⭐

**草案要点**：

- **改革 PSF 董事会**结构
- 加强 **社区参与**
- 推动 **PEP 流程现代化**
- 建立 **长期资金模型**

**为什么**：

- 社区越来越大，决策越来越难
- **老治理模型不适应新规模**
- 资金来源需要更多元化

**具体措施**：

- 董事会从 11 人扩到 15 人
- 增加 **国际董事** 席位
- 建立 **PEP 数字签名系统**
- 推动 **企业赞助多元化**

## 三、5 大具体措施

### 措施 1：AI 专项基金

- 资金：**100 万美元/年**
- 用途：AI 工具链、AI 安全、AI 教育
- **2027 年开始**

### 措施 2：Global PyCon Fund

- 资金：**50 万美元/年**
- 用途：支持发展中国家 PyCon
- **2026 年底启动**

### 措施 3：本地化翻译基金

- 资金：**20 万美元/年**
- 用途：翻译 Python 文档
- **2026 年 Q3 启动**

### 措施 4：Django Girls 升级

- 资金：**30 万美元/年**
- 用途：扩展到 50+ 国家
- **2026-2030 持续**

### 措施 5：CPython 驻场开发者

- 资金：**80 万美元/年**
- 用途：3 位驻场开发者
- **3.13 至今 1 位，3.14 计划 2 位**
