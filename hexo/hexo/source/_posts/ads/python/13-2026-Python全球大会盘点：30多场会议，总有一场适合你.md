## 一、PyCon 是什么？

**PyCon = Python Conference**。

**1994 年开始举办，**至今已有 30+ 年历史**。**

**是 Python 开发者最盛大的聚会**：

- 🌍 全球巡回
- 🎤 顶级专家演讲
- 🛠️ 实操工作坊
- 🤝 结交同行
- 🎁 各种周边
- **对 Python 社区最重要的事**

**PyCon 大会由各地 Python 社区独立组织**，**但都遵循 PSF 的指导**。

## 五、PyCon 上能学到什么？

### 收益 1：听顶级专家演讲

- Guido van Rossum（Python 之父）
- **核心库作者**（Django、Flask、FastAPI、NumPy）
- **顶级 Python 工程师**

### 收益 2：参与 Sprint

- 跟着核心开发者**一起写代码**
- 直接贡献 Python 本身
- **你的代码可能进入下个版本**

### 收益 3：实操工作坊

- **手把手**教你用 Python
- 现场解决你的问题
- 比起看视频，**实战 10 倍速**

### 收益 4：Networking

- 结识**全球 Python 开发者**
- 找工作、找合作、找灵感
- **很多顶级工作都是 PyCon 上找到的**

### 收益 5：最新消息

- **下个版本的 Python 特性**
- **新库发布**
- **社区动向**

### 收益 6：周边福利

- 各种贴纸、T恤
- 赞助商展位
- 免费午餐
- **晚上还有社交活动**

## 六、PyCon 怎么参加？

### 步骤 1：选会议

- 选你想去的会议
- 看时间、地点、议题
- **选 1-2 场，不要贪多**

### 步骤 2：买票

- 官方售票
- **早鸟票便宜**，**晚鸟票贵**
- 学生票、失业票**有折扣**

### 步骤 3：订酒店/机票

- 早订便宜
- **可以找 PyCon 群友拼房**

### 步骤 4：申请签证

- 国际会议需要签证
- **提前 1-3 个月办**

### 步骤 5：准备议题

- 如果你想演讲，**提前提交**
- 一般 6 个月前开放征集

### 步骤 6：参加

- 早到
- 多交流
- **穿 PyCon T 恤交朋友**

## 七、不去 PyCon 的 3 个替代方案

### 替代 1：PyCon 视频

- https://pyvideo.org/
- **所有 PyCon 演讲都有录像**
- 免费看
- **跟现场效果 90% 一样**

### 替代 2：Python 用户组

- https://www.python.org/community/usergroups/
- **本地 Python 开发者聚会**
- 免费
- 中国主要城市都有

### 替代 3：线上会议

- XtremePython 2026（11月17日）
- **远程参加**
- 不用出国
