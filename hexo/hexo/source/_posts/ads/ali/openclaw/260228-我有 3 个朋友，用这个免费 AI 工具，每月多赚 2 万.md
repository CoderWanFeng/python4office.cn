## 详细教程：从零开始做第一个项目

为了让你更快上手，我准备了第一个项目（AI 客服自动回复）的详细教程。

### 第一步：安装 OpenClaw（10 分钟）

按照上面的安装教程，完成 OpenClaw 的配置。

### 第二步：连接消息平台（5 分钟）

```bash
# 以微信为例
openclaw channels login
```

扫码登录你的微信账号。

### 第三步：配置自动回复（15 分钟）

在 OpenClaw 的工作区创建一个配置文件：

```json
{
  "autoReply": {
    "enabled": true,
    "rules": [
      {
        "keyword": "价格",
        "reply": "我们的产品价格是 XXX 元，具体可以咨询客服"
      },
      {
        "keyword": "发货",
        "reply": "我们一般在 24 小时内发货"
      }
    ]
  }
}
```

### 第四步：测试和优化（10 分钟）

找几个朋友测试一下，看看自动回复是否准确。

根据测试结果，优化回复模板。

### 第五步：开始接单（持续）
