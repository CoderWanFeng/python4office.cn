开通后，在百炼控制台复制你的 API Key。

---

### Step 2：在 Cursor 中配置阿里云百炼

1. 打开 Cursor 设置 → Models
2. 选择"Add Custom Model"或"Custom Provider"
3. 填写以下信息：

| 字段 | 内容 |
|------|------|
| API URL | `https://dashscope.aliyuncs.com/compatible-mode/v1` |
| API Key | 你的阿里云百炼 API Key |
| 模型名称 | `qwen-turbo` 或 `qwen-plus` |

4. 保存并设置为默认模型

---

### Step 3：开始使用

配置成功后，在 Cursor 中直接使用 AI 功能，响应速度比海外模型快很多。
