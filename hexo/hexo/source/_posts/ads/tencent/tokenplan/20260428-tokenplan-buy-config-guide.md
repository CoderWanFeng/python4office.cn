前几天有个朋友跟我吐槽："我用 Claude Code 才一周，信用卡账单直接爆了。"我问他怎么用的，他说："就是问几个问题嘛，能花多少钱？"

## 六、配置：7 步走

终于到了激动人心的时刻——把腾讯云接进你的 AI 编程工具！

### 支持的工具有这些

- Claude Code
- Cursor
- Cline
- CodeBuddy
- OpenClaw
- 各种 VS Code 插件

**你用哪个都行，配置方法都一样。**

### 配置步骤

#### 步骤 1：打开工具，找到「添加模型」

打开你的 AI 编程工具，在设置里找到「添加模型」或「添加自定义模型」。

![15-添加自定义模型配置.png](https://raw.atomgit.com/user-images/assets/5027920/8da83e1d-4acd-4341-84e8-6f56568448ce/15-添加自定义模型配置.png)

#### 步骤 2：选择腾讯云

在服务商列表里找到「腾讯云」。

如果没看到，直接选「自定义模型」或「使用其他模型」——腾讯云支持 OpenAI 兼容协议。

#### 步骤 3：复制 Model ID

回到腾讯云控制台：
1. 点击「快速接入指引」
2. 选「第二步」
3. 找到「Model ID」，点复制

> ⚠️ **重点**：必须点复制按钮，不能手动输入！手动输入容易出错，一旦输错，配置 100% 失败。

![16-快速接入指引步骤.png](https://raw.atomgit.com/user-images/assets/5027920/e0ad6eda-8e2e-467f-9125-18b2b48a52da/16-快速接入指引步骤.png)

#### 步骤 4：粘贴 Model ID

把刚才复制的 ID 粘贴到工具的「模型 ID」输入框。

![17-输入模型ID与API密钥.png](https://raw.atomgit.com/user-images/assets/5027920/2e0d0206-c53a-4f2e-8f47-6e14948b8564/17-输入模型ID与API密钥.png)

#### 步骤 5：粘贴 API Key

把之前保存的 API Key 复制过来，粘贴到「API Key」或「密钥」输入框。

#### 步骤 6：粘贴 URL

在腾讯云控制台「快速接入指引」第一步中，找到「OpenAI 兼容协议」的完整 URL，复制，粘贴到工具的「Base URL」或「请求地址」输入框。

> ⚠️ **重点**：URL 前后不能有空格，不能多输任何字符。

![19-获取URL配置信息.png](https://raw.atomgit.com/user-images/assets/5027920/8f8b33f9-f77d-4d3d-989a-63239213f512/19-获取URL配置信息.png)

#### 步骤 7：保存，完事！

配置成功后会显示：
- 状态：**打开** ✅
- 模型名称：如「混元 3」或「Hunyuan-3-Preview」

### 常见问题急救

| 问题 | 解决方法 |
|------|----------|
| 没有「自定义地址」选项 | 更新软件到最新版本 |
| 选项显示不完整 | 滚动鼠标，选项在下面藏着 |
| 配置失败，提示错误 | 重启软件，再试一次 |
| 显示请求失败 | 检查 URL 是否有多余空格 |
