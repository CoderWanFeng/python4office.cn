## 一、与CODING Plan对比

### Q1：TOKEN Plan和CODING Plan有什么区别？

**核心区别**：
- CODING Plan = 固定月费套餐
- TOKEN Plan = 按量计费服务

| 对比项 | CODING Plan | TOKEN Plan |
|:---|:---|:---|
| 付费方式 | 每月固定费用 | 用多少付多少 |
| 灵活性 | 受套餐限制 | 完全自由 |
| 最低消费 | 有（月费） | 无 |
| 适合人群 | 用量稳定 | 用量波动或追求灵活 |

### Q2：TOKEN Plan是CODING Plan的替代品吗？

**是的，而且是进化版**。

TOKEN Plan解决了CODING Plan的最大痛点——**套餐束缚**。

### Q3：用了TOKEN Plan还需要CODING Plan吗？

一般不需要。TOKEN Plan功能更灵活，覆盖CODING Plan的所有使用场景。

### Q4：哪个更便宜？

**看用量**：
- 用量波动大 → TOKEN Plan更省
- 每天稳定高用量 → 可能CODING Plan更划算
- 用量少 → TOKEN Plan完胜

## 二、费用相关

### Q5：TOKEN Plan怎么计费？

按Token计费：
- 输入Token（你的Prompt）
- 输出Token（AI的回复）

每千Token有公开透明的定价。

### Q6：有最低消费吗？

**没有**。用多少付多少，不用不花钱。

### Q7：费用怎么充值？

腾讯云账户充值，支持：
- 支付宝
- 微信
- 银行卡
- 对公转账（企业用户）

### Q8：可以设置预算上限吗？

可以。在控制台设置用量或费用上限，防止意外超支。

### Q9：费用比CODING Plan贵吗？

**不一定**。举例：

- 场景A：每天编程2小时 → TOKEN Plan可能只要几十元/月
- 场景B：每天编程8小时 → 可能和CODING Plan Pro差不多
- 场景C：偶尔用用 → TOKEN Plan完胜

### Q10：有免费额度吗？

## 三、使用相关

### Q11：支持哪些编程工具？

支持所有主流AI编程工具：
- Cursor
- Claude Code
- Cline（VS Code插件）
- Continue
- 其他OpenAI兼容接口的工具

### Q12：怎么配置Cursor？

1. 打开Cursor设置
2. 找到OpenAI API设置
3. 填入腾讯云TOKEN Plan的API Key和Base URL
4. 选择模型，开始使用

### Q13：支持哪些模型？

- 腾讯云自研代码模型
- 主流第三方模型
- 具体模型列表以官网为准

### Q14：可以切换模型吗？

可以。TOKEN Plan的灵活性之一就是**自由选择模型**。

### Q15：网络稳定吗？

腾讯云国内基础设施，网络稳定性和速度都有保障。

## 四、迁移相关

### Q16：从CODING Plan迁移麻烦吗？

不麻烦：
1. 注册腾讯云账户
2. 开通TOKEN Plan
3. 获取新的API Key
4. 在工具中更新配置

整个过程10分钟搞定。

### Q17：可以同时用两个吗？

可以。建议：
- 初期：保留CODING Plan作为备用
- 稳定后：完全切换到TOKEN Plan

### Q18：迁移后数据会丢失吗？

不会。AI编程工具的历史记录保存在本地，切换API不影响。
