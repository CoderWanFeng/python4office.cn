## 二、稿费怎么算？

| 项目 | 标准 |
|---|---|
| **基础稿费** | **0.3 元 / 阅读**（按文章发布后 72 小时的实际阅读数结算） |
| **单篇上限** | **100 元**（超过 100 元的部分不再叠加） |
| **爆款奖励** | 阅读数 **≥ 5000**，**额外 +100 元** |
| **结算时间** | 发布后 **72 小时内**打到你提供的收款账户（微信/支付宝） |
| **最低阅读门槛** | **≥ 30 个阅读**（低于 30 不结算，但文章保留） |
| **禁止事项** | **禁止刷量**（机器刷、互刷群刷、买量等），发现不结算 |

**举例**：

- 你写了一篇，阅读数 200 → 稿费 = 200 × 0.3 = **60 元**
- 阅读数 1000 → 稿费 = **100 元**（封顶）
- 阅读数 6000 → 稿费 = 100 + 100（爆款奖励）= **200 元**

## 三、稿子写什么？

**唯一的主题**：在 QClaw 里体验"个人网站 Skill"之后，**你的真实感受**。

具体怎么写，**角度不限**，可以参考下面这些：

### 角度 1：从 0 到 1 的建站小白
- 你之前完全不会建站，**5 分钟搞定一个网站**的体验
- 你第一次看到自己网站跑起来那一刻的心情
- 你踩了什么坑、QClaw 怎么帮你避开的

### 角度 2：被代码劝退过的非技术人
- 你以前尝试过 Hexo / WordPress / 各种建站工具，**全都放弃了**
- QClaw 跟你之前用过的工具比，**最大的区别是什么**
- 你最希望 QClaw 下一步加上什么功能

### 角度 3：折腾多年的技术玩家
- 你用过 GitHub Pages、Vercel、Nginx 各种部署方案
- QClaw 这种"对话建站"模式，**对你来说有什么价值**
- 你对 Skill 生态、ClawHub 的看法

### 角度 4：内容创作者 / 自媒体
- 你有自己的公众号 / 小红书 / 抖音，**为什么还需要一个个人网站**
- 用 QClaw 建站后，**你的内容分发链路发生了什么变化**
- QClaw 的"AI 推荐"机制，**有没有真的给你带来读者**

### 角度 5：直接讲故事
- 你那天晚上睡不着，**为什么突然想建个网站**
- 你用 QClaw 帮你爸妈 / 同事 / 朋友做了什么
- 你印象最深的一次对话（跟 QClaw 的）

> **只要是真人、真事、真感受，都欢迎。**
