## 1. What is iFLYTEK Spark Coding Plan?

iFLYTEK Spark is the AI large model brand under iFlytek, focusing on the integration capability of "Voice + Large Model."

### Core Features

| Feature | Description |
|---------|-------------|
| **Voice Interaction** | iFlytek's signature capability, speaking is more convenient than typing |
| **Spark Large Model** | Self-developed by iFlytek, top tier of domestic large models |
| **Voice Synthesis** | Special features like code broadcasting, voice broadcasting results |
| **Voice to Code** | Voice descriptions automatically generate code |

### Suitable Scenarios

- Programming scenarios that require voice interaction
- Chinese code generation and optimization
- Scenarios that require voice broadcasting of results
- Developers who prefer "listening" to code instead of "looking" at code

## 2. Why Choose iFLYTEK Spark Coding Plan?

### 1. Voice Capability Integration

This is iFlytek's unique advantage. Other vendors can only communicate with AI by typing, but iFlytek can "speak" to program.

Imagine:
- Using voice to have AI write code while driving
- Asking technical questions by voice while washing up
- Having AI read the code to you after it's written

### 2. Excellent Chinese Understanding

iFlytek has been deeply engaged in Chinese NLP for many years, with better understanding of Chinese context than many vendors.

When writing code with Chinese comments or describing Chinese requirements, the effect is even better.

### 3. Domestic Vendor Advantages

- Fast domestic access, fast speed
- No need for VPN
- Fast customer service response
- Compliant and secure
