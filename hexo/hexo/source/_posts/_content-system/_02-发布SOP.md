# 发布SOP

> 固定发布节奏，让创作更从容。
> 
> 核心原则：素材积累有方向，写作不焦虑。

---

## 周计划

| 星期 | 内容类型 | 发布时间 | 状态 |
|------|----------|----------|------|
| 周一 | 口播稿（AI热点） | 18:00 | 待写作 |
| 周二 | 技术教程 | 18:00 | 待写作 |
| 周三 | 公众号（深度） | 18:00 | 待写作 |
| 周四 | 工具测评 | 18:00 | 待写作 |
| 周五 | 休息/储备 | - | - |
| 周六 | 自由写作（日记） | 20:00 | 待写作 |
| 周日 | 复盘 | 22:00 | 待写作 |

---

## 内容状态流转

```
选题库 → 写作中 → 待发布 → 已发布 → 复盘
```

### 状态说明

| 状态 | 含义 |
|------|------|
| 选题库 | 待挑选的素材 |
| 写作中 | 正在写 |
| 待发布 | 已写完，等待发布 |
| 已发布 | 已发布到平台 |
| 复盘 | 已记录数据 |

---

## 写作时间建议

| 内容类型 | 写作时长 | 建议时段 |
|----------|----------|----------|
| 口播稿 | 60-90分钟 | 周六储备，或周一早上 |
| 技术教程 | 2-3小时 | 周末集中写 |
| 公众号 | 2小时 | 周二写，周三改 |
| 工具测评 | 1-2小时 | 边用边写 |
| 日记 | 30分钟 | 碎片时间 |

---

## 质量检查清单

发布前检查：

- [ ] 标题是否有吸引力
- [ ] 第一句是否能抓住读者
- [ ] 是否有明确的观点/结论
- [ ] 没有错别字
- [ ] 格式统一
- [ ] 配图合适
