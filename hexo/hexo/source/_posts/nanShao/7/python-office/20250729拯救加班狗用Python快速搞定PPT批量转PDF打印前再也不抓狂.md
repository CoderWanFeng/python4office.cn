## 01

虽然，我们这个功能的需求很好理解，就是批量转换PPT文件到PDF文件，但是关键是我们如何用Python代码来实现。

好吧，这里就不给大家卖关子了，我们直接看代码吧。

代码演示：
```
import office

# 填入你的ppt目录
office.ppt.ppt2pdf(path=r'./test_files/50-29-ppt2pdf',output_path=r'./test_files/50-29-ppt2pdf/output')
```
参数解释：
```
path：需要转换为PDF文件的PPT文件存放的路径位置
output_path：转换之后的文件存放的位置
```
