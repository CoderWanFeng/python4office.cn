## -01-

说实话，如果你现在还在做小绿书，而且如果你是做二创的话，那图片的水印问题一定是你要解决的问题之一。

其实，不只是做小绿书需要用到去除水印的功能，你做短视频，它的封面图也是需要精心设计，并且最好是没有水印的，不然显得多low对吧。

OK，那我们现在就来看一下一行代码实现图片去除水印的代码实现。

代码实现如下：
```
import office

office.image.del_watermark(input_image=r'./test_files/50-17-del_watermark/程序员晚枫img.png',
                           output_image=r'./test_files/50-17-del_watermark/del_water_mark.jpg')
```
参数解释：
```
input_image：需要去除水印的图片
output_image=去除水印之后的图片存在哪
```
