## 01

虽然，我们这个功能的需求很好理解，就是给图片添加水印，但是关键是我们如何用Python代码来实现。

好吧，这里就不给大家卖关子了，我们直接看代码吧。

代码演示：
```
import office

# office.image.add_watermark(file='./要添加水印的图片.png',mark='python办公自动化',output_path=r'已添加水印',opacity=0.2)

office.image.add_watermark(file='./要添加水印的图片.png',
                           mark='python办公自动化',
                           output_path=r'已添加水印',
                           color="#ff0000", size=80, opacity=0.5, space=600, angle=30)
```                           
参数解释：
```
file: 图片位置
mark: 水印内容
output_path: 输出位置
color: 水印颜色
size: 水印大小
opacity: 不透明度，0.01~1
space: 水印间距
angle: 水印角度
```
