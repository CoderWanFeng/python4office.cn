## 01


虽然，我们这个功能的需求很好理解，就是实现自动计算excel表格数据并对其进行填充的功能，但是关键是我们如何用Python代码来实现。

好吧，这里就不给大家卖关子了，我们直接看代码吧。

代码演示：

```python
import pandas as pd


# def add_8(x):
#    return x + 8

# index_col表示设置索引为编号所在的列
example = pd.read_excel(r"C:\Users\zzqyy\Desktop\Pandas玩转Excel\06函数填充，计算列\example.xlsx",index_col="编号")


# 利用series中的apply方法实现目录价格增加8元的操作
# lambda表示Python中的匿名函数
example["目录价格"] = example["目录价格"].apply(lambda x:x + 8)

print(example) 
```
