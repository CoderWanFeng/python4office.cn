## 01


虽然，我们这个功能的需求很好理解，就是实现绘制excel图表并对其进行操作的功能，但是关键是我们如何用Python代码来实现。

好吧，这里就不给大家卖关子了，我们直接看代码吧。

代码演示：

```python

# 导入pandas库，用于数据分析和操作
import pandas as pd
# 导入matplotlib.pyplot库，用于数据可视化（绘制图表）
import matplotlib.pyplot as plt

# 使用pandas的read_excel函数读取一个Excel文件
# 文件路径是C:\Users\zzqyy\Desktop\Pandas玩转Excel\09柱状图\example.xlsx
# 这里的路径是本地路径，你需要确保文件存在且路径正确
example = pd.read_excel(r"C:\Users\zzqyy\Desktop\Pandas玩转Excel\09柱状图\example.xlsx")

# 对读取的数据按照"Number"列进行排序
# inplace=True表示在原数据上直接修改，不创建新的数据框
# ascending=False表示按照降序排序（从大到小）
example.sort_values(by="Number", inplace=True, ascending=False)

# 打印排序后的数据框内容，方便查看数据
print(example)

# 下面是绘制柱状图的代码
# 使用matplotlib的bar函数绘制柱状图
# x轴为"Field"列的值，y轴为"Number"列的值
# 柱子的颜色设置为紫色


plt.bar(example.Field, example.Number, color="purple")

# 设置x轴的刻度标签为"Field"列的值，并将标签旋转90度，避免标签重叠
plt.xticks(example.Field, rotation=90)

# 设置x轴的标签为"Field"
plt.xlabel("Field")

# 设置y轴的标签为"Number"
plt.ylabel("Number")

# 设置图表的标题为"Students by Filed"，并设置字体大小为15
plt.title("Students by Filed", fontsize=15)

# 调整布局，确保图表内容不被遮挡
plt.tight_layout()

# 显示绘制的图表
plt.show()

```
