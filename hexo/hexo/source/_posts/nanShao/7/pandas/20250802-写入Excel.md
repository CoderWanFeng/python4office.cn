## 01

虽然，我们这个功能的需求很好理解，就是实现自动填充excel表格数据并对其进行操作的功能，但是关键是我们如何用Python代码来实现。

好吧，这里就不给大家卖关子了，我们直接看代码吧。

代码演示：

```python
import pandas as pd

# skiprows表示跳过Excel表格中的前三行，usecols表示只使用CDEF这四列数据，dtype表示限定CDEF四列的数据格式为字符串类型

books = pd.read_excel(r"C:\Users\zzqyy\Desktop\Pandas玩转Excel\04自动填充\程序员晚枫.xlsx",
                      skiprows=3,usecols="C:F",index_col=None,dtype={"ID":str,"InStore":str,"Date":str})

for i in books.index:
    books.at[i,"ID"]= i + 1
    books.at[i,"InStore"] = "是" if i%2==0 else "否"

books.set_index("ID",inplace=True)
print(books)
books.to_excel(r"C:\Users\zzqyy\Desktop\Pandas玩转Excel\04自动填充\程序员晚枫填充结果.xlsx")
print("完成！")
```
