## 01


虽然，我们这个功能的需求很好理解，就是实现自动计算excel表格数据并对其进行填充的功能，但是关键是我们如何用Python代码来实现。

好吧，这里就不给大家卖关子了，我们直接看代码吧。

代码演示：

```python
# 导入 pandas 模块，并给它一个简短的别名 pd，方便后续调用
import pandas as pd

# 使用 pandas 的 read_excel 函数读取 Excel 文件
# r"C:\Users\zzqyy\Desktop\Pandas玩转Excel\07排序，多重排序\example.xlsx" 是 Excel 文件的路径
# index_col="第几次" 表示将 "第几次" 这一列作为数据框的索引列


example = pd.read_excel(r"C:\Users\zzqyy\Desktop\Pandas玩转Excel\07排序，多重排序\example.xlsx", index_col="第几次")

# 对数据框进行排序操作
# by=["值不值","价格"] 表示按照 "值不值" 和 "价格" 这两列的值进行排序
# inplace=True 表示直接在原数据框上修改，而不是创建一个新的数据框
# ascending=[True,False] 表示 "值不值" 列按升序排列，"价格" 列按降序排列


example.sort_values(by=["值不值", "价格"], inplace=True, ascending=[True, False])

# 打印排序后的数据框，查看结果
print(example)

```
