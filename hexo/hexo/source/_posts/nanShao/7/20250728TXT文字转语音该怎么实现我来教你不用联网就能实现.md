## -02-


好的，现在我们来看一下TXT转语音的代码实现：



代码实现：
```
import office

res = office.video.txt2mp3(content='程序员晚枫',
                           file=r'test_files/50-12-txt2mp3/程序员晚枫.txt',
                           mp3=r'test_files/50-12-txt2mp3/程序员晚枫.mp3',
                           speak=True)
print(res)
```
参数解释：
```
content:需要转换的内容
file: 指定读取的文件，优先级最高
mp3: 需要保存的mp3位置和名称。填None不保存。
speak:是否阅读
```
