
## 🎬 Script Standards

Each script contains the following sections:

```
Title: (One sentence to attract clicks)
Cover copy: (Text matching the cover)
Duration estimate: (1-3 minutes / 3-5 minutes)
Script body: (Word-for-word, can record directly)
Publishing platform: (Douyin/Video Account/Xiaohongshu/Bilibili/Universal)
Filming suggestions: (Camera、画面/BGM tips)
Related article: (Link to corresponding blog article)
Update log:
- 2026-04-12: Script #001 completed
```

## 📌 Content Categories

- **Series 001**: Real blogger reflections/behind-the-scenes (trust-building)
- **Series 002**: AI tool hands-on reviews (trust-building)
- **Series 003**: Mass pain points + AI solutions (traffic-building)
- **Series 004**: Product/course promotion (conversion)

## ✍️ Writing Principles

1. Speak human language, don't be stiff
2. Every sentence can be read directly, no incomprehensible written language
3. Have real details, no empty big道理
4. End with "something to take away" or "action guide"
