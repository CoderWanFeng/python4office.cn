## 🎬 文案规范


## 📌 内容分类

- **001系列**：博主真实反思/幕后内容（信任型）
- **002系列**：AI工具实测/评测（信任型）
- **003系列**：大众痛点+AI解法（流量型）
- **004系列**：产品/课程推广（转化型）

## ✍️ 写作原则

1. 说人话，不要端着
2. 每句话都能直接读出来，不要写读不懂的书面语
3. 有真实细节，不讲空洞大道理
4. 结尾有"带走的东西"或"行动引导"
