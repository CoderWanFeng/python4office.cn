## 🎬 文案规范

每篇文案包含以下部分：

```
标题：（吸引点击的一句话）
封面文案：（配合封面的文字）
时长预估：（1-3分钟 / 3-5分钟）
脚本正文：（逐字稿，可直接录音）
发布平台：（抖音/视频号/小红书/B站/通用）
拍摄建议：（镜头/画面/背景音乐提示）
关联文章：（对应的博客图文版链接）
更新记录：
- 2026-04-12：001 号文案完成
```

## 📌 内容分类

- **001系列**：博主真实反思/幕后内容（信任型）
- **002系列**：AI工具实测/评测（信任型）
- **003系列**：大众痛点+AI解法（流量型）
- **004系列**：产品/课程推广（转化型）

## ✍️ 写作原则

1. 说人话，不要端着
2. 每句话都能直接读出来，不要写读不懂的书面语
3. 有真实细节，不讲空洞大道理
4. 结尾有"带走的东西"或"行动引导"
