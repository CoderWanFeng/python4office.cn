# AI Skills Column

> Focus on AI office scenario Skill creation, usage, and promotion

## Column Positioning

This is programmer Wan Feng's **AI Skill Column**, documenting the entire process of transitioning from Python office automation to the AI Skill ecosystem.

**Core Narrative**: Install one Skill, solve one troublesome task

## Directory Structure

```
ai-skills/
├── skill-tutorials/      # Skill usage tutorials (teaching users how to use)
│   └── How to install/configure/use various office Skills
├── platform-guides/      # Platform guides (platform usage guides)
│   └── Coze / OpenClaw / Feishu CLI platform reviews and tutorials
├── skill-dev/            # Skill development tutorials (teaching users how to make)
│   └── Develop Skills from scratch, advanced tips, open-source Skill breakdowns
├── office-automation/    # Office automation practice (scenario cases)
│   └── Skill solutions for Excel/PDF/PPT/Word and other specific scenarios
└── skill-reviews/        # Skill reviews and recommendations
    └── Weekly/monthly best Skill recommendations, comparison reviews
```
