大家好，我是正在实战各种 AI 项目的程序员晚枫。

每天下班前最痛苦的事是什么？**写日报**——回忆今天做了什么、整理数据、写总结，至少 30 分钟。

今天教你配置 **日报自动生成 Skill**，让 AI 自动帮你汇总数据、生成日报。

---

## 一、Skill 简介

### 功能概述
日报自动生成 Skill 可以：
- ✅ 自动汇总任务管理工具数据（飞书任务/钉钉/Notion 等）
- ✅ 自动读取日历事件
- ✅ 自动统计代码提交/Git 记录
- ✅ 自动读取邮件/消息摘要
- ✅ 生成结构化的日报文档
- ✅ 定时自动发送

### 适用人群
| 人群 | 收益 |
|---|---|
| 程序员 | 自动汇总代码提交、Bug 修复 |
| 产品经理 | 自动汇总需求进度、会议记录 |
| 销售人员 | 自动汇总客户跟进、订单数据 |
| 运营人员 | 自动汇总数据报表、活动进展 |
| 项目经理 | 自动汇总项目进度、风险预警 |

## 二、安装 Skill

### Coze 扣子安装
1. 打开 [Coze 官网](https://www.coze.cn)
2. 进入 "技能商店"
3. 搜索 "日报生成" 或 "工作汇报"
4. 点击 "安装"

### OpenClaw 安装
```bash
openclaw skills install daily-report
openclaw skills install calendar-tools
openclaw skills install git-tools
```

### 飞书 CLI
飞书 CLI 原生支持日历、任务、文档操作，可直接用于日报生成。

## 四、使用教程

### 基础用法：手动生成日报

```
你：生成今天的日报

AI：正在生成日报...
🔧 Using skill: daily-report
   action: generate
   date: 2024-04-06
   
📤 Result:
   日报生成完成！
   
   === 今日工作日报 ===
   日期: 2024年4月6日
   
   【今日完成】
   1. ✅ 完成用户登录模块开发
   2. ✅ 修复订单支付 Bug
   3. ✅ 参加产品需求评审会
   
   【会议记录】
   - 10:00 产品需求评审会
     - 确定了 Q2 产品规划
     - 讨论了用户增长策略
   - 14:00 技术分享会
     - 主题: AI 在自动化测试中的应用
   
   【代码提交】
   - 提交 5 次
   - 新增代码: 320 行
   - 修改代码: 150 行
   - 主要改动: 登录模块、支付接口
   
   【明日计划】
   1. 📝 开始用户中心页面开发
   2. 📝 跟进测试反馈
   3. 📝 准备周会汇报材料
   
   【遇到的问题】
   （待填写）
   
   是否需要发送给领导？
```

模板:
【今日提交】
{{git_commits}}

【完成任务】
{{completed_tasks}}

【参加会议】

## 七、常见问题

### Q1：数据源连接失败怎么办？
- 检查 API 权限是否开启
- 确认账号有访问权限
- 重新授权连接

### Q2：生成的日报不准确？
```
你：调整日报生成规则，只包含标记为"重要"的任务

AI：已调整规则，将只汇总高优先级任务。
```

### Q3：如何添加自定义字段？
```
你：在日报中添加"心情指数"字段，让我每天手动填写

AI：已添加自定义字段：
   - 字段名: 心情指数
   - 类型: 1-5 星评分
   - 填写方式: 手动
```

### Q4：如何导出历史日报？
```
你：导出 3 月份的所有日报为 Word 文档

AI：导出完成！
   共 22 篇日报
   已合并为: 3月工作月报.docx
```
