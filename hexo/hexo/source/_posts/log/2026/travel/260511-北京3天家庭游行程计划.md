# 北京3天家庭游行程计划（和爷爷奶奶出行）
---
## 📌 基础信息
- **出行人员**：祖孙三人
- **入住酒店**：鸟巢附近酒店
- **出行时间**：2026年5月11日-5月13日
- **核心原则**：节奏舒缓不赶路、全程打车不走回头路、所有景点仅走平地不爬山、预留充足休息时间
---
## 📅 每日行程
### 🔹 5月11日（周一） 抵达日
| 时间 | 行程安排 | 注意事项 |
|------|----------|----------|
| 上午 | 抵达北京 → 前往酒店办理入住 | 全程打车直达酒店 |
| 12:00-14:00 | 酒店午休 | 保证充足休息 |
| 14:00-17:00 | 颐和园 → 昆明湖划船 | 园内走平地游览路线，划船时穿好救生衣<br/>⚠️需提前预约：颐和园官网 https://yhy.yidyou.cn/#/ 或「颐和园」公众号预约（买大门票即可），昆明湖划船可现场购票 |
| 18:00 | 晚餐：北京烤鸭 | 建议提前大众点评取号 |
| 19:00 | 返回酒店休息 | 当天行程结束，早点休息 |
---
### 🔹 5月12日（周二） 市区休闲日
| 时间 | 行程安排 | 注意事项 |
|------|----------|----------|
| 8:00-8:30 | 酒店早餐 | 慢悠悠用餐不赶时间 |
| 8:30-9:00 | 地铁前往天安门广场 | 酒店步行至8号线奥体中心站/奥林匹克公园站 → 前门站（约25分钟），A口出站即到天安门 |
| 9:00-11:00 | 第一站：天安门广场游览 | 走平地游览路线，拍照打卡<br/>⚠️需提前1-7天预约：天安门广场预约参观小程序，或 https://yuyue2026.tamgw.beijing.gov.cn/web/index.html#/index，所有人员携带身份证原件 |
| 11:00-11:30 | 第二站：某办公机构门口打卡 | 从天安门顺路前往，无需绕路 |
| 11:30-12:30 | 打车前往清河站 | 预留足够时间进站候车 |
| 13:24 **🚄** | 清河站出发 → 八达岭长城站 | 高铁约23分钟到达，建议提前在12306购票 |
| 13:47-16:00 | 第三站：八达岭长城游览 | 仅乘坐南线地面缆车，单程约6分钟，直达南四楼游览<br/>⚠️需提前预约：八达岭长城官网 http://www.badaling.cn/ 或「长城内外旅游」小程序，建议预约南线缆车票减少排队 |
| 16:00-16:30 | 返回八达岭长城站 | 预留充足时间进站候车 |
| 16:30 **🚄** | 八达岭长城站出发 → 清河站 | 高铁约23分钟，建议提前购票或现场买票（余票充足） |
| 17:00-19:00 | 自由活动+晚餐 | 返回市区自由安排 |
| 19:00-20:30 | 鸟巢、水立方看夜景 | 步行即可到达，看完夜景直接回酒店休息<br/>✅无需预约，仅外部观赏夜景不需要进入场馆 |
| 20:30 | 返回酒店休息 | 当天行程结束 |
---
### 🔹 5月13日（周三） 返程日 ⚠️ 重点时间提醒
| 时间 | 行程安排 | 注意事项 |
|------|----------|----------|
| 10:30 | 退房出发 | 行李打包好直接带车上 |
| 10:30-11:00 | 前往清华大学 | 打车前往 |
| 11:00-12:00 | 第一站：清华西门打卡 → 北大东门打卡 | 顺路安排，无需进入校园，门口拍照即可<br/>✅无需预约，仅校门外部打卡不需要进入校园 |
| 12:00-13:30 | 第二站：清华食堂午餐 | 需提前确认是否对外开放的食堂窗口 |
| 13:30-14:00 | 前往恭王府 | 打车约30分钟 |
| 14:00-16:30 | 第三站：恭王府游览 | 全程平地游览，无需攀爬，参观和珅府邸清代王府景观<br/>⚠️需提前预约：恭王府官网 http://web.pgm.org.cn/login 或关注「恭王府博物馆」公众号预约，刷身份证入园 |
| 16:30-17:30 | 前往北京南站 | 全程打车直达，约1小时，预留充足候车时间 |
| 20:00 | 乘坐高铁返程 | |

## 💡 温馨提示
1. 所有行程均已按顺路原则规划，全程打车无需换乘公共交通
2. 每个景点都已确认有平地游览路线，不会安排任何需要爬山、走台阶的区域
3. 每日均预留了充足的午休和休息时间，不会出现赶路情况
4. 返程日18点出发去车站是硬性时间要求，请提前做好准备
5. 热门北京烤鸭门店建议提前在大众点评预约取号，避免用餐高峰长时间等待
