#  一、安装python
双击这个像两条蟒蛇的图标(正好今年是蛇年哈哈)

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747136310788.jpg)

双击后会跳出这样一个页面，然后点击 1 处打上勾，再点击2 自定义安装

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747123012691.jpg)

然后这个页面全部都要勾选，再点击 next

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747123236599.jpg)

接下来这个页面只勾选前面5项，后两项不选。
再点击 Browse 选择安装位置。不建议安装在C盘，除非你的电脑只有C盘

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747123340128.jpg)

安装位置选择好后点击 Install

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747123527599.jpg)

就会开始安装python

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747123656524.jpg)

安装完成后会出现这三个单词，最后点击 Close

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747124329047.jpg)


#  二、安装pycharm
双击这个图标

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747123783423.jpg)

直接点击 ”下一步“

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747123850337.jpg)

然后点击 ”浏览“ 选择一个安装位置，同样不建议安装在C盘

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747123925522.jpg)

选择完成后点击 ”下一步“

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747124034570.jpg)

这个画面要注意了，全部都要勾选上。再点击 ”下一步“

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747124106220.jpg)

然后这个画面不用改，直接点击 ”安装“

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747124191643.jpg)

就会进入安装界面

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747124260035.jpg)

安装完成后选择”否，我会在之后重新启动“，再点击”完成“

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747124493666.jpg)

#  三、配置pycharm
在桌面找到这个图标，双击打开它

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747124808348.jpg)

点击这个文件夹图标

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747124932102.jpg)

在这里填入我们保存代码的目录

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747125115732.jpg)

比如我的代码在这里面，把这个路径复制下来，粘贴到上图箭头的地方，再点 ok 

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747125696550.jpg)

然后点击右下角，因为我安装过，所以有记录，你们应该是无

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747126387645.jpg)

点击 ”添加新的解释器“ 或者”Add ...“

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747129692590.jpg)

再点击 这个

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747129811000.jpg)

在弹出的这个框，点击中间这个

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747136220130.jpg)

点击右边的三个点

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747130046718.jpg)

找到刚刚安装 python 的位置，然后选择 python.exe，再点击确定

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747130155641.jpg)

最后点击确定就完成配置了

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747130241825.jpg)
