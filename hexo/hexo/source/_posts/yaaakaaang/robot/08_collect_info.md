#  一、先打开微信群

首先点开要收集的微信群

#  二、直接运行代码

找到左边【收集群成员信息】 这个文件，鼠标右键单击它

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1748241956196.jpg)

然后找到有三角形这行，鼠标左键单击它

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1748242035924.jpg)

# 三、运行效果

首先会点开微信群信息

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747972312676.png)

最后会生成一个表格

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747972613637.jpg)

表格就存着群成员信息

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747972511747.jpg)
