#  一、自定义发送给谁

双击鼠标左键打开 群发对象 excel 表

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747904280723.jpg)

左边填写要发送的好友昵称，右边 True 表示要发送，False 表示不发送

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747904851335.jpg)

#  二、自定义发送内容

双击鼠标左键打开  群发内容.txt ，然后编辑发送内容

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747906453544.jpg)

#  三、运行代码

  找到左边  【群发消息】文件，鼠标右键单击它
  
![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1748241172552.jpg)
  
  然后找到有三角形这行，鼠标左键单击它
  
![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1748241290098.jpg)
  
  会跳出这么一个框来，接着点击加载
  
  ![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747906564366.png)
  
  先选择刚刚编辑的 群发对象 excel 表
  
  ![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747906637240.jpg)
  
  再选择  群发内容.txt 
  
  ![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747906728031.jpg)
  
  就会加载出要发送的好友。
  
  因为 兆锋  是设置的 False ，所以这里没加载出来。
  
  ![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747906788965.jpg)
  
  最后点击 群发  就可以了。
  
  ![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747906927437.jpg)
  
  # 四、运行效果
  
  会依次搜索好友，然后发送自定义的内容。
  
  ![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747906976046.png)
