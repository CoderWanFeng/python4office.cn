#  一、填写收信人、发送的文件名

找到左边【发文件】这个文件，鼠标左键双击它

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1748241327445.jpg)

然后在这里填收信人的昵称

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747896424491.jpg)

如果要发送的文件在这里

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747896599289.png)

那么就接着在这里填要发送的文件名

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747896520542.jpg)

如果文件在电脑其他地方，就填完整的存储路径，比如 

file=r'E:\poemail\tests\test_files\要发送的文件.docx'

#  二、运行代码

找到左边【发文件】这个文件，鼠标右键单击它

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1748241405562.jpg)

然后找到有三角形这行，鼠标左键单击它

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1748241456649.jpg)

# 三、运行效果

运行日志显示运行成功

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747896929191.jpg)

然后微信上也成功发送了文件

![](https://raw.atomgit.com/yaaakaaang/pic/raw/main/1747896971043.png)
