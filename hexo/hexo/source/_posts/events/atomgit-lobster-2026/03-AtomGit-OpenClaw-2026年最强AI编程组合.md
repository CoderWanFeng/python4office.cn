## 为什么是 AtomGit？

AtomGit（atomgit.com）是国内新兴的开源平台，背后是 GitCode 和白开水 AI 社区。

对于 AI 编程比赛来说，AtomGit 有几个独特的优势：

### 1. 「龙虾」云端开发环境

这是 AtomGit 最特色的功能之一。

传统的编程比赛，你需要：
- 自己装 Python 环境
- 自己配置开发工具
- 自己搞定各种依赖

但 AtomGit 的「龙虾」环境，直接在浏览器里给你一个完整的 Linux 开发环境。

> 打开网页 → 写代码 → 运行 → 提交作品

全程不需要在本地安装任何东西。

### 2. 免费算力 + 大模型支持

AtomGit 会为参赛者提供：

- 免费的大模型 API Token
- 免配置的 AI 辅助编程工具
- 完整的项目模板和示例代码

这意味着，**你只需要有一台能上网的电脑，就可以开始 AI 编程了**。

### 3. 代码托管 + 作品展示

AtomGit 原生支持 Git 版本控制，你的参赛作品可以直接托管在平台上，方便评审查看，也有利于赛后展示和二次传播。

## 为什么是 OpenClaw？

OpenClaw 是我一直在用的 AI 编程辅助工具，简单来说，它是一个**智能体（Agent）编程环境**。

### 1. 对话式编程

在 OpenClaw 中，你不需要记住复杂的命令和语法。

你只需要用自然语言描述你想做什么，它就会帮你：

- 写代码
- 调试 Bug
- 解释代码
- 重构代码
- 生成文档

### 2. 多工具链集成

OpenClaw 可以调用：

- 浏览器（自动搜索资料）
- 文件系统（读写文件）
- 终端（执行命令）
- API（调用外部服务）
- 代码解释器（运行 Python/JavaScript 等）

这意味着，你可以在一个界面里，完成从写代码到运行到发布的所有步骤。

### 3. 智能体能力

OpenClaw 不仅仅是回答问题，它能：

- 理解复杂的任务需求
- 拆解成多个步骤
- 自动调用合适的工具
- 逐步完成任务
- 自我纠错和优化

> 这就是所谓的「AI Agent」——能自主行动和决策的 AI 系统。
