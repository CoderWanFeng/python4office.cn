#### 1️⃣ **统一测试文件**
- ✅ 所有测试用例统一使用同一个 PPT 文件: `./ppt/test_ppt/晚枫-1211重庆.pptx`
- ✅ 通过类变量 `cls.test_ppt` 统一管理测试文件路径

#### 2️⃣ **规范化测试结构**
```python

```

#### 3️⃣ **完善的测试用例**

| 测试方法 | 功能 | 输出目录 |
|---------|------|---------|
| `test_ppt2pdf` | PPT转PDF | `./ppt/test_ppt/test_output_pdf` |
| `test_ppt2img_single` | PPT转独立图片 | `./ppt/test_ppt/test_output_img` |
| `test_ppt2img_merge` | PPT转长图 | `./ppt/test_ppt/test_output_img_merge` |
| `test_merge4ppt` | 合并PPT | `./ppt/test_ppt/test_output_merge` |

