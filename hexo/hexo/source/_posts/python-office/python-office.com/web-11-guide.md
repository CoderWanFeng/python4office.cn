## 源码使用

### 运行起来

按顺序，运行以下命令即可。
```
git clone https://github.com/CoderWanFeng/python-office.com.git
cd docs-pages
yarn install 
yarn run dev
```

运行后，可以在``http://localhost:18001``进行查看。

### 技术说明

因为没有逻辑操作，所以该网站目前使用了纯静态页面的形式，技术框架是：``vuepress``，文章内容编写只需要``md``和``html``即可。
