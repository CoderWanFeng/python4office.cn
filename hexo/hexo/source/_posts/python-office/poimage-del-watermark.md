## 一、代码运行，效果演示

举个例子，在阅读公众号文章的时候，经常会看到下图所示的水印，这个是公众号系统自带的。


```python

# pip install poimage
import poimage

# 支持jpg、png等所有图片格式
poimage.del_watermark(
    input_image=r"D:\test\程序员晚枫\加了水印的图片.jpg",
    output_image=r'D:\test\程序员晚枫\保存为去除了水印的图片.jpg')

```
