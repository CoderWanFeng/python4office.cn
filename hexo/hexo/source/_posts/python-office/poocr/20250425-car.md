## 直接上代码
【potx-cloud】库基于腾讯的API实现发票识别并且生成Excel文件的Python库。

```shell
pip install potx-cloud
```

```python
import potx

potx.ocr2excel.VehicleLicenseOCR2Excel(input_path='test.png', id=SecretId, key=SecretKey)
```

## 参数说明

- input_path：行驶证的位置
- id：SecretId，key：SecretKey
