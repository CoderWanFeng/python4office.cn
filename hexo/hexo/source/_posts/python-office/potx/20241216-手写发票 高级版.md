安装也很简单，在有python环境的电脑上，只需要执行下面这一行命令，就可以自动下载和安装了。


## 2. 识别手写发票

为了让大家更好的理解，本文会通过用腾讯云智能OCR，实现1行Python代码识别手写发票，来带大家体验一下这个功能：

直接上代码！

```python
import potx

res = potx.ocr.SmartStructuralPro(
    id=SecretId, 
    key=SecretKey, 
    img_path=r'D:\workplace\\程序员晚枫的手写发票.png'
    )

print(res)
```

运行以上代码，就可以得到识别结果了。
