## Dispute Point Analysis

The most disputed parts in this incident are 2 parts, which I marked with red dots, here's an impression, detailed explanation below:
- External parties question Huawei copied Alibaba's open source code.
- Huawei Pangu large model and Alibaba Qianwen large model have algorithmic similarities.

Based on currently available evidence, my judgment: The first point is purely unprofessional nitpicking, the second point cannot yet be used as conclusive evidence.

### 1. Alibaba Qianwen's Logo Text in Huawei's Code

This was the most hyped point:


![https://atomgit.com/ascend-tribe/pangu-pro-moe-model/blob/main/Open%20Source%20Software%20Notice](https://raw.atomgit.com/user-images/assets/5027920/7aa404d8-37fd-454b-9883-5b9edd722675/image.png)

But anyone who understands open source technology knows: Qianwen is itself open source, using the quite liberal Apache 2.0 protocol.

> Open source means: Alibaba Tongyi large model explicitly states in its code that **anyone can use my code, and can also modify my code, but must keep my name.**
