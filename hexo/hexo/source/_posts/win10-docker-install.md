## docker知识点

1、3要素
- 容器：根据镜像，生成的环境
- 镜像：一个模板
- 仓库：存放镜像的地方

2、编写dockerfile
可以打包一个镜像

docker是分层生成的

3、docker-compose
对于大量的镜像运行，已经运行顺序，进行编排
