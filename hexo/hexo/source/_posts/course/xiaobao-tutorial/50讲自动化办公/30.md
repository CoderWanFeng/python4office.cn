## 2、python-office速查表

| 功能 | 代码 |
|------|------|
| 读取Excel | `office.excel.read(path='file.xlsx')` |
| 写入Excel | `office.excel.write(path='file.xlsx', data=data)` |
| 合并Excel | `office.excel.merge(folder_path='folder')` |
| 读取Word | `office.word.read(path='file.docx')` |
| 写入Word | `office.word.write(path='file.docx', content='text')` |
| 发送邮件 | `office.email.send(email='to@example.com', title='title', content='text')` |
| OCR识别 | `office.ocr.text(path='image.jpg')` |
| PDF读取 | `office.pdf.read(path='file.pdf')` |
| 文件分类 | `office.file.classify(folder='folder', type_rule=True)` |
| 图片水印 | `office.image.add_text(path='img.jpg', text='watermark')` |

## 4、学习心得分享

### 我是怎么学Python的？

我是法学生转行程序员，零基础开始学的。

学习方法很简单：
1. **跟着课程敲代码**——不要只看，要动手敲
2. **遇到问题先搜索**——99%的问题别人都遇到过
3. **多动手做项目**——做一个小项目比看100行代码进步更快

### 普通人学Python的心态

- **不要追求成为程序员**——目的是提高效率，不是转行
- **不需要学得很深**——够用就行
- **坚持比天赋更重要**——每天学一点，1个月就能看到明显进步
