今天这讲，我们来搞定 **Windows 上安装 Python**。

## 1、为什么Windows要单独讲？

因为 Windows 用户是绝大多数，而且 Windows 的安装方式最简单——只需要下载一个安装包，一路点"下一步"就搞定了。

但简单归简单，有几个坑需要注意，看完这讲你就知道了。

## 2、去官网下载Python

打开浏览器，访问：https://www.python.org/downloads/

你会看到最新的 Python 版本（目前是 3.12.x）。

点击 **Download Python 3.12.x** 按钮，开始下载。

> ⚠️ 注意：下载的是 `.exe` 安装包，文件大小约 25MB。

## 3、安装Python（重要！）

找到下载的 `python-3.12.x-amd64.exe` 文件，双击运行。

**这里有几个关键步骤：**

### ✅ 勾选"Add Python to PATH"

这一步至关重要！

安装界面第一个页面，**一定要勾选底部的 "Add Python to PATH"**。

不勾选的话，后面运行 Python 会报错。

### ✅ 选择"Customize installation"

点击"Customize installation"，然后一路点"Next"就行。

### ✅ 等待安装完成

大约需要 2-3 分钟，看到 **"Setup was successful"** 的提示，说明安装成功了。

## 4、验证安装

按 `Win + R`，输入 `cmd`，回车，打开命令提示符。

输入：

```bash
python --version
```

看到 `Python 3.12.x` 这样的版本号，说明安装成功！

## 5、安装python-office

在命令提示符里输入：

```bash
pip install python-office
```

看到 `Successfully installed python-office` 这样的提示，说明安装成功。

> 💡 如果下载太慢，用国内镜像：
> ```bash
> pip install python-office -i https://mirrors.aliyun.com/pypi/simple/
> ```

## 6、第一个程序：验证环境

打开命令提示符，输入 `python`，回车，进入 Python 交互模式。

输入：

```python
import office
print("Hello，程序员晚枫！")
```

看到输出 `Hello，程序员晚枫！`，说明环境完全OK！

按 `exit()` 退出。

## 7、常见问题

**Q：提示"不是内部或外部命令"？**

A：可能是 PATH 没添加成功。重新运行安装程序，选择"Repair"修复一下。

**Q：下载太慢了？**

A：用国内镜像源，或者直接用微信联系我帮你远程解决。

**Q：Mac 用户看这个吗？**

A：Mac 用户请看第2讲，Windows 和 Mac 安装方式不一样。

准备好你的电脑，我们开始！
