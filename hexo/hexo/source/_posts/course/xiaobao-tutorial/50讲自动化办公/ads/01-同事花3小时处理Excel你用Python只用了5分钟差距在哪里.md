### 📊 自动化办公的核心逻辑

让我告诉你，Python处理Excel的效率差距有多夸张：

**场景1：合并多个Excel文件**

手动操作：打开文件A → 全选 → 复制 → 打开汇总表 → 粘贴 → 打开文件B → 全选 → 复制……重复12次，耗时约2小时。

Python代码：
```python
import pandas as pd
import glob

# 一行代码合并所有Excel
dfs = [pd.read_excel(f) for f in glob.glob("部门数据/*.xlsx")]
result = pd.concat(dfs, ignore_index=True)
result.to_excel("汇总.xlsx", index=False)
```

耗时：5秒钟。对，你没看错，5秒钟。

**场景2：批量添加公式**

手动操作：逐个单元格输入公式，拖拽填充，重复几百次，耗时约1小时。

Python代码：
```python
import openpyxl

wb = openpyxl.load_workbook("数据.xlsx")
ws = wb.active
for row in range(2, ws.max_row + 1):
    ws[f"C{row}"] = f"=A{row}*B{row}"  # 自动填充公式
wb.save("数据_计算完成.xlsx")
```

耗时：3秒钟。

**场景3：按条件筛选和分类**

手动操作：筛选、复制、新建工作表、粘贴……耗时约30分钟。

Python代码：
```python
# 按部门分类并保存到不同sheet
for dept, group in df.groupby("部门"):
    group.to_excel(f"部门_{dept}.xlsx", index=False)
```

耗时：2秒钟。
