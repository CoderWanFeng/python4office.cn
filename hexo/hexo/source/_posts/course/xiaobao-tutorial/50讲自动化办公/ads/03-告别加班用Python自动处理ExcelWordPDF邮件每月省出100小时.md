## 🔥 Python能自动化哪些办公操作？

让我给你一个更直观的感受：

### 1. Excel：从手动到自动

以前：每天花1.5小时做日报，手动拉数据、做透视表、填模板
现在：运行一个Python脚本，3分钟出报表

```python
import poexcel

# 从数据库导出数据 → 自动格式化 → 填入模板
poexcel.fake2excel(rows=100, columns=['name', 'age', 'salary'])
```

### 2. Word：从逐个修改到批量生成

以前：年终给200个客户发感谢信，改名字改了3个小时
现在：一个模板+一份名单，5分钟搞定

### 3. PDF：从人工拆分到一键处理

以前：收到一本300页的PDF，要提取其中30页，手动一页一页删
现在：指定页码范围，1秒钟搞定

```python
import popdf

popdf.split4pdf(input_path="大文件.pdf", output_folder="./拆分结果/", pages="1-30")
```

### 4. 邮件：从手动发收到自动管理

以前：每天花1小时归类邮件、下载附件
现在：自动下载、自动归类、自动回复
