## 1、什么是Git？

Git是一个版本控制工具，可以记录代码的修改历史，随时找回之前的版本。

## 2、安装Git

Windows、Mac、Linux都可以安装Git：
- Windows：去 git-scm.com 下载安装包
- Mac：brew install git
- Linux：sudo apt install git

配置用户名和邮箱：
<pre>git config --global user.name "你的名字"
git config --global user.email "你的邮箱"</pre>

## 3、初始化仓库

<pre>cd 我的项目文件夹
git init</pre>

## 4、基本操作

<pre>git status  # 查看状态
git add 文件名  # 添加文件
git commit -m "这次改了什么"  # 提交</pre>

## 5、查看历史

<pre>git log --oneline</pre>

## 6、撤销修改

<pre>git checkout -- 文件名  # 撤销工作区修改
git reset --hard HEAD^  # 回退到上一个版本</pre>

## 7、分支管理

<pre>git branch 新功能  # 创建分支
git checkout 新功能  # 切换分支
git merge 新功能  # 合并分支</pre>

## 8、实战：管理自动化脚本

<pre># 1. 创建项目
mkdir my_automation
cd my_automation
git init

# 2. 添加代码
git add .
git commit -m "初始版本"

# 3. 创建新功能分支
git checkout -b add_pdf

# 4. 开发完成，切换回主分支
git checkout main

# 5. 合并新功能
git merge add_pdf</pre>

## 9、常见问题

Q：提交说明怎么写？
A：简单描述本次修改，如：添加Excel读取功能

Q：代码改坏了怎么办？
A：用 git checkout -- 文件名 可以恢复
