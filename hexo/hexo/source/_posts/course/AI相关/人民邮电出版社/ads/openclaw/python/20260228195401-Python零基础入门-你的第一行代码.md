## 第一步：安装Python

### Windows用户
1. 访问 python.org
2. 下载最新版Python（3.10+）
3. 安装时勾选"Add Python to PATH"
4. 点击Install Now

### Mac用户
```bash
# 使用Homebrew安装
brew install python

# 或者从官网下载安装包
```

### 验证安装
打开命令行，输入：
```bash
python --version
```

看到版本号（如Python 3.10.0），说明安装成功！

## 第三步：写下第一行代码

创建一个文件 `hello.py`，写入：

```python
print("Hello, World!")
print("你好，Python！")
```

运行：
```bash
python hello.py
```

看到输出？恭喜你，你已经是程序员了！🎉

## 理解print函数

`print()`是Python最常用的函数之一，作用是在屏幕显示内容。

```python
# 打印文字
print("Hello")

# 打印数字
print(123)

# 打印多个内容
print("年龄:", 25)

# 自定义分隔符
print("A", "B", "C", sep="-")  # A-B-C

# 自定义结束符（默认是换行）
print("Loading", end="...")
print("Done!")  # Loading...Done!
```

---

## Python的注释

注释是给程序员看的，程序会忽略。

```python
# 这是单行注释

"""
这是多行注释
也叫文档字符串（docstring）
可以用来写函数说明
"""

print("Hello")  # 行末也可以加注释
```

**好习惯**：多写注释，未来的你会感谢现在的自己。

## 常见错误

### 错误1：拼写错误
```python
Print("Hello")  # 错误！Python区分大小写
print("Hello")  # 正确
```

### 错误2：引号不匹配
```python
print("Hello')   # 错误！引号要成对
print("Hello")   # 正确
```

### 错误3：缩进错误
```python
print("第一行")
  print("第二行")  # 错误！不要无故缩进
```
