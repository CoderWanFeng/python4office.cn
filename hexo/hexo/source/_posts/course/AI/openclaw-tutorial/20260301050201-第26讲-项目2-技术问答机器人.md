## 功能实现

### 1. 代码解释
用户发送代码片段，AI自动分析：

```
用户：@DevHelper 解释一下这段代码
```python
def fib(n):
    if n <= 1:
        return n
    return fib(n-1) + fib(n-2)
```

AI：这是一个递归实现的斐波那契数列函数。

**时间复杂度**：O(2^n)，因为存在大量重复计算
**空间复杂度**：O(n)，递归栈深度

**优化建议**：使用记忆化或迭代方式
```python
# 记忆化版本
def fib(n, memo={}):
    if n in memo:
        return memo[n]
    if n <= 1:
        return n
    memo[n] = fib(n-1, memo) + fib(n-2, memo)
    return memo[n]
```
```

### 2. 错误诊断
```
用户：报错了，怎么解决？
AttributeError: 'NoneType' object has no attribute 'append'

AI：这个错误表示你在None值上调用了append方法。

常见原因：
1. 变量未正确初始化
2. 函数返回了None而不是列表
3. 条件判断导致变量未被赋值

检查你的代码中哪个变量可能是None，确保在使用前正确初始化。
```

### 3. 文档查询
```
用户：Python的dataclass怎么用？

AI：[搜索Python官方文档]

`@dataclass`是Python 3.7+的装饰器，用于自动生成特殊方法。

基本用法：
```python
from dataclasses import dataclass

@dataclass
class User:
    name: str
    age: int = 0

user = User("Alice", 30)
print(user)  # User(name='Alice', age=30)
```

自动生成的方法：
- __init__
- __repr__
- __eq__
...
```
