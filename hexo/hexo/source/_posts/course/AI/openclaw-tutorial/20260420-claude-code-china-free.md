## Claude Code是什么？

Claude Code是Anthropic官方推出的终端编程工具，可以：
- 在命令行中直接与Claude对话
- 读写文件、执行命令
- 帮你写代码、debug、重构
- 接入各种大模型

---

## 如何免费使用？

### 方法1：通过OpenClaw（推荐）

OpenClaw支持Claude Code，可以：
- 接入Claude模型（需要API Key）
- 接入便宜的国产大模型（DeepSeek、Kimi等）
- 在微信/Telegram等平台使用Claude Code

### 方法2：通过claude-code包

```bash
npm install -g @anthropic-ai/claude-code
claude-code
```

---

## 接入国产大模型

如果你觉得Claude太贵，可以接入便宜的国产模型：

### 配置示例

```json
{
  "providers": {
    "deepseek": {
      "enabled": true,
      "apiKey": "your-deepseek-key",
      "baseUrl": "https://api.deepseek.com",
      "defaultModel": "deepseek-chat"
    }
  }
}
```

### 成本对比

| 模型 | 价格 | 性价比 |
|------|------|--------|
| Claude 3.5 Sonnet | $3/千Token | 中等 |
| DeepSeek V3 | ¥1/百万Token | 极高 |
