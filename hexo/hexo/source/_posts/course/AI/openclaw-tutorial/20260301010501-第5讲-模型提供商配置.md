## 获取API密钥

### Anthropic (Claude)
1. 访问 https://console.anthropic.com/
2. 注册账号并登录
3. 进入Settings → API Keys
4. 点击Create Key，复制保存

**注意**：新账号有$5免费额度。

### OpenAI
1. 访问 https://platform.openai.com/
2. 注册并绑定信用卡（必需）
3. 进入API Keys页面
4. 创建新的Secret Key

**注意**：需要海外信用卡，国内卡可能无法绑定。

### Moonshot (Kimi)
1. 访问 https://platform.moonshot.cn/
2. 用国内手机号注册
3. 进入API Key管理
4. 创建Key并保存

**优势**：国内直接访问，新用户有15元免费额度。

## 配置Anthropic (Claude)

编辑`~/.openclaw/openclaw.json`：

```json
{
  "providers": {
    "anthropic": {
      "enabled": true,
      "apiKey": "sk-ant-api03-xxxxx",
      "baseUrl": "https://api.anthropic.com",
      
      "defaultModel": "claude-3-opus",
      "defaultParams": {
        "max_tokens": 4096,
        "temperature": 0.7
      },
      
      "models": {
        "claude-3-opus": {
          "id": "claude-3-opus-20240229",
          "name": "Claude 3 Opus",
          "description": "最强推理能力",
          "maxTokens": 4096,
          "contextWindow": 200000,
          "inputPrice": 0.015,
          "outputPrice": 0.075
        },
        "claude-3-sonnet": {
          "id": "claude-3-sonnet-20240229",
          "name": "Claude 3 Sonnet",
          "description": "平衡性能与成本",
          "maxTokens": 4096,
          "contextWindow": 200000,
          "inputPrice": 0.003,
          "outputPrice": 0.015
        },
        "claude-3-haiku": {
          "id": "claude-3-haiku-20240307",
          "name": "Claude 3 Haiku",
          "description": "快速响应，成本低",
          "maxTokens": 4096,
          "contextWindow": 200000,
          "inputPrice": 0.00025,
          "outputPrice": 0.00125
        }
      }
    }
  }
}
```

**价格说明**（每1000 tokens）：
- Opus：输入$0.015，输出$0.075
- Sonnet：输入$0.003，输出$0.015
- Haiku：输入$0.00025，输出$0.00125

## 多模型负载均衡配置

同时配置多个提供商，实现故障自动切换：

```json
{
  "routing": {
    "strategy": "failover",
    "providers": ["anthropic", "moonshot", "openai"],
    "timeout": 30000,
    "retry": 3,
    "fallback": "moonshot"
  },
  
  "providers": {
    "anthropic": {
      "enabled": true,
      "priority": 1,
      "apiKey": "${ANTHROPIC_KEY}"
    },
    "moonshot": {
      "enabled": true,
      "priority": 2,
      "apiKey": "${MOONSHOT_KEY}"
    },
    "openai": {
      "enabled": true,
      "priority": 3,
      "apiKey": "${OPENAI_KEY}"
    }
  }
}
```

**策略说明**：
- `failover`：主模型失败时切换到备用
- `round-robin`：轮询使用各模型
- `random`：随机选择

## 测试模型连接

配置完成后，测试是否成功：

```bash
# 查看已配置的提供商
openclaw providers list

# 测试特定提供商
openclaw providers test anthropic

# 在Control UI中测试
openclaw dashboard
```

在Chat界面发送测试消息：
```
你好，请介绍一下你自己
```

如果能收到回复，说明配置成功！

## 成本优化技巧

### 1. 按场景选择模型
```json
{
  "routing": {
    "rules": [
      {
        "condition": "code in prompt",
        "provider": "anthropic",
        "model": "claude-3-sonnet"
      },
      {
        "condition": "length(prompt) > 50000",
        "provider": "moonshot",
        "model": "moonshot-v1-128k"
      },
      {
        "default": true,
        "provider": "moonshot",
        "model": "moonshot-v1-8k"
      }
    ]
  }
}
```

### 2. 设置用量限制
```json
{
  "usage": {
    "limits": {
      "daily": 1000000,
      "monthly": 20000000
    },
    "alerts": {
      "threshold": 0.8,
      "webhook": "https://your-webhook.com/alert"
    }
  }
}
```
