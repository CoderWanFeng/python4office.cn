## 项目目标

构建一个智能监控系统：
- 🔍 定时检查服务器状态
- 📊 收集性能指标（CPU、内存、磁盘）
- 🚨 异常时自动告警
- 🤖 AI分析故障原因
- 📱 多渠道通知（Telegram/微信）

## 监控脚本

### health_check.sh
```bash
#!/bin/bash
# 服务器健康检查脚本

ALERTS=""

# 检查CPU使用率
CPU_USAGE=$(top -bn1 | grep "Cpu(s)" | awk '{print $2}' | cut -d'%' -f1)
if (( $(echo "$CPU_USAGE > 80" | bc -l) )); then
  ALERTS="${ALERTS}⚠️ CPU使用率过高: ${CPU_USAGE}%\n"
fi

# 检查内存使用率
MEM_USAGE=$(free | grep Mem | awk '{printf "%.0f", $3/$2 * 100.0}')
if [ "$MEM_USAGE" -gt 90 ]; then
  ALERTS="${ALERTS}⚠️ 内存使用率过高: ${MEM_USAGE}%\n"
fi

# 检查磁盘空间
DISK_USAGE=$(df / | tail -1 | awk '{print $5}' | sed 's/%//')
if [ "$DISK_USAGE" -gt 85 ]; then
  ALERTS="${ALERTS}⚠️ 磁盘空间不足: ${DISK_USAGE}%\n"
fi

# 检查服务状态
if ! systemctl is-active --quiet nginx; then
  ALERTS="${ALERTS}🚨 Nginx服务已停止\n"
fi

# 发送告警
if [ -n "$ALERTS" ]; then
  curl -X POST http://localhost:18789/webhook/alert \
    -H "Content-Type: application/json" \
    -d "{\"message\": \"$ALERTS\", \"severity\": \"warning\"}"
fi
```

## OpenClaw配置

### Webhook接收告警
```json
{
  "webhooks": {
    "handlers": [
      {
        "name": "server_alerts",
        "endpoint": "/webhook/alert",
        "script": "handle_alert.js"
      }
    ]
  }
}
```

### 告警处理脚本
```javascript
// handle_alert.js
exports.handler = async (payload) => {
  const { message, severity } = payload;
  
  // 根据严重程度选择通道
  const channels = severity === 'critical' 
    ? ['telegram', 'wechat'] 
    : ['telegram'];
  
  for (const channel of channels) {
    await openclaw.message.send({
      channel,
      text: `🚨 服务器告警\n\n${message}\n\n时间: ${new Date().toLocaleString()}`
    });
  }
  
  // AI分析并给出建议
  if (severity === 'critical') {
    const analysis = await openclaw.agent.run(
      `分析以下服务器问题并给出解决方案:\n${message}`
    );
    
    await openclaw.message.send({
      channel: 'telegram',
      text: `🤖 AI分析:\n${analysis}`
    });
  }
};
```

## 日报生成

每天上午9点自动生成系统报告：

```bash
#!/bin/bash
# daily_report.sh

REPORT="📊 系统日报 $(date +%Y-%m-%d)\n\n"

# 系统负载
LOAD=$(uptime | awk -F'load average:' '{print $2}')
REPORT+="系统负载: $LOAD\n"

# 内存使用
MEM=$(free -h | grep Mem | awk '{print "已用: " $3 "/" $2}')
REPORT+="内存使用: $MEM\n"

# 磁盘使用
DISK=$(df -h / | tail -1 | awk '{print $5}')
REPORT+="磁盘使用: $DISK\n\n"

# 发送报告
curl -X POST http://localhost:18789/message/send \
  -H "Content-Type: application/json" \
  -d "{\"channel\": \"telegram\", \"text\": \"$REPORT\"}"
```
