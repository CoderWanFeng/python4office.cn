## 日志系统

### 日志文件位置
```
~/.openclaw/state/logs/
├── gateway.log       # Gateway主日志
├── channels.log      # 通道相关日志
├── sessions.log      # 会话日志
├── tools.log         # 工具调用日志
├── audit.log         # 审计日志
└── error.log         # 错误日志
```

### 查看实时日志
```bash
# 查看Gateway日志
tail -f ~/.openclaw/state/logs/gateway.log

# 查看特定通道日志
tail -f ~/.openclaw/state/logs/channels.log | grep telegram

# 查看错误日志
tail -f ~/.openclaw/state/logs/error.log
```

### 日志级别配置
```json
{
  "logging": {
    "level": "info",
    "levels": {
      "gateway": "debug",
      "channels": "warn",
      "tools": "info"
    }
  }
}
```

日志级别：debug < info < warn < error

## 常用诊断命令

### 系统状态检查
```bash
# 检查Gateway状态
openclaw gateway status

# 查看系统健康
openclaw doctor

# 检查配置有效性
openclaw config validate

# 查看资源使用
openclaw system stats
```

### 通道诊断
```bash
# 列出所有通道
openclaw channels list

# 测试特定通道
openclaw channels test telegram

# 重新连接通道
openclaw channels reconnect whatsapp
```

## 常见问题解决

### 问题1：Gateway无法启动
**症状**：`openclaw gateway start` 失败

**排查步骤**：
```bash
# 1. 检查端口占用
lsof -i :18789

# 2. 检查配置文件
openclaw config validate

# 3. 查看详细错误
cat ~/.openclaw/state/logs/gateway.log | tail -50

# 4. 尝试前台启动看错误
openclaw gateway --port 18789 --foreground
```

**常见原因**：
- 端口被占用
- 配置文件格式错误
- Node.js版本过低

### 问题2：AI不回复消息
**排查步骤**：
```bash
# 1. 检查API密钥
openclaw providers test anthropic

# 2. 检查通道状态
openclaw channels status telegram

# 3. 查看消息日志
grep "message received" ~/.openclaw/state/logs/channels.log

# 4. 检查用户权限
grep "access denied" ~/.openclaw/state/logs/audit.log
```

### 问题3：工具调用失败
**排查步骤**：
```bash
# 1. 检查工具权限
cat ~/.openclaw/openclaw.json | jq '.tools'

# 2. 查看工具日志
tail -f ~/.openclaw/state/logs/tools.log

# 3. 测试简单命令
openclaw tool exec --command "echo test"
```

## 性能优化

### 内存优化
```json
{
  "sessions": {
    "maxHistory": 50,
    "ttl": 3600,
    "pruneInterval": 300
  },
  "memory": {
    "compaction": {
      "enabled": true,
      "interval": 86400
    }
  }
}
```

### 连接池优化
```json
{
  "providers": {
    "anthropic": {
      "connectionPool": {
        "min": 2,
        "max": 10,
        "idleTimeout": 30000
      }
    }
  }
}
```

### 缓存配置
```json
{
  "cache": {
    "enabled": true,
    "ttl": 3600,
    "maxSize": "100MB"
  }
}
```
