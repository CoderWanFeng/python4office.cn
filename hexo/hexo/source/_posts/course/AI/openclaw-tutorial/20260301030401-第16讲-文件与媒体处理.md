## 支持的多媒体类型

| 类型 | 格式 | 功能 |
|------|------|------|
| **图片** | JPG, PNG, GIF, WebP | 图像识别、OCR、分析 |
| **音频** | MP3, WAV, OGG, M4A | 语音转文字、语音识别 |
| **视频** | MP4, AVI, MOV | 帧提取、内容分析 |
| **文档** | PDF, DOCX, TXT, MD | 文本提取、摘要生成 |
| **表格** | CSV, XLSX, JSON | 数据分析、可视化 |

---

## 图片分析与OCR

### 场景1：识别图片内容
发送一张图片给AI，它可以：
- 描述图片内容
- 识别图中文字（OCR）
- 分析图表数据
- 识别物体和场景

示例对话：
```
[用户发送一张包含图表的图片]

你：分析一下这张图表

AI：我看到这是一张销售趋势图，显示了2024年各季度的销售额：
- Q1: ¥120万
- Q2: ¥150万  
- Q3: ¥180万
- Q4: ¥210万

整体呈上升趋势，Q4相比Q1增长了75%。
```

### 配置图片处理
```json
{
  "tools": {
    "image": {
      "enabled": true,
      "maxSize": 10485760,
      "supportedFormats": ["jpg", "png", "gif", "webp"],
      "ocr": {
        "enabled": true,
        "language": "chi_sim+eng"
      }
    }
  }
}
```

## 语音转文字

### 场景：语音消息处理
在支持的通道（如Telegram、WhatsApp）中发送语音，AI可以：
- 将语音转为文字
- 理解语音内容并回复
- 支持多种语言

示例：
```
[用户发送语音消息]

AI：🎤 语音转文字：
"帮我查一下明天北京的天气"

明天北京天气预报：
- 天气：晴转多云
- 温度：5°C ~ 15°C
- 风力：北风3级
```

### 配置语音识别
```json
{
  "tools": {
    "audio": {
      "enabled": true,
      "speechToText": {
        "enabled": true,
        "model": "whisper",
        "language": "zh"
      },
      "maxDuration": 300
    }
  }
}
```

## 文档读取与分析

### 场景1：PDF文档摘要
```
你：[上传一份PDF报告]
请总结一下这份报告的核心观点

AI：我来阅读这份PDF文档...
🔧 Using tool: read_document
   Format: PDF
   Pages: 25
   
📤 核心观点摘要：
1. 市场增长：2024年AI市场规模增长35%
2. 技术趋势：大模型成为主流方向
3. 投资建议：关注基础设施和应用层
4. 风险提示：监管政策不确定性
```

### 场景2：Excel数据分析
```
你：[上传sales_data.xlsx]
分析一下销售数据，找出增长最快的产品

AI：正在分析Excel文件...
🔧 Using tool: read_excel
   Sheets: 3
   Rows: 1250
   
📤 分析结果：
增长最快的产品Top 3：
1. 产品A - 环比增长45%
2. 产品B - 环比增长32%
3. 产品C - 环比增长28%

建议加大产品A的营销投入。
```

## 代码文件处理

### 场景：代码审查
```
你：[上传main.py]
帮我检查一下这段代码有没有问题

AI：正在分析Python代码...
🔧 Using tool: read
   Path: main.py
   Lines: 156
   
📤 代码审查结果：
✓ 整体结构清晰
⚠️ 第23行：未使用的导入
⚠️ 第45行：硬编码的API密钥，建议使用环境变量
✗ 第78行：可能存在SQL注入风险

建议修改：
```python
# 原代码
cursor.execute(f"SELECT * FROM users WHERE id = {user_id}")

# 修改为
cursor.execute("SELECT * FROM users WHERE id = %s", (user_id,))
```
```
