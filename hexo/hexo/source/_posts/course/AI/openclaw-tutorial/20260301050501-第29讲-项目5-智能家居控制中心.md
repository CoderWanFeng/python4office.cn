---

## 项目目标

构建一个智能家居控制中心：
- 🎤 语音唤醒和控制
- 💡 控制灯光、空调、窗帘
- 🌡️ 监控温湿度
- 🔒 管理智能门锁
- 📱 手机App远程控制

---

## 硬件准备

| 设备 | 品牌/协议 | 接入方式 |
|------|----------|---------|
| 智能音箱 | 小爱/天猫精灵 | 开发者API |
| 智能灯泡 | Yeelight/MiLight | WiFi/Zigbee |
| 空调伴侣 | 小米/博联 | 红外遥控 |
| 温湿度传感器 | 小米/DHT22 | WiFi/GPIO |
| 智能插座 | TP-Link/小米 | WiFi |

---

## 架构设计

```
用户语音 → OpenClaw Node (手机) → Gateway → Home Assistant
                                          ↓
                                    设备控制指令
                                          ↓
                              ┌─────┬─────┼─────┐
                              ↓     ↓     ↓     ↓
                            灯光   空调   窗帘   插座
```

## Home Assistant集成

### 安装Home Assistant
```bash
# Docker安装
docker run -d \
  --name homeassistant \
  --privileged \
  --restart=unless-stopped \
  -e TZ=Asia/Shanghai \
  -v /path/to/config:/config \
  --network=host \
  ghcr.io/home-assistant/home-assistant:stable
```

### 配置设备
```yaml
# configuration.yaml
light:
  - platform: yeelight
    devices:
      192.168.1.100:
        name: Living Room Light

climate:
  - platform: xiaomi_miio
    name: Bedroom AC
    host: 192.168.1.101
    token: your_token

sensor:
  - platform: dht
    pin: 4
    monitored_conditions:
      - temperature
      - humidity
```

## OpenClaw控制脚本

### 设备控制Skill
```javascript
// smart_home.js
const axios = require('axios');

const HA_URL = 'http://homeassistant.local:8123';
const HA_TOKEN = 'your_long_lived_token';

async function control_light(entity_id, action) {
  const service = action === 'on' ? 'turn_on' : 'turn_off';
  
  await axios.post(
    `${HA_URL}/api/services/light/${service}`,
    { entity_id },
    {
      headers: {
        'Authorization': `Bearer ${HA_TOKEN}`
      }
    }
  );
  
  return `已${action === 'on' ? '打开' : '关闭'}灯光`;
}

async function set_temperature(entity_id, temp) {
  await axios.post(
    `${HA_URL}/api/services/climate/set_temperature`,
    {
      entity_id,
      temperature: temp
    },
    {
      headers: {
        'Authorization': `Bearer ${HA_TOKEN}`
      }
    }
  );
  
  return `已将温度设置为${temp}度`;
}

module.exports = {
  control_light,
  set_temperature
};
```

## 语音控制

### 唤醒词配置
```json
{
  "nodes": {
    "voiceWake": {
      "enabled": true,
      "wakeWord": "小助手",
      "sensitivity": 0.8
    }
  }
}
```

### 自然语言理解
```
用户：小助手，把客厅灯打开

AI：[解析意图]
动作: control_light
位置: living_room
状态: on

[执行命令]
🔧 Using tool: control_light
   entity_id: light.living_room
   action: on
   
✓ 已打开客厅灯光
```

### 复杂场景
```
用户：我要睡觉了

AI：好的，正在执行睡前模式：
- 关闭客厅灯 ✓
- 打开卧室夜灯 ✓
- 空调设为睡眠模式（26度）✓
- 关闭窗帘 ✓

晚安！
```

## 自动化场景

### 回家模式
```json
{
  "cron": {
    "jobs": [
      {
        "name": "arrive_home",
        "trigger": "geofence",
        "condition": "user enters home",
        "actions": [
          "light.turn_on: living_room",
          "climate.set_preset_mode: home",
          "media_player.play: welcome_music"
        ]
      }
    ]
  }
}
```

```
- 扫地机器人开始工作 ✓

一路平安！
```
