## Slack配置

Slack是企业协作的热门选择，特别是在海外公司。

### 创建Slack App

1. 访问 https://api.slack.com/apps
2. 点击 **Create New App** → **From scratch**
3. 输入App名称，选择Workspace

### 设置权限

在OAuth & Permissions页面，添加以下Scopes：
- `chat:write` - 发送消息
- `im:history` - 读取私聊历史
- `channels:history` - 读取频道历史
- `app_mentions:read` - 读取@提及

### 安装到Workspace

点击 **Install to Workspace**，复制 **Bot User OAuth Token**。

### OpenClaw配置

```json
{
  "channels": {
    "slack": {
      "enabled": true,
      "token": "xoxb-your-token",
      "signingSecret": "your-signing-secret",
      "allowFrom": [],
      "requireMention": true
    }
  }
}
```

## iMessage配置（macOS）

iMessage是苹果设备的原生消息应用。

### 前提条件
- macOS系统
- 已登录iMessage账号
- 开启辅助功能权限

### 配置步骤
```json
{
  "channels": {
    "imessage": {
      "enabled": true,
      "platform": "macos",
      "allowFrom": [],
      "autoReply": false
    }
  }
}
```

> ⚠️ 注意：iMessage需要macOS保持运行，不能部署在服务器上。

## 飞书（Feishu/Lark）配置

飞书是字节跳动出品的企业协作平台，在很多国内公司流行。

### 创建飞书应用

1. 访问 https://open.feishu.cn/app
2. 创建企业自建应用
3. 获取 **App ID** 和 **App Secret**

### 配置权限

在权限管理中添加：
- `im:message:send` - 发送消息
- `im:message.group_msg` - 群组消息
- `im:chat:readonly` - 读取会话

### OpenClaw配置

```json
{
  "channels": {
    "feishu": {
      "enabled": true,
      "appId": "cli_xxxxx",
      "appSecret": "xxxxxxxx",
      "encryptKey": "optional",
      "verificationToken": "optional"
    }
  }
}
```

## 钉钉配置

钉钉是阿里巴巴出品的企业通讯平台。

### 创建钉钉机器人

1. 打开群设置 → 智能群助手
2. 添加机器人 → 自定义
3. 获取 **Webhook URL** 和 **Secret**

### OpenClaw配置

```json
{
  "channels": {
    "dingtalk": {
      "enabled": true,
      "webhook": "https://oapi.dingtalk.com/robot/send?access_token=xxxxx",
      "secret": "SECxxxxx",
      "atAll": false
    }
  }
}
```

## 实战建议

### 组合方案推荐

**方案1：出海应用**
- 主通道：Telegram + Discord
- 备用通道：WhatsApp + Slack

**方案2：国内企业**
- 主通道：飞书/钉钉
- 备用通道：企业微信

**方案3：通用覆盖**
- 海外：Discord + Slack
- 国内：飞书 + 企业微信

### 配置检查清单

在部署前，检查以下几点：
- [ ] 所有通道的Token已配置
- [ ] 白名单用户已设置
- [ ] 速率限制已配置
- [ ] 告警通知已设置
- [ ] 测试消息已通过
