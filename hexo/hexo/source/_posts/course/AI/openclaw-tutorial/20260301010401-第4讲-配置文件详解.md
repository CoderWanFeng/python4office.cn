大家好，我是正在实战各种<a href="https://www.bilibili.com/cheese/play/ss982042944" target="_blank">AI项目</a>的程序员晚枫。

今天我们来深入**OpenClaw的配置系统**。配置文件是OpenClaw的灵魂，掌握它你就能完全掌控Gateway的行为。

---

## 配置文件位置

主配置文件位于：
```
~/.openclaw/openclaw.json
```

使用以下命令查看当前配置：
```bash
openclaw config show
```

编辑配置：
```bash
# 用默认编辑器打开
openclaw config edit

# 或手动编辑
nano ~/.openclaw/openclaw.json
```

---

## 配置文件结构概览

```json
{
  "gateway": {          // Gateway核心配置
    "port": 18789,
    "host": "127.0.0.1",
    "auth": {...}
  },
  "providers": {        // AI模型提供商
    "anthropic": {...},
    "openai": {...}
  },
  "channels": {         // 消息通道
    "telegram": {...},
    "whatsapp": {...}
  },
  "tools": {            // 工具配置
    "exec": {...},
    "browser": {...}
  },
  "sessions": {         // 会话管理
    "maxHistory": 100,
    "ttl": 3600
  },
  "logging": {          // 日志配置
    "level": "info",
    "file": true
  }
}
```

## 核心配置项详解

### 1. Gateway配置

```json
{
  "gateway": {
    "port": 18789,              // 服务端口
    "host": "127.0.0.1",        // 绑定地址
    "publicUrl": null,          // 公网访问地址（可选）
    
    "auth": {
      "type": "token",          // 认证方式
      "token": "your-secret-token"  // 访问令牌
    },
    
    "cors": {
      "enabled": true,          // 是否允许跨域
      "origins": ["*"]          // 允许的域名
    }
  }
}
```

**安全建议**：生产环境务必设置强token，并限制origins。

### 2. 模型提供商配置

#### Anthropic (Claude)
```json
{
  "providers": {
    "anthropic": {
      "enabled": true,
      "apiKey": "sk-ant-api03-xxxxx",
      "defaultModel": "claude-3-opus-20240229",
      "models": {
        "claude-3-opus": {
          "id": "claude-3-opus-20240229",
          "maxTokens": 4096,
          "contextWindow": 200000
        },
        "claude-3-sonnet": {
          "id": "claude-3-sonnet-20240229",
          "maxTokens": 4096,
          "contextWindow": 200000
        }
      }
    }
  }
}
```

#### OpenAI (GPT)
```json
{
  "providers": {
    "openai": {
      "enabled": true,
      "apiKey": "sk-xxxxx",
      "baseUrl": "https://api.openai.com/v1",
      "defaultModel": "gpt-4",
      "models": {
        "gpt-4": {"id": "gpt-4", "maxTokens": 8192},
        "gpt-3.5-turbo": {"id": "gpt-3.5-turbo", "maxTokens": 4096}
      }
    }
  }
}
```

#### 国内模型示例（Moonshot）
```json
{
  "providers": {
    "moonshot": {
      "enabled": true,
      "apiKey": "your-moonshot-key",
      "baseUrl": "https://api.moonshot.cn/v1",
      "defaultModel": "moonshot-v1-8k"
    }
  }
}
```

### 3. 通道安全配置

```json
{
  "channels": {
    "telegram": {
      "enabled": true,
      "botToken": "xxxxxx:yyyyyy",
      "allowFrom": [],           // 白名单用户ID
      "groups": {
        "requireMention": true,  // 群组中需要@才响应
        "allowedGroups": []      // 允许的群组ID
      }
    },
    
    "whatsapp": {
      "enabled": true,
      "sessionName": "default",
      "allowFrom": ["+86138xxxxxxxx"],  // 只允许特定手机号
      "pairingCode": null
    }
  }
}
```

### 4. 工具权限配置

```json
{
  "tools": {
    "exec": {
      "enabled": true,
      "sandbox": true,           // 沙盒模式
      "allowedCommands": ["ls", "cat", "grep", "curl"],
      "blockedCommands": ["rm", "dd", "mkfs"],
      "workingDirectory": "~/.openclaw/sandbox"
    },
    
    "browser": {
      "enabled": true,
      "headless": true,          // 无头模式
      "timeout": 30000,          // 超时时间(ms)
      "allowedDomains": ["*"],   // 允许访问的域名
      "blockedDomains": []
    },
    
    "read": {
      "enabled": true,
      "allowedPaths": ["~/projects", "~/documents"],
      "blockedPaths": ["~/.ssh", "~/.aws"]
    },
    
    "write": {
      "enabled": false,          // 默认禁用写入
      "allowedPaths": ["~/projects/temp"],
      "requireApproval": true    // 需要人工确认
    }
  }
}
```

### 5. 会话与记忆配置

```json
{
  "sessions": {
    "maxHistory": 100,           // 最大历史消息数
    "ttl": 86400,               // 会话过期时间(秒)
    "pruneInterval": 3600,      // 清理间隔
    
    "memory": {
      "enabled": true,
      "storage": "file",        // file/redis
      "path": "~/.openclaw/state/memory"
    }
  }
}
```

## 常用配置模板

### 模板1：本地开发环境
```json
{
  "gateway": {
    "port": 18789,
    "auth": {"type": "none"}
  },
  "providers": {
    "anthropic": {
      "enabled": true,
      "apiKey": "${ANTHROPIC_API_KEY}"
    }
  },
  "tools": {
    "exec": {"enabled": true, "sandbox": false},
    "write": {"enabled": true}
  }
}
```

### 模板2：生产服务器
```json
{
  "gateway": {
    "port": 18789,
    "host": "0.0.0.0",
    "auth": {
      "type": "token",
      "token": "${GATEWAY_TOKEN}"
    }
  },
  "providers": {
    "anthropic": {"enabled": true, "apiKey": "${ANTHROPIC_KEY}"},
    "openai": {"enabled": true, "apiKey": "${OPENAI_KEY}"}
  },
  "tools": {
    "exec": {
      "enabled": true,
      "sandbox": true,
      "allowedCommands": ["ls", "cat", "ps"]
    },
    "write": {"enabled": false}
  }
}
```

### 模板3：多模型负载均衡
```json
{
  "routing": {
    "strategy": "failover",     // failover/round-robin/random
    "providers": ["anthropic", "openai", "moonshot"],
    "fallback": "openai"
  }
}
```

## 配置热重载

修改配置后，无需重启Gateway：

```bash
# 发送重载信号
openclaw gateway reload

# 或发送SIGHUP
kill -HUP $(pgrep -f openclaw-gateway)
```

部分配置（如port、host）仍需重启生效。

## 环境变量注入

敏感信息建议使用环境变量：

```bash
# .bashrc 或 .zshrc
export ANTHROPIC_API_KEY="sk-ant-xxxxx"
export OPENAI_API_KEY="sk-xxxxx"
export GATEWAY_TOKEN="my-secret-token"
```

配置文件中引用：
```json
{
  "providers": {
    "anthropic": {
      "apiKey": "${ANTHROPIC_API_KEY}"
    }
  }
}
```
