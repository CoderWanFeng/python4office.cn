## 项目目标

打造一个7x24小时在线的个人AI助理，具备：
- ✅ 多通道接入（Telegram+微信）
- ✅ 长期记忆（记住你的偏好）
- ✅ 工具调用（执行命令、查资料）
- ✅ 定时任务（每日简报）
- ✅ 远程访问（随时随地使用）

## 部署步骤

### 1. 准备环境
```bash
# 安装Node.js 22+
nvm install 22

# 安装OpenClaw
npm install -g openclaw@latest

# 创建配置目录
mkdir -p ~/.openclaw/memory
```

### 2. 配置API密钥
```bash
# 设置环境变量
export ANTHROPIC_KEY="sk-ant-xxxxx"
export MOONSHOT_KEY="sk-xxxxx"
export TG_BOT_TOKEN="123456:ABCdef"

# 添加到.bashrc使其永久生效
echo 'export ANTHROPIC_KEY="sk-ant-xxxxx"' >> ~/.bashrc
```

### 3. 启动服务
```bash
# 初始化
openclaw onboard

# 连接通道
openclaw channels connect telegram
openclaw channels connect wechat

# 启动Gateway
openclaw daemon start

# 检查状态
openclaw gateway status
```

### 4. 测试验证
发送测试消息：
```
你好，请介绍一下你自己
```

预期回复：
```
你好！我是你的个人AI助理。我可以帮你：
- 回答问题
- 执行命令
- 搜索信息
- 管理任务

有什么可以帮你的吗？
```

## 运维监控

### 设置开机自启
```bash
sudo systemctl enable openclaw
```

### 健康检查脚本
```bash
#!/bin/bash
# health_check.sh

if ! openclaw gateway status | grep -q "running"; then
  openclaw daemon restart
  echo "$(date): Gateway restarted" >> ~/logs/openclaw_health.log
fi
```

添加到cron：
```bash
*/5 * * * * /path/to/health_check.sh
```
