## 关键词研究工具

### 1. Google 关键词规划师（免费）

**简介**：Google Ads 附带的免费关键词工具

**功能**：
- 查看关键词搜索量
- 发现相关关键词
- 查看关键词竞争度

**使用方法**：

1. 登录 [Google Ads](https://ads.google.com)
2. 工具与设置 → 关键词规划师
3. 选择"发现新关键词"
4. 输入核心关键词
5. 筛选地域关键词

**技巧**：

```
输入：Python 培训

筛选：
- 地区：中国
- 语言：中文

结果：
- Python 培训 北京
- Python 培训 上海
- Python 培训 班
...
```

### 2. 百度指数（免费）

**简介**：百度官方的关键词趋势分析工具

**网址**：https://index.baidu.com

**功能**：
- 查看关键词搜索趋势
- 地域分布分析
- 人群画像分析

**使用场景**：

```
查询：Python 培训

地域分布：
- 北京：1,200
- 上海：980
- 深圳：860
- 广州：750
- 成都：620

用途：
- 确定重点优化城市
- 制定地域关键词策略
```

### 3. 5118（付费，有免费功能）

**简介**：国内知名 SEO 工具平台

**网址**：https://www.5118.com

**功能**：
- 长尾关键词挖掘
- 关键词排名监控
- 内容创作建议

**特色功能**：

| 功能 | 说明 |
|------|------|
| 长尾词挖掘 | 挖掘地域长尾关键词 |
| 移动端流量 | 查看移动端搜索数据 |
| 竞争分析 | 分析关键词竞争度 |

**使用示例**：

```
查询：Python 培训

长尾词推荐：
- Python 培训 北京 哪家好
- Python 培训 上海 价格
- Python 培训 深圳 周末班
...
```

### 4. Ahrefs（付费）

**简介**：国际顶级 SEO 工具

**网址**：https://ahrefs.com

**功能**：
- 关键词研究
- 竞品分析
- 反向链接分析
- 内容分析

**适用场景**：
- 针对海外市场的 GEO 优化
- 竞争激烈的行业

## 排名追踪工具

### 1. Google Search Console（免费）

**功能**：查看网站在 Google 搜索中的平均排名

**查看方法**：

```
GSC → 效果 → 查询

按排名排序：
- 查看哪些关键词排名较好
- 发现排名下降的关键词
```

### 2. 5118 排名监控

**功能**：监控百度搜索排名

**使用方法**：

```
1. 添加网站
2. 添加关键词
3. 设置监控频率
4. 查看排名变化
```

## 竞品分析工具

### 1. Ahrefs Site Explorer

**功能**：分析竞争对手网站的 SEO 策略

**使用方法**：

```
Site Explorer → 输入竞品网站

查看：
- 有机流量
- 关键词排名
- 反向链接
- 热门页面
```

**GEO 分析场景**：

```
分析竞品的地区流量分布：

有机搜索 → 按国家

结果：
- 中国：60%
- 美国：20%
- 日本：10%
...

判断竞品的主攻市场
```

### 2. SimilarWeb（部分免费）

**网址**：https://www.similarweb.com

**功能**：
- 网站流量分析
- 流量来源分析
- 地理分布分析
