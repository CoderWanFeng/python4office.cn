# 第 03 讲｜公司注册全流程

> **别被代理忽悠了！自己注册，一天搞定，省下一千五。**
> 
> 这一讲，我手把手教你注册公司，每个坑都给你标出来。

---

## 🎯 开篇：我为什么要讲这个？

先说个真实故事。

我有个学员，想注册一人公司。

找了个代理，花了两千块。

**结果呢？**

地址是假的，被工商局查了。
章程没写对，后来改章程又花了五百。
银行开户被拒，因为代理没帮他预约。

**最后折腾了两个月，才把公司办好。**

他找到我，说："晚枫，早认识你就好了。"

**我说："不晚，现在认识也不迟。"**

**这就是我今天要讲的内容——公司注册全流程。**

我注册过三家公司，踩过所有的坑。

今天我把这些经验都给你，让你少走弯路。

**看完这一讲，你自己就能注册公司，省下一千五代理费。**

---

## 😭 先说个扎心的真相

**大部分代理，就是在赚信息差的钱。**

他们做的事，你自己也能做。

而且做得更好，因为这是你自己的公司，你比他们上心。

**代理能做什么？**
- 帮你填表（表格网上都有）
- 帮你跑腿（你自己也能跑）
- 帮你预约（你自己也能约）

**代理不能做什么？**
- 不能帮你决定公司名称
- 不能帮你决定经营范围
- 不能帮你决定注册资本
- 不能帮你承担法律责任

**那你还花这个钱干嘛？**

当然，如果你时间特别宝贵，或者人在外地，找代理也合理。

**但如果你有时间，自己注册，真不难。**

## 📋 注册前准备（别跳过，很重要）

### 1. 想好公司名称（准备 3 个备选）

**为什么准备 3 个？**

因为好名字可能被人注册了。

我第一个公司，想了个特别牛的名字，"未来科技有限公司"。

去工商局一查，被人注册了。

**当时那个心情，你能体会吗？**

所以一定要准备备选。

**名称格式：**
```
行政区划 + 字号 + 行业 + 组织形式
```

**示例：**
- 深圳市 + 晚枫 + 科技 + 有限公司
- 北京市 + 创客 + 信息科技 + 有限公司
- 上海市 + 星辰 + 文化传播 + 有限公司

**注意事项：**
- 字号不能和已有公司重复
- 不能用敏感词（国家、中国、国际等）
- 不能用名人名字
- 最好两个字或三个字，好记

**我的建议：**
- 先在网上查一下有没有重名
- 准备 3 到 5 个备选
- 选个朗朗上口的，别太复杂

### 2. 确定注册资本（别写太高）

**现在是认缴制，不用实缴。**

什么意思？

就是你写一百万，不用真往公司账户里存一百万。

**那写多少合适？**

写太高，责任大。写太低，客户不信任。

**我的建议：十万到五十万之间。**

**真实案例：**

我有个学员，为了显得有实力，写了一千万。

后来公司欠债，法院判他在一千万范围内承担责任。

**他当时就傻了。**

"我就是写写而已，没想到真要赔这么多。"

**记住：注册资本是你的责任上限，不是面子。**

### 3. 确定经营范围（别贪多）

**经营范围怎么写？**

网上有模板，照着抄就行。

**注意事项：**
- 别写太多，写主要的就行
- 有些经营范围需要许可证（比如食品、医疗）
- 写错了可以改，别太纠结

**我的建议：**
- 参考同行，看他们怎么写
- 写你实际要做的业务
- 以后要加业务，可以变更

### 4. 确定注册地址（最关键的环节）

**这是最容易出问题的环节。**

**地址要求：**
- 必须是商业地址或商住两用
- 纯住宅不行（大部分城市）
- 地址必须真实存在

**地址来源：**
1. **自己的房子**（如果是商业性质）
2. **租办公室**（成本高，适合真需要办公的）
3. **地址挂靠**（成本低，适合在家办公的）

**地址挂靠是怎么回事？**

就是有些园区、孵化器，提供地址给你注册。

你不用真去那里办公，只是用这个地址注册。

**成本：一年两千到五千不等。**

**注意事项：**
- 一定要找正规的园区
- 问清楚能不能配合工商检查
- 问清楚能不能接收工商信函
- 别贪便宜，太便宜的可能有问题

**踩坑案例：**

我那个学员，就是找了个不靠谱的挂靠地址。

五百块一年，便宜吧？

结果工商局一查，地址不存在。

公司被列入经营异常，后来花了两千才移出来。

**我的建议：**
- 找当地正规的孵化器或园区
- 问清楚服务内容包括什么
- 签正式合同，别口头约定

### 5. 确定法人、监事、财务负责人

**一人公司，你是股东。**

但还需要：
- **法人**：可以是股东本人
- **监事**：不能是股东本人（需要找个人）
- **财务负责人**：可以是股东本人

**监事找谁？**

找个信任的人，家人、朋友都行。

监事不参与经营，就是个名义。

**注意事项：**
- 监事不能是股东
- 监事需要身份证信息
- 监事不需要到场

## 📝 注册流程（手把手教学）

### 第 1 步：网上核名（10 分钟）

**去哪里？**

当地工商局官网，或者"一网通办"平台。

**怎么做？**
1. 注册账号
2. 进入"企业核名"
3. 输入你想好的公司名称
4. 系统会自动查重

**结果：**
- 通过 → 进入下一步
- 不通过 → 换备选名称

**我当时的情况：**

第一个名字，不通过。
第二个名字，不通过。
第三个名字，通过了。

**所以一定要准备备选。**

### 第 2 步：提交材料（30 分钟）

**需要填什么？**
1. 公司名称
2. 注册资本
3. 经营范围
4. 注册地址
5. 股东信息
6. 法人信息
7. 监事信息
8. 财务负责人信息

**需要上传什么？**
1. 身份证照片（股东、法人、监事、财务）
2. 地址证明（房产证或租赁合同）
3. 公司章程（网上有模板）

**注意事项：**
- 信息要准确，别填错
- 照片要清晰
- 章程可以下载模板修改

### 第 3 步：电子签名（10 分钟）

**材料提交后，需要电子签名。**

**怎么做？**
1. 下载当地工商 APP
2. 实名认证
3. 对提交的材料进行签名

**谁需要签名？**
- 股东
- 法人
- 监事

**大家不用到场，各自用手机签就行。**

### 第 4 步：等待审核（1 到 3 个工作日）

**签完名，就等审核。**

**审核结果：**
- 通过 → 领取营业执照
- 驳回 → 按意见修改后重新提交

**我被驳回过一次，原因是经营范围写得有问题。**

修改后重新提交，第二天就通过了。

### 第 5 步：领取营业执照（1 小时）

**审核通过后，去工商局领营业执照。**

**需要带什么？**
- 身份证原件
- 受理通知书（网上下载）

**可以代领吗？**

可以，需要授权委托书。

**现在也可以选邮寄，直接寄到你家。**

### 第 6 步：刻章（1 小时）

**拿到营业执照后，需要刻章。**

**需要刻什么章？**
1. 公章
2. 财务章
3. 法人章
4. 发票章（可选）

**去哪里刻？**

公安局指定的刻章点。

**成本：三百到八百不等。**

**有些城市首次注册免费送章，问清楚。**

### 第 7 步：银行开户（1 到 2 小时）

**拿着营业执照和章，去银行开公司账户。**

**需要带什么？**
1. 营业执照正副本
2. 公章、财务章、法人章
3. 法人身份证
4. 公司章程

**注意事项：**
- 提前预约
- 法人必须到场
- 不同银行收费不同，多问几家

**成本：**
- 开户费：零到五百
- 年费：零到一千
- 网银费：零到五百

**我的建议：**
- 选四大行（工商、农业、中国、建设）
- 问清楚收费标准
- 选离你近的网点

### 第 8 步：税务报到（1 小时）

**开完户，去税务局报到。**

**需要带什么？**
1. 营业执照
2. 公章
3. 法人身份证
4. 银行开户许可证

**做什么？**
- 登记公司信息
- 核定税种
- 领取发票

**现在也可以网上办理，不用跑税务局。**

## ⚠️ 常见坑点（都是真金白银买来的教训）

### 坑 1：地址不实

**后果：** 被列入经营异常，影响征信。

**怎么避免：** 找正规园区，问清楚服务内容。

### 坑 2：注册资本写太高

**后果：** 责任上限太高，风险大。

**怎么避免：** 写十万到五十万，别贪高。

### 坑 3：经营范围乱写

**后果：** 有些业务需要许可证，没证就是违法。

**怎么避免：** 只写实际要做的业务，别贪多。

### 坑 4：不按时记账报税

**后果：** 罚款，列入异常，严重了吊销执照。

**怎么避免：** 找个代账公司，一个月两三百。

### 坑 5：公私不分

**后果：** 财产混同，无限责任。

**怎么避免：** 公司账户和个人账户严格分开。

## 💰 成本核算（自己注册 vs 找代理）

### 自己注册

| 项目 | 费用 |
|------|------|
| 核名 | 免费 |
| 营业执照 | 免费 |
| 刻章 | 三百到八百（有些城市免费） |
| 银行开户 | 零到五百 |
| 地址挂靠 | 两千到五千每年（可选） |
| 代账 | 两千到四千每年（可选） |
| **合计** | **三千到一万（首年）** |

### 找代理

| 项目 | 费用 |
|------|------|
| 代理费 | 一千到两千 |
| 刻章 | 三百到八百 |
| 银行开户 | 零到五百 |
| 地址挂靠 | 两千到五千每年 |
| 代账 | 两千到四千每年 |
| **合计** | **五千到一万五（首年）** |

**差价：一千五到三千。**

**这就是代理赚的钱。**

## 💬 常见问题（快问快答）

**Q：注册需要本人到场吗？**

A：大部分城市不用，全程网上办理。银行开户需要法人到场。

**Q：注册需要多长时间？**

A：顺利的话，3 到 7 个工作日。

**Q：注册后多久需要记账报税？**

A：拿到营业执照后 30 天内，去税务局报到。之后按月或按季报税。

**Q：公司不经营了怎么办？**

A：要么注销，要么转让。别放着不管，会被列入异常。

**Q：一个人可以注册几家公司？**

A：一人公司只能注册一家。但可以用这家公司的名义再投资其他公司。
