# 第 11 讲｜案例 2：AI 课程变现

> **课程，是最适合一人公司的产品。**
> 
> 这一讲，我把课程变现的完整路径掏给你。

---

## 🎯 开篇：从"没人买"到卖了300万，我经历了什么？

2020年，我第一次做课程。

**信心满满地开发了一个Python爬虫课，定价199。**

**结果？**

上架第一个月，卖了7份。

**7份。**

我朋友调侃我："晚枫，你这课的收入还不够请一顿火锅的。"

说实话，当时挺挫败的。

但我没放弃，而是开始研究：**为什么卖不动？**

我花了两个月分析，发现了3个致命问题：

```
❌ 问题1：课程是我"想教"，不是用户"想学"
❌ 问题2：定价199太低，反而没人买（太便宜=质量差）
❌ 问题3：没有信任基础，没人知道我是谁
```

我调整策略：

```
✅ 重新选题：从"爬虫"改为"Python自动化办公"（用户痛点更大）
✅ 定价调到599（提高感知价值）
✅ 先写150篇技术文章，建立信任
✅ 在文章末尾自然引流到课程
```

**调整后的结果：**

| 课程 | 定价 | 销量 | 总收入 |
|------|------|------|--------|
| Python爬虫课（第一次） | ¥199 | 7份 | ¥1,393 |
| Python自动化办公课（改进后） | ¥599 | 5,000+份 | ¥300万+ |

**同一个我，不同的策略，收入差了2000倍。**

**这就是课程变现的威力——做对了，一人公司也能赚大钱。**

## 📊 课程变现的4条路径

### 路径1：录播课（最适合一人公司）

**为什么推荐录播课？**
- 录一次，卖无数次（边际成本趋近于0）
- 不需要协调时间，用户随时学
- 你不在也能赚钱（真正的"睡后收入"）

**平台对比：**

| 平台 | 价格 | 优势 | 劣势 |
|------|------|------|------|
| 小鹅通 | ¥3999/年起 | 功能最全，分销体系好 | 贵 |
| 知识星球 | ¥0（付费圈） | 社群+内容一体 | 课程功能弱 |
| 网易云课堂 | 免费（按比例分成） | 平台自带流量 | 分成高（30-50%） |
| Udemy | 免费（按比例分成） | 全球流量 | 价格战严重 |
| 自建网站 | 服务器¥100/年 | 自主可控 | 需要技术能力 |

**我的建议：** 起步用免费平台（网易云课堂/知识星球），收入稳定后迁移到自有平台。

**收入预期：**

| 阶段 | 课程数 | 单课均价 | 月销量 | 月收入 |
|------|--------|---------|--------|--------|
| 起步期 | 1-2 | ¥99-199 | 10-50 | ¥1,000-10,000 |
| 成长期 | 3-5 | ¥299-599 | 50-200 | ¥15,000-120,000 |
| 成熟期 | 5+ | ¥599-1999 | 100-500 | ¥60,000-1,000,000 |

### 路径2：直播课/训练营

**和录播课的区别：**

| 维度 | 录播课 | 直播训练营 |
|------|--------|-----------|
| 开发成本 | 低（录一次） | 高（每次都要讲） |
| 互动性 | 弱 | 强 |
| 价格天花板 | ¥599-1999 | ¥999-9999 |
| 时间投入 | 一次投入，持续收益 | 持续投入 |
| 适合 | 起步期、规模化 | 高客单价、强服务 |

**训练营的典型玩法：**
```
周期：21天 / 30天 / 90天
定价：¥999-9999
模式：每天直播/录播 + 作业打卡 + 社群答疑
核心卖点：不是内容（录播也有），是"陪伴+监督+反馈"
```

**我的经验：** 训练营的完课率和满意度远高于录播课。但你也更累。

### 路径3：1对1咨询

**适合场景：**
- 你是某个领域的专家
- 问题比较个性化，不能标准化
- 用户愿意为"定制"付费

**定价参考：**

| 你的背景 | 新手价 | 成熟价 |
|----------|--------|--------|
| 3年以下经验 | ¥200-500/小时 | ¥500-1000/小时 |
| 3-5年经验 | ¥500-1000/小时 | ¥1000-2000/小时 |
| 5年以上/有影响力 | ¥1000-3000/小时 | ¥3000+/小时 |

**关键技巧：** 咨询不是"聊天"，要有明确的交付物：
```
✅ 一次1小时咨询 + 1份咨询报告（含具体建议）
✅ 不是按时间收费，是按"价值"收费
```

### 路径4：企业内训

**这是很多人忽略的高利润路线。**

**企业为什么找你？**
- 内部培训预算充足（一场1-5万）
- 需要外部专家的视角
- 一个人公司反而灵活（响应快、定制强）

**企业内训的定价：**

| 你的知名度 | 半天（3h） | 全天（6h） |
|-----------|-----------|-----------|
| 不知名 | ¥5,000-10,000 | ¥10,000-20,000 |
| 有一定影响力 | ¥10,000-30,000 | ¥20,000-50,000 |
| 行业知名 | ¥30,000-80,000 | ¥50,000-150,000 |

**怎么接到企业内训？**
1. 在公众号/知乎持续输出专业内容
2. 主动联系目标企业HR（linkedin/脉脉）
3. 让现有学员帮你推荐（转介绍最有效）

## 🤖 AI如何帮你做课程？

### 课程开发的完整流程（AI辅助版）

```
第1步：选题验证（1-2天）
├── 用AI搜索分析市场热度
├── 用AI分析竞品课程的优缺点
├── 在社群/朋友圈做调研
└── 确定选题 + 目标学员画像

第2步：大纲设计（半天）
├── 告诉AI你的主题和学员
├── AI生成课程大纲
├── 你调整结构（加入个人经验）
└── 确定每讲的标题和目标

第3步：内容制作（1-3周）
├── 每讲先写文字稿（AI辅助）
├── 制作PPT/演示文稿
├── 录制讲解（OBS录屏 / 手机拍摄）
├── AI辅助剪辑（剪映）
└── 每讲控制在15-30分钟

第4步：包装上线（3-5天）
├── 设计课程封面（AI生成）
├── 写课程介绍页（AI辅助）
├── 上传到平台
└── 设置价格和优惠

第5步：推广转化（持续）
├── 写引流文章（AI辅助）
├── 制作推广视频
├── 社群运营
└── 学员转介绍机制
```

### AI提示词模板

**选题验证：**
```
我想做一个 [主题] 的在线课程。
帮我分析：
1. 这个方向的市场热度如何？
2. 目标学员是谁？痛点是什么？
3. 市面上有哪些竞品课程？优缺点是什么？
4. 建议的课程定位和差异化方向
```

**大纲生成：**
```
我要做一个 [主题] 课程，目标学员是 [人群]。
课程目标：学完后能 [具体能力]。
请帮我设计一个15讲的课程大纲，要求：
1. 从零基础到实战
2. 每讲有明确的标题和学习目标
3. 最后一讲是综合实战项目
```

**内容初稿：**
```
请根据以下大纲写第 [N] 讲的详细文字稿：
标题：[标题]
目标学员：[人群]
学习目标：[目标]
风格：口语化、有案例、有代码示例

要求：
- 开篇用场景引入
- 正文讲清楚原理和实操
- 结尾有总结和作业
```

## 📈 定价策略：为什么不是越便宜越好？

**这是90%的新手都搞错的地方。**

**常见误区：**
```
❌ "我刚开始做，定价低一点吧" → 199、99甚至免费
❌ "便宜点，买家才多" → 越便宜越好
❌ "等有口碑了再涨价" → 结果一直涨不上去
```

**定价的心理学原理：**
```
¥99  → "才99？估计内容一般"
¥199 → "200不到，可以试试"
¥399 → "这个价格有点内容了"
¥599 → "有点贵，但应该是好课"
¥999 → "贵，但应该很专业"
¥2999 → "高端课，得好好研究"
```

**我的定价建议：**

| 阶段 | 建议定价 | 原因 |
|------|---------|------|
| 第一个课 | ¥199-399 | 积累口碑，降低决策门槛 |
| 第二三个课 | ¥399-599 | 有口碑后，适当提价 |
| 第四个课+ | ¥599-1999 | 品牌成熟，追求利润 |

**关键原则：** 定价定的是你的品牌位置，不是内容的"实际价值"。

## ⚠️ 新手做课最容易踩的坑

| 坑 | 表现 | 避坑方法 |
|------|------|---------|
| 闭门造车 | 不调研就开做 | 先问目标学员要什么 |
| 内容太长 | 一讲60分钟 | 控制在15-30分钟 |
| 追求完美 | 录了3个月还没上线 | 先上线再迭代 |
| 定价太低 | 99块的课没人买 | 起步199起步 |
| 只卖不服务 | 卖完就不管 | 至少提供社群答疑 |
| 没有引流 | 做了课没人知道 | 内容先行，课是变现工具 |
