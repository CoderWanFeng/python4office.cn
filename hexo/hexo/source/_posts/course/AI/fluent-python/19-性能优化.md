## 🔍 Profiling 工具

### cProfile

```python
import cProfile
import pstats

# 代码性能分析
def slow_function():
    result = []
    for i in range(10000):
        result.append(i ** 2)
    return result

# 方法 1：命令行
# python -m cProfile -s cumulative script.py

# 方法 2：代码内
profiler = cProfile.Profile()
profiler.enable()
slow_function()
profiler.disable()
stats = pstats.Stats(profiler)
stats.sort_stats('cumulative').print_stats(10)
```

### line_profiler

```bash
# 安装
pip install line_profiler

# 使用
kernprof -l -v script.py
```

## ⚡ 优化技巧

| 技巧 | 效果 |
|------|------|
| 使用内置函数 | C 实现更快 |
| 列表推导式 | 比循环快 |
| 局部变量查找 | 避免全局查找 |
| 使用__slots__ | 减少内存占用 |
| C 扩展 | 突破性能瓶颈 |

```python
# 1. 使用内置函数（C 实现）
result = sum(range(1000))  # 快

# 2. 列表推导式代替循环
squares = [x**2 for x in range(1000)]  # 快

# 3. 局部变量查找更快
def fast():
    local_append = result.append  # 缓存方法
    for i in range(1000):
        local_append(i)
```
