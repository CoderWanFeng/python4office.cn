# 《流畅的 Python》共读课销售方案

**目标**：首发文章即成交，3个月内完成第一批 200-500 名付费学员
**预算**：几乎为零，纯内容+私域驱动
**核心逻辑**：引流 → 信任 → 转化 → 交付验证 → 口碑复购

---

## 一、产品分层：不是一门课，是一套产品线

先把课程拆成三层，不同层次承担不同任务：

```
┌─────────────────────────────────────────────┐
│  第1层：引流产品（免费 / 极低价）              │
│  任务：获取潜在用户联系方式                    │
│  交付：《Python 进阶知识点速查表》PDF          │
│  价格：免费，或 9.9 元（筛选意愿用户）          │
└─────────────────────────────────────────────┘
              ↓ 领取后进入私域微信
┌─────────────────────────────────────────────┐
│  第2层：核心产品（试运营期优惠）                │
│  任务：直接成交                              │
│  交付：20讲直播共读 + 社群陪伴 + 回放           │
│  价格：试运营 299 元（原价 499 元）            │
└─────────────────────────────────────────────┘
              ↓ 学完后自然产生
┌─────────────────────────────────────────────┐
│  第3层：口碑复购（未来）                       │
│  任务：拉新 + 提升 LTV                        │
│  交付：《Fluent Python》第2版+第3本共读        │
│  价格：待定                                  │
└─────────────────────────────────────────────┘
```

**关键设计**：第一层是门票，不是替代品。很多人领了免费资料就跑了——没关系，他已经在你的微信里了，以后还会看到你的内容。

## 二、引流节奏：四篇文章，分两周发出

### 第1周（引流 + 建信任）

| 发布时间 | 文章 | 任务 |
|---------|------|------|
| 第1天 | `260408-lead.md`（首发文） | 建立认知：戳"你以为你懂了其实没有"，获取种子用户 |
| 第3天 | `260408-badhabits.md`（避坑文） | 深化信任：展示具体知识内容，让人想学 |
| 第5天 | `260408-interview.md`（面试文） | 精准引流：直接戳面试痛点，筛选高意愿用户 |

### 第2周（成交）

| 发布时间 | 文章 | 任务 |
|---------|------|------|
| 第8天 | `260408-community.md`（社群文） | 收网转化：展示"有人陪"的价值，发布限时优惠 |
| 第9-10天 | **限量优惠截止** | 制造紧迫感 |
| 第10天后 | 每日学员案例分享 | 社会证明维持热度 |

## 三、首发文章特别改造（最重要的一步）

`260408-lead.md` 需要加一个**行动入口**，让它不只是"看完就走"的公众号文章：

### 在文章结尾增加这个模块：

```markdown
---

### 📋 限时福利：免费领取《Python 进阶知识点速查表》

读到这里的你，如果感觉自己 Python 基础还行，但底层逻辑一直没打通——
我为你准备了一份**《Python 进阶知识点速查表》**，覆盖：

- 数据模型 & 特殊方法的调用链路
- 描述符协议与 @property 底层关系
- 生成器与迭代器的执行时机图解
- Python 装饰器的完整执行流程
- GIL 释放的 4 种场景

**免费领取方式**：
1. 搜索微信号 `aiwf365` 添加
2. 备注「**速查表**」
3. 我会直接发给你

> 👉 限量 300 份，领完即止

---

**如果这篇文章对你有触动，想系统学完这本书**——
我正在做《流畅的 Python》直播共读课，原价 499 元，试运营期间 **299 元**，报名方式同上。
```

**为什么这样设计**：
1. 免费福利在前，降低读者"什么都没得到就走"的损失感
2. 领取福利需要加微信——这是最重要的转化动作
3. 加完微信后，看到课程信息，**转化才算真正发生**
4. 限量 300 份是心理锚，不是真的限量

## 四、微信私域转化：接住流量

读者加了微信之后，如果没人理，80% 会沉默流失。设计一套自动化的接住流程：

### 标准欢迎语（自动回复）

```
👋 你好！我是晚枫，《流畅的 Python》课程作者。

你已经领取了《Python 进阶知识点速查表》，我稍后会发给你。

关于共读课，简单说几句：
- 共读周期：约 8-10 周，每周 1-2 次直播精讲
- 试运营价格：299 元（原价 499 元）
- 适合人群：有 1-3 年 Python 经验，想打通底层逻辑

如果你感兴趣，我可以简单介绍一下课程的节奏和内容，
或者你也可以先看看第一讲的免费公开课，再决定。

有其他问题随时问我 👍
```

### 分层运营策略

```
潜在用户（领了资料但没买课）
├── 每天下午 6 点推送：当日课程相关知识点
├── 每周一次：学员学习进度分享
└── 有问必答：建立信任感

高意愿用户（有明确购买意向）
├── 直接发送课程详细介绍 + 报名链接
├── 说明"前 N 名优惠"（紧迫感）
└── 如果 24 小时内没回复，再次触达一次

沉默用户（3 天无互动）
└── 3 天后发一篇历史好文链接（不是广告）
```

## 五、定价策略：不是降价，是锚定

| 定价策略 | 说明 |
|---------|------|
| **原价锚定** | 页面始终显示"原价 499 元，试运营 299 元"，让读者感知到价值 |
| **早鸟名额** | 前 50 名报名额外赠送《考点速查表》+ 晚枫 1 对 1 答疑 1 次 |
| **限时窗口** | 首发期结束后恢复原价 499 元，不做长期低价 |

**注意**：不要做"永久折扣"，一旦价格锚定在 299 元，以后涨到 499 元会引发已购用户反感。

## 六、紧迫感设计：让读者现在就行动

这是很多个人博主最容易忽略的一步。内容再好，没有紧迫感，读者会想"先收藏，等有空再说"——然后就没有然后了。

### 三层紧迫感叠加

**第一层（文章内）**：
- "限量 300 份速查表，领完即止"
- "前 50 名报名额外赠送……"

**第二层（社群内）**：
- 每日发布已报名人数截图："今天是第 37 位同学加入"
- 让已报名学员截图发朋友圈，形成外部传播

**第三层（私信内）**：
- "活动本周五截止，之后恢复原价"
- "如果名额满了，你也可以先进群等下一期"

## 七、社会证明：让观望的人看到别人已经在做了

首发期间最缺的就是社会证明。以下内容需要**提前准备**：

### 必须提前做的一件事

**在正式发布文章之前，先招募 20-30 名内测学员**：

- 价格降到 99 元或免费，换取真实反馈
- 要求内测学员完成至少 3 讲学习后，截图发朋友圈或推荐给朋友
- 用内测学员的真实反馈作为首发文章的社会证明

**内测学员反馈模板（引导收集）**：

```
"第 X 讲讲到的 XXX 内容，
我之前一直用 XXX（错误方式），
学完之后终于理解了 XXX（正确逻辑）。

建议 XXX（人群）的人来听。"
```

### 每日社会证明内容库

提前准备 7-10 条学员反馈/截图，发文章期间每天放一条：

```
Day 1："今天已经有 23 位同学加入了，我们第 2 讲直播见 👏"
Day 2："昨天报名的@小李说：'听了第一讲，数据模型那章以前全靠背，现在终于理解了'"
Day 3："本周第 3 讲的直播回放已生成，报名后可无限次回看 📹"
...
```

## 八、复购与裂变：把学员变成推广者

### 学员推广机制

学员完成课程后，设计一个"推荐返现"机制：

```
推荐 1 位朋友报名 → 返现 30 元
推荐 3 位朋友报名 → 返现 100 元 + 晚枫一对一代码review 1次
```

这不是分销（分销在中国有法律风险），是**学员推荐奖励**——合规且有温度。

### 裂变内容自动流

每学完一讲，学员在社群里分享笔记 → 截图发朋友圈 → 吸引新的观望者。

这个循环跑通之后，招生会越来越轻松。

## 十、关键指标：每天盯这三个数字

| 指标 | 目标（首发期） | 含义 |
|------|-------------|------|
| 文章阅读量 | 1000+ | 基本传播量，低于 500 说明标题/选题有问题 |
| 加微信人数 | 阅读量的 3-5% | 正常转化率，低于 3% 说明引流入口不够突出 |
| 微信→付费转化 | 10-15% | 私域转化率，低于 10% 说明信任不够或价格太高 |

**如果前两项达标、第三项不达标**：优先提升信任（多发学员反馈、少发广告感内容）。
**如果前两项不达标**：回头改标题和文章开头，而不是继续写更多内容。

---

## 十一、总结：核心原则

1. **不要一次性把所有文章都发了**——每篇之间留 2-3 天，让每篇都有独立热度
2. **加微信是所有转化的入口**——不管领资料还是买课，第一步都是加微信
3. **限量不等于欺骗**——300 份速查表是真的限量，50 个早鸟名额也是真的先到先得
4. **信任是累计的**——内测学员的 20 条真实反馈，比你自己写 200 字推荐语有用 100 倍
5. **不要在文章里直接挂购买链接**——微信生态不支持，正确的路径是：文章 → 加微信 → 私信成交
