## 批量处理 Excel 文件

### 场景 1：合并多个 Excel 文件
**问题**：有 50 个销售员的 Excel 文件，需要合并成一个

**用集简云配置**：
1. 打开 集简云（jianjiyun.com）
2. 创建新流程
3. 触发器："文件夹中有新文件"
4. 动作："合并 Excel 文件"
5. 配置参数，保存

**提示词**：
```
帮我配置一个自动化流程：
1. 监控"销售数据"文件夹
2. 当有新 Excel 文件时自动读取
3. 合并所有文件到一个表格
4. 添加"来源文件"列标识数据来源
5. 保存为"汇总.xlsx"
```

### 场景 2：批量修改格式
**提示词**（用 AI 生成 Python 代码）：
```
你是一名 Python 专家。
我需要批量处理 Excel 文件：
1. 读取指定文件夹下所有.xlsx 文件
2. 统一格式：字体微软雅黑、字号 11、表头加粗
3. 自动调整列宽
4. 保存覆盖原文件
请帮我生成 Python 代码。
```

## 自动抓取网络数据

### 抓取竞争对手价格
**提示词**：
```
帮我设计一个自动化流程：
1. 每天上午 9 点自动抓取指定网页
2. 提取产品名称和价格
3. 保存到 Excel 文件
4. 如果价格变化超过 10%，发送邮件提醒
```

### 抓取汇率/股票数据
**提示词**：
```
帮我设计一个自动化流程：
1. 每小时抓取一次美元/人民币汇率
2. 记录到 Excel 文件，包含时间戳和汇率
3. 生成趋势图表
```

## 自动发送邮件报表

### 每周自动发送销售报表
**用集简云配置**：
1. 触发器："定时触发"（每周一 9 点）
2. 动作："读取 Excel 数据"
3. 动作："AI 生成邮件内容"
4. 动作："发送邮件"

**邮件内容提示词**：
```
你是一名专业助理。
根据以下销售数据，写一份周报邮件：
- 本周总销售额：[数据]
- 环比增长：[数据]
- Top3 产品：[数据]
- 需要关注的问题：[数据]
要求：语气专业简洁，包含关键数据，提出建议，字数 300 字左右。
```
