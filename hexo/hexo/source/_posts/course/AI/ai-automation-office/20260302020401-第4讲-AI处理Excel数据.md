## 常见的脏数据问题

### 问题 1：格式不统一
```
❌ 原始：张三 10000 / 李四，20000 / 王五  30000
✅ 清洗后：张三，10000 / 李四，20000 / 王五，30000
```

### 问题 2：缺失值
```
❌ 原始：张三，10000 / 李四，/ 王五，30000
✅ 清洗后：张三，10000 / 李四，0 / 王五，30000
```

### 问题 3：重复数据
```
❌ 原始：张三，10000 / 张三，10000 / 李四，20000
✅ 清洗后：张三，10000 / 李四，20000
```

## 用 AI 清洗数据

### 方法 1：用 ChatExcel
1. 打开 chatexcel.com
2. 上传 Excel 文件
3. 输入指令："帮我清洗数据，删除重复行，填充缺失值为 0"
4. 下载清洗后的文件

### 方法 2：用 Claude/ChatGPT
1. 复制数据到 AI 对话框
2. 输入清洗指令
3. AI 返回清洗后的数据
4. 复制回 Excel

**提示词**：
```
我有以下 CSV 数据：[粘贴数据]
请帮我：
1. 删除重复行
2. 填充缺失值为 0
3. 统一格式
4. 输出 CSV 格式
```

## AI 生成 Excel 公式

### VLOOKUP 查询
**提示词**：
```
你是一名 Excel 专家。
我有两个表：表 1（A 列姓名，B 列销售额），表 2（A 列姓名，需要查询销售额）。
请帮我写一个 VLOOKUP 公式，从表 1 查询销售额到表 2。
```

**AI 返回**：
```
在表 2 的 B2 单元格输入：
=VLOOKUP(A2, 表 1!A:B, 2, FALSE)
```

### SUMIF 条件求和
**提示词**：
```
你是一名 Excel 专家。
A 列是产品名称，B 列是销售额。需要计算每个产品的总销售额。
请帮我写一个 SUMIF 公式。
```

**AI 返回**：
```
在 D 列列出产品名称，在 E 列计算：
=SUMIF(A:A, D2, B:B)
```

### IF 条件判断
**提示词**：
```
你是一名 Excel 专家。
A 列是销售额。销售额>=10000 显示"达标"，否则显示"未达标"。
请帮我写一个 IF 公式。
```

**AI 返回**：
```
在 B2 单元格输入：
=IF(A2>=10000, "达标", "未达标")
```
