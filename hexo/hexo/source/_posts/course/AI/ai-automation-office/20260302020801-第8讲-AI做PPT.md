## 用 Gamma 生成 PPT

### 步骤
1. 打开 gamma.app，注册账号
2. 点击"Create New" → "Generate"
3. 输入 PPT 主题和提示词
4. AI 生成 PPT 大纲，确认后生成完整 PPT
5. 在线编辑和调整
6. 导出为 PDF/PPTX 或生成分享链接

### 提示词示例（工作总结）
```
帮我生成一个季度工作总结 PPT，包含以下内容：
1. 封面：2026 年 Q1 工作总结
2. 关键业绩：销售额 500 万，同比增长 20%
3. 亮点工作：签下 3 个大客户、新产品上线
4. 问题分析：人手不足、客户需求变更频繁
5. 下季度计划：销售额目标 600 万、团队扩充
要求：专业商务风格，8-10 页，每页有标题和要点，包含图表建议。
```

### 提示词示例（产品介绍）
```
帮我生成一个产品介绍 PPT，包含以下内容：
1. 封面：XX 产品介绍
2. 产品定位：面向中小企业的 CRM 系统
3. 核心功能：客户管理、销售跟进、数据分析
4. 产品优势：易用、便宜、功能全
5. 价格方案：基础版 99 元/月，专业版 199 元/月
6. 客户案例：XX 公司使用后销售额提升 30%
7. 联系方式
要求：现代简洁风格，10-12 页，多用图表和图示。
```

## 用 WPS AI 生成 PPT

### 步骤
1. 打开 WPS Office
2. 新建演示文稿
3. 点击"AI 写作"按钮
4. 选择"生成 PPT"
5. 输入要求，AI 生成 PPT
6. 在线编辑内容，调整模板和配色
7. 保存为.pptx 文件或导出为 PDF

### 提示词示例
```
生成一份月度销售汇报 PPT：
- 包含销售数据、环比分析、下月计划
- 10 页左右
- 商务风格
```

## 用 AI 美化 PPT

### 优化现有 PPT
**提示词**：
```
我有以下 PPT 内容：[粘贴内容]
请帮我重新设计：
- 现代商务风格
- 统一配色
- 每页布局优化
- 添加合适的图标和图示
```

### 生成图表
**提示词**：
```
我有以下销售数据：
1 月：100 万 / 2 月：120 万 / 3 月：90 万 / 4 月：150 万 / 5 月：180 万 / 6 月：200 万
请帮我生成一个柱状图，展示月度销售趋势。配色要专业，适合商务汇报。
```

### 生成图示
**提示词**：
```
帮我生成一个流程图，展示销售流程：
1. 客户咨询 → 2. 需求分析 → 3. 方案报价 → 4. 合同签订 → 5. 项目实施 → 6. 售后服务
要求：横向流程图，每步有图标，配色专业。
```

## PPT 内容优化

### 用 AI 优化文字
**提示词**：
```
请帮我优化以下 PPT 文字，使其更简洁有力：

原文字："我们公司在今年第一季度取得了非常不错的销售业绩，总共完成了 500 万元的销售额，相比去年同期增长了 20% 左右，这是一个非常令人鼓舞的成绩。"

要求：精简到 50 字以内，突出关键数据，适合 PPT 展示。
```

**AI 返回**：
```
2026 Q1 销售业绩
• 销售额：500 万元
• 同比增长：20%
• 创历史新高
```

### 用 AI 生成演讲备注
**提示词**：
```
我有以下 PPT 页面内容：[粘贴 PPT 内容]
请帮我生成演讲备注：
- 每页 100 字左右
- 口语化表达
- 包含过渡语句
- 标注重点强调内容
```
