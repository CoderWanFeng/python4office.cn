---

## 提示词万能公式

**公式：角色 + 任务 + 要求 + 格式**

### 示例 1：写邮件
```
角色：你是一名专业的人力资源专员
任务：写一封面试邀请邮件
要求：语气友好、包含面试时间地点
格式：邮件格式，包含主题和正文
```

### 示例 2：做 Excel 公式
```
角色：你是一名 Excel 专家
任务：帮我写一个 VLOOKUP 公式
要求：从 Sheet2 的 A 列查找姓名，返回 B 列的工资
格式：直接给出公式，并解释每个参数
```

### 示例 3：生成 PPT 大纲
```
角色：你是一名专业的 PPT 设计师
任务：为"季度工作总结"生成 PPT 大纲
要求：包含 5-8 页，每页有标题和要点
格式：列表格式，标注每页的核心内容
```

## 办公场景提示词模板

### 写工作报告
```
你是一名 [职位]。
我需要写一份 [报告类型]。
报告包含以下内容：[列出要点]。
语气要 [正式/轻松/专业]，字数 [X] 字左右。
```

### 写邮件
```
你是一名 [职位]。
我需要给 [收件人] 写一封邮件。
目的是：[邮件目的]。
要包含：[关键信息]。
语气要 [正式/友好/紧急]。
```

### 做 PPT
```
你是一名 PPT 设计师。
我需要做一个关于 [主题] 的 PPT。
听众是：[目标听众]。
包含以下内容：[列出要点]。
请生成：PPT 大纲，每页的标题和要点。
```

## 提示词优化技巧

### 技巧 1：给 AI 角色
❌ 差：写一份报告
✅ 好：你是一名资深财务分析师，写一份财务分析报告

### 技巧 2：提供背景信息
❌ 差：帮我分析数据
✅ 好：这是 Q2 销售数据，需要向管理层汇报，找出增长点和问题

### 技巧 3：指定输出格式
❌ 差：给我一些建议
✅ 好：用列表格式输出 5 条可执行建议，每条不超过 50 字
