## 用 AI 写工作报告

### 周报模板
**提示词**：
```
你是一名专业助理。
我需要写一份周报，包含以下内容：

本周工作：
1. 完成 A 项目第一阶段开发
2. 与客户 B 开会确认需求
3. 修复 5 个线上 bug

下周计划：
1. 开始 A 项目第二阶段
2. 跟进客户 C 的合同
3. 优化系统性能

遇到的问题：
1. 人手不足，进度有延迟
2. 客户需求变更频繁

请帮我写一份专业的周报，语气正式，字数 500 字左右。
```

### 月报模板
**提示词**：
```
你是一名专业助理。
我需要写一份月度工作报告。

本月关键数据：
- 销售额：500 万，环比增长 15%
- 新客户：20 个
- 项目交付：5 个

亮点工作：
1. 签下行业大客户 A
2. 新产品上线，反馈良好
3. 团队扩充 3 人

下月目标：
- 销售额目标：600 万
- 新客户：25 个
- 完成 B 项目交付

请帮我写一份专业的月报，包含数据分析和展望，字数 800 字左右。
```

## 用 AI 生成合同模板

### 服务合同模板
**提示词**：
```
你是一名专业律师。
我需要一份技术服务合同模板。
合同包含以下内容：
1. 合同双方信息（甲方、乙方）
2. 服务内容
3. 服务费用及支付方式
4. 服务期限
5. 双方权利义务
6. 保密条款
7. 违约责任
8. 争议解决
请生成一份规范的技术服务合同，条款完整，用词专业。
```

### 保密协议模板
**提示词**：
```
你是一名专业律师。
我需要一份保密协议模板。
协议包含：
1. 保密信息的定义
2. 保密义务
3. 保密期限
4. 违约责任
5. 协议生效
请生成一份规范的保密协议，条款简洁明了。
```

## 批量生成文档

### 批量生成报价单
**提示词**：
```
你是一名专业助理。
我有以下客户数据：[粘贴 Excel 数据]
请帮我为每个客户生成一份报价单，包含：
- 客户名称
- 产品信息
- 价格
- 有效期
- 联系方式
输出格式：每个客户的报价单内容，方便我复制到 Word。
```

### 批量生成邀请函
**提示词**：
```
你是一名专业助理。
我需要为 50 个客户生成会议邀请函。

会议信息：
- 会议名称：2026 产品发布会
- 时间：2026 年 3 月 15 日 14:00
- 地点：北京市朝阳区 XX 酒店
- 联系人：张三 138xxxxxxx

客户数据：[粘贴 Excel 数据]
请为每个客户生成一份邀请函，语气正式友好，每份 200 字左右。
```

## 用 WPS AI 写文档

### WPS AI 功能
1. **智能写作**：输入主题，自动生成文档
2. **智能润色**：优化语言表达，修正语法错误
3. **智能续写**：根据上下文自动续写

### 操作步骤
1. 打开 WPS
2. 新建文档
3. 点击"AI 写作"按钮
4. 输入写作要求
5. AI 生成内容
6. 编辑和保存
