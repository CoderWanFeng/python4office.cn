## 批量生成文档

### 场景 1：批量生成合同
**问题**：有 50 个客户需要签合同

**提示词**：
```
你是一名专业助理。
我有以下客户数据：[粘贴 Excel 数据]
合同模板：[粘贴合同模板]
请帮我为每个客户生成一份合同：
- 替换客户信息
- 替换合同金额
- 替换服务期限
- 输出为独立文档
输出格式：每个合同的完整内容，标注文件名。
```

### 场景 2：批量生成报价单
**提示词**：
```
你是一名专业助理。
我有以下产品报价数据：[粘贴 Excel 数据]
报价单模板包含：
- 客户名称
- 产品列表（名称、数量、单价、小计）
- 总计金额
- 报价有效期
- 联系方式
请为每个客户生成一份报价单，格式规范，方便复制到 Word。
```

### 场景 3：批量生成邀请函
**提示词**：
```
你是一名专业助理。
我需要为 50 个客户生成会议邀请函。
会议信息：
- 会议名称：2026 产品发布会
- 时间：2026 年 3 月 15 日 14:00
- 地点：北京市朝阳区 XX 酒店
- 联系人：张三 138xxxxxxx
客户数据：[粘贴 Excel 数据]
请为每个客户生成一份邀请函，语气正式友好，每份 200 字左右。
```

## 文档模板管理

### 创建模板库
```
文档模板库/
├── 合同模板/
│   ├── 技术服务合同.docx
│   ├── 劳动合同.docx
│   └── 保密协议.docx
├── 报告模板/
│   ├── 周报模板.docx
│   ├── 月报模板.docx
│   └── 述职报告模板.docx
├── 商务文档/
│   ├── 报价单模板.docx
│   ├── 邀请函模板.docx
│   └── 通知模板.docx
└── PPT 模板/
    ├── 工作总结 PPT.pptx
    ├── 产品介绍 PPT.pptx
    └── 融资路演 PPT.pptx
```

### 用 AI 优化模板
**提示词**：
```
你是一名专业文档设计师。
我有以下文档模板：[粘贴模板内容]
请帮我优化：
1. 格式规范化
2. 语言专业化
3. 添加占位符（如 [客户名称]、[日期]）
4. 添加使用说明
输出优化后的模板。
```

## 自动归档和命名

### 文件命名规范
**建议格式**：`日期_文档类型_客户名称_版本.docx`
**示例**：
- 20260302_合同_XX 公司_v1.docx
- 20260302_周报_销售部_2026W09.docx
- 20260302_报价单_XX 公司_v2.docx

### 用 AI 生成文件名
**提示词**：
```
你是一名专业助理。
请根据以下信息生成规范的文件名：
文档类型：合同
客户名称：北京 XX 科技有限公司
日期：2026 年 3 月 2 日
版本：第 1 版
文件名格式：日期_文档类型_客户名称_版本.docx
```

**AI 返回**：
```
20260302_合同_北京 XX 科技有限公司_v1.docx
```

### 自动归档流程
**用集简云配置**：
1. 触发器："新文件保存到文件夹"
2. 动作："重命名文件"
3. 动作："移动到归档文件夹"

**提示词**：
```
帮我设计一个文档自动归档流程：
1. 监控"待归档"文件夹
2. 当有新文件时：
   - 读取文件内容
   - 识别文档类型和客户名称
   - 生成规范文件名
   - 重命名文件
   - 移动到对应归档文件夹
归档文件夹结构：合同/2026/03/、报告/2026/03/、报价单/2026/03/
请给出详细的流程配置步骤。
```

## 文档搜索和检索

### 建立文档索引
**用 Excel 建立索引**：

| 文件名 | 文档类型 | 客户名称 | 日期 | 存放位置 | 备注 |
|--------|----------|----------|------|----------|------|
| 20260302_合同_XX 公司_v1.docx | 合同 | XX 公司 | 2026-03-02 | 合同/2026/03/ | 技术服务 |

### 用 AI 生成索引
**提示词**：
```
你是一名专业助理。
我有以下文档列表：[列出所有文档]
请帮我生成一个 Excel 索引表格，包含：
- 文件名
- 文档类型
- 客户名称
- 日期
- 存放位置
- 备注
输出 CSV 格式，方便我导入 Excel。
```
