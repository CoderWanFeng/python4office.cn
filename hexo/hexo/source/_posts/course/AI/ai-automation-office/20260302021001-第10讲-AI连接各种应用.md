- 官网：jianjiyun.com

### 场景 1：钉钉审批自动记录
**流程**：钉钉新审批 → 提取审批数据 → 添加到 Excel

**提示词**：
```
帮我配置一个自动化流程：
触发：钉钉有新审批
动作：将审批数据添加到 Excel
需要提取的字段：审批类型、申请人、申请时间、审批金额、审批状态
Excel 表格结构：A 列审批类型、B 列申请人、C 列申请时间、D 列审批金额、E 列审批状态
请给出详细的配置步骤。
```

### 场景 2：飞书消息自动创建待办
**流程**：飞书收到消息 → 判断是否包含关键词 → 创建待办事项 → 发送确认消息

**提示词**：
```
帮我配置一个自动化流程：
触发：飞书收到消息
条件：消息包含"待办"或"任务"关键词
动作 1：在飞书中创建待办事项（标题：消息内容，截止时间：明天 18:00）
动作 2：发送确认消息给发送者（内容："已为您创建待办：[消息内容]"）
请给出详细的配置步骤。
```

### 场景 3：企业微信客户自动记录
**流程**：企业微信收到消息 → 判断是否为新客户 → 提取客户信息 → 添加到 CRM → 发送欢迎消息

**提示词**：
```
帮我配置一个自动化流程：
触发：企业微信收到外部联系人消息
动作 1：在 CRM 中创建客户记录（姓名：从企业微信获取，来源：企业微信，首次咨询时间：当前时间）
动作 2：发送欢迎消息（内容："您好，感谢咨询！我们会尽快回复您。"）
请给出详细的配置步骤。
```

### 场景 4：钉钉考勤数据自动同步到 Excel 工资表
**流程**：定时触发（每天 18 点） → 获取钉钉考勤数据 → 计算出勤天数 → 更新 Excel 工资表

**提示词**：
```
帮我配置一个自动化流程：
触发：每天 18:00 定时执行
动作 1：获取钉钉考勤数据（时间范围：当天，字段：姓名、部门、上班时间、下班时间、状态）
动作 2：计算出勤天数（正常：计 1 天，迟到/早退：计 0.5 天，缺勤：计 0 天）
动作 3：更新 Excel 工资表（匹配字段：姓名，更新字段：出勤天数）
请给出详细的配置步骤。
```
