## 常见报表类型

### 销售报表
- 本月销售额、环比增长、同比增长
- Top10 产品/销售
- 区域分布

### 财务报表
- 收入汇总、支出明细
- 利润计算、现金流分析

### 运营报表
- 用户数据、活跃度和留存
- 转化漏斗、渠道分析

---

## 用 AI 自动汇总数据

### 用 ChatExcel
**提示词**：
```
帮我生成一份月度销售报表：
1. 计算本月总销售额
2. 计算环比增长（与上月对比）
3. 计算同比增长（与去年同月对比）
4. 列出 Top10 产品
5. 按区域汇总销售额
6. 生成汇总表格
```

### 用 Claude/ChatGPT
**提示词**：
```
我有以下销售数据（1-12 月）：[粘贴数据]
请帮我：
1. 计算全年总销售额
2. 计算月平均销售额
3. 找出销售最高的月份
4. 计算下半年比上半年的增长率
5. 输出汇总表格
```

---

## AI 生成图表

### 柱状图（对比数据）
**提示词**：
```
我有以下数据：
1 月：100 万 / 2 月：120 万 / 3 月：90 万 / 4 月：150 万 / 5 月：180 万 / 6 月：200 万
请帮我生成一个柱状图，展示月度销售趋势。
```

### 折线图（趋势分析）
**提示词**：
```
我有 2025 年和 2026 年各月销售数据：[粘贴数据]
请帮我生成一个折线图，两条线分别代表两年，便于对比趋势。
```

### 饼图（占比分析）
**提示词**：
```
我有以下产品销售数据：
产品 A：30% / 产品 B：25% / 产品 C：20% / 产品 D：15% / 其他：10%
请帮我生成一个饼图，展示产品销售占比。
```

## AI 做数据透视表

**提示词**：
```
你是一名 Excel 专家。
我有以下数据：A 列日期、B 列产品名称、C 列销售员、D 列销售额、E 列区域。
请帮我创建一个数据透视表：
- 行：产品名称
- 列：区域
- 值：销售额（求和）
- 筛选器：销售员
```

## 本讲作业

1. 准备一份原始数据

3. 用 AI 制作 3 种图表（柱状图、折线图、饼图）
4. 创建一个数据透视表
