## 1. 大小写转换（3个）

```python
text = "Hello World"

print(text.upper())      # HELLO WORLD
print(text.lower())      # hello world
print(text.title())      # Hello World（每个单词首字母大写）
print(text.capitalize()) # Hello world（只有第一个字母大写）
print(text.swapcase())   # hELLO wORLD（大小写互换）
```

**实战**：处理用户输入时不区分大小写
```python
answer = input("是否继续？(yes/no): ").lower().strip()
if answer in ['yes', 'y']:
    print("继续执行")
```

## 2. 查找与替换（4个）

```python
text = "Python is easy, Python is powerful"

# find - 查找子串位置（找不到返回-1）
print(text.find("Python"))     # 0
print(text.find("Java"))       # -1

# index - 查找子串位置（找不到报错）
print(text.index("Python"))    # 0

# count - 统计出现次数
print(text.count("Python"))    # 2

# replace - 替换
print(text.replace("Python", "JavaScript"))
# JavaScript is easy, JavaScript is powerful

# 只替换前N次
print(text.replace("Python", "JS", 1))
# JS is easy, Python is powerful
```

---

## 3. 判断方法（6个）

```python
# 判断是否以某字符串开头/结尾
print("report.pdf".startswith("report"))  # True
print("report.pdf".endswith(".pdf"))      # True
print("report.pdf".endswith((".pdf", ".docx")))  # True（支持元组）

# 判断字符串类型
print("123".isdigit())      # True（纯数字）
print("abc".isalpha())      # True（纯字母）
print("abc123".isalnum())   # True（字母或数字）
print("   ".isspace())      # True（纯空格）
print("HELLO".isupper())    # True（全大写）
print("hello".islower())    # True（全小写）
```

**实战**：验证文件名
```python
def is_valid_image(filename):
    return filename.lower().endswith(('.jpg', '.jpeg', '.png', '.gif'))

print(is_valid_image("photo.JPG"))   # True
print(is_valid_image("document.pdf")) # False
```

## 4. 去除空白（3个）

```python
text = "  hello world  \n\t"

print(text.strip())    # "hello world"（两边都去掉）
print(text.lstrip())   # "hello world  \n\t"（只去左边）
print(text.rstrip())   # "  hello world"（只去右边）

# 去掉指定字符
print("...hello...".strip('.'))  # "hello"
```

---

## 5. 对齐与填充（3个）

```python
text = "Python"

print(text.center(20))       # "       Python       "（居中）
print(text.center(20, '-'))  # "-------Python-------"
print(text.ljust(20, '*'))   # "Python**************"（左对齐）
print(text.rjust(20, '*'))   # "**************Python"（右对齐）

# 数字补零
print("42".zfill(5))         # "00042"
```

**实战**：格式化输出表格
```python
products = [
    ("Apple", 5.5),
    ("Banana", 3.0),
    ("Cherry", 12.5)
]

for name, price in products:
    print(f"{name.ljust(10)} ${price:.2f}")

# 输出：
# Apple      $5.50
# Banana     $3.00
# Cherry     $12.50
```

## 6. 分割与连接（进阶用法）

```python
# split - 默认按任意空白分割
print("a  b\tc\nd".split())  # ['a', 'b', 'c', 'd']

# splitlines - 按行分割
text = """Line 1
Line 2
Line 3"""
print(text.splitlines())  # ['Line 1', 'Line 2', 'Line 3']

# partition - 分成三部分
print("key=value".partition("="))  # ('key', '=', 'value')

# rpartition - 从右边开始分
print("a,b,c".rpartition(","))  # ('a,b', ',', 'c')
```

## 7. 其他实用方法

```python
# encode/decode - 编码转换
text = "中文"
encoded = text.encode('utf-8')
print(encoded)  # b'\xe4\xb8\xad\xe6\x96\x87'
print(encoded.decode('utf-8'))  # "中文"

# translate - 字符映射转换
table = str.maketrans('aeiou', '12345')
print("hello".translate(table))  # "h2ll4"

# format - 格式化（虽然现在多用f-string）
print("{} + {} = {}".format(1, 2, 3))  # "1 + 2 = 3"
```
