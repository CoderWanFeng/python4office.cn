## 什么是变量？

变量就像一个盒子，用来存放数据。

```python
name = "Alice"      # 把"Alice"放进name这个盒子
age = 25            # 把25放进age这个盒子
is_student = True   # 把True放进is_student这个盒子
```

**命名规则：**
- 只能包含字母、数字、下划线
- 不能以数字开头
- 区分大小写（Name和name是不同的）
- 不能用Python关键字（如if、for、class等）

**好习惯：**
- 见名知意：`user_name`比`x`好
- 小写字母+下划线：`my_variable`

## 7大基本数据类型

### 1. 整数（int）
```python
age = 25
count = -10
big_number = 1000000

# 支持大数运算
print(2 ** 100)  # 2的100次方，Python轻松处理
```

### 2. 浮点数（float）
```python
price = 19.99
temperature = -3.5
pi = 3.14159

# 注意精度问题
print(0.1 + 0.2)  # 0.30000000000000004，不是精确的0.3
```

### 3. 字符串（str）
```python
name = "Alice"
message = 'Hello World'
multiline = """这是
多行
字符串"""

# 字符串操作
print(len(name))           # 5，长度
print(name.upper())        # ALICE，转大写
print(name[0])             # A，索引访问
print("Hello " + name)     # Hello Alice，拼接
```

### 4. 布尔值（bool）
```python
is_valid = True
is_empty = False

# 比较运算产生布尔值
print(5 > 3)   # True
print(5 == 3)  # False
print(5 != 3)  # True
```

### 5. 空值（None）
```python
result = None  # 表示"没有值"或"未知"

# 常用于函数默认返回值
 def find_user(user_id):
     if user_id in database:
         return database[user_id]
     return None  # 没找到返回None
```

### 6. 列表（list）
```python
fruits = ["apple", "banana", "cherry"]
mixed = [1, "hello", 3.14, True]  # 可以混存不同类型

# 常用操作
print(fruits[0])        # apple，索引访问
fruits.append("orange") # 添加元素
fruits.remove("banana") # 删除元素
print(len(fruits))      # 长度
```

### 7. 字典（dict）
```python
person = {
    "name": "Alice",
    "age": 25,
    "city": "Beijing"
}

# 常用操作
print(person["name"])       # Alice，通过key访问
person["email"] = "alice@example.com"  # 添加键值对
del person["age"]           # 删除键值对
```

## 类型转换

```python
# 字符串转整数
num_str = "123"
num = int(num_str)  # 123

# 整数转字符串
num = 123
num_str = str(num)  # "123"

# 转浮点数
price = float("19.99")  # 19.99

# 转列表
chars = list("hello")  # ['h', 'e', 'l', 'l', 'o']
```

---

## 查看数据类型

```python
print(type(123))       # <class 'int'>
print(type("hello"))   # <class 'str'>
print(type([1, 2, 3])) # <class 'list'>
```

## 实战练习

创建一个学生信息卡片：

```python
# 学生信息
student_name = "张三"
student_age = 20
student_scores = [85, 90, 78, 92]  # 各科成绩
student_info = {
    "major": "计算机科学",
    "grade": "大二",
    "is_scholarship": True
}

# 计算平均分
average = sum(student_scores) / len(student_scores)

# 打印报告
print(f"学生姓名: {student_name}")
print(f"年龄: {student_age}")
print(f"专业: {student_info['major']}")
print(f"平均成绩: {average:.2f}")
```
