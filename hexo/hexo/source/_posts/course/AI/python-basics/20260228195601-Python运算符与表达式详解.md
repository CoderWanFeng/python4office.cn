## 算术运算符

```python
a = 10
b = 3

print(a + b)   # 13，加法
print(a - b)   # 7，减法
print(a * b)   # 30，乘法
print(a / b)   # 3.333...，除法（结果是浮点数）
print(a // b)  # 3，整除（取整数部分）
print(a % b)   # 1，取余（模运算）
print(a ** b)  # 1000，幂运算（10的3次方）
```

### 实用技巧
```python
# 判断奇偶
num = 7
if num % 2 == 0:
    print("偶数")
else:
    print("奇数")  # 输出这个

# 获取个位数
num = 123
print(num % 10)  # 3

# 四舍五入到整数
price = 19.7
rounded = int(price + 0.5)  # 20
```

## 比较运算符

```python
x = 5
y = 10

print(x == y)  # False，等于
print(x != y)  # True，不等于
print(x > y)   # False，大于
print(x < y)   # True，小于
print(x >= y)  # False，大于等于
print(x <= y)  # True，小于等于
```

### 链式比较
```python
# 判断是否在范围内
age = 25
print(18 <= age <= 60)  # True，Python支持这种写法！

# 等价于
print(18 <= age and age <= 60)  # True
```

---

## 赋值运算符

```python
x = 10

x += 5   # 等价于 x = x + 5，x现在是15
x -= 3   # 等价于 x = x - 3，x现在是12
x *= 2   # 等价于 x = x * 2，x现在是24
x /= 4   # 等价于 x = x / 4，x现在是6.0
x //= 2  # 等价于 x = x // 2，x现在是3.0
x %= 2   # 等价于 x = x % 2，x现在是1.0
x **= 3  # 等价于 x = x ** 3，x现在是1.0
```

**好处**：代码更简洁，尤其是变量名很长时。

## 逻辑运算符

```python
# and：两个都为True，结果才为True
print(True and True)   # True
print(True and False)  # False

# or：只要有一个为True，结果就是True
print(True or False)   # True
print(False or False)  # False

# not：取反
print(not True)   # False
print(not False)  # True
```

### 实际应用
```python
age = 25
income = 8000

# 判断是否符合贷款条件
is_qualified = (age >= 18 and age <= 60) and (income >= 5000)
print(is_qualified)  # True

# 判断是否周末
is_weekend = day == "Saturday" or day == "Sunday"
```

### 短路求值
```python
# and：如果第一个为False，直接返回False，不判断第二个
result = False and expensive_function()  # expensive_function不会执行

# or：如果第一个为True，直接返回True，不判断第二个
result = True or expensive_function()    # expensive_function不会执行
```

## 成员运算符

```python
fruits = ["apple", "banana", "cherry"]

print("apple" in fruits)      # True
print("orange" in fruits)     # False
print("apple" not in fruits)  # False

# 字典中检查key
person = {"name": "Alice", "age": 25}
print("name" in person)   # True
print("Alice" in person)  # False（检查的是key，不是value）
```

## 身份运算符

```python
a = [1, 2, 3]
b = a        # b和a指向同一个对象
c = [1, 2, 3]  # c是新创建的相同内容的对象

print(a is b)      # True，同一个对象
print(a is c)      # False，不同对象
print(a == c)      # True，内容相同

print(a is not c)  # True
```

**区别**：
- `==` 比较值是否相等
- `is` 比较是否是同一个对象（内存地址相同）

## 运算符优先级

```python
# 从高到低
# 1. 括号 ()
# 2. 幂运算 **
# 3. 正负号 +x, -x
# 4. 乘除取余 * / // %
# 5. 加减 + -
# 6. 比较 > < >= <= == !=
# 7. 身份 is, is not
# 8. 成员 in, not in
# 9. 逻辑 not
# 10. 逻辑 and
# 11. 逻辑 or

# 例子
result = 2 + 3 * 4      # 14，先乘后加
result = (2 + 3) * 4    # 20，括号优先
result = 2 ** 3 ** 2    # 512，从右往左算（2**(3**2)）
```

**建议**：不确定优先级时，用括号明确顺序。

## 实战：计算器程序

```python
# 简单计算器
def calculator():
    num1 = float(input("请输入第一个数字: "))
    operator = input("请输入运算符 (+, -, *, /): ")
    num2 = float(input("请输入第二个数字: "))
    
    if operator == "+":
        result = num1 + num2
    elif operator == "-":
        result = num1 - num2
    elif operator == "*":
        result = num1 * num2
    elif operator == "/":
        if num2 != 0:
            result = num1 / num2
        else:
            return "错误：不能除以零！"
    else:
        return "错误：未知的运算符"
    
    return f"结果: {result}"

print(calculator())
```
