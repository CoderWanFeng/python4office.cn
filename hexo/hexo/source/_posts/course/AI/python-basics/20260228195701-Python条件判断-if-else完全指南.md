大家好，我是正在实战各种AI项目的程序员晚枫。

上一篇我们学习了运算符。今天来学习**条件判断（if-else）**，这是让程序拥有"决策能力"的关键。

有了它，你的程序就能根据不同情况执行不同操作，真正"智能"起来。

---

## 基本语法：if语句

```python
age = 18

if age >= 18:
    print("你已经成年了")
    print("可以考驾照了")

print("程序结束")
```

**注意**：
- `if`后面跟条件表达式
- 条件为`True`时执行缩进的代码块
- Python用**缩进**（通常是4个空格）表示代码块

---

## if-else结构

```python
score = 75

if score >= 60:
    print("及格了！")
else:
    print("不及格，需要补考")
```

**流程**：
- 条件成立 → 执行if块
- 条件不成立 → 执行else块

---

## if-elif-else多分支

```python
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"你的等级是: {grade}")  # B
```

**要点**：
- `elif` = else if，可以有多个
- 从上到下依次判断，满足一个就跳出
- `else`可选，放在最后

## 嵌套条件

```python
is_member = True
purchase_amount = 200

if is_member:
    if purchase_amount >= 100:
        discount = 0.8  # 会员满100打8折
    else:
        discount = 0.9  # 会员不满100打9折
else:
    if purchase_amount >= 200:
        discount = 0.95  # 非会员满200打95折
    else:
        discount = 1.0   # 非会员无折扣

final_price = purchase_amount * discount
print(f"最终价格: ¥{final_price}")
```

**建议**：嵌套不要超过3层，否则可读性很差。

## 条件表达式（三元运算符）

简单的if-else可以写成一行：

```python
age = 20

# 传统写法
if age >= 18:
    status = "成年"
else:
    status = "未成年"

# 简洁写法
status = "成年" if age >= 18 else "未成年"

print(status)  # 成年
```

**格式**：`值1 if 条件 else 值2`

---

## 常见条件模式

### 模式1：判断是否为空
```python
name = ""

# 不推荐
if name == "":
    print("名字不能为空")

# 推荐（空字符串会被视为False）
if not name:
    print("名字不能为空")
```

### 模式2：多重条件组合
```python
age = 25
has_license = True

# 同时满足多个条件
if age >= 18 and has_license:
    print("可以租车")

# 满足任一条件
if age < 13 or age > 65:
    print("享受优惠票价")
```

### 模式3：判断是否在范围内
```python
score = 85

# Python支持链式比较
if 60 <= score <= 100:
    print("成绩有效")
```

### 模式4：判断元素是否存在
```python
fruits = ["apple", "banana"]

if "apple" in fruits:
    print("有苹果")

if "orange" not in fruits:
    print("没有橙子")
```

## 避坑指南

### ❌ 错误1：忘记缩进
```python
if age >= 18:
print("成年")  # 错误！需要缩进
```

### ❌ 错误2：混淆=和==
```python
x = 5

if x = 10:   # 错误！这是赋值，不是比较
    pass

if x == 10:  # 正确
    pass
```

### ❌ 错误3：逻辑错误
```python
# 这个条件永远为False
if age > 18 and age < 12:
    pass
```

## 实战：智能客服系统

```python
def customer_service():
    print("欢迎使用智能客服系统")
    query = input("请输入您的问题: ").lower()
    
    if "价格" in query or "多少钱" in query:
        response = "我们的基础版免费，专业版¥99/月"
    elif "退款" in query or "退货" in query:
        response = "7天无理由退款，请联系客服处理"
    elif "账号" in query or "登录" in query:
        response = "请尝试重置密码，或联系技术支持"
    elif "谢谢" in query or "感谢" in query:
        response = "不客气，有其他问题随时找我！"
    else:
        response = "抱歉，我不太理解。请转人工客服：400-123-4567"
    
    print(f"客服: {response}")

customer_service()
```
