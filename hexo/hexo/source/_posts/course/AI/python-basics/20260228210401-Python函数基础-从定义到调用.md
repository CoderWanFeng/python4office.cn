## 什么是函数？

函数是一段**可重复使用的代码块**，用来完成特定任务。

就像洗衣机：你不需要知道内部原理，只需要把衣服放进去，按下按钮，就能得到干净的衣服。

```python
# 定义函数
def greet():
    print("Hello, World!")

# 调用函数
greet()  # Hello, World!
greet()  # Hello, World!（可以重复使用）
```

---

## 函数的基本结构

```python
def 函数名(参数):
    """文档字符串（说明函数做什么）"""
    # 函数体
    return 返回值  # 可选
```

### 示例：带参数的函数
```python
def greet(name):
    """向指定的人打招呼"""
    print(f"Hello, {name}!")

greet("Alice")  # Hello, Alice!
greet("Bob")    # Hello, Bob!
```

### 示例：带返回值的函数
```python
def add(a, b):
    """计算两个数的和"""
    result = a + b
    return result

sum = add(3, 5)
print(sum)  # 8
```

## 参数类型

### 位置参数
```python
def introduce(name, age, city):
    print(f"我叫{name}，今年{age}岁，来自{city}")

introduce("Alice", 25, "北京")
# 必须按顺序传参
```

### 关键字参数
```python
introduce(age=25, city="北京", name="Alice")
# 顺序不重要，可读性更好
```

### 默认参数
```python
def greet(name, greeting="Hello"):
    print(f"{greeting}, {name}!")

greet("Alice")           # Hello, Alice!
greet("Bob", "Hi")       # Hi, Bob!
```

**注意**：默认参数要放在非默认参数后面！

## 返回值

### 返回单个值
```python
def square(x):
    return x ** 2

result = square(4)  # 16
```

### 返回多个值（实际上是元组）
```python
def get_min_max(numbers):
    return min(numbers), max(numbers)

min_val, max_val = get_min_max([3, 1, 4, 1, 5])
print(min_val, max_val)  # 1 5
```

### 没有return默认返回None
```python
def say_hello():
    print("Hello")

result = say_hello()  # Hello
print(result)         # None
```

## 变量作用域

```python
global_var = "我是全局变量"

def my_function():
    local_var = "我是局部变量"
    print(global_var)  # 可以访问全局变量
    print(local_var)   # 可以访问局部变量

my_function()
print(global_var)      # 可以访问
# print(local_var)     # 错误！外部不能访问局部变量
```

### 修改全局变量
```python
count = 0

def increment():
    global count  # 声明使用全局变量
    count += 1

increment()
print(count)  # 1
```

## 函数作为参数（高阶函数）

```python
def apply_operation(x, y, operation):
    """接收函数作为参数"""
    return operation(x, y)

def add(a, b):
    return a + b

def multiply(a, b):
    return a * b

# 使用
result = apply_operation(3, 4, add)      # 7
result = apply_operation(3, 4, multiply) # 12
```

## lambda表达式（匿名函数）

简单的函数可以写成一行：

```python
# 普通函数
def square(x):
    return x ** 2

# lambda等价写法
square = lambda x: x ** 2

# 使用
print(square(5))  # 25

# 常用场景：排序
students = [("Alice", 85), ("Bob", 92), ("Charlie", 78)]
students.sort(key=lambda x: x[1], reverse=True)
print(students)  # [('Bob', 92), ('Alice', 85), ('Charlie', 78)]
```

## 递归函数

函数调用自己：

```python
def factorial(n):
    """计算阶乘：n! = n * (n-1) * ... * 1"""
    if n <= 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # 120
```

**注意**：递归要有终止条件，否则会无限循环！

## 实战：工具函数库

```python
def is_valid_email(email):
    """验证邮箱格式"""
    import re
    pattern = r'^[\w\.-]+@[\w\.-]+\.\w+$'
    return re.match(pattern, email) is not None

def format_price(price):
    """格式化价格"""
    return f"¥{price:.2f}"

def truncate_text(text, max_length=100):
    """截断长文本"""
    if len(text) <= max_length:
        return text
    return text[:max_length] + "..."

# 使用
print(is_valid_email("test@example.com"))  # True
print(format_price(19.9))                  # ¥19.90
print(truncate_text("这是一段很长的文本...", 10))
```
