# 搭一个个人网站，到底需要买什么？6样东西，全给你列清楚了

## 先说结论

**搭个人网站，你需要 6 样东西：**

1. **域名** —— 你的网址（比如 python4office.cn）
2. **DNS** —— 把域名指向服务器的"导航"
3. **云服务器** —— 放网站代码的地方
4. **控制台** —— 管理上面所有东西的面板
5. **CDN** —— 让你的网站访问更快
6. **内容/程序** —— 你的网站本身（博客、导航页、官网……）

就这么简单。**少一样都跑不起来。**

---

## 逐个说

### 一、域名（Domain）

**就是你的网址。**

没有域名，别人只能用 IP 地址访问你——一串数字，谁记得住？

| 要点 | 说明 |
|------|------|
| 怎么买 | 腾讯云/阿里云/Namesilo 都行 |
| 价格 | .cn 域名首年几块钱到几十块；.com 大概 50-70 元/年 |
| 怎么选 | 越短越好、越容易拼越好、别用特殊字符 |
| 我的 | `python4office.cn` |

**选域名的原则：能打字说出来，别人就能拼出来。**

---

### 二、DNS（域名解析）

**域名是名字，DNS 是电话簿。**

用户在浏览器输入 `python4office.cn`，DNS 负责告诉浏览器："这个域名对应的服务器 IP 是 xxx.xxx.xxx.xxx"

| 记录类型 | 干什么用的 |
|----------|-----------|
| **A 记录** | 域名 → IP 地址（最常用）|
| **CNAME** | 域名 → 另一个域名（常用于 CDN）|
| **MX 记录** | 邮件服务器（如果你要用域名邮箱）|

**大多数情况，你只需要配一条 A 记录就够了。**

---

### 三、云服务器（CVM / ECS）

**你网站的家。**

代码、数据库、图片……全都放在这里。

| 配置建议 | 个人网站够用 |
|---------|-------------|
| CPU | 1-2 核 |
| 内存 | 2GB 起 |
| 带宽 | 1-3Mbps（如果用了 CDN 可以更低）|
| 系统 | Ubuntu / CentOS / Debian 都行 |
| 价格 | 新用户活动价大概 30-100 元/月 |

**个人博客/导航站/作品集，最低配就够用。别上来就买 8 核 16G，浪费钱。**

---

### 四、控制台

**就是你登录云服务商后台的那个网页。**

腾讯云控制台长这样：

![腾讯云控制台](https://images.unsplash.com/photo-1551288049-bebda4e38f71?auto=compress&cs=tinysrgb&w=800)

你在里面能干这些事：

- 🖥️ 开通/管理**云服务器**
- 🌐 配置**域名和 DNS**
- ⚡ 开启/配置 **CDN**
- 💳 充值/查看账单
- 🔒 设置安全组（防火墙）

**所有操作都在这里完成。这就是"控制台"。**

---

### 五、CDN（内容分发网络）

**让全国各地的用户打开你的网站都快得飞起。**

没有 CDN：

```
用户(北京) → 你的服务器(广州) = 跨大半个中国，慢
```

有 CDN：

```
用户(北京) → CDN节点(北京) = 就近取，快
```

| 场景 | 需要 CDN 吗？ |
|------|--------------|
| 个人小站，日访问 < 500 | 可选，不急 |
| 有静态资源（图片/JS/CSS）| 推荐，效果明显 |
| 全国/全球用户都有 | 必须上 |
| 只是本地/朋友圈看 | 不需要 |

**腾讯云 CDN 有免费额度，新用户先白嫖再说。**

---

### 六、内容 / 程序

**前面 5 样是基础设施，这才是你的网站本身。**

常见选择：

| 类型 | 方案 | 难度 |
|------|------|------|
| 博客 | Hexo / Hugo / WordPress | ⭐ 简单 |
| 导航站 | 前端静态页面 | ⭐ 简单 |
| 官网 | Vue / React | ⭐⭐ 中等 |
| 全栈应用 | Django / Spring Boot | ⭐⭐⭐ 较难 |

**新手建议从静态博客或导航站开始。不用数据库，不用后端，部署超简单。**

---

## 它们之间的关系

用一个图说明这 6 样东西怎么配合：

```
用户浏览器
    │
    ▼
[DNS] "python4office.cn 在哪？"
    │
    ▼ 返回 IP 或 CNAME
[CDN] （可选）"就近给内容"
    │  或者直接到
    ▼
[云服务器] "这里是你的网站文件"
    │
    ▼
[控制台] ← 你在这里管理以上所有东西
    │
[域名] ← 用户记住了这个名字
```

**缺谁都不行。**

## 新手踩坑提醒

### 坑 1：域名买了没解析

域名 ≠ 网站。买了域名只是有了名字，**还得配 DNS 指向服务器**。

### 坑 2：服务器开了端口没放行

云服务器的防火墙（安全组）默认只开 22 和 80/443。

如果你用别的端口（比如 8080、3000），**记得去控制台放行**。

### 坑 3：备案！

在国内用域名绑定服务器，**必须做 ICP 备案**。

不备案？域名会被运营商拦截。

备案周期一般 1-2 周，提前准备。

### 坑 4：HTTPS 证书

现在浏览器会标记 HTTP 为"不安全"。

申请免费的 SSL 证书（Let's Encrypt 或腾讯云免费证书），**一步搞定 HTTPS**。
