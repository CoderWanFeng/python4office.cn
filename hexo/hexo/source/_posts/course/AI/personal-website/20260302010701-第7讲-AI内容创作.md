## 为什么需要持续更新内容？

### 内容的作用

1. **吸引流量** - 每篇文章都是一个流量入口
2. **建立专业形象** - 展示你的知识和经验
3. **SEO 优化** - 更多内容=更多搜索关键词
4. **转化客户** - 好的内容能说服访客变成客户

### 更新的频率

**建议**：
- 最少：每周 1 篇
- 理想：每周 2-3 篇
- 激进：每天 1 篇

**用 AI 后**：
- 写 1 篇的时间从 3 小时降到 10 分钟
- 每周写 5-10 篇成为可能

## 用 AI 写文章的完整流程

### 第一步：确定选题（5 分钟）

**选题来源**：
1. 客户常问的问题
2. 行业热点话题
3. 竞争对手的热门文章
4. 关键词研究工具

**用 AI 生成选题**：

```
我是 [职业/行业]，目标用户是 [用户画像]。

请帮我列出 20 个他们最关心的问题，
这些问题会成为我博客文章的选题。

要求：
1. 问题要具体，不要太宽泛
2. 覆盖不同难度（新手/进阶/高级）
3. 有搜索量，有人关心
```

**示例输出**（以心理咨询师为例）：
```
1. 焦虑症有哪些症状？
2. 心理咨询一次多少钱？
3. 抑郁症能治好吗？
4. 如何判断自己是否需要心理咨询？
5. 心理咨询的流程是什么样的？
...
```

### 第二步：生成大纲（2 分钟）

**用 AI 生成大纲**：

```
我要写一篇关于"[选题]"的文章。
目标读者是 [用户画像]。

请帮我生成文章大纲，包括：
1. 吸引人的标题（5 个备选）
2. 引言（如何开头）
3. 正文结构（3-5 个小标题）
4. 结尾（如何总结 + CTA）
```

**示例输出**：
```
标题备选：
1. 焦虑症的 5 个症状，你中了几个？
2. 如何判断自己是否焦虑？专业心理咨询师告诉你
3. 焦虑症自查指南：这 5 个信号要警惕

大纲：
一、引言：焦虑的普遍性
二、焦虑症的 5 个典型症状
三、如何区分正常焦虑和焦虑症
四、什么时候需要寻求专业帮助
五、结语 + 预约咨询 CTA
```

### 第三步：撰写正文（5 分钟）

**用 AI 生成初稿**：

```
根据上面的大纲，帮我写完整文章。

要求：
1. 语气亲切、专业但不生硬
2. 每段 200 字以内
3. 加入案例和故事
4. 结尾引导读者 [行动]
5. 全文 1500-2000 字
```

### 第四步：修改润色（3 分钟）

**AI 写的文章需要修改**：
- ✅ 加入个人经历和案例
- ✅ 调整语气，更像"人话"
- ✅ 检查事实准确性
- ✅ 加入内部链接（链接到你网站的其他文章）

**用 AI 润色**：

```
请帮我润色这篇文章：
1. 让语言更自然、更像人写的
2. 加入一些口语化表达
3. 确保逻辑清晰
4. 检查有没有重复内容
```

### 第五步：添加配图（2 分钟）

**AI 配图工具**：
- **Midjourney** - 质量最好
- **DALL-E 3** - 理解能力强
- **Stable Diffusion** - 免费
- **Canva AI** - 简单好用

**配图提示词**：

```
[文章主题] related illustration, 
minimalist style, clean background, 
professional, suitable for blog post
```

## 不同类型文章的写法

### 1. 科普类文章

**结构**：
- 什么是 X？
- X 的特点/分类
- 如何判断/识别
- 如何应对/解决

**示例**：《焦虑症的 5 个症状，你中了几个？》

### 2. 教程类文章

**结构**：
- 问题/痛点引入
- 准备工作
- 步骤 1、2、3...
- 常见问题解答
- 总结

**示例**：《如何用 AI 写文章：5 步搞定》

### 3. 案例类文章

**结构**：
- 客户背景
- 遇到的问题
- 解决方案
- 结果/效果
- 经验总结

**示例**：《帮助一位创业者摆脱焦虑的 3 个月》

### 4. 观点类文章

**结构**：
- 现象/问题引入
- 你的观点
- 论据 1、2、3
- 反驳可能的质疑
- 总结 + 行动建议

**示例**：《为什么我不建议焦虑的人自己硬扛》

## 内容批量创作技巧

### 一次生成多篇文章

**用 AI 生成系列内容**：

```
我要做一个关于"[主题]"的系列文章，共 5 篇。

请帮我规划：
1. 每篇的主题和标题
2. 5 篇之间的逻辑关系
3. 每篇的核心要点

要求：由浅入深，适合新手阅读
```

### 一篇文章多平台分发

**用 AI 改写内容**：

```
请把这篇 2000 字的文章改写成：
1. 公众号版本（3000 字，更详细）
2. 小红书版本（500 字，加 emoji）
3. 知乎版本（1500 字，更专业）
4. 微博版本（140 字，精简）
```
