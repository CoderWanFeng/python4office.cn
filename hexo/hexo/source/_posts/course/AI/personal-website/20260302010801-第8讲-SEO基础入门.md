## 什么是 SEO？

**SEO（Search Engine Optimization）**：搜索引擎优化

简单说：让你的网站在 Google 搜索结果中排名更靠前。

### 为什么 SEO 重要？

**搜索流量的价值**：
- ✅ 被动流量 - 睡着也有人来
- ✅ 精准流量 - 用户主动搜索，需求明确
- ✅ 免费流量 - 不需要投广告
- ✅ 持续流量 - 好内容可以排名好几年

**数据说话**：
- Google 搜索结果第 1 名：点击率 31.7%
- 第 2 名：点击率 24.7%
- 第 3 名：点击率 18.7%
- 第 10 名：点击率只有 3.1%

**结论**：排名越靠前，流量越多。

---

## SEO 核心概念

### 1. 关键词（Keyword）

用户搜索时输入的词，比如：
- "心理咨询师"
- "焦虑症症状"
- "如何缓解压力"

### 2. 搜索意图（Search Intent）

用户搜索背后的目的：
- **信息型**：想了解某事（"什么是焦虑症"）
- **导航型**：想找特定网站（"简单心理官网"）
- **交易型**：想购买（"心理咨询预约"）

### 3. 页面权重（Page Authority）

Google 认为你的页面有多重要，取决于：
- 内容质量
- 外部链接数量
- 用户停留时间
- 网站整体权威性

## 关键词研究

### 用 AI 找关键词

**提示词**：

```
我是 [职业/行业]，目标用户是 [用户画像]。

请帮我列出 50 个他们可能会搜索的关键词，
包括：
1. 核心关键词（2-3 个字）
2. 长尾关键词（4 个字以上）
3. 问题型关键词（"如何..."、"为什么..."）

按搜索量从高到低排序
```

### 关键词工具

**免费工具**：
- Google 关键词规划师
- Ubersuggest（免费版有限制）
- AnswerThePublic（问题型关键词）

**付费工具**：
- Ahrefs（$99/月，最强大）
- SEMrush（$119/月）
- Moz Pro（$99/月）

**国内工具**：
- 5118.com
- chinaz.com
- aizhan.com

### 如何选择关键词？

**好关键词的标准**：
- ✅ 有搜索量（至少 100 次/月）
- ✅ 竞争度低（新手能做）
- ✅ 与你的业务相关
- ✅ 有商业价值（能转化）

**新手策略**：
1. 先做长尾关键词（竞争小）
2. 积累权重后再做核心词
3. 每个页面聚焦 1-2 个关键词

## 页面 SEO 优化

### 1. 标题优化（Title Tag）

**重要性**：⭐⭐⭐⭐⭐

标题是 SEO 最重要的因素之一。

**优化技巧**：
- 包含目标关键词
- 控制在 60 个字符以内
- 吸引点击（加入数字、疑问、承诺）

**示例**：
```
❌ 差：心理咨询
✅ 好：心理咨询师推荐：10 年经验的真心建议
✅ 好：焦虑症怎么办？专业心理咨询师告诉你 5 个方法
```

### 2. 描述优化（Meta Description）

**重要性**：⭐⭐⭐

描述不影响排名，但影响点击率。

**优化技巧**：
- 包含关键词
- 控制在 150-160 个字符
- 写清楚文章价值
- 加入行动号召

**示例**：
```
有焦虑情绪怎么办？10 年资深心理咨询师分享 5 个实用方法，
帮你快速缓解焦虑。免费自测量表 + 预约入口。
```

### 3. 内容优化

**重要性**：⭐⭐⭐⭐⭐

**优化技巧**：
- 关键词自然出现（不要堆砌）
- 小标题包含关键词
- 内容要有深度（1500 字以上）
- 加入图片、视频等多媒体
- 内部链接（链接到其他相关文章）

### 4. URL 优化

**重要性**：⭐⭐

**优化技巧**：
- 简短清晰
- 包含关键词
- 用小写字母和连字符

**示例**：
```
❌ 差：website.com/?p=123
✅ 好：website.com/anxiety-symptoms
```

### 5. 图片优化

**重要性**：⭐⭐

**优化技巧**：
- 文件名包含关键词
- 添加 Alt 文本
- 压缩图片大小

## 用 AI 优化 SEO

### 优化文章标题

```
我写了一篇关于"[主题]"的文章。

请帮我生成 10 个 SEO 友好的标题，要求：
1. 包含关键词"[关键词]"
2. 控制在 60 个字符以内
3. 吸引点击（加入数字、疑问、承诺等）
```

### 优化文章描述

```
请根据这篇文章，写一个 SEO 友好的 Meta Description：
1. 包含关键词"[关键词]"
2. 控制在 150-160 个字符
3. 写清楚文章价值
4. 吸引用户点击
```

### 优化文章内容

```
请帮我优化这篇文章的 SEO：
1. 检查关键词密度（目标 1-2%）
2. 建议在哪里加入关键词
3. 优化小标题
4. 建议内部链接位置
```

## 技术 SEO 基础

### 1. 网站速度

**检查工具**：pagespeed.web.dev

**优化建议**：
- 压缩图片
- 使用 CDN
- 减少插件

### 2. 移动端适配

Google 优先索引移动版网站。

**检查**：用手机访问网站，确保正常显示。

### 3. HTTPS

确保网站使用 HTTPS（安全协议）。

大部分主机商免费提供 SSL 证书。

### 4. 网站地图（Sitemap）

帮助 Google 发现你网站的所有页面。

**生成工具**：
- XML-Sitemaps.com（免费）
- WordPress 插件：Yoast SEO

## 提交 Google

### Google Search Console

**是什么**：Google 官方的网站管理工具

**作用**：
- 提交网站给 Google
- 查看搜索排名
- 发现索引问题
- 查看流量数据

**设置步骤**：
1. 打开 search.google.com/search-console
2. 添加你的网站
3. 验证网站所有权
4. 提交 Sitemap
