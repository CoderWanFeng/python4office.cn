## 第一部分：域名注册

### 什么是域名？

域名就是你网站的地址，比如：
- baidu.com
- google.com
- wanfeng.ai

### 如何取一个好域名？

**好域名的标准**：
- ✅ 简短好记（最好 10 个字母以内）
- ✅ 容易拼写（避免生僻词）
- ✅ 与个人品牌相关
- ✅ 优先选.com 后缀

**取域名的技巧**：
- 用你的名字：wanfeng.com
- 用你的名字 + 职业：designerwan.com
- 用你的品牌名：pythonoffice.cn
- 用动词 + 名词：learnai.com

**避免的坑**：
- ❌ 太长（超过 15 个字母）
- ❌ 连字符（wan-feng.com 不好记）
- ❌ 数字（除非有特殊意义）
- ❌ 生僻单词

### 用 AI 帮你取域名

我用 Claude 写了一个提示词：

```
我想注册一个个人网站域名，我的情况是：
- 名字/昵称：[填写你的名字]
- 职业/领域：[填写你的领域]
- 目标用户：[填写目标用户]

请帮我推荐 10 个域名建议，要求：
1. 简短好记（10 个字母以内）
2. 优先.com 或.cn 后缀
3. 与我的个人品牌相关
4. 每个域名说明含义
```

### 域名注册平台推荐

**国际平台**（推荐）：
- Namecheap：便宜，界面友好
- GoDaddy：最大，但续费贵
- Cloudflare：成本价，推荐

**国内平台**：
- 阿里云：国内最大
- 腾讯云：也不错
- 西部数码：便宜

**价格对比**：
- .com：60-80 元/年
- .cn：30-50 元/年
- .ai：400-600 元/年（科技类推荐）
- .io：300-400 元/年（技术类推荐）

### 注册域名步骤（以 Namecheap 为例）

1. 打开 namecheap.com
2. 搜索你想要的域名
3. 加入购物车，注册账号
4. 填写信息，付款
5. 完成！

**注意**：
- 关闭域名保护（省钱）
- 关闭自动续费（避免忘记取消）
- 记得到期时间，手动续费

## 第二部分：主机选择

### 什么是主机？

主机就是存放网站文件的地方，相当于网站的"房子"。

### 主机类型

**共享主机**（Shared Hosting）
- 多人共享一台服务器
- 便宜：100-300 元/年
- 适合：个人网站、小流量

**云服务器**（VPS）
- 独立虚拟服务器
- 中等：300-1000 元/年
- 适合：中等流量、需要更多控制

**托管主机**（Managed Hosting）
- 服务商帮你维护
- 较贵：500-2000 元/年
- 适合：不想折腾的人

### 主机推荐

**国际主机**（推荐）：
- SiteGround：WordPress 官方推荐，300 元/年起
- Bluehost：便宜，适合新手，200 元/年起
- Vercel/Netlify：免费，适合静态网站

**国内主机**：
- 阿里云：200-500 元/年
- 腾讯云：类似阿里云
- 华为云：也不错

**免主机方案**：
- Notion + Super：不需要主机
- Framer：包含主机
- GitHub Pages：免费静态托管

### 选择主机的考虑因素

1. **流量**：预计每月有多少访问？
2. **技术**：想不想自己维护？
3. **预算**：愿意花多少钱？
4. **位置**：用户主要在国内还是国外？

**我的建议**：
- 新手 → 用免主机方案（Notion/Framer）
- 用 WordPress → SiteGround 或阿里云
- 技术博客 → GitHub Pages（免费）

## 第三部分：网站备案（国内）

### 什么是备案？

在中国大陆，网站需要向工信部备案才能访问。

### 需要备案吗？

**需要备案**：
- 主机在国内
- 面向国内用户

**不需要备案**：
- 主机在国外（推荐新手）
- 用免主机方案（Notion/Framer 等）

### 备案流程（以阿里云为例）

1. 购买国内主机
2. 登录阿里云备案系统
3. 填写个人信息
4. 上传身份证照片
5. 拍照核验
6. 等待审核（7-20 天）
7. 备案成功

### 备案注意事项

- 备案免费
- 需要身份证
- 个人不能做商业网站
- 备案期间网站不能访问

**我的建议**：
- 新手先用国外主机，免备案
- 等网站稳定了再考虑备案

## 第四部分：域名解析

### 什么是域名解析？

把域名指向你的主机，让用户能通过域名访问网站。

### 解析步骤

1. 登录域名注册商后台
2. 找到 DNS 设置
3. 添加 A 记录或 CNAME 记录
4. 等待生效（几分钟到几小时）

### 常见记录类型

**A 记录**：域名 → IP 地址
```
主机记录：@
记录类型：A
记录值：1.2.3.4（你的主机 IP）
```

**CNAME 记录**：域名 → 另一个域名
```
主机记录：www
记录类型：CNAME
记录值：yourname.framer.website
```
