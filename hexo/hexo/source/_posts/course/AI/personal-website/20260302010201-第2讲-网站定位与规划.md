这一讲，我们要做一件最重要的事：**想清楚你的网站要做什么**。

很多人建站失败，不是因为技术不行，而是因为没想清楚就动手。结果网站做好了，没人看，也没用。

---

## 第一步：确定你的网站目标

个人网站的目标，无非以下几种：

### 目标 1：展示型 - 让别人了解你

**适合人群**：求职者、自由职业者、专业人士

**核心内容**：
- 个人简介
- 作品集/案例展示
- 联系方式
- 客户评价

**成功案例**：
我有个做 UI 设计的朋友，网站上放了 20 个设计案例。HR 看到后主动联系，现在薪资涨了 50%。

### 目标 2：获客型 - 让客户找到你

**适合人群**：小老板、创业者、咨询师、律师、医生

**核心内容**：
- 服务介绍
- 成功案例
- 预约/咨询表单
- 联系方式

**成功案例**：
一个做企业咨询的学员，网站上线 3 个月，通过 Google 搜索来了 15 个客户，成交 8 个，客单价 2 万。

### 目标 3：内容型 - 积累粉丝和影响力

**适合人群**：内容创作者、博主、讲师

**核心内容**：
- 博客/文章
- 视频/播客
- 邮件订阅
- 付费内容

**成功案例**：
一个写技术博客的开发者，坚持写了一年，现在每月有 5 万 PV，靠广告和付费专栏月入 3 万+。

### 目标 4：电商型 - 直接卖货

**适合人群**：手工艺人、小店主、带货达人

**核心内容**：
- 商品展示
- 购物车
- 支付系统
- 订单管理

## 第二步：分析你的目标用户

想清楚谁会来看你的网站。

### 用 AI 帮你分析用户画像

我用 Claude 写了一个提示词，帮你快速分析：

```
我打算做一个个人网站，我的情况是：
- 职业/身份：[填写你的职业]
- 提供的产品/服务：[填写你提供什么]
- 目标：[展示/获客/内容/电商]

请帮我分析：
1. 我的目标用户是谁？（年龄、职业、痛点）
2. 他们最关心什么问题？
3. 他们会在什么场景下访问我的网站？
4. 网站应该包含哪些内容才能吸引他们？
```

**示例输出**（以心理咨询师为例）：

```
目标用户：
- 25-45 岁，城市白领
- 面临工作压力、情感问题、焦虑抑郁
- 有付费意愿，但担心效果

最关心的问题：
- 咨询师专不专业
- 咨询费用多少
- 咨询有没有效果
- 隐私能不能保护

访问场景：
- 深夜焦虑时搜索"心理咨询"
- 朋友推荐后了解咨询师背景
- 对比多个咨询师后做决定

网站内容建议：
- 专业资质展示（证书、培训经历）
- 咨询流程和费用说明
- 客户评价（匿名处理）
- 免费心理测评工具
- 预约咨询入口
```

## 第三步：规划网站内容框架

根据目标和用户，规划网站需要哪些页面。

### 标准个人网站结构

```
首页（Home）
├── 关于我（About）
├── 服务/产品（Services/Products）
├── 案例/作品（Portfolio）
├── 博客/文章（Blog）
├── 联系方式（Contact）
└── 其他页面（根据需要）
```

### 不同类型网站的内容建议

**展示型网站**：
- 首页：一句话介绍你是谁 + 核心价值
- 关于我：详细背景、资质、经历
- 作品集：5-10 个代表作品
- 联系方式：邮箱、微信、电话

**获客型网站**：
- 首页：解决什么问题 + 成功案例
- 服务介绍：服务内容、流程、价格
- 客户评价：真实反馈、案例展示
- 预约入口：表单、日历预约

**内容型网站**：
- 首页：最新文章 + 热门内容
- 博客：分类清晰的文章列表
- 关于我：为什么值得关注我
- 订阅入口：邮件列表、付费会员

## 第四步：用 AI 生成网站地图

继续用 AI 帮你规划具体内容：

```
我的网站定位是：[填写你的定位]
目标用户是：[填写用户画像]

请帮我规划：
1. 网站需要哪些页面
2. 每个页面的核心内容
3. 首页的文案框架（标题、副标题、CTA）
```

## 第五步：确定技术路线

根据你的技术水平和预算，选择建站方案。

### 方案对比

| 方案 | 难度 | 成本 | 适合人群 |
|------|------|------|----------|
| Notion + Super | ⭐ | 免费 -100 元/年 | 完全零基础，快速上线 |
| Framer | ⭐⭐ | 500-1000 元/年 | 想要更好设计感 |
| WordPress | ⭐⭐⭐ | 300-800 元/年 | 想要完全控制 |
| 自定义开发 | ⭐⭐⭐⭐⭐ | 2000 元+/年 | 有技术基础 |

**我的建议**：
- 第一次建站 → 选 Notion + Super，1 天就能上线
- 想要更专业 → 选 Framer，设计感更好
- 长期运营 → 选 WordPress，扩展性强
