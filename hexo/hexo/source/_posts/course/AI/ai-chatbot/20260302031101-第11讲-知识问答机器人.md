## 知识库准备

### 文档类型
**1. Word/PDF 文档**：员工手册、制度文件、产品手册
**2. Excel 表格**：产品参数表、价格表、常见问题表
**3. PPT 文档**：培训课件、产品介绍
**4. 网页内容**：官网文档、帮助页面

### 文档处理
**方法 1：直接上传**
扣子支持直接上传文档：
1. 进入"知识库"
2. 点击"上传文档"
3. 选择文件
4. AI 自动解析

**支持的格式**：PDF、Word、Excel、PPT、TXT

**方法 2：提取内容**
用 AI 提取文档内容：

**提示词**：
```
你是一名专业的文档整理专家。
我有以下文档内容：[粘贴文档内容]
请帮我：
1. 提取关键知识点
2. 生成常见问题（FAQ）
3. 为每个问题写标准答案
4. 输出结构化格式
要求：
- 问题真实，是用户常问的
- 答案简洁，100-200 字
- 适合机器人回复
```

**方法 3：批量处理**
多个文档批量处理：

**提示词**：
```
你是一名专业的知识整理专家。
我有多个文档，主题是 [主题]。
请帮我：
1. 整合所有文档内容
2. 分类整理（按主题/部门/产品）
3. 提取常见问题（50-100 个）
4. 为每个问题写标准答案
输出结构化知识库。
```

## 扣子配置

### 上传文档
**步骤**：
1. 进入"知识库"
2. 点击"上传文档"
3. 选择文件（PDF/Word/Excel 等）
4. 等待 AI 解析
5. 检查解析结果

### 优化知识库
**检查要点**：
- 文档解析是否完整
- 问题提取是否准确
- 答案是否清晰

**优化方法**：
- 手动补充问题
- 优化答案内容
- 添加相似问法

### 人设设定
**企业知识库机器人**：
```
【基本信息】
名字：小知
身份：企业知识库助手
性格：专业、准确、高效

【回复原则】
1. 答案准确，不瞎编
2. 不知道的问题引导人工
3. 涉及机密不回答
4. 回复简洁，引用原文

【语言风格】
- 正式、专业
- 简洁明了
- 适当引用文档

【标志性表达】
开场白：您好，我是知识库助手小知，请问有什么可以帮您？
结束语：如有其他问题，随时问我。
```

**培训机器人**：
```
【基本信息】
名字：小培
身份：培训助手
性格：耐心、鼓励、友好

【回复原则】
1. 鼓励用户学习
2. 答错不批评，引导再试
3. 记录学习进度
4. 适时给予奖励

【语言风格】
- 亲切、鼓励
- 适当用 emoji
- 有互动性
```

## 高级功能

### 多轮对话
**场景：培训考试**
```
机器人：欢迎参加产品知识考试，共 10 题，准备好了吗？
用户：好了
机器人：第 1 题：[问题]
用户：[回答]
机器人：[判断对错] 第 2 题：[问题]
...
机器人：考试完成，得分 8/10，不错哦！
```

### 上下文记忆
**场景：连续追问**
```
用户：产品 A 的价格是多少？
机器人：[回答]
用户：那产品 B 呢？
机器人：[理解是问价格，回答产品 B 价格]
用户：有优惠吗？
机器人：[理解是问产品 B 的优惠]
```

### 权限控制
**场景：不同人看不同内容**
```
普通员工：只能看公共知识库
管理人员：可以看管理文档
HR：可以看人事文档
```

**实现方法**：
- 用工作流判断用户身份
- 根据身份返回不同内容
- 或配置多个机器人
