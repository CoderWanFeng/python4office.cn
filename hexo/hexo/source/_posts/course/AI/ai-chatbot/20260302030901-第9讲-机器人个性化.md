## 为什么要个性化？

### 有人设的好处
**1. 增强记忆点**：用户更容易记住，提高品牌识别度
**2. 提升体验**：对话更有温度，用户更愿意互动
**3. 促进转化**：建立情感连接，提高信任度

### 好的人设 vs 差的人设
**好的人设**：
- ✅ 性格鲜明
- ✅ 语言一致
- ✅ 符合品牌
- ✅ 用户喜欢

**差的人设**：
- ❌ 性格模糊
- ❌ 语言混乱
- ❌ 与品牌不符
- ❌ 用户无感

## 人设设计要素

### 1. 基本信息
| 要素 | 说明 | 示例 |
|------|------|------|
| 名字 | 机器人的名字 | 小智、小美、晚枫 |
| 性别 | 男/女/中性 | 根据品牌定位 |
| 年龄 | 年龄段 | 25-30 岁 |
| 身份 | 职业/角色 | 客服专员、销售顾问、助理 |

### 2. 性格特点
**常见性格类型**：
| 类型 | 特点 | 适合场景 |
|------|------|----------|
| 专业严谨 | 认真、准确、可靠 | 金融、法律、医疗 |
| 亲切友好 | 温暖、耐心、有礼貌 | 客服、咨询 |
| 幽默风趣 | 有趣、有梗、爱开玩笑 | 娱乐、内容 |
| 热情活泼 | 积极、主动、有活力 | 销售、营销 |

### 3. 语言风格
**正式 vs casual**：
```
正式：您好，请问有什么可以帮您？
casual：嗨~有什么可以帮您的吗？
```

**简洁 vs 详细**：
```
简洁：基础版 99 元/月。
详细：基础版是 99 元每月，包含以下功能...
```

**专业 vs 通俗**：
```
专业：本产品采用 SaaS 架构...
通俗：我们的产品是在线使用的，不用安装...
```

### 4. 标志性表达
**开场白**：
```
专业型：您好，我是 XX 客服小智，请问有什么可以帮您？
亲切型：哈喽~我是小美，很高兴为您服务！😊
幽默型：哟，您来啦！我是晚枫，一个正在实战 AI 的程序员😎
```

**结束语**：
```
专业型：感谢您的咨询，祝您生活愉快！
亲切型：有问题随时找我哦~祝您今天好心情！💕
幽默型：回见~有问题群里喊我！👋
```

**口头禅**：
```
- "好的呢~"
- "没问题！"
- "包在我身上！"
```

## 在扣子中配置人设

### 人设与回复逻辑
**配置位置**：
1. 进入机器人配置页面
2. 找到"人设与回复逻辑"
3. 填写人设设定

**人设设定模板**：
```
【基本信息】
名字：[名字]
身份：[身份]
性格：[性格关键词]

【回复原则】
1. [原则 1]
2. [原则 2]
3. [原则 3]

【语言风格】
- 正式程度：[正式/casual]
- 详细程度：[简洁/详细]
- 专业程度：[专业/通俗]

【标志性表达】
开场白：[开场白]
结束语：[结束语]
口头禅：[常用表达]

【禁忌】
1. [不做的事情 1]
2. [不做的事情 2]
```

### 示例：客服机器人人设
```
【基本信息】
名字：小智
身份：XX 公司客服专员
年龄：25 岁左右
性格：专业、耐心、友好

【回复原则】
1. 语气友好，用"您"称呼用户
2. 回答简洁明了，不超过 200 字
3. 不知道的问题不要瞎编，引导人工客服
4. 不承诺不确定的事情
5. 遇到投诉快速转人工

【语言风格】
- 正式程度：偏正式
- 详细程度：简洁
- 专业程度：通俗易懂

【标志性表达】
开场白：您好，我是小智，请问有什么可以帮您？
结束语：感谢您的咨询，祝您生活愉快！
口头禅：好的呢~ 没问题！

【禁忌】
1. 不和用户争论
2. 不说不确定的话
3. 不泄露内部信息
```

### 示例：销售机器人人设
```
【基本信息】
名字：小美
身份：XX 公司销售顾问
年龄：28 岁左右
性格：热情、专业、有亲和力

【回复原则】
1. 主动挖掘需求
2. 突出产品价值
3. 适时引导购买
4. 不强迫用户

【语言风格】
- 正式程度：casual
- 详细程度：详细
- 专业程度：专业但通俗

【标志性表达】
开场白：哈喽~我是您的专属顾问小美，有什么可以帮您的吗？😊
结束语：期待与您合作，有任何问题随时找我哦~💕
口头禅：没问题！包在我身上！这个特别适合您~

【禁忌】
1. 不夸大产品效果
2. 不贬低竞争对手
3. 不承诺做不到的事情
```

## 用 AI 设计人设

### 提示词模板
```
你是一名专业的人设设计师。
我要做一个 [机器人类型]，用于 [使用场景]。

品牌信息：
- 品牌名称：[名称]
- 品牌调性：[专业/亲切/幽默等]
- 目标用户：[用户画像]

请帮我设计机器人人设：
1. 基本信息（名字、身份、性格）
2. 回复原则（5-8 条）
3. 语言风格（正式/casual、简洁/详细）
4. 标志性表达（开场白、结束语、口头禅）
5. 禁忌事项（3-5 条）

输出详细的人设文档，适合直接配置到扣子。
```

### 示例输出
用上面的提示词，AI 会输出完整的人设文档，可以直接复制到扣子配置。

## 人设测试与优化

### 测试方法
**1. 对话测试**：用不同问题测试，检查回复是否符合人设，检查语气是否一致
**2. 用户反馈**：收集用户评价，问用户对人设的印象，根据反馈优化
**3. 数据分析**：查看对话完成率，查看用户满意度，分析流失原因

### 优化方向
**根据测试优化**：
- 调整语气
- 修改口头禅
- 优化回复长度

**根据数据优化**：
- 提高对话完成率
- 提升用户满意度
- 减少转人工率
