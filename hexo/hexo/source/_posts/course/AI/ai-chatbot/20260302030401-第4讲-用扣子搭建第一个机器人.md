大家好，我是正在实战各种 AI 项目的程序员晚枫。

这一讲，手把手教你用扣子搭建第一个机器人。不需要任何技术，1 小时就能上线。

---

## 第一步：注册账号

1. 打开扣子官网：coze.cn
2. 点击"登录/注册"
3. 用抖音/头条/手机号注册
4. 完成实名认证

---

## 第二步：创建机器人

1. 登录后，点击"创建 Bot"
2. 填写基本信息：
   - 名字：[你的机器人名字]
   - 描述：[机器人的用途]
   - 头像：上传 Logo 或 AI 生成
3. 点击"创建"

---

## 第三步：配置人设和回复

### 人设设定
在"人设与回复逻辑"中，填写：

**示例（客服机器人）**：
```
名字：小智
身份：XX 公司客服专员
性格：专业、耐心、友好

回复原则：
1. 语气友好，有礼貌
2. 回答简洁明了
3. 不知道的问题引导人工客服
4. 不承诺不确定的事情

开场白：您好，我是小智，请问有什么可以帮您？
```

### 用 AI 生成人设
**提示词**：
```
你是一名专业的机器人设计师。
我要做一个客服机器人，用于 [行业]。
请帮我写人设设定：
- 名字、身份、性格
- 回复原则（5 条左右）
- 开场白和结束语
要求：适合 [目标用户]，语气 [专业/亲切]
```

---

## 第四步：添加知识库

### 方法 1：手动添加
1. 点击"知识库"
2. 点击"添加知识"
3. 选择"文本"
4. 填写问题和答案

**示例**：
```
问题：你们的产品多少钱？
答案：我们的产品价格如下：
- 基础版：99 元/月
- 专业版：199 元/月
- 企业版：面议
具体可以根据您的需求推荐合适的版本。
```

### 方法 2：批量导入
1. 准备 Excel 文件
2. 格式：A 列问题，B 列答案
3. 点击"批量导入"
4. 上传文件

### 方法 3：用 AI 生成
**提示词**：
```
你是一名专业的客服培训专家。
我是一家 [行业] 公司，产品/服务是 [描述]。
请帮我生成 20 个常见问题和答案：
- 覆盖产品、价格、服务、售后
- 答案简洁明了，适合机器人回复
- 每条答案 100-200 字
输出表格格式，方便导入。
```

---

## 第五步：配置工作流（可选）

### 什么是工作流？
**工作流**：机器人的处理逻辑

**简单场景**：不需要工作流，用知识库就够了

**复杂场景**：需要工作流，比如：
- 需要调用外部 API
- 需要多轮对话
- 需要条件判断

### 配置简单工作流
**示例：预约机器人**
1. 点击"工作流"
2. 点击"创建工作流"
3. 添加节点：开始 → 获取用户姓名 → 获取预约时间 → 获取联系方式 → 确认预约 → 结束
4. 配置每个节点的回复

---

## 第六步：调试和测试

### 在线调试
1. 点击"调试与预览"
2. 在右侧对话框测试
3. 检查回复是否准确

### 测试清单
- [ ] 常见问题能正确回答
- [ ] 开场白正常显示
- [ ] 多轮对话流畅
- [ ] 不知道的问题有引导
- [ ] 语气和人设一致

### 优化建议
根据测试结果优化：
- 补充知识库
- 调整人设
- 优化回复

---

## 第七步：发布机器人

### 选择发布渠道
1. 点击"发布"
2. 选择发布渠道：
   - 抖音
   - 飞书
   - 微信（需要额外配置）
   - 网页嵌入

### 发布到抖音
1. 选择"抖音"
2. 关联抖音账号
3. 设置为私信机器人或评论区机器人
4. 点击"发布"

### 发布到飞书
1. 选择"飞书"
2. 关联飞书账号
3. 创建飞书机器人
4. 邀请到群聊

### 发布到微信
**注意**：扣子不直接支持微信，需要：
- 用腾讯云 HiFlow 中转
- 或用企业微信

### 网页嵌入
1. 选择"网页嵌入"
2. 复制嵌入代码
3. 粘贴到网站 HTML 中

---

## 本讲作业

1. **创建第一个机器人**：注册扣子账号，创建机器人，配置人设和知识库，测试并发布
2. **准备知识库**：整理 20 个常见问题，写标准答案，导入到机器人
3. **测试和优化**：测试 10 个以上问题，记录回答不准确的问题，优化知识库
