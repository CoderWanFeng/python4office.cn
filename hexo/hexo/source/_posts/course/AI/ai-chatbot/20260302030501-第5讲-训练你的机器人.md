## 理解机器人的"学习"机制

### 机器人如何"学习"？
**扣子机器人**的学习方式：
1. **知识库匹配**：用户问题 → 匹配知识库 → 返回答案
2. **大模型理解**：没有匹配 → 大模型理解 → 生成回答

### 影响准确率的因素
**1. 知识库质量**：问题覆盖是否全面、答案是否准确、表达是否清晰
**2. 问题匹配**：用户问法和知识库问题是否匹配、同义词是否覆盖
**3. 大模型能力**：模型的理解能力、人设设定的清晰度

---

## 提升准确率的方法

### 方法 1：完善知识库
**问题分析**：
- 收集用户真实提问
- 找出回答不准确的问题
- 补充到知识库

**提示词**：
```
你是一名专业的客服培训专家。
我有以下用户提问和机器人回答：[粘贴对话记录]
请帮我分析：
1. 哪些回答不准确
2. 可能的原因是什么
3. 如何优化知识库
输出优化建议。
```

**优化步骤**：
1. 添加相似问法
2. 补充答案细节
3. 添加关联问题

**示例**：
```
原问题：你们的产品多少钱？
答案：基础版 99 元，专业版 199 元。

优化后：
问题 1：你们的产品多少钱？
问题 2：价格是多少？
问题 3：怎么收费？
问题 4：费用多少？
答案：我们的产品价格如下：
- 基础版：99 元/月，包含基础功能
- 专业版：199 元/月，包含全部功能
- 企业版：面议，有定制服务
可以根据您的需求推荐合适的版本。
```

### 方法 2：优化人设设定
**问题**：机器人回答太机械

**优化**：
```
原设定：
- 你是一个客服机器人
- 回答用户问题

优化后：
- 你是小智，XX 公司的客服专员
- 性格耐心友好，说话有温度
- 不知道的问题不要瞎编，引导人工客服
- 回答简洁，不要长篇大论
- 用"您"称呼用户，有礼貌
```

### 方法 3：添加反问逻辑
**场景**：用户需求不明确

**示例**：
```
用户：我想了解一下
    ↓
机器人：好的，请问您想了解哪方面呢？
1. 产品功能
2. 价格方案
3. 服务流程
4. 其他
    ↓
用户：价格
    ↓
机器人：好的，我们的价格如下...
```

**配置方法**：
1. 在工作流中添加条件判断
2. 设置反问回复
3. 根据用户选择继续对话

## 处理不知道的问题

### 策略 1：诚实承认
**设定**：
```
如果你不知道答案：
1. 诚实承认不知道
2. 不要瞎编
3. 引导用户联系人工客服
4. 留下联系方式
```

**回复示例**：
```
抱歉，这个问题我暂时不太清楚。
您可以联系我们的客服专员：
- 微信：xxxxx
- 电话：xxxxx
他们会为您详细解答。
```

### 策略 2：收集问题
**设定**：
```
如果遇到不知道的问题：
1. 记录用户问题
2. 告诉用户会后续回复
3. 转人工跟进
```

**回复示例**：
```
您好，这个问题我需要确认一下。
方便留下您的联系方式吗？
确认后会第一时间回复您。
```

### 策略 3：引导到其他话题
**设定**：
```
如果回答不了：
1. 承认不知道
2. 介绍能回答的问题
3. 引导用户提问
```

**回复示例**：
```
抱歉，这个问题我暂时回答不了。
不过我可以为您介绍：
- 产品功能
- 价格方案
- 服务流程
您想了解哪方面呢？
```

## 优化对话体验

### 技巧 1：控制回复长度
**原则**：
- 简短回复优先
- 复杂信息分条发送
- 避免大段文字

**设定**：
```
回复原则：
- 每条回复不超过 200 字
- 复杂信息分点列出
- 重要信息加粗或 emoji 标注
```

### 技巧 2：添加 emoji
**作用**：
- 增加亲和力
- 突出重点
- 让对话更生动

**示例**：
```
原回复：您好，我们的产品有三个版本。

优化后：您好！我们的产品有三个版本哦~ 😊
- 💚 基础版：99 元/月
- 💙 专业版：199 元/月
- 💜 企业版：面议
```

### 技巧 3：设置上下文记忆
**场景**：用户连续提问

**示例**：
```
用户：你们的产品多少钱？
机器人：[回答价格]

用户：那功能呢？
机器人：[应该理解"那"指的是产品，回答产品功能]
```

**配置方法**：
1. 在扣子中开启"多轮对话"
2. 设置上下文轮数（建议 3-5 轮）

## 数据分析与优化

### 查看对话数据
**扣子后台**：
1. 进入"数据分析"
2. 查看：总对话量、常见问题、unanswered 问题、用户满意度

### 分析未回答问题
**步骤**：
1. 导出未回答问题列表
2. 分析问题类型
3. 补充到知识库

**提示词**：
```
你是一名专业的数据分析师。
我有以下用户未回答问题列表：[粘贴问题列表]
请帮我分析：
1. 问题分类（产品/价格/服务/其他）
2. 哪些问题应该补充到知识库
3. 优先级排序（高频/重要）
输出表格格式。
```

### 持续优化
**每周优化**：
- 查看对话数据
- 补充新问题
- 优化不准确回答

**每月优化**：
- 分析整体效果
- 更新产品信息
- 优化人设和回复
