## 机器人类型

### 按功能分类

**1. 问答机器人**
- 回答用户问题
- 基于知识库
- 示例：客服机器人、知识问答机器人

**2. 任务机器人**
- 帮用户完成任务
- 需要调用外部服务
- 示例：预约机器人、下单机器人

**3. 聊天机器人**
- 陪用户聊天
- 需要有趣的人设
- 示例：情感陪伴机器人、IP 人设机器人

**4. 营销机器人**
- 引导用户购买
- 需要销售话术
- 示例：销售机器人、带货机器人

## 对话流程设计

### 基本对话结构
```
用户：[问题/需求]
    ↓
机器人：[理解意图]
    ↓
机器人：[回复/追问]
    ↓
用户：[进一步信息]
    ↓
机器人：[完成任务/回答问题]
```

### 简单问答流程
```
用户：你们的产品多少钱？
    ↓
机器人：我们的产品价格如下：
        - 基础版：99 元/月
        - 专业版：199 元/月
        - 企业版：面议
        请问您想了解哪个版本？
```

### 多轮对话流程
```
用户：我想预约演示
    ↓
机器人：好的，请问您贵姓？
    ↓
用户：我姓王
    ↓
机器人：王先生您好，请问什么时间方便？
    ↓
用户：明天下午 2 点
    ↓
机器人：好的，已为您预约明天下午 2 点。请问您的电话是？
```

### 用 AI 设计对话流程
**提示词**：
```
你是一名专业的对话设计师。
我要做一个 [机器人类型]，用于 [使用场景]。
目标用户是：[用户画像]
主要功能是：[功能列表]
请帮我设计对话流程：
1. 列出常见的用户问题（10-20 个）
2. 为每个问题设计回复
3. 设计多轮对话逻辑
4. 标注可能的分支情况
输出表格格式。
```

## 知识库准备

### 什么是知识库？
**知识库**：机器人回答问题的依据

**知识库包含**：
- 常见问题（FAQ）
- 产品信息
- 服务流程
- 价格政策
- 联系方式

### 知识库结构
```
知识库/
├── 产品知识/
│   ├── 产品介绍
│   ├── 功能说明
│   ├── 价格政策
│   └── 使用教程
├── 服务知识/
│   ├── 服务流程
│   ├── 常见问题
│   ├── 售后政策
│   └── 联系方式
└── 公司知识/
    ├── 公司介绍
    ├── 团队信息
    └── 荣誉资质
```

### 用 AI 整理知识库
**提示词**：
```
你是一名专业的知识整理专家。
我有以下零散的资料：[粘贴资料]
请帮我整理成知识库：
1. 分类整理（产品/服务/公司）
2. 提取常见问题（FAQ）
3. 为每个问题写标准答案
4. 输出结构化文档
要求：答案简洁明了，适合机器人回复，每条答案 100-200 字。
```

## 机器人人设打造

### 为什么要有人设？
**有人设的好处**：
- 更有温度，不像机器
- 用户更容易记住
- 增强品牌识别度

### 人设要素
**1. 基本信息**：名字、性别、年龄（可选）、职业/身份
**2. 性格特点**：专业严谨、亲切友好、幽默风趣、热情活泼
**3. 语言风格**：正式/casual、简洁/详细、专业/通俗
**4. 口头禅/标志性表达**：开场白、结束语、常用表达

### 人设示例

**示例 1：专业客服**
```
名字：小智
身份：XX 公司客服专员
性格：专业、耐心、友好
语言风格：正式、清晰、有礼貌
开场白：您好，我是小智，请问有什么可以帮您？
结束语：感谢您的咨询，祝您生活愉快！
```

**示例 2：销售顾问**
```
名字：小美
身份：XX 公司销售顾问
性格：热情、专业、有亲和力
语言风格：亲切、有说服力
开场白：哈喽~我是您的专属顾问小美，有什么可以帮您的吗？
结束语：期待与您合作，有任何问题随时找我哦~
```

**示例 3：IP 人设**
```
名字：晚枫
身份：AI 编程讲师
性格：幽默、专业、接地气
语言风格：casual、有梗、爱用 emoji
开场白：嗨，我是晚枫，一个正在实战 AI 项目的程序员😎
结束语：有问题随时找我，咱们群里见~👋
```

### 用 AI 设计人设
**提示词**：
```
你是一名专业的人设设计师。
我要做一个 [机器人类型]，用于 [使用场景]。
目标用户是：[用户画像]
品牌调性是：[专业/亲切/幽默等]
请帮我设计机器人人设：
1. 基本信息（名字、身份）
2. 性格特点（3-5 个关键词）
3. 语言风格（正式/casual、简洁/详细）
4. 标志性表达（开场白、结束语、口头禅）
输出详细的人设文档。
```

## 本讲作业

1. **确定机器人类型**：根据你的需求，确定机器人类型（问答/任务/聊天/营销），列出主要功能（3-5 个），描述使用场景
2. **设计对话流程**：列出 10-20 个常见问题，为每个问题设计回复，设计 2-3 个多轮对话场景
3. **准备知识库**：整理产品/服务资料，提取常见问题（FAQ），为每个问题写标准答案
4. **设计机器人人设**：确定基本信息，设计性格特点，设计语言风格，写开场白和结束语
