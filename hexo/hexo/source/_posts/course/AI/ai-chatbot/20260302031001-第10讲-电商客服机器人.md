## 电商客服场景

### 常见问题类型
**1. 商品咨询（40%）**
- 这个有货吗？
- 什么尺寸？
- 什么材质？
- 有什么颜色？

**2. 物流查询（25%）**
- 什么时候发货？
- 发什么快递？
- 几天能到？
- 我的快递到哪了？

**3. 售后问题（20%）**
- 能退货吗？
- 怎么换货？
- 质量问题怎么办？
- 保修多久？

**4. 优惠活动（10%）**
- 有优惠吗？
- 能便宜点吗？
- 有什么活动？
- 会员有什么优惠？

**5. 其他（5%）**
- 发票怎么开？
- 能货到付款吗？
- 怎么联系客服？

## 知识库准备

### 商品知识
**模板**：
```
商品名称：[名称]
价格：[价格]
库存：[有货/缺货]
尺寸：[尺寸信息]
材质：[材质信息]
颜色：[颜色选项]
发货时间：[发货时间]
快递：[快递公司]
```

**示例**：
```
商品名称：XX 品牌 T 恤
价格：99 元
库存：有货
尺寸：S/M/L/XL
材质：100% 纯棉
颜色：白色/黑色/灰色
发货时间：24 小时内发货
快递：默认中通，可备注其他
```

### 物流知识
**模板**：
```
发货时间：[发货时间]
快递公司：[快递列表]
配送时效：[各地区时效]
运费：[运费政策]
物流查询：[查询方式]
```

**示例**：
```
发货时间：付款后 24 小时内发货
快递公司：默认中通，可发圆通/申通/顺丰
配送时效：
- 江浙沪：1-2 天
- 其他地区：3-5 天
- 偏远地区：5-7 天
运费：满 99 包邮，不满 10 元
顺丰：到付或补差价
```

### 售后知识
**模板**：
```
退货政策：[退货条件]
换货政策：[换货条件]
保修政策：[保修期限]
质量问题：[处理流程]
联系方式：[客服联系方式]
```

**示例**：
```
退货政策：
- 7 天无理由退货
- 商品未使用，吊牌完整
- 买家承担退货运费

换货政策：
- 质量问题免费换货
- 非质量问题买家承担运费

保修政策：
- 非人为损坏保修 1 年
- 人为损坏付费维修

质量问题处理：
1. 拍照留证
2. 联系客服
3. 寄回检测
4. 换货或退款

客服联系方式：
- 微信：xxxxx
- 电话：xxxxx
- 工作时间：9:00-21:00
```

### 用 AI 生成知识库
**提示词**：
```
你是一名专业的电商客服专家。
我是一家 [行业] 店铺，主要卖 [产品]。
请帮我生成电商客服知识库：
1. 商品知识（20 个常见问题）
2. 物流知识（10 个常见问题）
3. 售后知识（15 个常见问题）
4. 优惠活动（5 个常见问题）
要求：
- 问题真实，是用户常问的
- 答案简洁，适合机器人回复
- 每条答案 100 字以内
输出表格格式，方便导入。
```

## 工作流设计

### 场景 1：商品咨询
**流程**：
```
用户：这个有货吗？
    ↓
识别商品（用户发链接或说名称）
    ↓
查询库存
    ↓
回复库存状态
    ↓
引导下单
```

### 场景 2：物流查询
**流程**：
```
用户：我的快递到哪了？
    ↓
询问订单号/手机号
    ↓
调用物流 API
    ↓
返回物流状态
    ↓
安抚用户（如有延迟）
```

### 场景 3：售后处理
**流程**：
```
用户：我要退货
    ↓
询问退货原因
    ↓
判断是否符合退货政策
    ↓
符合：发送退货地址和流程
    ↓
不符合：解释原因，提供其他方案
```

## 扣子配置

### 人设设定
```
【基本信息】
名字：小淘
身份：XX 店铺客服专员
性格：热情、耐心、专业

【回复原则】
1. 语气亲切，用"亲"称呼用户
2. 回答简洁，突出关键信息
3. 主动引导下单
4. 遇到投诉快速转人工

【语言风格】
- 电商风格，用"亲"
- 适当用 emoji
- 热情但不骚扰

【标志性表达】
开场白：亲，您好~我是小淘，有什么可以帮您的吗？😊
结束语：感谢亲的支持，祝您购物愉快！💕
口头禅：亲~ 没问题的~ 包您满意~
```

### 知识库配置
1. 整理商品知识
2. 整理物流知识
3. 整理售后知识
4. 批量导入扣子

### 工作流配置
**订单查询工作流**：
```
触发：用户问订单/物流
    ↓
询问订单号
    ↓
调用物流 API
    ↓
返回物流信息
    ↓
结束
```

**售后处理工作流**：
```
触发：用户说退货/换货
    ↓
询问原因
    ↓
判断类型
    ↓
质量问题是 → 发送退货地址
    ↓
非质量问题 → 解释政策
    ↓
结束
```

## 实战案例

### 案例 1：服装店客服机器人
**背景**：淘宝服装店，每天 200+ 咨询

**配置**：
- 商品知识：50 个商品，每个 10 个问题
- 物流知识：10 个常见问题
- 售后知识：15 个常见问题

**效果**：
- 自动回答率：85%
- 转人工率：15%
- 客户满意度：4.8/5

### 案例 2：美妆店客服机器人
**背景**：拼多多美妆店，每天 500+ 咨询

**配置**：
- 商品知识：100 个商品
- 肤质推荐工作流
- 成分查询 API

**效果**：
- 自动回答率：80%
- 转化率提升：20%
- 客服人力节省：60%
