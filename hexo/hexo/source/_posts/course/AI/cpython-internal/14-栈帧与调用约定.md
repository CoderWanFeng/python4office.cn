## 📚 栈帧结构详解

```c
struct _frame {
    PyObject_VAR_HEAD
    
    // 链式结构
    struct _frame *f_back;      // 调用者的栈帧
    PyCodeObject *f_code;       // 代码对象
    PyObject *f_builtins;       // 内置命名空间
    PyObject *f_globals;        // 全局命名空间
    PyObject *f_locals;         // 局部命名空间
    
    // 执行状态
    PyObject **f_valuestack;    // 值栈底
    PyObject **f_stacktop;      // 值栈顶
    
    // 指令指针
    int f_lasti;                // 最后执行的指令偏移
    
    // 局部变量 + 单元格（闭包）
    PyObject *f_localsplus[1];  // 变长数组
};
```

---

## 🔄 调用过程

```
调用函数 → 创建新栈帧 → 压入栈 → 执行字节码 → 返回结果 → 弹出栈帧
```

**栈帧结构**：链式结构，包含命名空间、执行状态。

**调用过程**：创建→执行→销毁的完整流程。
