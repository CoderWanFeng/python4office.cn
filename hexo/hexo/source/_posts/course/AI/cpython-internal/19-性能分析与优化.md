## 🔍 cProfile

```python
import cProfile
import pstats

# 分析代码
profiler = cProfile.Profile()
profiler.enable()
# ... 你的代码 ...
profiler.disable()

# 输出结果
stats = pstats.Stats(profiler)
stats.sort_stats('cumulative')
stats.print_stats(20)  # 显示前 20 个
```

## ⚡ 常见优化技巧

| 技巧 | 效果 |
