**函数和类在 C 层面是什么结构？理解 Python 的对象系统。**

---

## 🔧 PyFunctionObject

```c
// Include/funcobject.h
typedef struct {
    PyObject_HEAD
    PyObject *func_code;        // 代码对象
    PyObject *func_globals;     // 全局命名空间
    PyObject *func_defaults;    // 默认参数
    PyObject *func_closure;     // 闭包变量
    PyObject *func_doc;         // 文档字符串
} PyFunctionObject;
```

---

## 🏛️ PyTypeObject

```c
// 类型对象的结构
typedef struct _typeobject {
    PyObject_VAR_HEAD
    const char *tp_name;        // 类型名称
    Py_ssize_t tp_basicsize;    // 实例大小
    
    // 方法族
    PyNumberMethods *tp_as_number;
    PySequenceMethods *tp_as_sequence;
    PyMappingMethods *tp_as_mapping;
    
    // 属性访问
    getattrfunc tp_getattr;
    setattrfunc tp_setattr;
} PyTypeObject;
```

---

## 🎯 本讲总结

**PyFunctionObject**：函数对象的 C 结构体。

**PyTypeObject**：类型对象的结构，定义类型特性。
