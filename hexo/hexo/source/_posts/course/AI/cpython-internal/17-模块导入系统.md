## 🔍 导入流程

```
import foo
    ↓
1. 检查 sys.modules（已导入缓存）
    ↓
2. 创建模块对象
    ↓
3. 执行模块代码
    ↓
4. 加入 sys.modules
    ↓
5. 绑定到本地命名空间
```

---

## 🎯 Finder 与 Loader

```python
# Python 层面的导入钩子
import sys

class MyFinder:
    def find_module(self, fullname, path=None):
        print(f"Finding {fullname}")
        return None  # 没找到

sys.meta_path.insert(0, MyFinder())
```

---

## 🎯 本讲总结

**导入流程**：缓存检查→创建→执行→绑定。

**Finder 与 Loader**：自定义导入钩子。
