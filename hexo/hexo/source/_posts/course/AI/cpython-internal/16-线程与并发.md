## 🧵 PyThreadState

```c
// Include/pystate.h
typedef struct _ts {
    struct _ts *prev;           // 线程链表
    struct _ts *next;
    PyInterpreterState *interp; // 解释器状态
    
    struct _frame *frame;       // 当前栈帧
    int recursion_depth;        // 递归深度
    
    // 异常状态
    PyObject *exc_type;
    PyObject *exc_value;
    PyObject *exc_traceback;
    
    // 线程 ID
    unsigned long thread_id;
} PyThreadState;
```

---

## 🔄 线程调度

```
线程 A 执行 → 遇到 IO/时间片用完 → 释放 GIL
                ↓
            线程调度器
                ↓
线程 B 获取 GIL → 开始执行
```

---

## 🎯 本讲总结

**PyThreadState**：线程状态结构体。

**线程调度**：GIL 释放与获取的交互。
