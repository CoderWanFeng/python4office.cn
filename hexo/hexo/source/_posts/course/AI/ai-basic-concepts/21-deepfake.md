## 🎯 一句话先说清楚

::: tip 核心结论
**AI换脸 = 用AI把你的脸"贴"到别人身上**

AI换脸就是用AI技术，把A的脸"换"到B的脸上，做出假的视频。以前这是专业特效团队才能做的事，现在随便一个人用手机App就能做，而且真假难辨。
:::

## 🤔 什么是Deepfake？

### 名字的由来

**Deepfake = Deep Learning（深度学习）+ Fake（伪造）**

用深度学习做的假视频/假图片。

### AI换脸的本质

```
目标视频：某明星在说话
  ↓
AI提取人脸特征
  ↓
替换成你的脸
  ↓
输出：你"说"了明星说的话
```

## 🔧 AI换脸是怎么做的？

### Step 1：收集照片

```
上传你的照片（或从网上找）
AI需要几十到几百张照片来"认识"这张脸

照片要求：
- 正脸、侧脸都要有
- 不同光线、不同角度
- 眼睛、嘴巴要清晰
```

### Step 2：训练模型

```
AI开始学习这张脸的特征：
  ↓
眼睛多大、什么形状
  ↓
鼻子什么形状、多高
  ↓
腮型什么样
  ↓
脸型、肤色、表情特点

训练时间：几分钟到几小时
```

### Step 3：生成换脸视频

```
把目标视频里的人脸
  ↓
替换成"你的脸"
  ↓
调整光线、表情、口型
  ↓
输出：看起来真假的假视频

生成时间：几分钟
```

## ⚠️ AI换脸的危害

::: danger 这是重点！以下行为违法！
:::

### 1. 诈骗

```
案例：
朋友发视频说"我出事了，急用钱"
  ↓
其实是AI换的假视频
  ↓
骗钱成功

如何防范：
- 涉及钱的事，一定要打电话确认
- 问个只有你们俩知道的问题
- 不要轻信视频内容
```

### 2. 假新闻

```
案例：
名人"说"了从没说过的话
  ↓
假新闻视频满天飞
  ↓
影响舆论

如何防范：
- 从官方渠道确认
- 看发布时间和来源
- 不要轻信短视频
```

### 3. 谣言

```
案例：
用假视频抹黑别人
  ↓
毁人名誉
  ↓
可能涉及法律问题
```

### 4. 侵权

```
案例：
用AI换脸制作假色情视频
  ↓
受害者完全不知情
  ↓
严重侵犯肖像权
```

## 🔍 如何识别AI换脸？

### 实用技巧

**1. 看脸和脖子交界处**

```
假视频：
- 边缘模糊
- 颜色不自然
- 有明显的"拼接感"
```

**2. 看眨眼/表情**

```
假视频：
- 眨眼不自然
- 表情僵硬
- 口型和声音不匹配
```

**3. 看光线**

```
假视频：
- 脸部光线和背景不协调
- 阴影方向不一致
- 脸部反光异常
```

**4. 问只有真人才知道的事**

```
视频通话时：
- 问个只有你们俩知道的问题
- 问近期发生的事
- 看对方能否回答
```

**5. 百度/抖音搜搜图**

```
截图搜索：
- 看是不是旧图被P的
- 看有没有原图
- 看是否是二次加工
```

## 🎯 实战技巧：识别AI换脸的方法

### 技巧1：一招保命——约定家庭暗号

```
最好的防骗方法：约定只有家人知道的暗号

家庭暗号示例：
- "咱家那只狗叫什么名字？"
- "上次暑假我们去哪玩了？"
- "妈/爸的生日是什么时候？"

遇到借钱/要敏感信息的电话：
→ 问这个暗号
→ AI声音再像，也回答不出只有真人才知道的事
```

### 技巧2：多重验证

```
遇到"领导/老师/朋友"突然找你：

❌ 直接相信 → 可能被骗

✅ 多重验证：
   1. 问只有双方知道的问题
   2. 挂断，主动打回去（用通讯录号码）
   3. 不要回拨刚才的号码
   4. 涉及钱的事情，视频+电话+文字都要确认
```

### 技巧3：看细节辨真假

```
AI换脸破绽（仔细看）：

1. 眼睛：
   → 眨眼频率不对
   → 眼神光不自然
   → 瞳孔大小不一致

2. 脸和脖子边缘：
   → 可能有模糊/色差
   → 头发边缘不自然

3. 说话时：
   → 嘴型可能不同步
   → 说话时表情僵硬

4. 光线：
   → 脸上光线方向不一致
   → 阴影不自然

5. 手：
   → 手部经常出问题（多手指、变形）
```

### 技巧4：警惕这些场景

```
🚨 极度紧急的情况：
   "快！赶紧转账，不然来不及了！"
   → 越紧急越要冷静，越可能是诈骗

🚨 涉及钱：
   "转账/汇款/支付宝/微信"
   → 任何涉及钱的要求，必须多重验证

🚨 不正常的请求：
   "这件事不要告诉别人"
   → 诈骗分子常用这话术，不让你核实
```
