## 🎯 一句话先说清楚

::: tip 核心结论
**AI画图 = 你说一句话，AI画一幅画**

AI画图就是：你输入一段文字描述（Prompt），AI就能生成一张图片。你说"一只穿西装的猫在弹钢琴"，AI就给你画出来。不需要你会画画，1分钟出图。
:::

## 🎨 AI画图是什么？

### 以前做图 vs 现在做图

**以前：**
```
需要一张图 → 找设计师
  ↓
沟通需求（几天）
  ↓
等待设计（几天到几周）
  ↓
修改调整（几轮）
  ↓
终于拿到图
```

**现在：**
```
需要一张图 → 输入描述
  ↓
等待1分钟
  ↓
拿到图
```

**效率提升了几十倍！**

## ✍️ 描述词（Prompt）很重要

**AI画图的质量，90%取决于你的描述。**

### 好的描述

```
正面示例：
一只可爱的柴犬，橘色背景，穿着蓝色和服，
坐在樱花树下，樱花飘落，动漫风格，
新海诚画风，8K高清，细节丰富，光线柔和
```

**包含要素：**
- 主体：柴犬
- 细节：穿着和服
- 背景：樱花树下
- 风格：动漫风格、新海诚画风
- 质量：8K高清、细节丰富、光线柔和

### 糟糕的描述

```
负面示例：
狗
```

**问题：** 太简单，AI不知道你想要什么！

## ⚠️ AI画图的问题

### 1. 👌 手指问题

```
问题：AI画手经常多一根或少一根手指

解决：在描述词里加上"perfect hands"等
      或后期PS修复
```

### 2. ✏️ 文字问题

```
问题：AI画文字经常写错

解决：不要让AI画文字
      生成图后，用PS加文字
```

### 3. 📜 版权问题

```
问题：AI画的图能不能商用？

解决：各平台政策不同，注意看用户协议
      Midjourney：付费用户可商用
      Stable Diffusion：开源，相对宽松
```

**❌ 不完全对！**

## 🎯 实战技巧：AI画图提示词进阶指南

### 技巧1：主体+环境+风格+光效四要素

```
完美提示词 = 主体 + 环境 + 风格 + 光效

✅ 好例子：
"一只柴犬（主体），穿着日式厨师服，在东京街头（环境），
动漫风格，宫崎骏画风（风格），黄金时段柔和光线（光效）"

❌ 差例子：
"柴犬"
→ AI随机发挥，效果不可控
```

### 技巧2：用负面提示词避免瑕疵

```
所有AI画图工具都支持"负面提示词"（不想出现的内容）：

常用负面提示词：
- 畸形、变形
- 低质量
- 多余的手指
- 文字、错误的水印
- 模糊、失焦

示例：
"一只帅气的男人" + "负面提示词：畸形、低质量"
→ 减少生成畸形图片的概率

效果：废片率降低50%
```

### 技巧3：保持角色一致性

```
场景：做一个故事系列，需要同一个角色

方法：
1. 先生成一张满意的角色图
2. 保存这张图的"Seed值"或"参考图"
3. 后续所有图都用这个参考

效果：同一个角色在不同场景里长得一样
```

### 技巧4：图生图控制细节

```
场景：有一张图，想让AI参考风格但不完全一样

方法：
1. 上传原图
2. 调整"参考强度"（0-100%）
3. 写新的提示词

参考强度：
- 0% = 全新创作
- 30% = 参考风格，形态自由
- 70% = 接近原图，只改细节
- 100% = 几乎一样

效果：既保留参考，又有新创作
```
