## 🎯 一句话先说清楚

::: tip 核心结论
**AI客服 = 24小时不睡觉的机器人客服**

AI客服就是用AI代替人工客服，自动回答客户问题。它24小时在线，永不疲倦，还能同时接待上万人。现在你打电话/上网问客服，背后可能已经是一个AI在回答你了。

```
人工客服的工作时间：8小时/天
```

## 💬 AI客服能做什么？

### 1. 回答常见问题

```
客户：你们的退货政策是什么？
AI：我们支持7天无理由退货，运费由我们承担，
    请在收到商品7天内申请。

客户：怎么申请售后？
AI：您可以：1.打开APP → 2.我的 → 3.订单 → 
    4.选择订单 → 5.申请售后
```

### 2. 引导客户操作

```
客户：怎么改收货地址？
AI：请按以下步骤操作：
    1. 点击"我的订单"
    2. 选择需要修改的订单
    3. 点击"修改地址"
    4. 输入新地址并确认

（还能直接帮客户跳转到对应页面）
```

### 3. 处理投诉

```
客户：东西坏了！太差了！
AI：非常抱歉给您带来困扰，我完全理解您的心情。
    我已经帮您创建了一个售后工单，工单号是CS20240416001。
    稍后会有专人联系您处理，预计24小时内解决。
```

### 4. 主动营销

```
客户：最近有什么优惠？
AI：您好！您是我们尊贵的VIP会员，
    正好有一张100元优惠券即将过期（4月30日失效）。
    要不要看看适用商品？这次全场满500可用。
```

## ⚠️ AI客服的局限性

### AI做不好的事

```
❌ 情绪化投诉
客户：你们太恶心了！我要投诉到底！
AI：抱歉...（客户更生气了）
需要：人共情、安抚、解决情绪

❌ 复杂情况
客户：我订单寄错了，但已经拆了...
需要：多方协调、特批处理

❌ 特殊情况
超出知识库，AI"不知道怎么办"

❌ 议价/砍价
客户：能不能再便宜点？
需要：谈判技巧、灵活决策
```

### 最佳方案：人机协作

```
AI处理80%的常见问题
  ↓
遇到复杂问题 → 转人工客服
  ↓
人工客服专注处理20%的复杂问题
  ↓
效率最高，成本最低
```

## 🔥 新闻里那些AI客服术语，到底是什么意思？

### "AI客服替代300个人工"
= 某电商用AI客服，减少了大部分人工客服

**原因：**
- 80%的问题是常见的，AI能处理
- 节省大量人力成本
- 人工专注处理复杂问题

### "AI客服被骂哭"
= AI处理不了情绪化问题，引发投诉

**问题：**
- AI只会按规则回答
- 没有人情味
- 情绪化客户更生气

**解决：** 遇到情绪问题，及时转人工。

## ⚠️ 常见误区避坑

### ❌ 误区1："AI客服能完全替代人工"
**❌ 错！**
- AI处理不了复杂问题
- AI没有人情味
- 最佳方案：AI+人工协作

### ❌ 误区2："AI客服不需要维护"
**❌ 错！**
- 需要更新知识库
- 需要优化回答质量
- 需要监控性能

### ❌ 误区3："AI客服成本极低"
**❌ 不完全对！**
- 初期开发/购买成本高
- 需要持续优化
- 但长期看成本远低于人工
