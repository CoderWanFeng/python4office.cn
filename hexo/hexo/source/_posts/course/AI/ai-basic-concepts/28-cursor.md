# 什么是Cursor？

**Cursor = AI帮你写代码的编辑器**

> **作者：程序员晚枫**

::: tip 一句话解释
Cursor是一个代码编辑器，但内置了AI。你写代码时AI帮你自动补全、帮你找bug、帮你解释代码、甚至帮你写整个功能。程序员用它，写代码速度翻倍，不会的直接问AI。
:::

## Cursor是什么？

### 类比理解

```
普通编辑器 = 打字机
  → 你写什么，它显示什么
  
VS Code = 智能打字机
  → 有代码高亮、自动补全

Cursor = 会写代码的助手
  → 你说需求，AI帮你写
  → 不会的代码，AI帮你解释
  → 出了bug，AI帮你修
```

### 为什么火？

```
以前写代码：
1. 打开搜索引擎
2. 搜"怎么实现XXX功能"
3. 看Stack Overflow答案
4. 复制粘贴改一改
5. 报错继续搜...
时间：30分钟起步

Cursor：
1. 描述你要做什么
2. AI给你代码
3. 直接用
时间：5分钟
```

## Cursor vs 其他工具

| 对比 | 普通编辑器 | GitHub Copilot | Cursor |
|------|-----------|----------------|--------|
| 代码补全 | 基础关键词 | AI补全整行 | AI补全整块 |
| 写新功能 | 自己写 | AI写一部分 | AI写完整功能 |
| 找bug | 自己debug | 有限 | 很强 |
| 聊天解释 | 不能 | 不能 | 能 |
| 价格 | 免费 | $10/月 | $20/月 |

## Cursor核心功能

### 1. AI自动补全（Tab键）
```
你写：def calculate
Cursor自动补全：
def calculate_total(items):
    return sum(item['price'] for item in items)
→ 按Tab直接接受
```

### 2. AI聊天（Ctrl+L）
```
你问：这段代码为什么报错？
AI答：因为类型不一致，
      第23行需要转成int

你问：帮我写一个用户登录功能
AI答：给出一套完整的登录代码
```

### 3. AI自动改代码（Apply）
```
选中代码 → 说"改成XXX"
→ AI自动改完
→ 不满意可以继续改
```

### 4. 代码理解
```
光标停在任何代码上
→ AI解释这段代码是干嘛的
→ 不用问同事了
```

```
✅ 程序员：主力编程工具，效率翻倍
✅ 学生：写作业、毕业设计神器
✅ 转行者：不会就问AI，快速学习
✅ 产品经理：想验证想法，自己写demo
❌ 非技术人员：不如用无代码工具
```

## 为什么Cursor比Copilot更火？

```
GitHub Copilot（微软）：
  → 补全做得好，但只是补全
  → 工具思维：帮你写，但还是要你主导

Cursor：
  → 补全 + 聊天 + 修改 融为一体
  → 助手思维：AI主导，你审核
  → 更符合"AI写代码"的体验
```

## 使用技巧

### 新手用法
```
1. 打开Cursor → 新建文件
2. 按Ctrl+L打开AI聊天
3. 说"帮我写一个计算器"
4. 拿走用
```

### 进阶用法
```
1. 光标放在报错代码上
2. 按Ctrl+K说"修复这个bug"
3. AI自动修复 → Tab接受

批量修改：
1. 选中多行代码
2. 按Ctrl+K说"把这里的变量名改成驼峰命名"
3. 批量改完
```

### 高手用法
```
1. 打开整个项目
2. 按Ctrl+L问"这个项目的架构是怎样的"
3. AI分析整个项目
4. 继续问"帮我加一个新功能XXX"
5. AI理解上下文，生成符合项目风格的代码
```

## Cursor的局限性

```
✅ 速度快：秒级生成代码
✅ 质量高：符合编程规范
✅ 易上手：会说话就会用

❌ 需要审核：AI写的代码可能有bug
❌ 复杂逻辑：AI理解有时会出错
❌ 需要基础：完全不懂编程还是有点难用
```
