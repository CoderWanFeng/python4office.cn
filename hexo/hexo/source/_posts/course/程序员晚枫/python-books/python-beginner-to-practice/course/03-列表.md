## 1. 什么是列表

列表是有序的**可变**序列，可以存储任意类型的元素。

```python
bicycles = ['trek', 'cannondale', 'redline', 'specialized']
print(bicycles)
# ['trek', 'cannondale', 'redline', 'specialized']
```

> 官方文档：[5.1. More on Lists](https://docs.python.org/3/tutorial/datastructures.html#more-on-lists) — 列表是可变序列。

## 2. 访问列表元素

```python
bicycles = ['trek', 'cannondale', 'redline', 'specialized']

print(bicycles[0])        # trek（首字母大写）
print(bicycles[0].title())  # Trek

print(bicycles[-1])       # specialized（最后一个）
print(bicycles[-2])       # redline（倒数第二个）
```

## 3. 修改列表元素

```python
motorcycles = ['honda', 'yamaha', 'suzuki']
print(motorcycles)  # ['honda', 'yamaha', 'suzuki']

motorcycles[0] = 'ducati'  # 修改第一个元素
print(motorcycles)  # ['ducati', 'yamaha', 'suzuki']
```

## 4. 添加元素

```python
# 末尾添加
motorcycles.append('ducati')
print(motorcycles)  # ['honda', 'yamaha', 'suzuki', 'ducati']

# 指定位置插入
motorcycles.insert(0, 'bmw')
print(motorcycles)  # ['bmw', 'honda', 'yamaha', 'suzuki', 'ducati']
```

## 5. 删除元素

```python
motorcycles = ['honda', 'yamaha', 'suzuki', 'ducati']

# del 删除（不返回值）
del motorcycles[0]
print(motorcycles)  # ['yamaha', 'suzuki', 'ducati']

# pop 删除末尾（或指定位置），返回被删元素
motorcycles = ['honda', 'yamaha', 'suzuki']
popped = motorcycles.pop()
print(popped)       # suzuki
print(motorcycles)  # ['honda', 'yamaha']

# pop(index) 从任意位置删除
first = motorcycles.pop(0)
print(first)        # honda

# remove 按值删除（只删第一个）
motorcycles = ['honda', 'yamaha', 'suzuki', 'ducati']
motorcycles.remove('ducati')
print(motorcycles)  # ['honda', 'yamaha', 'suzuki']
```

> 官方文档：[list.append()](https://docs.python.org/3/library/stdtypes.html#list.append)、[list.pop()](https://docs.python.org/3/library/stdtypes.html#list.pop)、[list.remove()](https://docs.python.org/3/library/stdtypes.html#list.remove)

## 6. 组织列表

### 永久排序 sort()

```python
cars = ['bmw', 'audi', 'toyota', 'subaru']
cars.sort()
print(cars)  # ['audi', 'bmw', 'subaru', 'toyota']

cars.sort(reverse=True)
print(cars)  # ['toyota', 'subaru', 'bmw', 'audi']
```

### 临时排序 sorted()

```python
cars = ['bmw', 'audi', 'toyota', 'subaru']
print(sorted(cars))           # 临时排序，不改变原列表
print(cars)                   # 原列表不变
```

### 反转列表

```python
cars.reverse()
print(cars)  # 永久反转顺序
print(len(cars))  # 列表长度
```

## 7. 避免索引错误

```python
motorcycles = ['honda', 'yamaha', 'suzuki']

# ❌ motorcycles[3] 会报错：IndexError
# ✅ 安全访问最后一个元素
print(motorcycles[-1])  # suzuki

# ✅ 遍历前检查
if motorcycles:
    print(motorcycles[0])

# ✅ 用 try-except 捕获（后面会学）
```

> 官方文档：[Common Sequence Operations](https://docs.python.org/3/library/stdtypes.html#typesseq-common) — 序列索引和切片操作规范。
