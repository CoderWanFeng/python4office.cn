# 《Python入门到实践》课程结构建设

## 任务
根据推广文章 `20260428-python-book-course.md` 中的课程大纲，在 `python-beginner-to-practice/course/` 目录下建立课程主干文件。

## 完成的操作
1. 在 `python-books/` 下新建目录 `python-beginner-to-practice/`
2. 用户已预先建立 `ads/` 和 `course/` 两个子目录
3. 将推广文章移到 `ads/20260428-python-book-course.md`
4. 在 `course/` 下建立课程文档

## 课程文档结构
```
python-beginner-to-practice/
├── ads/
│   └── 20260428-python-book-course.md  ← 推广文章
└── course/
    ├── 大纲.md                          ← 课程大纲
    ├── 01-hello-world.md                ← 第1讲
    ├── 02-变量与数据类型.md
    ├── 03-列表.md
    ├── 04-元组与字典.md
    ├── 05-if语句.md
    ├── 06-while循环.md
    ├── 07-函数.md
    ├── 08-类与对象.md
    ├── 09-继承与多态.md
    ├── 10-文件与异常.md
    ├── 11-测试.md
    ├── project-01-alien-invasion.md     ← 项目一
    ├── project-02-data-visualization.md ← 项目二
    └── project-03-web-app.md            ← 项目三
```

## 大纲内容
- **基础语法**：11讲（HelloWorld → 测试）
- **项目实战**：3个项目（外星人入侵、数据可视化、Web应用）
