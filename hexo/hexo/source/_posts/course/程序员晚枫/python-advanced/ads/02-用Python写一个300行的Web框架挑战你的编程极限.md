## 为什么要写Web框架？

一个Web框架需要用到几乎所有高级特性：

| 特性 | 在框架中的作用 |
|------|-------------|
| 元类 | 自动注册路由 |
| 装饰器 | 中间件、权限控制 |
| 描述符 | 请求参数校验 |
| 上下文管理器 | 数据库连接池 |
| 生成器 | 流式响应 |
| 异步编程 | 高并发处理 |

**写完一个框架，这些特性你就真正理解了。**
