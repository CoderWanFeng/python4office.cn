## 我现在在做什么

我做的事情用一句话讲清楚就是：

**我帮企业研发负责人、数据团队 leader，把 AI Coding 和 AI Agent 真正用起来——不是装个样子，是落成一个能跑、有 ROI、能考核的团队生产力。**

具体三件事：

**1. AI Coding 工具选型与落地陪跑**

帮 5-50 人的研发团队从 0 选型、试点、铺开 AI Coding 工具链（Cursor / Claude Code / Codex / WorkBuddy / Trae / Copilot），做出可衡量的 ROI。

**2. AI Agent 工程化**

把 AI Agent 落地为真实业务能力——MCP 协议、Function Calling、多 Agent 协作、效果评估、风险控制、企业合规。

**3. 团队 AI 培训与考核体系**

让研发/数据团队从「自己瞎试」升级到「可培训、可评估、可复用」的标准化生产力，包含工具部署、流程改造、KPI 重设。

**可控、可培训、可评估**——是我做每件事的硬指标。

## 为什么是我？

**1. 真做过企业落地，不只是写文章**

我从 2025 年开始给企业研发团队做 AI Coding 落地陪跑。

涉及 5-50 人研发团队、Cursor / Claude Code 工具链部署、研发流程改造、KPI 体系重设。

不是 demo，是真刀真枪的项目。

**2. 懂工具、懂团队、懂老板**

- **懂工具**：Cursor、Claude Code、Codex、WorkBuddy、Trae、Copilot、MCP、Function Calling 全部跑过真实项目
- **懂团队**：5-50 人研发团队怎么推、怎么分工、怎么算账、怎么培训
- **懂老板**：能从 ROI、合规、风险、KPI 4 个维度给老板讲清楚价值

**3. 有"说真话"的表达气质**

我做内容有个习惯：**先告诉企业不该做什么，再讲该做什么**。

比如：
- 我会先告诉你"什么团队千万别一上来就上 Cursor"
- 我会先告诉你"AI Coding 落地前先回答的 5 个管理问题"
- 我会先告诉你"AI Coding 越免费，研发团队越要小心"

听起来像劝退？其实不是。

**说真话，是为了让你少踩坑。少踩坑，是为了你能走得更快。**

## 我相信什么？

**1. AI Coding 真正值钱的不是写代码，是改你的研发流程。**

**2. AI Agent 在企业落地的关键不是模型强，是工程化做得好。**

**3. AI 不会取代研发团队，但用 AI 的研发团队会取代不用 AI 的。**

**4. 工具免费不可怕，可怕的是「免费 = 你的代码 / 你的数据被训练」。**

**5. B 端 AI 落地，第一年最值钱的是选型、流程、培训，不是 demo。**
