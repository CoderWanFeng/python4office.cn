## 2、接单变现

### 在哪里接单？

- **程序员晚枫的派单群**：我搭建的AI编程派单平台
- **程序员客栈**：老牌程序员外包平台
- **猪八戒**：综合类外包平台
- **GitHub**：开源项目可能被企业看中

### 报价策略

```python
def calculate_price(task_type, complexity):
    """AI编程任务报价"""
    base_prices = {
        "AI聊天机器人": 2000,
        "AI数据分析工具": 3000,
        "AI自动化脚本": 800,
        "AI微信机器人": 1500,
        "网站集成AI": 5000,
    }
    
    complexity_multiplier = {
        "简单": 1.0,
        "中等": 1.5,
        "复杂": 2.0
    }
    
    base = base_prices.get(task_type, 1000)
    return base * complexity_multiplier[complexity]

print(calculate_price("AI聊天机器人", "中等"))
# 输出：3000
```

## 5、企业培训变现

### 培训报价参考

```python
def training_price(hours, is_ai_topic):
    """培训报价"""
    base = 2000  # 基础报价/小时
    
    if is_ai_topic:
        base *= 1.5  # AI课程溢价50%
    
    # 大客户优惠
    discount = 0.9 if hours >= 8 else 1.0
    
    return base * hours * discount

print(f"AI培训2天（每天6小时）：{training_price(12, True):.0f}元")
# 输出：AI培训2天（每天6小时）：32400元
```

### 企业培训注意事项

- 提前了解企业背景和需求
- 准备案例要贴合企业业务
- 留好联系方式，后续跟进
