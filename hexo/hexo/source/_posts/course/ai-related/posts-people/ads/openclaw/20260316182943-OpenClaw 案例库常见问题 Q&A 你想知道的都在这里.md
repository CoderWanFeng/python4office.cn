## 📌 基础问题

### Q1: 案例库是免费的吗？

**A**: 完全免费！

- 浏览案例：免费
- 复制代码：免费
- 商用：免费（遵守开源协议）

### Q2: 需要注册账号吗？

**A**: 不需要。

案例库完全开放，无需注册即可使用。

### Q3: 案例库有多少个案例？

**A**: 目前 130+ 案例，每周更新 5-10 个。

分类分布：
- AI 办公自动化：35+
- AI Agent 开发：28+
- 数据分析：22+
- 网页爬虫：18+
- 图像处理：15+
- 其他：12+

### Q4: 案例可以在公司用吗？

**A**: 可以。

大部分案例是 MIT 协议，允许商用。

**注意**：
- 保留原作者声明
- 遵守开源协议
- 具体查看每个案例的许可证

## 🔧 使用问题

### Q5: 代码复制后运行报错怎么办？

**A**: 按以下步骤排查：

1. **检查 Python 版本**
   ```bash
   python --version
   # 需要 3.9+
   ```

2. **检查依赖包**
   ```bash
   pip install -r requirements.txt
   ```

3. **查看错误信息**
   - ModuleNotFoundError → 缺包
   - FileNotFoundError → 路径错
   - 其他 → 搜索错误信息

4. **查看案例说明**
   - 是否有特殊要求
   - 是否需要配置 API 密钥

### Q6: 如何修改案例适配自己的场景？

**A**: 3 步走：

1. **理解代码逻辑**
   - 看懂每个函数的作用
   - 理解数据流向

2. **找到关键参数**
   - 输入路径
   - 输出路径
   - 配置选项

3. **修改并测试**
   - 小步修改
   - 每步测试
   - 有问题回退

### Q7: 案例可以修改后发布吗？

**A**: 可以，但有条件。

**允许**：
- 修改后内部使用
- 基于案例开发新项目
- 学习参考

**要求**：
- 保留原作者声明
- 遵守开源协议
- 如开源需使用相同协议

### Q8: 如何收藏喜欢的案例？

**A**: 有几种方式：

1. **浏览器书签**
   - 最方便
   - 支持分类

2. **本地保存**
   - 复制代码到本地
   - 添加注释
   - 分类管理

3. **截图保存**
   - 保存关键信息
   - 方便快速查阅

## 💻 技术问题

### Q9: 需要安装什么软件？

**A**: 基础要求：

- Python 3.9+
- pip（Python 包管理）
- 代码编辑器（VS Code、PyCharm 等）

部分案例需要：
- 特定 Python 包
- 数据库
- API 密钥

### Q10: API 密钥在哪里获取？

**A**: 不同平台不同：

- **阿里云百炼**：https://bailian.console.aliyun.com/
- **OpenAI**：https://platform.openai.com/
- **其他**：查看案例说明文档

### Q11: 案例运行慢怎么办？

**A**: 可能原因：

1. **网络问题**
   - API 调用慢
   - 解决：检查网络

2. **数据量大**
   - 处理时间长
   - 解决：分批处理

3. **性能问题**
   - 代码效率低
   - 解决：优化代码

### Q12: 支持 Windows/Mac/Linux 吗？

**A**: 都支持！

案例库的代码都是跨平台的。

**注意**：
- 路径分隔符不同（Windows 用\）
- 部分命令不同
- 案例说明会标注

## 🤝 贡献问题

### Q16: 如何贡献自己的案例？

**A**: 4 步走：

1. **整理案例**
   - 完整代码
   - 使用说明
   - 示例数据

2. **编写文档**
   - 功能介绍
   - 使用方法
   - 注意事项

3. **提交审核**
   - 发送到官方邮箱
   - 或提交 GitHub PR

4. **等待发布**
   - 官方审核
   - 修改完善
   - 正式发布
