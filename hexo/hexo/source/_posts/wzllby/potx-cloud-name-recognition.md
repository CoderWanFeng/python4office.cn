# 1. 2步实现"・"识别！代码小白也能轻松上手

## 📍 1.1 安装库
> 【potx-cloud】库基于腾讯的API实现发票识别并且生成Excel文件的Python库。
```python
pip install potx-cloud
```
## 📍 1.2 两行代码
```python
import potx

potx.ocr2excel.IDCardOCR2Excel(input_path='test.jpg', id=SecretId, key=SecretKey)
```
只需这2步，就可以轻松解决这个问题啦！
