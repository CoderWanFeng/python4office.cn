## 一、这次"撞墙"，代价是什么

帮她算一笔账：

| 项目 | 「撞墙」 | 「绕过去」 |
|---|---|---|
| 提示词 | 100+ 字模糊描述 | 30 字精准指令 |
| 试错次数 | 5-10 次 | 1 次 |
| Token 消耗 | 几千个 | 几百个 |
| 折算金钱（按 GPT-4 计）| ¥5-15 | ¥0.5-1.5 |
| 占用时间 | 半小时 | 2 分钟 |
| 效果 | 不达标 | 一次过 |

**一次撞墙的代价 = 几百 token + 半小时 + 一张不能用的图。**

**一次绕过去的回报 = 节省 90% token + 节省 90% 时间 + 拿到能用的图。**

这只是**一次撞墙**。

而每个上班族，每年撞这种墙，**少说 100 次**。

**光算钱：100 × ¥10 = ¥1000。**
**光算时间：100 × 30 分钟 = 50 小时 = 6 个工作日。**
**光算效果：每次都不达标，最后只能继续请人加班。**

**一次撞墙 = 烧钱 + 烧时间 + 烧效果。**
