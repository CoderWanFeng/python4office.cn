Here's a list of money-making open source projects, with brief introductions:

## 1. list-of-wechat-mini-program-list

This is an open-source WeChat Mini Program list, containing many Mini Programs that can help you make money.

- **King of Glory Story Station**: A Mini Program with open-source crawler code for game-related content sites.
- **Mask Photo Mini Program**: Built with uniapp + Vue, accumulated 100,000+ users, earned over ¥4,000 in ad revenue.
- **Zhihu Daily**: Use Mini Programs for content, target trending topics, do SEO well.
- **Coupon Mini Program Template**: E-commerce coupon direction, close to money, sustainable earnings.
- **Meituan/Ele.me Red Packet**: Food delivery red packets, CPS commission — others order via your red packet, you get commission.
- **WeChat Red Packet Cover**: Users watch video ads or invite users to get WeChat red packet serial numbers.
- **Account Helper Mini Program**: A Mini Program for storing accounts and passwords, built on WeChat Mini Program cloud development, no server needed.
- **Lunar Calendar Mini Program**: Supports checking favorable/unfavorable activities — fortune-telling is a huge industry with big profits.
- **Succulent Stories**: A succulent plant encyclopedia Mini Program — succulents are addictive, with huge industry potential.

[Project Address](https://github.com/shaoxiongdu/list-of-wechat-mini-program-list)

## 2. mfish-nocode

Mfish No-Code platform aims to build a low-code/no-code platform usable by both programmers and non-technical people.

- **Tech Stack**: Backend based on Spring Cloud Alibaba ecosystem; frontend based on Vue-Vben-Admin.

[Project Address](https://gitee.com/qiufeng9862/mfish-nocode)
