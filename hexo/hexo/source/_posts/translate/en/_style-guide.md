# English Style Guide

This guide defines the style conventions for all English translations.

## Tone

- **Conversational and friendly** — match the original Chinese tone
- **First person** — use "I" / "we" naturally
- **Active voice** — prefer active over passive constructions
- **Contractions OK** — "I'm", "don't", "we've" are fine for natural feel

## Formatting

### Paragraph Length

- **Maximum 3 lines** per paragraph (mobile-friendly)
- **One idea per paragraph**
- Use blank lines to separate paragraphs

### Headings

- Use `#` for title (one per file)
- Use `##` for main sections
- Use `###` for subsections
- **No skipping levels**

### Emphasis

- `**bold**` for keywords, warnings, important numbers
- `*italic*` for terms or definitions
- Use sparingly — too much emphasis dilutes impact

### Lists

- Use `-` (NOT `*` or `+`)
- Keep list items parallel in structure
- Maximum 7 items per list (cognitive load)

### Links

- Descriptive link text — never "click here"
- External links should have a clear destination hint

## Punctuation

| Element | Rule |
|---------|------|
| Quotes | Use straight `"..."` for quotes; smart quotes only in titles |
| Em dash | Use `—` (with spaces) — like this — |
| Ellipsis | Use `...` (three periods, NOT the Unicode ellipsis …) |
| Comma | Inside the quote for US style: "Hello," she said. |
| Period | Inside the quote for US style |

## Numbers

- Spell out one through nine
- Use digits for 10 and above
- Always use digits for: percentages, money, dates, time, addresses

## Date Format

- Prefer ISO 8601: `2026-06-14`
- Long form: `April 28, 2026` (month day, year)
- Avoid formats like `04/28/2026` (ambiguous)

## Capitalization

- **Title Case** for article titles and section headings (major words capitalized)
- **Sentence case** for body text
- Product names: as written (GitHub, ChatGPT, etc.)

## Forbidden Patterns

- ❌ Chinglish / direct word-for-word translation
- ❌ Mixing Chinese characters in English articles (except proper nouns)
- ❌ Adding content not in the original
- ❌ Removing content from the original
- ❌ Code / programming jargon (this is a general-audience blog, not a technical blog)

## Translation Memory Aids

|---------|-------------------|

| 今天我就[做某事] | Today, I'll [do something] |

| 这篇文章... | This article / This post |
| 记住一点 | One thing to remember |
