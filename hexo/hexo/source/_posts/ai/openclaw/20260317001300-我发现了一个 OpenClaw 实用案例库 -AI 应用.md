大家好，我是正在实战各种 AI 项目的程序员晚枫。

今天汇总**技能库里的 AI 应用技能**。

## 🤖 AI 应用技能分类

### 1. AI 对话类

**技能**：
- ai-chatbot：通用对话
- customer-service：客服对话
- qa-bot：问答机器人

**使用场景**：
- 客户咨询
- 内部问答
- 聊天陪伴

**安装**：
```bash
clawhub install ai-chatbot
```

### 2. 内容生成类

**技能**：
- content-generator：文章生成
- email-writer：邮件撰写
- report-writer：报告生成

**使用场景**：
- 文章写作
- 邮件回复
- 报告撰写

**安装**：
```bash
clawhub install content-generator
```

### 3. 图像识别类

**技能**：
- image-recognizer：图像识别
- ocr-skill：文字识别
- face-detector：人脸检测

**使用场景**：
- 图片分类
- 票据识别
- 人脸验证

**安装**：
```bash
clawhub install image-recognizer
```

### 4. 语音处理类

**技能**：
- speech-to-text：语音转文字
- text-to-speech：文字转语音
- voice-command：语音命令

**使用场景**：
- 会议记录
- 语音播报
- 语音控制

**安装**：
```bash
clawhub install speech-to-text
```

### 5. 数据分析类

**技能**：
- data-analyzer：数据分析
- trend-predictor：趋势预测
- anomaly-detector：异常检测

**使用场景**：
- 销售分析
- 趋势预测
- 风险预警

**安装**：
```bash
clawhub install data-analyzer
```

### 客服场景
```
ai-chatbot + customer-service + qa-bot
```

### 内容创作
```
content-generator + email-writer + report-writer
```

```
data-analyzer + trend-predictor + anomaly-detector
```
