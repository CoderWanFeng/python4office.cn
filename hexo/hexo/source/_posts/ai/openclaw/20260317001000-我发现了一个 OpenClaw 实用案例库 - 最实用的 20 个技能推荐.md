大家好，我是正在实战各种 AI 项目的程序员晚枫。

这是 OpenClaw 实用案例库发现系列的第 4 篇。

今天给大家**推荐 20 个最实用的技能**，都是我亲自测试过的。

## 🏆 TOP 20 技能推荐

### 办公自动化类（1-5）

#### 1. Excel 自动报表 ⭐⭐⭐⭐⭐

**功能**：自动生成 Excel 报表

**场景**：
- 每日销售报表
- 周报月报生成
- 数据汇总

**安装**：
```bash
clawhub install excel-report-generator
```

**使用**：
```python
from openclaw import Skill
excel = Skill('excel-report-generator')
excel.generate(data='./sales.xlsx', output='./report.xlsx')
```

**评价**：每天节省 1 小时，必装！

---

#### 2. 邮件自动发送 ⭐⭐⭐⭐⭐

**功能**：自动发送邮件

**场景**：
- 日报发送
- 通知邮件
- 批量邮件

**安装**：
```bash
clawhub install email-sender
```

**使用**：
```python
email = Skill('email-sender')
email.send(to='boss@company.com', subject='日报', body='今日完成...', attachment='./report.xlsx')
```

**评价**：配合 Excel 技能，完全自动化。

---

#### 3. PDF 转 Word ⭐⭐⭐⭐

**功能**：PDF 转 Word 文档

**场景**：
- 合同编辑
- 报告修改
- 文档转换

**安装**：
```bash
clawhub install pdf-to-word
```

**使用**：
```python
pdf = Skill('pdf-to-word')
pdf.convert('contract.pdf', 'contract.docx')
```

**评价**：比在线工具快，还安全。

---

#### 4. 文件批量重命名 ⭐⭐⭐⭐

**功能**：批量重命名文件

**场景**：
- 照片整理
- 文档归档
- 图片管理

**安装**：
```bash
clawhub install file-renamer
```

**使用**：
```python
renamer = Skill('file-renamer')
renamer.batch('./photos/', prefix='trip_', date_format='%Y%m%d')
```

**评价**：整理文件神器。

---

#### 5. 定时任务管理 ⭐⭐⭐⭐⭐

**功能**：定时执行任务

**场景**：
- 定时报表
- 定期备份
- 自动同步

**安装**：
```bash
clawhub install scheduler
```

**使用**：
```python
scheduler = Skill('scheduler')
scheduler.every().day.at("09:00").do(generate_report)
```

**评价**：解放双手，必备。
