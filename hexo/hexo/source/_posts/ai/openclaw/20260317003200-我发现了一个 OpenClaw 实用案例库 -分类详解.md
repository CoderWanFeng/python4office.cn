大家好，我是正在实战各种 AI 项目的程序员晚枫。

今天详解**分类体系**。

## 📂 一级分类

### 1. productivity（生产力）
- 办公自动化
- 日程管理
- 文件管理

### 2. development（开发）
- 代码生成
- 测试工具
- 版本控制

### 3. data-science（数据科学）
- 数据处理
- 数据分析
- 数据可视化

### 4. web-scraping（爬虫）
- 数据采集
- 监控工具
- API 工具

### 5. ai-applications（AI 应用）
- 对话机器人
- 内容生成
- 图像识别

### 6. integrations（集成）
- GitHub
- Slack
- Gmail

## 📂 二级分类

以 productivity 为例：
- email/
- calendar/
- document/
- file-management/
- automation/

## 💡 分类意义

1. **便于查找**：快速定位技能
2. **便于学习**：系统学习某类技能
3. **便于管理**：技能分类管理
