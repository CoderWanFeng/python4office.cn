大家好，我是正在实战各种 AI 项目的程序员晚枫。

今天聊聊**如何贡献技能**。

## 🎯 贡献流程

### 第 1 步：开发技能

1. 确定技能功能
2. 编写代码
3. 测试验证
4. 编写文档

### 第 2 步：发布到 ClawHub

1. 注册 ClawHub 账号
2. 创建技能仓库
3. 上传技能代码
4. 填写技能信息

### 第 3 步：提交到 awesome 列表

1. Fork 仓库
2. 添加技能链接
3. 提交 PR
4. 等待审核

## 📋 技能要求

- [ ] 功能完整
- [ ] 文档完善
- [ ] 代码规范
- [ ] 测试通过
- [ ] 无安全问题

## 💡 贡献建议

1. **解决实际问题**：技能要有实用价值
2. **保持更新**：及时修复 Bug
3. **响应用户**：积极回复 Issue
4. **持续优化**：根据反馈改进
