## 🎯 3 分钟选工具法

### 第 1 分钟：确定需求

```
问自己 3 个问题：

1. 我要解决什么问题？
   - 写作？绘画？编程？视频？

2. 我的预算是多少？
   - 免费？$10/月以内？不限？

3. 我的使用频率？
   - 偶尔用？每天用？团队用？
```

### 第 2 分钟：筛选对比

```
打开 AI 导航：

1. 选择分类（如"AI 写作"）
2. 筛选价格（如"免费"）
3. 筛选评分（如"4 星以上"）
4. 对比 Top 3 工具

结果：从 50+ 个缩小到 3 个
```

### 第 3 分钟：查看评价

```
仔细看用户评价：

✓ 好评内容（确认优点）
✓ 差评内容（看能否接受）
✓ 最新评价（看近期质量）
✓ 长期评价（看稳定性）

选综合评分最高的！
```

## 💡 选工具避坑指南

### 坑 1：只看价格

```
❌ 错误：选最便宜的
✅ 正确：看性价比

便宜的可能：
- 功能少
- 限制多
- 服务差

贵的可能：
- 功能过剩
- 用不上

选最适合的！
```

### 坑 2：只看评分

```
❌ 错误：选评分最高的
✅ 正确：看评价内容

评分可能：
- 刷出来的
- 样本太少
- 近期下滑

仔细看评价内容！
```

### 坑 3：只看功能

```
❌ 错误：选功能最多的
✅ 正确：看需要的功能

功能多可能：
- 学习成本高
- 界面复杂
- 价格贵

选够用的！
```

### 坑 4：不看更新

```
❌ 错误：选了就不管
✅ 正确：定期查看更新

工具可能：
- 价格调整
- 功能变化
- 服务降级

定期回顾！
```
