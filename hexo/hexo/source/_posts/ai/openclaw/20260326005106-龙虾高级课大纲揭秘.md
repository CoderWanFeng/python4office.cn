## 📖 第 1-2 周：Python 高级特性深挖

### 第 1 讲：装饰器的高级用法

```
✓ 装饰器基础回顾（快速过）
✓ 带参数的装饰器

✓  functools.wraps 的作用
✓ 实战：用装饰器实现日志、鉴权、缓存

```
✓ 描述符协议（__get__、__set__、__delete__）

✓ @property 的本质
✓ 实战：实现一个 ORM 字段类

```

### 第 5 讲：生成器与协程

```
