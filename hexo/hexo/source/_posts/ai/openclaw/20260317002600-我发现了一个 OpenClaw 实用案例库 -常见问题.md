大家好，我是正在实战各种 AI 项目的程序员晚枫。

今天解答**常见问题**。

## ❓ 安装问题

### Q: 技能安装失败？

**A**: 检查网络和权限。

```bash
# 检查网络
ping clawhub.ai

# 检查权限
ls -la ~/.openclaw/skills/
```

### Q: 技能无法运行？

**A**: 检查配置和依赖。

```bash
# 检查配置
cat ~/.openclaw/config.yaml

# 安装依赖
pip install -r requirements.txt
```

## ❓ 使用问题

### Q: 技能效果不好？

**A**: 调整参数或更换技能。

### Q: 多个技能冲突？

**A**: 检查名称和权限，卸载冲突技能。

## ❓ 安全问题

### Q: 技能安全吗？

**A**: 从官方渠道安装，检查代码。

### Q: 会泄露数据吗？

**A**: 限制权限，定期审计。

## ❓ 费用问题

### Q: 技能收费吗？

**A**: 大部分免费，部分付费。

### Q: API 调用收费吗？

**A**: 根据 API 提供商定价。
