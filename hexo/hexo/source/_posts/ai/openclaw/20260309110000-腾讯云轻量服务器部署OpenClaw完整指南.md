## 🚀 为什么选择腾讯云轻量服务器？

腾讯云轻量应用服务器（Lighthouse）是专为中小企业和开发者设计的云产品，具有以下优势：

1. **开箱即用**：预装热门软件，一键部署应用
2. **高性价比**：相比普通云服务器，套餐价格更优惠
3. **简单易用**：可视化控制台，无需复杂配置
4. **国内合规**：完全符合中国网络安全和数据安全法规

## 🔧 部署步骤详解

### 第一步：购买腾讯云轻量服务器

1. 访问[腾讯云轻量应用服务器购买页](https://buy.cloud.tencent.com/lighthouse)
2. 选择合适的配置（建议2核4G起步）
3. 选择操作系统（推荐Ubuntu 22.04 LTS）
4. 设置登录密码和安全组规则

### 第二步：安装OpenClaw

通过SSH连接到你的服务器，执行以下命令：

```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装Node.js（OpenClaw运行环境）
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# 安装OpenClaw
npm install -g openclaw

# 验证安装
openclaw --version
```

### 第三步：配置国内AI模型

为了确保合规性，我们推荐使用国内大模型服务：

```bash
# 配置通义千问API（需要在阿里云申请API Key）
openclaw config set model.default qwen-max
openclaw config set model.qwen-max.apiKey your_qwen_api_key

# 或者配置文心一言（需要在百度智能云申请）
openclaw config set model.default ernie-bot
openclaw config set model.ernie-bot.apiKey your_ernie_api_key
```

### 第四步：启动OpenClaw服务

```bash
# 启动OpenClaw网关服务
openclaw gateway start --port 8080

# 设置开机自启（可选）
sudo systemctl enable openclaw
```

## 🛡️ 安全合规配置

### 网络安全配置

1. **防火墙设置**：只开放必要的端口（8080、443等）
2. **IP白名单**：限制访问来源IP
3. **HTTPS加密**：配置SSL证书确保数据传输安全

### 数据安全措施

1. **本地数据存储**：敏感数据存储在本地，不上传到境外服务器
2. **日志审计**：开启操作日志记录，便于安全审计
3. **定期备份**：设置自动备份策略，防止数据丢失
