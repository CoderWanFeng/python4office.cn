## 📋 Step 1: Confirm If You Qualify

Tencent's free deployment this time is not unconditional. Let's see if you qualify:

### ✅ Basic Conditions

| Condition | Requirement | Description |
|-----------|-------------|-------------|
| Region | 17 designated cities | Shanghai, Guangzhou, Shenzhen, Beijing, etc. (see complete list below) |
| Industry | Aquatic processing/circulation enterprise | Lobsters, crabs, fish, etc. |
| Scale | Daily processing ≥10,000 units | Too small scale not cost-effective |
| Site | Has independent sorting area | Need to install cameras and light sources |
| Network | Has stable broadband | For data upload |
| Power | 220V stable power supply | Equipment needs power |

## 📝 Step 2: Prepare Application Materials

Application materials are not complicated, but need careful preparation.

### Required Materials

**1. Business License Scan**
- Color scan
- Within validity period
- PDF format, <5MB

**2. Site Photos**
- Sorting area panoramic photo (1)
- Assembly line close-up photos (2-3)
- Power/network interface location photo (1)
- JPG format, each <3MB

**3. Daily Processing Volume Proof**
- Recent 3 months shipping receipt scans
- Or production record forms
- Used to prove scale meets conditions

**4. Contact Information**
- Name
- Position
- Mobile phone
- WeChat
- Email

### Optional Materials (Bonus Points)

- Existing QC process description
- Pain point description (e.g., high labor costs, high misjudgment rate)
- Expected results (e.g., want to improve efficiency, reduce costs)

## 🌐 Step 3: Apply Online

### Method 1: Tencent Cloud Official Website Application

1. Open Tencent Cloud official website: https://cloud.tencent.com
2. Search "Lobster AI Inspection" or "AI Vision Inspection"
3. Click "Apply for Free Deployment"
4. Fill in application form
5. Upload application materials
6. Submit and wait for review

Tencent has set up offline service points in all 17 cities. Scanning QR code can directly contact local representatives.
