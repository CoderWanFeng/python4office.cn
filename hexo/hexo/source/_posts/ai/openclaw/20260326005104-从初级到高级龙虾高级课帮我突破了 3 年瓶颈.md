## 😫 瓶颈期的 5 个症状

### 症状 1：代码能跑，但很丑

```python
# 我以前的代码
def get_user_data(user_id):
    user = User.objects.get(id=user_id)
    if user:
        orders = Order.objects.filter(user_id=user_id)
        result = []
        for order in orders:
            item = {}
            item['id'] = order.id
            item['name'] = order.name
            item['price'] = order.price
            result.append(item)
        return {'user': user.name, 'orders': result}
    else:
        return None

# 问题：
# - 冗余的 if-else
# - 手动构建字典
# - 没有用序列化
```

### 症状 2：看不懂优秀代码

```
打开 Django/Flask 源码：
"这写的啥？"
"为什么要这样设计？"
"这个装饰器啥意思？"

然后关掉，继续用现成的。
```

### 症状 3：性能问题只会加机器

```
接口慢？→ 加服务器
数据库慢？→ 升级配置
并发高？→ 负载均衡

治标不治本，成本还高。
```

### 症状 4：设计能力弱

```
拿到需求直接开写
写到哪里算哪里
最后发现：
- 耦合严重
- 难以扩展
- 想重构不敢动
```

### 症状 5：技术自信不足

```
技术评审不敢说话
代码 Review 怕被说
面试大厂心里没底
```

**中 3 条以上，说明你遇到瓶颈了。**

## 🦞 龙虾高级课给我的 3 个突破

### 突破 1：理解 Python 的本质

#### 上课前

```
Python = 语法 + 库
会用就行，不用懂原理
```

#### 上课后

```
Python = 对象模型 + 元编程 + 并发模型
理解本质，才能用好
```

#### 具体收获

```
✓ 理解了描述符 → 看懂了 ORM 框架
✓ 理解了元类 → 能设计自己的框架
✓ 理解了协程 → 能写高并发程序
✓ 理解了 GIL → 知道什么时候用多进程
```

### 突破 2：掌握了性能优化方法论

#### 上课前

```
性能优化 = 瞎猜 + 试错
"可能是这里慢？改一下……"
"好像快了点？继续改……"
```

#### 上课后

```
性能优化 = 测量 + 分析 + 优化

1. 用 cProfile 定位瓶颈
2. 分析是 CPU 还是 IO 问题
3. 针对性优化
4. 再测量验证
```

#### 实战案例

```
公司有个批量处理任务，跑 2 小时：

第 1 步：测量
→ cProfile 发现 80% 时间在数据库查询

第 2 步：分析
→ N+1 查询问题，每次循环都查数据库

第 3 步：优化
→ 用 prefetch_related 预加载
→ 用批量查询代替单次查询

第 4 步：验证
→ 从 2 小时降到 10 分钟

领导：你做了什么？
我：只是用了正确的方法。
```

### 突破 3：建立了架构思维

#### 上课前

```
设计代码 = 实现功能
能跑就行，以后再说
```

#### 上课后

```
设计代码 = 考虑扩展 + 维护 + 复用

1. 先画类图，理清关系
2. 确定接口边界
3. 考虑扩展点
4. 遵循设计模式
5. 再开始编码
```

#### 实战案例

```
重构一个老模块：

以前：
- 1 个文件 2000 行
- 函数最长 300 行
- 全局变量到处用
- 想加功能不敢动

重构后：
- 拆成 10 个模块
- 函数不超过 30 行
- 依赖注入，方便测试
- 加新功能只需扩展

结果：
- 代码量少了 30%
- bug 率降低了 50%
- 开发效率提升了 2 倍
```

## 🎯 给瓶颈期同学的建议

### 1. 承认瓶颈，不要逃避

```
很多人 3 年后就躺平了：
"差不多就行了，那么拼干嘛"

但你不突破，就被淘汰。
```

### 2. 系统学习，不要碎片化

```
今天看个文章，明天看个视频
学了一年，还是零散知识点

不如花 2 个月，系统学一门课。
```

```
看懂了 ≠ 会写了
会写了 ≠ 能用了

必须动手写，必须被评审。
```

```
自学 1 年，不如名师指点 1 月。

- 讲师走过你走的路
```
