OpenClaw case library is released (https://www.python-office.com/openclaw/), but many people don't know how to use it.

Today's **beginner-friendly tutorial** takes you from scratch to learn how to use the case library.

## 📋 Preparation Work

### 1. Environment Requirements

| Item | Requirement |
|------|-------------|
| Python | 3.9+ |
| OpenClaw | Latest version |
| Network | Can access internet |

### 2. Install OpenClaw

```bash
# Install using pip
pip install openclaw

# Verify installation
openclaw --version
```

### 3. Configure API Key

```bash
# Create configuration file
cat > ~/.openclaw/config.yaml << EOF
model:
  provider: bailian
  api_key: your_API_key
EOF
```

## 🔧 Run Your First Case

### Case: Excel Auto Report

**Step 1: Copy Code**

```python
import pandas as pd
from openclaw import Agent

def generate_report():
    # Read data
    df = pd.read_excel('data.xlsx')

    # Data processing
    summary = df.groupby('category').sum()

    # Generate report
    summary.to_excel('report.xlsx')
    print("Report generated!")

if __name__ == '__main__':
    generate_report()
```
