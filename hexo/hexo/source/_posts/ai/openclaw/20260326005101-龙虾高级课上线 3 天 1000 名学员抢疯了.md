## 📚 课程核心：为什么叫"龙虾"？

### 名字由来

```
龙虾 = L.O.X.P. = Learn Open-source X Python

寓意：
- L：Learning（持续学习）
- O：Open-source（开源精神）
- X：Expert（专家级）
- P：Python（Python 生态）
```

### 课程定位

| 维度 | 定位 |
|------|------|
| 目标人群 | 1-3 年 Python 开发者 |
| 前置要求 | 掌握 Python 基础，有项目经验 |
| 课程目标 | 从"会用"到"精通" |
| 学习周期 | 8 周，每周 10 小时 |
| 课程形式 | 直播 + 录播 + 实战 + 答疑 |

```
✓ 装饰器的高级用法（带参数、类装饰器）

## 💡 课程特色：为什么值得学？

### 特色 1：不是教语法，是教思维

```
普通课程：
"装饰器的语法是@decorator，这样写……"

龙虾高级课：
"为什么需要装饰器？解决什么问题？
如何在你的项目中设计类似的机制？"
```

### 特色 2：每个知识点都有实战

```
学完装饰器 → 改造你现有的项目代码
学完协程 → 写一个高并发爬虫
学完元类 → 设计一个 ORM 框架
```

### 特色 3：代码评审环节

```
每周一次代码评审：
- 学员提交作业

✓ 讲师 + 助教团队在线答疑
