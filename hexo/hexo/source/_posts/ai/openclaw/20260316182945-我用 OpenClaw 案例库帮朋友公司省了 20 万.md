## 🛠️ 实施过程

### 第 1 周：订单自动化

**原案例**：Excel 自动报表生成

**修改内容**：
1. 对接 ERP 接口
2. 添加订单验证
3. 自动录入系统

**代码核心**：
```python
def process_orders(excel_path):
    # 读取订单 Excel
    orders = pd.read_excel(excel_path)
    
    # 验证订单
    valid_orders = validate_orders(orders)
    
    # 录入 ERP
    for order in valid_orders:
        erp.create_order(order)
    
    # 发送通知
    send_notification(valid_orders)
```

**效果**：
- 处理时间：4 小时 → 10 分钟
- 错误率：5% → 0.1%
- 人力节省：1 人/天

### 第 2 周：智能客服

**原案例**：智能客服系统

**修改内容**：
1. 导入产品知识库
2. 训练常见问题
3. 对接微信/网站

**代码核心**：
```python
class CustomerService:
    def __init__(self):
        self.kb = load_knowledge_base()
        self.agent = Agent(model='qwen3.5-plus')
    
    def reply(self, question):
        # 先查知识库
        answer = self.kb.search(question)
        if answer:
            return answer
        
        # 再问 AI
        return self.agent.chat(question)
```

**效果**：
- 自动回复率：70%
- 响应时间：30 秒 → 即时
- 人力节省：2 人/天
