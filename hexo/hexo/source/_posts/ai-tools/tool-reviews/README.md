```
ai-tools/
└── tool-reviews/           ← 单个工具的深度评测文章
    ├── README.md
    ├── xxx-tool-review.md  ← 示例：xxx-tool.md
    └── ...
```

## 🎯 专栏定位

不做广告，只说实话。

每篇文章评测一个 AI 工具（或同类工具横评），核心回答三个问题：

1. **这东西是干什么的？** —— 0基础也能看懂
2. **我用了一段时间，真实感受是什么？** —— 好不好用、哪里坑、哪里值
3. **适合谁？不适合谁？** —— 帮人判断要不要用

## 📌 写作原则

1. **先自己用，再写评测** —— 没用过的不写，没资格说好或不好
2. **好就是好，烂就是烂** —— 不因为给没给钱改变评价
3. **说清楚适合谁** —— 没有最好的工具，只有最适合的工具
4. **避免夸大** —— 不用"震惊""绝了""太牛了"这类词
5. **图文配合** —— 有截图最好，没有也行，文字要够扎实
