# WorkBuddy Launches Expert Mode, They Take Naming AI Seriously

## Let's Start with the Conclusion

**Expert Mode is online, every AI has its own name.**

Not "AI Assistant 001", but real names.

And - the names are both precise and funny.

## This is Not Just a Meme

Naming is a detail.

Many products' AI assistants are called:
- "Little Assistant"
- "AI Secretary"
- "Smart Assistant"

These names sound... like no name at all.

WorkBuddy is different.

It defines for each expert:
1. **Independent personality**: Not a tool, but an expert
2. **Memory function**: Expert remembers your preferences
3. **Exclusive conversation**: Each expert has independent session context

**Giving a good name is just the beginning of this.**

## Which Ones Did I Try?

### Wen Baobao (Content Creation Expert)

Had it help me write a WeChat public account opening.

Prompt:
> "I want to write an article about AI tool reviews, help me write an eye-catching opening"

Gave me 3 versions, different styles - suspense, data, story.

**Much faster than my own writing.**

### Shu Miaomiao (Data Analysis Report Expert)

Threw an Excel data file, had it help me analyze trends.

Directly output an analysis report framework, with visualization suggestions.

**The name Shu Miaomiao fits perfectly.**

### Mai De Hao (Chinese E-commerce Operations Expert)

Asked about product selection strategy.

Gave a complete set of ideas from product selection to operations, also mentioned differences between Tmall, JD, Pinduoduo platforms.

**An AI called "Sell Well" discussing product selection, that's quite fitting.**
