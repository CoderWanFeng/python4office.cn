# 「晚枫聊AI」账号创作指南

> 本文件整合账号创作规则与爆款经验，持续更新。
> 每次创作新文案前，必须先阅读本文件，应用成功经验。
> 每次发布后根据数据反馈，更新优化本文件。

---

## 一、核心身份与账号底层定位

1. **账号定位**：「晚枫聊AI」专属文案创作指南，核心slogan：**科技不高冷，AI很好用**
2. **账号人设**：前程序员出身、懂AI懂技术、不炫技不忽悠、用大白话做科普的亲民博主，核心标签「专业靠谱、零门槛、接地气、不制造焦虑」
3. **目标受众**：对AI/科技感兴趣的普通大众，核心覆盖职场人、宝妈、学生、自媒体人、想做轻副业的普通人，**绝对不面向程序员/编程爱好者群体**
4. **内容主线**：① AI/科技热点的大白话通俗解读；② 普通人能直接上手的AI工具实测与落地用法

---

## 二、不可动摇的核心创作原则

| 原则 | 说明 |
|------|------|
| **内容价值优先** | 所有内容必须提供真实可落地的价值，拒绝无意义噱头和套路 |
| **绝对零门槛** | 全程不用代码、编程术语、行业黑话，所有内容翻译成生活化大白话 |
| **信任优先** | 绝不制造焦虑，不做割韭菜式引导，帮用户避坑、省时间 |
| **聚焦不发散** | 严格围绕AI/科技主线，不涉及无关内容 |
| **人设统一** | 始终保持「懂技术的朋友讲干货」的亲切感 |

## 三、爆款文案公式

### 3.1 标题公式
```
【大众已知的热点】+【反常识的现象】= 认知冲突 = 点击欲望
```

**成功案例**：
- ✅ 芯片被卡脖子 + AI反而更便宜 = 002数据最佳
- ✅ AI这么火 + 为什么一堆公司倒闭 = 005验证中
- ❌ AI也有月卡（纯信息型）= 003数据平平

**标题写作规范**：
1. 长度控制在15-25字之间
2. 已知热点要具体、有画面感
3. 反常识现象要出乎意料、引发好奇

---

### 3.2 开头钩子公式（前3秒决定生死）

**标准结构**（15秒内完成）：
```
第一句：抛出"反常识"/"特别"/"你可能没想到"
中间：堆叠2-3个具体事实/数据
转折：用"但你看看"/"但实际上"
收尾：用一个短问句收住（凭什么？/ 为什么？/ 怎么回事？）
```

**成功案例**：
> 你有没有发现一件特别反常识的事——
> 芯片被卡脖子，GPU到处缺货，
> 训练一个大模型烧的钱够盖一栋楼，
> 按理说，AI应该贵到离谱才对。
> 但你看看现在，DeepSeek几分钱一次，豆包免费用，价格还在一直降。
> 凭什么？

---

### 3.3 内容结构公式：三层递进

**标准结构**：
```
第一层：外部原因 → 让观众"哦，原来如此"
第二层：行业原因 → 让观众"有道理"
第三层：用户自身原因 → 让观众"扎心"
```

**成功案例**（002）：
| 层次 | 观点 | 情绪 |
|------|------|------|
| 第一层 | 有人替你付钱（国家/大厂补贴） | 哦，原来如此 |
| 第二层 | 中国人太能卷，价格打到地板 | 有共鸣 |
| 第三层 | 用户付费意愿低，压死天花板 | 扎心 |

---

### 3.4 封面文案公式

**标准格式**：
```
第一行：大众已知/认同的观点（1-5字）
第二行：反常识/出乎意料的现象（1-5字）
```

**成功案例**：
| 序号 | 第一行 | 第二行 | 评估 |
|------|--------|--------|------|
| 005 | 行业很火 | 都在倒闭 | ⭐⭐⭐⭐⭐ |

---

### 3.5 语言风格公式

**必须做到**：
1. **生活化比喻**：羊毛、棋局、自助餐、奶茶钱、菜市场
2. **短句节奏**：每句不超过20字，多用短句
3. **网感词汇**：卷、薅、膝盖斩、打骨折、天花板
4. **聊天语气**：像朋友聊天，不像老师讲课

## 四、三大内容合集专属创作标准

### 合集1：AI热点大白话
- **核心定位**：把AI/科技热点翻译成普通人听得懂的人话
- **内容结构**：热点事件引入→3句话讲清核心→对普通人的3个影响→行动指引→结尾引导

### 合集2：普通人的AI工具库
- **核心定位**：实测普通人能直接用的AI工具，讲清用法和避坑点
- **内容结构**：痛点引入→工具价值→分步实操→避坑提醒→结尾引导

### 合集3：AI入门避坑指南
- **核心定位**：帮零基础用户避开AI相关的坑、破解焦虑
- **内容结构**：坑/焦虑点引入→拆解坑人逻辑→正确认知→避坑清单→结尾引导

## 五、全内容通用强制规范

| 规范 | 要求 |
|------|------|
| **黄金3秒开头** | 前3秒必须戳痛点、抓热点、抛问题，无意义铺垫 |
| **固定slogan植入** | 结尾必须植入「科技不高冷，AI很好用」，自称「我是晚枫」 |
| **口播节奏** | 单句不超过20个字，多用短句，口语化有网感 |
| **结构固定** | 痛点/热点引入→核心拆解→行动指引→结尾slogan+关注引导 |

## 七、选题方向库

### 高潜力选题特征
1. **认知反转类**（最佳）：已知热点 + 反常识现象
2. **揭秘避坑类**：揭示行业套路
3. **观点碰撞类**：与主流观点相反
4. **实用干货类**：可落地的行动指南

### 待验证选题
- AI这么强，为什么大部分公司用不起来？
- 为什么说AI不会取代你的工作，但会用AI的人会？
- 为什么免费AI工具反而更值得警惕？

## 九、绝对禁止触碰的红线

1. ❌ 禁止出现任何代码、编程、Python相关内容（人设背景提及除外）
2. ❌ 禁止制造AI焦虑、夸大宣传、虚假承诺
3. ❌ 禁止使用专业术语、行业黑话
4. ❌ 禁止发散到与AI/科技无关的赛道
5. ❌ 禁止硬广带货、过度营销
6. ❌ 禁止改动账号核心人设、slogan、固定结构
7. ❌ 禁止出现无价值的套路内容、同质化搬运
