```

├── README.md                  ← 本文件
```

## 使用说明

### 🎬 口播稿（口播稿文件夹）
- **用途**：用于视频号、抖音等短视频平台的逐字口播脚本
- **内容格式**：包含标题建议、封面文案、时长预估、逐字脚本、拍摄建议等
- **命名规范**：`序号-标题.md`
- **发布平台**：视频号、抖音、B站等短视频平台

## 使用规范

- **文件命名**：使用序号前缀（如 001-xxx.md），方便管理

## 创作规范文件

- **创作指南.md** — 创作指南与爆款经验总结（**每次创作前必读**）
