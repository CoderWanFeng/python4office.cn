# 视频脚本库

> 所有短视频口播脚本和对应的公众号文章存放于此

## 目录结构

```
video-scripts/
├── README.md                  ← 本文件
├── 口播稿/                    ← 视频号、抖音等短视频平台的口播稿
│   ├── 001-做了X年AI博主，我想聊一个很多人不会讲的问题.md
│   ├── 002-明明国内算力更稀缺，为什么反而更便宜.md
│   └── ...                   ← 更多口播稿
└── 公众号文章/               ← 对应的公众号文章
    ├── 001-做了X年AI博主，我想聊一个很多人不会讲的问题.md
    ├── 002-明明国内算力更稀缺，为什么反而更便宜.md
    └── ...                   ← 更多公众号文章
```

## 使用说明

### 🎬 口播稿（口播稿文件夹）
- **用途**：用于视频号、抖音等短视频平台的逐字口播脚本
- **内容格式**：包含标题建议、封面文案、时长预估、逐字脚本、拍摄建议等
- **命名规范**：`序号-标题.md`
- **发布平台**：视频号、抖音、B站等短视频平台

### 📱 公众号文章（公众号文章文件夹）
- **用途**：用于微信公众号、博客等图文平台的正式文章
- **内容格式**：包含完整的图文内容、排版优化、引用来源等
- **命名规范**：`序号-标题.md`（与口播稿保持相同的文件名，方便对应）
- **发布平台**：微信公众号、知乎、博客等图文平台

## 使用规范

- **文件对应关系**：口播稿和公众号文章应使用相同的文件名前缀（如 001-xxx.md），以便相互关联
- **内容同步**：口播稿的核心观点应与公众号文章保持一致，但表述方式应根据平台特点调整
- **更新流程**：修改口播稿时，应同步更新对应的公众号文章，保持内容一致性

## 创作规范文件

- **CREATION_RULES.md** — 给Agent的创作指令（账号定位、内容原则、输出格式）
- **爆款文案经验总结.md** — 爆款经验总结（**每次创作前必读**）

> ⚠️ 重要：每次创作新文案前，必须先阅读「爆款文案经验总结.md」，应用成功经验，根据数据持续优化。
