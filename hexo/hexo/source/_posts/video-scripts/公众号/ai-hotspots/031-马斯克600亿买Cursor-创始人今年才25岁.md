大家好，我是程序员晚枫。

最近有个新闻把我看沉默了。

马斯克宣布，要花 **600亿美元** 收购一家叫 Cursor 的公司。

这家公司的创始人，今年 **25岁**。

四个00后做的产品，卖600亿美元。这是什么概念？相当于卖掉了三分之一个腾讯。

今天就来聊聊这事背后的门道。

---

## 四个00后，做了什么值钱的东西？

先看看 Cursor 是个啥：

| 维度 | 数据 |
|------|------|
| **做什么的** | AI编程工具，你跟它说话，它帮你写代码 |
| **创始人年龄** | 全员00后，CEO才25岁 |
| **成立时间** | 2022年，不到4年 |
| **年化收入** | 超过 **20亿美元** |
| **客户覆盖** | 《财富》世界500强里，**67%** 在用 |
| **每天生成代码量** | **1.5亿行** |

简单说就是：以前写代码得自己一行行敲，现在你用自然语言说需求，AI帮你生成。

全球最大的公司都在用它来干活。

所以马斯克为什么要花600亿买它？

---

## 马斯克的算盘

马斯克的逻辑其实很清晰：

他手里有 **xAI**（做ChatGPT竞品Grok的公司），有 **SpaceX**（造火箭的），还有算力集群。

但他缺一样东西——**最好的AI编程工具入口**。

买了 Cursor 之后，他的版图就完整了：

| 他有的 | 作用 |
|--------|------|
| xAI 的 Grok 模型 | 提供AI大脑 |
| SpaceX 的卫星网络 | 未来部署太空数据中心 |
| Colossus 超算集群 | 提供海量算力 |
| **Cursor（待收购）** | **面向所有开发者的工具入口** |

而且马斯克还跟 Anthropic 签了个协议，每月付人家 **12.5亿美元** 租算力，一直租到2029年。

说白了就是：**有钱任性，全都要。**

但更值得玩味的是——为什么 Cursor 愿意卖？或者说，Cursor 为什么需要卖？

---

## 光鲜背后的问题

Cursor 看起来风光无限，但其实压力很大：

**第一，竞争对手太强了。**

Anthropic 推出了 Claude Code，OpenAI 有 Codex，这两家都是模型厂商自己做工具。它们可以亏本卖，因为工具只是它们引流的方式。但 Cursor 不是模型厂商，每调用一次都要花钱买别人的模型。

**第二，算力不够用。**

训练自己的大模型需要巨量算力，而这块恰恰是 Cursor 最缺的。没有算力，就没法做出比别人更强的AI。

**第三，定价难做。**

用户越多，成本越高。不像卖软件，复制一份几乎零成本。AI工具是**用一次花一次钱**，规模越大越烧钱。

所以当马斯克拿着600亿美元敲门的时候，Cursor 的选择就很现实了：

要么独立苦撑，跟巨头硬刚；
要么加入马斯克的帝国，直接拥有顶级算力。

换你怎么选？

---

## 这跟我们普通人有啥关系？

你可能觉得，几个亿万富翁的事，关我什么事？

有关系。

这件事释放了一个明确信号：**AI编程工具是未来3-5年最确定的赛道之一。**

你看，马斯克愿意花600亿美元买，说明他认为这个赛道未来价值远超这个数。

对普通人来说，这意味着几件事：

**第一，学会使用AI工具的人，竞争力在快速提升。**

以前一个程序员一天写200行代码，现在用AI辅助可以写到2000行。效率差10倍，老板不是傻子。

**第二，不会用AI的人，正在被边缘化。**

斯坦福最新的报告显示，美国22-25岁的初级开发者岗位已经**减少了近20%**。不是因为不需要人干了，而是因为一个熟练使用AI的新人，能顶过去三个。

**第三，门槛在降低，但天花板在升高。**

入门确实变容易了——不会写代码也能做出App。但要成为真正的高手，你需要懂的不只是语法，还有架构思维、产品设计、项目管理。

AI替代的是"搬砖"，替代不了"设计图纸"。

有些人看到这里就开始焦虑了："完了完了，我要失业了。"

别慌。当年Excel出来的时候也有人说会计要完蛋，结果呢？会计没完蛋，不会Excel的会计完蛋了。

**工具变了，活儿还在。区别是谁来干、怎么干。**

## 总结

- 马斯克计划 **600亿美元** 收购 Cursor，创始人是4个00后
