## 第一步：先把网站准备好

这一步最容易卡住，因为很多人**根本没网站**。

Google广告审核的不是你的Google账号，是**你的网站**。

所以你得先有一个：

- **属于你自己的域名**（不能是 xxx.csdn.net、xxx.cnblogs.com 这种）
- **能正常访问的网站**（不能打开是空白页或404）
- **有一定数量的原创内容**（建议至少20-30篇）
- **有基础页面**（关于我、隐私政策、联系方式）

我的网站 python4office.cn 是用 **Hexo + AI** 搭的。

如果你完全没基础，最简单的方式是：

1. **买一个域名**（阿里云、腾讯云都行，10多块钱一年）
2. **用 GitHub Pages + Hexo 搭静态网站**（免费）
3. **用 AI 帮你写内容**（每天三篇不是梦）

**特别提醒一点：备案的事**。

如果你的网站服务器在国内，需要备案。

如果用 GitHub Pages 或 Vercel 这种海外托管，不需要备案，但访问速度会慢一些。

**建议海外服务起步，先把流程跑通**，赚钱之后再考虑迁移。

## 第二步：注册 Google AdSense 账号

网站准备好之后，正式开始申请。

访问地址：

👉 [https://adsense.google.com](https://adsense.google.com)

需要的东西：

- **一个Google账号**（没有的就先注册一个）
- **你的网站地址**
- **你所在的国家/地区**
- **付款币种**（建议选美元USD）

填表的过程很简单，跟着提示走就行。

整个过程大概**3分钟**。

但有几个坑要提醒你：

1. **国家/地区一旦选定，后面改不了**，一定要选对。
2. **收款人姓名要和身份证一致**（后面提现要用，别填昵称）。
3. **地址要写真实地址**，谷歌后续会寄PIN码信件来验证。

![](https://raw.atomgit.com/user-images/assets/5027920/93b40b9e-72cd-441a-8cf2-a285dcf35b0c/image.png 'image.png')

## 第三步：把验证代码贴到网站

注册完之后，谷歌会给你一段代码，大概长这样：

```html
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-xxxxxxxxxx" crossorigin="anonymous"></script>
```

![](https://raw.atomgit.com/user-images/assets/5027920/9f30f314-5fc6-41e0-ad61-44b84e732782/image.png 'image.png')

你要做的事很简单：

**把这段代码放到你网站每个页面的 `<head>` 标签里。**

不同的建站方式，放法不一样：

| 建站方式 | 放置位置 |
|---------|---------|
| **Hexo** | 主题目录下的 `_partial/head.ejs` 或类似文件 |
| **WordPress** | 主题的 `header.php`，或用插件一键搞定 |
| **HTML** | 直接编辑 `<head>` |
| **Vue/React** | `public/index.html` 的 `<head>` |

放完之后，回到 AdSense 后台，点"**我已粘贴代码**"。

剩下的事情，就是等。

---

## 第四步：耐心等审核

这一步**最磨人**。

谷歌不会告诉你什么时候审完。

我等了大概**一周**，邮箱才收到通过通知。

也有人等了几个月。

**审核期间，你只需要做一件事：保持网站正常运营**。

- 别下架文章
- 别频繁改版
- 别让网站挂掉
- 持续更新内容

如果被拒了，邮件里会告诉你原因，最常见的几种：

| 拒绝原因 | 怎么解决 |
|---------|---------|
| **内容不足** | 多写文章，至少20篇以上 |
| **内容质量低** | 别全是采集，写原创 |
| **网站结构不完整** | 加上"关于我"、"隐私政策" |
| **流量来源有问题** | 别买流量，别刷量 |
| **网站无法访问** | 检查服务器、域名解析 |

被拒了别慌，改完之后还能再提交，不限次数。

## 第五步：审核通过，挂广告

通过之后，AdSense 后台会出现"**广告**"选项。

![](https://raw.atomgit.com/user-images/assets/5027920/968f10a1-55f5-4790-ac6b-d0b1ae6919f8/image.png 'image.png')

你有两种选择：

### 1. 自动广告（推荐新手）

谷歌帮你**自动选位置、自动展示**。

你只要开启开关，剩下的全部交给AI。

**优点**：省事，一键开启。

**缺点**：广告位置可能不那么好看。

### 2. 手动广告位

自己选广告大小、自己选位置。

**优点**：可控，不影响阅读体验。

**缺点**：要懂一点点 HTML。

我自己用的是**自动广告**。

因为流量太小，纠结这个意义不大。

**先跑通流程比追求完美重要100倍**。

## 第六步：达到100美元，谷歌打钱

最后一步，也是最爽的一步。

**Google AdSense 的提现门槛是 100 美元。**

账户余额超过100美元，下个月谷歌就会**自动打款**。

![](https://raw.atomgit.com/user-images/assets/5027920/b94ec16f-26e4-4228-ac1f-294776ce9677/image.png 'image.png')

打款方式：

- **电汇到银行账户**（推荐，国内主流银行都支持）
- **支票**（已经很少用了）

**特别注意**：

- 余额没到100美元，钱就一直挂在账户里累积，**不会清零**。
- 流量越大、内容越多，到账速度越快。
- 老站收益普遍比新站好——所以**早开通，早开始累积**。

所以这事的关键不是"什么时候能提现"，而是**今天就把广告位先挂上**。

## 我踩过的坑，提醒你别踩

讲几个我自己或身边朋友踩过的坑：

### 1. 千万别点自己的广告

谷歌的算法**非常聪明**。

你点一次，账号就有可能被标记。

**封号比拒审更可怕**——封了基本没救。

### 2. 别找朋友帮你点

同样的IP、同样的设备、同样的浏览器指纹。

谷歌**一眼就能识别**出来。

封号没商量。

### 3. 别在内容很短的页面挂广告

谷歌的政策是"**内容为主，广告为辅**"。

一个页面只有几行字，挂三个广告位，会被警告。

严重的话会被取消资格。

### 4. 越早开通越好

很多人想等"流量大一点再开通"。

恰恰相反——**审核通过和流量大小关系不大**，但开通后每天都在累积广告数据。

你越早挂上，谷歌越早了解你的网站，**后期匹配的广告单价也会越合理**。

晚开一个月，少累积一个月的数据，少分一个月的钱。
