# 口播稿创作指南

> 每次创作前必须读一遍。每条稿子发布后根据数据更新。
> **本文不涉及公众号文章，相关规范见「公众号文章创作指南.md」。**

---

## 一、账号定位

- **名称**：程序员晚枫
- **定位**：AI行业的真相挖掘者
- **slogan**：科技不高冷，AI很好用
- **人设**：程序员底色（查数据/看代码）+ 真相挖掘（给你不知道的角度）+ 不忽悠（不好用直接说）
- **受众**：普通大众，有付费能力的职场人/宝妈/想做副业的人
- **绝不针对程序员群体**

**内容比例（2026年6月数据验证）**：
| 类型 | 占比 | 作用 |
|------|------|------|
| AI热点大白话（信息差） | 40% | 拉播放量 |
| AI大佬故事/趋势预判/深度思考 | 45% | **涨粉核心**，条均5.75粉 |
| AI工具实测/入门教学 | 15% | 搜索流量补充 |

> ⚠️ **硬性约束**（不遵守 = 播放量再高也不涨粉）：
> - 连续发3条热点大白话之后，第4条**必须**是深度思考/趋势预判
> - 单周热点大白话不得超过4条
> - 违反此规则，流量再高也不涨粉——数据已验证

## 二、最高优先级规则：第一句话必须把人留住

> ⚠️ **这是整个指南里最重要的一条，没有之一。**
> 前3秒留不住人，后面写得再好也没用——用户已经划走了。

### 核心要求

**第一句话（前3秒）必须完成以下任意一项**：

| 钩子类型 | 效果 | 示例 |
|---------|------|------|
| 反常识冲突 | 用户"诶？不对吧"→ 想继续看 | "芯片被卡脖子，但AI反而更便宜了" |
| 信息差暗示 | 用户"他有我不知道的东西"→ 想验证 | "腾讯文档大裁员，我查到了真相" |
| 数字锚点 | 用户被具体数字吸引 → 想了解 | "OpenAI亏了100亿，免费GPT要没了" |
| 直接打脸 | 用户"他在说我不对？"→ 想反驳 | "淘汰你的不是AI，是你身边那个会用AI的同事" |

**禁止使用的开头（一律删除）**：
- ❌ "今天教大家…"
- ❌ "你有没有发现…"
- ❌ "大家好，我是…"
- ❌ 任何铺垫词（"其实啊""话说回来""大家都知道"）

### 检查标准

写完开头第一句，问自己三个问题：
1. 用户刷到这条，会在3秒内划走吗？→ 会 = 重写
2. 第一句能让用户想"然后呢？"吗？→ 不能 = 重写
3. 如果把第一句删掉，后面的内容还能成立吗？→ 能 = 第一句没用，重写

### 成功案例（完播率最高）

**案例1（023视频，完播率12.6%）**：
> 第一句："腾讯文档大裁员，我查到了真相。"
> → 用户想：什么真相？你说出来看看对不对。

**案例2（002视频，完播率5.8%）**：
> 第一句："芯片被卡脖子，GPU到处缺货，按理说AI应该贵到离谱才对。"
> → 用户想：对啊，然后呢？怎么不对了？

## 三、人设锚点规则：每篇必须让用户记住你是谁

> ⚠️ **流量高但不涨粉，核心原因就是缺这条。**
> 用户看完觉得"讲得不错"→ 但不记得是谁讲的 → 不关注。

### 核心要求

**每篇稿子必须至少有一个"人设锚点"**——让用户看完知道"这是那个程序员晚枫讲的"。

| 插入位置 | 写法示例 |
|---------|---------|
| 开头钩子之后（推荐） | "我是晚枫，专门挖AI行业不愿意让你知道的事" |
| 反转之后 | "我是晚枫，只讲AI行业里别人不敢讲的真话" |
| 结尾互动之前 | "关注晚枫，AI行业的真话只在这里讲" |

**注意**：不是每篇都写一样的话，而是每篇至少有一处——人设锚点可以换着说，但不能没有。

### 反面案例

| ❌ 没有锚点的结尾 | 问题 |
|---------------------|------|
| "收费了，你还用吗？" | 用户评论完就走了，不记得是谁 |
| "你怎么看？" | 没有任何人设记忆点 |

### 正面案例（竞品参考）

| 账号 | 锚点手法 | 效果 |
|------|---------|------|
| 老爸评测 | "较真、死磕"每期都出现 | 强人设记忆 |
| 直男财经 | "我是直男"每期开场 | 强关注转化 |

## 四、时长与节奏：≤45秒，每句话必须有信息

### 时长铁律

| 内容类型 | 最长时长 | 说明 |
|---------|---------|------|
| 单一信息差/单反转 | **≤45s** | 最佳，完播率最高 |
| 2-3个递进观点 | ≤60s | 可接受，每10秒一个价值点 |
| ＞60s | ❌ 禁止 | 完播率断崖式下跌 |

**判断标准**：录完后问自己——"如果只能留30秒，我最想讲什么？"→ 那30秒就是核心，其余全砍。

### 删掉所有冗余

| ❌ 必须删除 | 原因 |
|-------------|------|
| "你有没有发现/有没有这种感觉" | 铺垫词，浪费前3秒 |
| "其实啊/话说回来/更重要的是" | 口头禅，没信息量 |
| 连续超过2句的陈述 | 用户走神，必须插入钩子 |
| 单句超过20字 | 太长，观众记不住 |

**节奏模板（45秒版）**：
```
0-3s   ：第一句话钩子（反常识/信息差/数字）
3-15s  ：抛出2-3个具体事实/数据
15-30s ：反转/出乎意料的点 + 人设锚点
30-40s ：行动指引/认知启发
40-45s ：互动问题（带关注动机）+ 直视镜头停顿2秒
```

## 五、内容质量：绑定具体产品，禁止空讲理论

### 必须绑定具体产品/事件

| ❌ 禁止（空讲理论） | ✅ 必须（绑定具体） |
|---------------------|---------------------|
| "AI很厉害，大家要学" | "豆包宣布收费500/月，普通人怎么办" |
| "AI可以帮你做副业" | "用ChatGPT做小红书文案，我试了2周" |
| "免费AI有风险" | "DeepSeek免费，但它在用你的数据训练" |

**数据支撑**：绑定具体产品的视频，平均播放量是纯理论内容的**3.6倍**。

### 情绪反转：至少翻转一次

```
负面/争议开头 → 反转成正面/另类角度 → 情绪曲线丰富
```

**反转手法**：
| 手法 | 示例 |
|------|------|
| 对比反转 | 豆包敢定500→ 国产AI有底气了 |
| 视角反转 | 收费不是坏事 → 逼你算账，值不值 |
| 时间反转 | 短期是坏消息 → 长期倒逼你学会用AI |

**禁止**：一路负面到底（用户划走）/ 一路正能量到底（用户觉得鸡汤）。

### 互动问题：必须带关注动机

> ⚠️ **评论多 ≠ 关注多。互动问题必须让用户有"关注理由"。**

**核心公式**：
```
互动问题 + "关注我，[具体下期预告/持续追踪这个话题]"
```

| ✅ 好问题（带关注动机） | ❌ 坏问题（只有评论动机） |
|------------------------|------------------------|
| "收费了，你还用吗？**关注我，下期告诉你哪款还能免费用**" | "收费了，你还用吗？"（评论完就走） |
| "你会怎么选？**关注晚枫，AI省钱方案持续更新**" | "你会怎么选？"（无关注理由） |

**底线要求**：
- 互动问题 ≤10字（主体部分）
- 关注动机 ≤15字
- 表演：直视镜头 + 停顿2秒

## 六、语言风格

1. **口语化**：像跟朋友聊天，不像老师讲课
2. **短句**：每句不超过20字
3. **零术语**：不出现任何代码/编程/技术黑话
4. **生活化比喻**：用观众脑子里已有的东西做类比（点外卖/开保险箱/放大镜）

**禁止医疗/药品类比**（合规红线）：
| ❌ 禁止 | ✅ 替换 |
|--------|--------|
| "处方药不需要处方" | "保险箱不需要密码，谁都能开" |
| "AI让人上瘾" | "用惯了就离不开，像离不开手机" |

## 七、标题与封面

### 标题公式
```
【数字/时间锚点】+【事件/产品】+【悬念/预测】
```
- 长度：15-25字
- 必须含具体数字（30天/1亿/500元）
- 有悬念感（"就要""居然""终于"）

### 封面公式

封面文案共4项，必须按以下顺序和类型标注：

| 项 | 类型 | 说明 |
|----|------|------|
| 第一行 | 📝文案 | 大众已知观点（1-5字） |
| 第二行 | 📝文案 | 反常识/扎心结论（1-5字） |
| 第三项 | 😐人物表情 | 主播面部情绪（如：震惊/冷笑/释然） |
| 第四项 | 🤳人物动作 | 主播肢体动作（如：手指屏幕/摇头/摊手） |

- 第一行+第二行越扎心、越有争议越好
- 用户刷到，2行看完，不点进去难受

**品牌展现规则**：

> ⚠️ 稿子里提到具体品牌/产品时，封面文案**必须加第五项**：

| 项 | 类型 | 说明 |
|----|------|------|
| 第五项 | 🏷️品牌展现 | 品牌Logo的位置、大小、与人物动作的关系 |

**品牌展现写法示例**：

| 品牌场景 | 品牌展现写法 |
|---------|-------------|
| 提到单一品牌 | "画面右侧1/3处放豆包Logo，主播手指指向Logo" |
| 对比多个品牌 | "画面左上角ChatGPT Logo vs 右上角豆包Logo，主播双手各指一侧" |
| 品牌为负面对象 | "画面左下角小字DeepSeek Logo，主播摇头摆手表示否定" |
| 品牌为正面对象 | "画面右侧半透明豆包Logo，主播竖大拇指，Logo与手势呼应" |

**注意**：
- 品牌Logo不得遮挡主播面部
- Logo尺寸不超过画面1/4
- 如果稿子没有提到任何品牌，不写第五项

## 八、合规红线（一律禁止）

1. ❌ 药品/医疗类比（处方药/治病/诊断）
2. ❌ 金融投资建议（稳赚/ guarantees/荐股）
3. ❌ 夸大承诺（第一/最强/保证（学会））
4. ❌ 涉政/社会敏感内容
5. ❌ 嘲讽具体的人（只嘲讽现象/行为）

## 十、发布前自检流程（每条必做）

```
第一步：开头3秒测试
   └→ 第一句话是钩子吗？（反常识/信息差/数字）
   └→ 不是 → 重写

第二步：人设锚点检查
   └→ 稿子里有"晚枫"/"我是"相关的锚点吗？
   └→ 没有 → 不加进去不让发布

第三步：时长检查
   └→ ≤45秒？（单一观点）或 ≤60秒？（多观点）
   └→ 超过 → 砍内容

第四步：冗余检查
   └→ 删掉所有"你有没有发现/其实啊/话说回来"
   └→ 每句不超过20字

第五步：互动问题检查
   └→ 有"关注我/关注晚枫"的关注动机吗？
   └→ 没有 → 补上再发布

第六步：合规检查
   └→ 无药品/医疗类比
   └→ 无夸大承诺
   └→ 无针对具体的人

第七步：自己打分
   └→ "如果我是刷到的用户，这条我会关注吗？"
   └→ <8分 → 重写
```
