按内存分类： 静态链表（分配在栈区）   动态链表（分配在堆区） 

定义一个链表的结构体

**备注：链表的增删改查的操作几乎都是用两个指针来实现，有时还用会用到第三个指针充当中间变量来完成**
~~~
struct LinkNode 
{
		int num; //数据域
        struct LinkNode * next; //指针域
}
