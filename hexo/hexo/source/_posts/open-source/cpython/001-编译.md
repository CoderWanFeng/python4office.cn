## 编译

接下来把它拉到本地，进行编译。

```python
git clone --branch 3.9 https://github.com/CoderWanFeng/cpython.git

```

因为我是Windows电脑，编译命令如下：

```python
cd cpython/PCBuild

build.bat -p x64 -c Debug # 测试版本
build.bat -p x64 -c Release # 正式版本
```
编译完成后，会在`PCBuild/amd64`目录下生成`python_d.exe`和`python.exe`文件。
