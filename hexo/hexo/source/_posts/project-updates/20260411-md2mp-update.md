## 💡 关于 md2mp

md2mp 是一款专为微信公众号排版设计的 Markdown 编辑器。

**解决的问题：**
- 微信公众号编辑器不支持 Markdown 语法
- 手动排版费时费力
- 代码块在公众号显示效果差

**使用流程：**
1. 用 Markdown 写文章
2. 选择喜欢的主题样式
3. 一键复制到公众号编辑器
4. 直接发布

如果你写技术博客需要发布到微信公众号，md2mp 能大大提升你的排版效率！
