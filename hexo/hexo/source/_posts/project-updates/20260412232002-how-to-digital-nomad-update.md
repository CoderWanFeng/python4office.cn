本次更新对 **数字游民签证大全** 进行了全面的 SEO 优化，并新增了多个实用页面。

## 🎯 本次更新了什么？

### 1. 新增 robots.txt 和 sitemap.xml

为网站添加了搜索引擎友好的配置文件：

- **robots.txt** - 告诉搜索引擎哪些可以爬取、哪些要跳过
- **sitemap.xml** - 包含所有重要页面的 URL，方便搜索引擎发现内容

**为什么要这样做？**

之前网站缺少这些文件，搜索引擎可能无法完整收录网站内容。现在有了 sitemap.xml：
- 搜索引擎能更快发现所有页面
- 重要页面（签证列表、城市指南等）有更高的优先权
- 收录速度会明显提升

### 2. 新增完整 SEO 组件

新增了 `SEO.tsx` 和 `JsonLd.tsx` 组件，提供：

| 功能 | 说明 |

| Twitter Card | Twitter 分享优化 |
| 结构化数据 | 让搜索引擎更好地理解页面内容 |
| Canonical URL | 避免重复内容问题 |
