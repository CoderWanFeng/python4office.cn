# 🦞 OpenClaw 更新：新增火山引擎ArkClaw + 网站联动

本次更新为 **OpenClaw** 带来了两个重要改进：**新增火山引擎 ArkClaw 部署平台**，以及**完整的网站间内容联动**。

## 🎯 本次更新了什么？

### 1. 新增火山引擎 ArkClaw 部署平台

在「一键部署」页面，新增了 **ArkClaw（火山引擎）** 的支持。

**为什么需要这个？**

之前用户只能通过第三方推广链接访问火山引擎服务，这次我们直接对接了火山引擎控制台，让用户可以：
- 一键直达 [火山引擎控制台](https://console.volcengine.com/ark/region:ark+cn-beijing/openManagement)
- 方便查看和管理已部署的 AI 应用
- 更清晰地了解用量和费用情况

同时在「编程计划」页面，也为腾讯云和火山引擎都添加了用量查询入口。

### 2. 完整的网站间联动

新增了 `projects-showcase.html` 组件，并在 Footer 添加了跨站导流链接：

- 📝 **技术博客** - python4office.cn
- 🧭 **AI导航** - nav.python4office.cn
- 🐍 **python-office** - www.python-office.com
- 📝 **md2mp** - md.python4office.cn
- 🌍 **数字游民** - digital.python4office.cn
- 🏗️ **建站教程** - opc.python4office.cn

### 3. 代码优化

在 `deploy/page.tsx` 中移除了 `import React from "react"` 这个不再需要的导入语句，让代码更符合 Next.js 最新实践。

## 💡 为什么要做这些改动？

OpenClaw 的理念是**让 AI 应用的使用和部署变得更加简单**。

这次更新：
- 🌐 **扩展部署选择** - 让用户多了一个可靠的部署方案
- 🔗 **打通内容生态** - 让 OpenClaw 成为了解作者所有项目的入口之一
- ⚡ **优化代码质量** - 跟随 Next.js 最新实践
