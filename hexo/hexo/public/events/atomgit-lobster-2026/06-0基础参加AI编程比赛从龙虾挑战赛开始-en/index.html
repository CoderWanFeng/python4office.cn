Participate in AI Programming Competition with Zero Foundation? Start from Lobster Challenge!

发表于2026-04-14|更新于2026-04-30|Beginner Guide
|阅读量:


Hello everyone, I'm Programmer Wanfeng.

Recently many fans asked me:

Wanfeng, I want to participate in an AI programming competition, but I don't know programming at all. Can I participate?

My answer is: Yes.

Moreover, I suggest you start from AtomGit AI Lobster Challenge.

1. I've Seen Too Many "Zero Foundation" People Succeed

In 2017, I started learning Python from scratch.

At that time, I didn't even know how to install Python, it took me 3 days to set up the environment.

Later I learned that many people can't even get past this step.

But the problem is: Installation and configuration is just the simplest step, what's really important is starting to do.

I've seen too many people spend half a year "preparing", but end up not completing a single project.

I've also seen other people make a bad project in 2 weeks, then grow quickly through practice.

The difference is: the latter started doing, the former kept "preparing".

2. Why is AI Lobster Challenge Suitable for Zero Foundation?

1. Free resources, lower barriers

Participants receive:

ResourceDescription
AtomGit Cloud EnvironmentWrite code in browser, no local configuration needed
Large Model API TokenFree calls, no need to buy yourself
OpenClaw Tool PermissionsAI programming assistant, helps you write code
Technical Support TeamAsk questions anytime

You only need a computer with internet access.

2. No background restrictions, compete together

  • Students can participate
  • Working professionals can participate
  • Non-CS majors can participate
  • Complete zero foundation can also participate

Competition judges project quality, not background.

3. Free direction, large creative space

Competition doesn't restrict directions, you can make:

  • Automation office tools (direction I strongly recommend)
  • Data analysis tools
  • AI chatbots
  • Content creation assistants
  • Any creative project you can think of

No standard answer, your imagination is the limit.

4. Bonuses, certificates

  • First Prize ¥1000
  • Second Prize ¥700
  • Third Prize ¥300
  • Several Excellent Work Awards
  • Official certificates

Even if you don't win a big prize, the participation experience itself has value.

3. Specific Steps for Zero Foundation Participation

Step 1: Understand the Competition (1 hour)

Spend 1 hour reading through all competition information:

  • Read recruitment post, understand rules and scoring standards
  • Join event exchange group, ask questions and communicate
  • Look at previous award-winning works, find inspiration

👉 Click to view competition details

Step 2: Determine Direction (1 day)

Think of a problem you want to solve.
