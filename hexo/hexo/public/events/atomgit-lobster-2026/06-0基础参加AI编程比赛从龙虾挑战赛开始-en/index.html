Participate in AI Programming Competition with Zero Foundation? Start from Lobster Challenge!


Hello everyone, I'm Programmer Wanfeng.

Recently many fans asked me:

Wanfeng, I want to participate in an AI programming competition, but I don't know programming at all. Can I participate?

My answer is: Yes.

Moreover, I suggest you start from AtomGit AI Lobster Challenge.

1. I've Seen Too Many "Zero Foundation" People Succeed

In 2017, I started learning Python from scratch.

At that time, I didn't even know how to install Python, it took me 3 days to set up the environment.

Later I learned that many people can't even get past this step.

But the problem is: Installation and configuration is just the simplest step, what's really important is starting to do.

I've seen too many people spend half a year "preparing", but end up not completing a single project.

I've also seen other people make a bad project in 2 weeks, then grow quickly through practice.

The difference is: the latter started doing, the former kept "preparing".
