0基础参加AI编程比赛？从龙虾挑战赛开始！

发表于2026-04-14|更新于2026-04-30|新手指南
|阅读量:


大家好，我是程序员晚枫。

最近很多粉丝问我：

晚枫哥，我想参加 AI 编程比赛，但我完全不会编程，能参加吗？

我的回答是：能。

而且，我建议你从 AtomGit AI 龙虾挑战赛开始。

一、我见过太多「零基础」的人做到了

2017 年，我从零开始学 Python。

那时候我连 Python 怎么安装都不知道，花了 3 天时间才把环境配好。

后来我才知道，很多人连这一步都过不了。

但问题是：安装配置只是最简单的一步，真正重要的是开始做。

我见过太多人，花了半年时间「准备」，结果一个项目都没做出来。

也见过另一些人，用 2 周时间做出一个烂项目，然后在实战中快速成长。

区别在于：后者开始做了，前者在一直「准备」。

二、AI 龙虾挑战赛为什么适合零基础？

1. 免费资源，降低门槛

参赛就能获得：

资源说明
AtomGit 云端环境浏览器里写代码，不用本地配置
大模型 API Token免费调用，不用自己买
OpenClaw 工具权限AI 编程助手，帮你写代码
技术支持团队有问题随时问

你只需要有一台能上网的电脑就够了。

2. 不限背景，共同竞争

  • 学生可以参加
  • 职场人可以参加
  • 非计算机专业可以参加
  • 完全零基础也可以参加

比赛评判的是项目质量，不是背景。

3. 方向自由，发挥空间大

比赛不限制方向，你可以做：

  • 自动化办公工具（我强烈推荐的方向）
  • 数据分析工具
  • AI 聊天机器人
  • 内容创作助手
  • 任何你想到的有创意的项目

没有标准答案，你的想象力就是上限。

4. 有奖金，有证书

  • 一等奖 ¥1000
  • 二等奖 ¥700
  • 三等奖 ¥300
  • 优秀作品奖若干
  • 官方证书

就算没拿大奖，参与经历本身也是有价值的。

三、零基础参赛的具体步骤

第一步：了解比赛（1 小时）

先花 1 小时，把比赛信息都看一遍：

  • 阅读招募帖，了解规则和评分标准
  • 加入活动交流群，提问和交流
  • 看看往期获奖作品，找找灵感

👉 点击查看比赛详情

第二步：确定方向（1 天）

想一个你要解决的问题。

不要想太复杂的，就想：

我每天在做什么重复性的工作？能不能用 AI 自动化？

例如：

  • 每天要写周报？
  • 每天要整理 Excel 数据？
  • 每天要发类似的邮件？

这个痛点，就是你的项目方向。

第三步：用 AI 工具起步（3-7 天）

这是最关键的一步。

把你想到的问题，用自然语言告诉 AI 工具，让它帮你写代码。

不要怕写出来的代码不完美——先跑通，再优化。

我推荐的工作流：

  1. 描述需求，让 AI 生成基础代码
  2. 运行代码，看看能不能跑起来
  3. 有问题，再问 AI 修复
  4. 逐步添加新功能
  5. 优化界面和用户体验

第四步：完成项目并提交（1-2 天）

项目做到能演示的程度，就可以提交了。

不需要完美，能解决你一开始提出的问题就行。

提交时准备：

  • 项目代码（上传到 AtomGit 仓库）
  • README 说明文档
  • 演示截图或录屏
  • 项目介绍 PPT（如果有）

第五步：参加线下活动（比赛当天）

如果你在重庆，一定要来参加线下专场！

线下活动包括：

  • 开场宣讲（了解规则）
  • 90-120 分钟实战开发
  • 作品展示和路演
  • 评审和颁奖

线下最大的好处是：你身边都是同行，可以认识很多朋友。

四、常见问题 FAQ

Q：我是学生，没有项目经验，能拿奖吗？

A：能。 比赛评判的是项目本身，不是你的背景。很多获奖者都是学生。

Q：我完全不会编程，能参加吗？

A：能，但需要学一点基础。 至少要会：

  • 怎么打开浏览器
  • 怎么注册账号
  • 怎么用 AI 工具问问题

剩下的，AI 工具会帮你。

Q：比赛时间来得及吗？

A：来得及。 距离报名截止还有时间，距离比赛还有一段时间。零基础的话，建议每天花 2-3 小时，7-14 天足够做出一个能参赛的项目。

Q：我的项目创意不够好，怎么办？

A：创意不够，实用来凑。 不需要多么创新的想法，能解决一个真实问题就行。有时候，最简单的项目反而最受欢迎。

Q：一个人参赛还是组队参赛？

A：都可以。 如果你是零基础，建议先一个人试试，感受一下整个流程。如果有队友，可以分工合作，做更复杂的项目。

五、我的一点建议

作为一个从零基础走过来的人，我想给你一个建议：

不要等到准备好了才参加比赛，而是在比赛中准备。

没有人是完全准备好的。

那些拿奖的人，很多都是第一次参加比赛。

他们的优势不是「什么都准备好了」，而是「敢于开始」。
