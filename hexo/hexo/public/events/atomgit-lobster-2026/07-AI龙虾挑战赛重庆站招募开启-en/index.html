Hello everyone, I'm Programmer Wanfeng.

Shrimp raising masters competition is here! AtomGit AI Lobster Challenge Chongqing station recruitment is open!

This competition relies on the AtomGit open source platform, creating online national track + offline Chongqing special dual-track shrimp raising masters, allowing developers to quickly implement ideas in an immersive programming atmosphere using large models + agents, together experiencing the infinite possibilities of AI-powered coding!

⚠️ Bonus trigger conditions: Once 30 people register and 20 valid submissions are received, bonus distribution starts immediately!

Core Timeline

  • 2026/04/08: Registration begins
  • 2026/04/25: Review time
  • 2026/04/26: Registration deadline
  • 2026/04/28: Project submission deadline
  • 2026/04/30: Competition ends
  • 2026/05/30: Bonus distribution

Award Settings

  • First Prize 1 team: Cash ¥1000 + Official Certificate + Computing Power Gift Package
  • Second Prize 1 team: Cash ¥700 + Official Certificate + Merchandise Gift Package
  • Third Prize 1 team: Cash ¥300 + Official Certificate + Customized Gift
  • Excellent Work Award (several): Electronic Certificate + Platform Resource Rights
  • Participation Award (all participants): Electronic Memorial Certificate + Event Merchandise

Minimum winning score: 60 points, no award if below 60 points.

1. What is AI Lobster Challenge

AI Lobster Challenge = Cloud zero-configuration environment + Large model free calls + Timed practical development + Excellent project awards

  • No environment setup needed: AtomGit Lobster one-click start, ready to use out of box
  • Free computing power support: Official provides large model Tokens, OpenClaw toolchain permissions
  • Online and offline same competition: National online development + Chongqing offline special practical exchange
  • Lightweight and efficient: Short cycle, heavy practice, fast output, suitable for individuals/small teams

2. Participants

  • AI developers, Python developers, students, tech enthusiasts
  • Individual participation supported; or 2-4 person small teams (no teams of 5 or more accepted)
  • No senior background needed, just willing to use AI tools

3. Competition Format

Online track: All national developers can participate, submit projects online.

Offline Chongqing special: Chongqing (offline practice, project display, live roadshow, award ceremony).

4. Competition Rules

Technical Requirements

  • Can use local OpenClaw or AtomGit Lobster environment for development
  • Python tech stack prioritized, extra points for using Python

Project Directions (no mandatory restrictions, practicality/creativity prioritized)

  • AI code generation/auxiliary development tools
  • AI document processing, intelligent summary, knowledge base assistant
  • Python + AI automation scripts, efficiency tools
  • AI data visualization, small tools, interactive applications and other innovative projects

Originality Requirements

  • Plagiarism, theft, multiple award-winning projects prohibited
  • Cheating discovered results in immediate disqualification and成绩 cancellation
  • Final interpretation rights belong to the competition organizing committee

5. Scoring Standards (Total 100 points + Special bonus 10 points)

  • AI tool usage depth (20 points): Lobster environment, large model, OpenClaw usage degree and innovation
  • Creative innovation (20 points): Project novelty, technical breakthrough, problem-solving approach
  • Practical value (20 points): Landing feasibility, scenario urgent need, user experience
  • Project completion (20 points): Function complete, running stable, demonstrable
  • Technical difficulty (10 points): Technical threshold, architecture rationality, code standards
  • Special bonus (10 points): Using Python tech stack + fitting AtomGit ecosystem
