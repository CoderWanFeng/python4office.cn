## 📝 标题规范

- "考研失败，加入国企当程序员，真香！"

### 标题公式：
```
情感冲突 + 身份标签 + 反转/结果 = 点击率爆表
```

### ❌ 避免的标题：
- "我的职场经历"（太平淡）
- "程序员故事"（没有情感冲突）

## 💡 故事创作SOP

### 步骤1：确定冲突
- 裁员/降薪/职场困境
- 考研失败/求职受挫
- 技术瓶颈/职业迷茫

### 步骤2：设置反转
- 意外转机（入职国企、转行成功）
- 自我突破（学AI、副业变现）
- 认知升级（从迷茫到清晰）

### 步骤3：引发共鸣
- 真实细节（具体数字、场景）
- 情感真实（不刻意煽情）
- 可复制性（读者也能做到）

### 步骤4：行动号召
- 引导关注公众号
- 引导加入社群
- 引导购买课程

---

## 🎭 故事结构模板

### 模板1：困境 → 转折 → 成功

```
1. 开头：困境描述（引发共鸣）
   "30岁那年，我被裁员了..."

2. 中间：转折点（制造悬念）
   "就在我迷茫时，一个意外的机会..."

3. 结尾：成功结果 + 经验总结（价值输出）
   "现在我年薪50w，我想告诉大家..."

4. 行动号召（引流）
   "如果你想了解更多，欢迎关注我的公众号..."
```

### 模板2：质疑 → 尝试 → 逆袭

```
1. 开头：质疑/否定（制造冲突）
   "大家都说程序员35岁会被淘汰..."

2. 中间：尝试新方向（引发好奇）
   "我开始学习AI，每天下班后..."

3. 结尾：逆袭成功（打破质疑）
   "现在我成了AI工程师，收入翻倍..."

4. 行动号召（引流）
   "我把学习路径整理成了课程..."
```
