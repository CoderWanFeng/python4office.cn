# AI Skills 专栏

> 专注 AI 办公场景 Skill 的制作、使用与推广

## 专栏定位

这里是程序员晚枫的 **AI Skill 专栏**，记录从 Python 自动化办公向 AI Skill 生态转型的全过程。

**核心叙事**：装一个 Skill，搞定一件麻烦事

## 目录结构

```
ai-skills/
├── skill-tutorials/      # Skill 使用教程（教用户怎么用）
│   └── 如何安装/配置/使用各类办公 Skill
├── platform-guides/      # 平台攻略（各平台使用指南）
│   └── Coze / OpenClaw / 飞书 CLI 等平台评测与教程
├── skill-dev/            # Skill 开发教程（教用户怎么做）
│   └── 从零开发 Skill、进阶技巧、开源 Skill 拆解
├── office-automation/    # 办公自动化实战（场景案例）
│   └── Excel/PDF/PPT/Word 等具体场景的 Skill 解决方案
└── skill-reviews/        # Skill 评测与推荐
    └── 每周/每月最佳 Skill 推荐、对比评测
```
