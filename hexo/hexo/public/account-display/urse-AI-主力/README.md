## 💡 创作SOP

### 步骤1：选题
- 从用户痛点出发（重复工作、效率低）
- 结合热点事件（马斯克、AI新工具发布）
- 查看搜索需求（百度指数、知乎热榜）

### 步骤2：标题
- 突出"AI+具体场景+价值"
- 控制在15-30字
- 包含核心关键词

### 步骤3：结构
1. 开头：痛点场景（引发共鸣）
2. 中间：解决方案（AI工具+实操步骤）
3. 结尾：行动号召（关注/购买课程）

### 步骤4：SEO优化
- meta描述完整
- 关键词自然分布
- 内链到课程页

### 步骤5：多平台分发
- 公众号：完整文章
- 知乎：问答形式
- 小红书：图文版
- B站：视频教程
