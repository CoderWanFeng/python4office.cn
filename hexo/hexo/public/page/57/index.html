potx-cloud 库全面升级:让发票识别体验直接起飞

2025-04-15

大家好，这里是码匠er,我是【potx-cloud】库的核心开发。今天我很高兴与大家分享近期对【potx-cloud】库的优化成果。这个库最初基于腾讯的API实现发票识别,然后进一步封装为一个方便生成Excel文件的Python库。随着使用场景不断拓宽，我注意到库内有一些设计和实现细节还可以进一步优化，对于这个库的易用性和性能要求也越来越高，于是我启动了本次全面优化计划今天就给大家讲讲这些改进点，咱们一探背后的技术细节。

1. 【potx-cloud】库

简单来说，这是我基于腾讯 OCR 接口封装的工具库，专门解决「发票识别 + 数据整理」的痛点。

你只需要传入发票图片、PDF 或者在线链接，它就能自动识别发票信息，生成带表格的 Excel 文件！

2. 六大优化点，全是干货！

2.1 通用发票识别逻辑优化

原版通用发票识别接口在不传入sub_type参数时不会生成数据，用户必须先知道发票类型才能用。这显然不是理想的体验。此次升级改进如下：

  • 若未传入sub_type，系统将识别图片中所有的发票

在处理数据的过程中，如果sub_type是None，就不跳过继续处理。

  • 生成的Excel文件会根据发票的类型分别生成不同的sheet，结构更清晰

生成excel过程中，查看当前类型的sheet是否存在，如果不存在，就创建一个新的sheet保存数据；如果存在，就将数据追加保存在之前已经存在的sheet中。

如果存在发票中有需要明细列，重复保存会有很多冗余数据。我本次将公共的代码参数合并。

  • 对于发票识别中列表数据（比如多行商品明细），公共参数会被合理合并到单元格内，确保数据展示更加整洁美观
  1. 由于存在明细列的话，腾讯的api会返回把数据以list类型的数据返回，所以先获取list类型和非list类型的数据；

  1. 将公共数据和list列表中的数据拼接起来，组成excel中一行数据，如果没有list类型数据，则不需要处理；

  1. 所有数据写入之后，获取公共参数的列，将所有的公共列进行合并，然后将之前的数据垂直居中，更好的显示数据；

2.3 文件格式判断前置

之前，文件类型的判断逻辑是交给调用层自己处理，用户需要根据文件不同类型传入不同的参数，容易出错。现在，我将这种判断提前封装到了do_api.py中：

  • 系统在内部根据文件内容自动判断文件类型
  • 用户只需要提供统一的file参数

不管你传的是图片还是PDF，我们的库会自己解析文件类型，你只需要关心「文件在哪」，不用再纠结「该传哪个参数」。这种方式大大降低了使用门槛，同时也让代码逻辑更加清晰。

随着文件格式判断逻辑的优化，用户无需再关注文件类型的细节。为此，我对部分参数名称做了重新设计：

  • 将原来的img_path改为file_path
  • 将原来的img_url改为file_url

名称的统一更直观地传达出参数的意义，让新老用户都能快速上手。

2.4 参数处理

老版本要求必须传input_path参数，但处理线上文件时，这个参数根本用不上，只需传入img_url参数即可完成，导致用户得写一堆假路径绕过校验。因此，优化后的版本在处理线上文件时对input_path和img_url参数做了更加智能的判断和补偿，用户体验更加友好。

RAG = Retrieval Augmented Generation（检索增强生成）

简单说： 让大模型在回答问题之前，先从知识库里检索相关信息，然后再生成答案。

RAG的解决方案：

  • ✅ 先检索企业文档
  • ✅ 基于检索结果生成答案
  • ✅ 可溯源、可验证

客户背景

  • 中型互联网公司，500+员工

  • 新员工培训成本高，老员工查资料效率低

  1. 知识问答：员工可以用自然语言提问，快速找到答案

一、GPT-5的三大核心升级

1. 真正的多模态：文本+图像+音频+视频

之前的GPT-4也能处理图片，但GPT-5实现了真正的”原生多模态”：

  • 输入：可以同时输入文本、图片、音频、视频
  • 理解：能理解视频内容，比如”这段视频里的人在做什么”
  • 生成：可以生成图像、音频，甚至简单的视频

我用AI做副业，月入过万：5个普通人也能上手的AI变现方法

2025-04-05

先声明：这不是一篇”教你轻松月入十万”的割韭菜文章。

我是一个写了6年代码、做了5年自媒体的程序员，AI工具我用了2年多。

今天分享的，是我真实验证过的AI变现方法。

说到这儿，想起我刚开始做自媒体的时候。那是2020年，我还在广州做法务工作，白天处理合同，晚上写技术文章。

那时候没有AI，一篇文章从选题到发布，至少要3小时。我坚持了2年，粉丝才涨到1万。

转机出现在2023年。我开始用AI辅助创作，效率提升了10倍。现在我的公众号有40万+粉丝，副业收入已经超过了主业。

我不是天赋异禀，我只是赶上了AI这波红利。

有的月入几千，有的月入过万，有的还在摸索。

但都是普通人能上手、不需要大额投入的。

一、AI写作/自媒体：最成熟的变现路径

怎么做？

  1. 选领域：技术教程、职场经验、AI工具测评
  2. 定平台：公众号、知乎、头条、小红书
  3. 用AI提效：选题→大纲→初稿→排版→发布，全流程AI辅助
  4. 变现方式：流量主、广告、知识付费

我的实操数据

  • 公众号粉丝：40万+
  • 月均收入：2-3万（流量主+广告+课程）
  • AI提效：10分钟产出8篇文章

适合谁？

  • 有一定写作能力
  • 愿意持续输出
  • 能接受前期低收入（前6个月可能只有几百）

关键心法

AI是杠杆，不是替代品。

AI能帮你写初稿，但选题、观点、个人风格，还是要你自己来。

二、AI编程接单：技术人的变现捷径

怎么做？

  1. 学AI编程工具：Cursor、Claude Code、Copilot
  2. 接小单练手：爬虫、数据处理、自动化脚本
  3. 积累案例：做成作品集
  4. 提价接大单：网站开发、小程序、数据分析

我的实操数据

  • 接单平台：猪八戒、程序员客栈、闲鱼
  • 客单价：500-5000元/单
  • 完成时间：用AI辅助，效率提升3-5倍

适合谁？

  • 有编程基础
  • 愿意接小单练手
  • 能独立交付项目

关键心法

不要和AI比速度，要和AI比质量。

AI能写代码，但需求沟通、项目管理、代码审查，还是要人来。

三、AI数字人/短视频：新的流量红利

怎么做？

  1. 选赛道：知识科普、情感语录、书单推荐
  2. 做数字人：用HeyGen、D-ID生成虚拟形象
  3. 批量生产：AI写脚本→AI配音→数字人出镜→批量发布
  4. 变现方式：流量分成、带货、知识付费

朋友的实操数据

  • 平台：抖音、视频号、小红书
  • 粉丝：10万+
  • 月均收入：1-2万（流量+带货）
  • 投入时间：每天2小时

适合谁？

  • 愿意出镜或能接受数字人
  • 有一定的内容敏感度
  • 能接受试错（前10个视频可能都没流量）

关键心法

数量比质量更重要。

AI让内容生产成本趋近于0，谁能批量生产，谁就能抢占流量。

四、AI提示词工程：小众但高客单价

怎么做？

  1. 学Prompt Engineering：系统学习提示词写法
  2. 做案例库：整理各行业的优质Prompt
  3. 卖服务：为企业定制AI工作流
  4. 卖课程：教别人写Prompt

我的实操数据

  • 客单价：3000-10000元/企业
  • 服务内容：定制AI写作工作流、客服机器人、数据分析流程
  • 复购率：60%（企业用了效果好，会续费或扩量）

适合谁？

  • 对AI工具很熟
  • 有业务理解能力
  • 能和企业客户沟通

关键心法

Prompt本身不值钱，解决业务问题才值钱。

不要卖”Prompt模板”，要卖”用AI提升效率的解决方案”。

五、AI工具测评/ affiliate：被动收入

怎么做？

  1. 用AI工具：深度使用各种AI产品
  2. 写测评：发文章、发视频、发小红书
  3. 挂affiliate链接：用户通过你的链接购买，你拿佣金
  4. 积累被动收入：内容一直在，收入一直有

我的实操数据

  • 测评过的工具：50+
  • Affiliate收入：月均3000-5000元
  • 投入时间：前期多，后期维护少

适合谁？

  • 喜欢用新工具
  • 愿意写测评
  • 能接受长尾效应（前期收入低，后期稳定增长）

关键心法

真实比夸大更重要。

测评要客观，优缺点都要说，才能建立信任。

很多人看到这个收入数字，可能会觉得”哇，好赚钱”。但我想告诉你的是：这背后是5年的积累。

2020-2021年，我基本没收入，纯粹是兴趣驱动，免费分享技术文章。

2025年，现在每个月副业收入稳定在4-6万。

所以我的建议是：不要急于求成。AI降低了门槛，但不代表不需要积累。

前2年打好基础，后面才能享受复利。

七、给想做AI副业的人的建议

1. 不要辞职all in

副业就是副业，先验证可行性，再考虑全职。

2. 选择比努力重要

不是所有AI副业都能赚钱，要选：

  • 你擅长的
  • 有市场需求的
  • 能持续做的

3. 前期不要在意收入

前6个月可能只有几百块，这是正常的。

重点是积累案例、建立信任、打磨产品。

4. 用AI提效，但不要依赖AI

AI能帮你做执行，但：

  • 选题要你自己想
  • 观点要你自己有
  • 质量要你自己把控

AI是杠杆，你是支点。

八、写在最后

AI时代，做副业的机会确实比过去多了。

但机会多，不代表容易。

真正能赚到钱的，还是那些愿意学习、愿意执行、愿意坚持的人。

AI是杠杆，不是对手。

AI降低了门槛，但提高了天花板。

谁能用好AI，谁就能在这个时代获得超额收益。

选择权在你手里。
