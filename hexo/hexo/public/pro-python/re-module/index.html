三、专业方法

subn方法

subn和sub的方法类似，区别在于：subn()方法返回一个元组，其中包含所有替换的总数以及新字符串。

看过源码👇才知道，区别就是那个n。

四、注意事项

  • match只能从头开始匹配
  • match和search的区别：search可以从全部内容中匹配
  • 所有的匹配方法，都有一个属性：flags：
    • 标志位，用于控制正则表达式的匹配方式，如：是否区分大小写，多行匹配等等
