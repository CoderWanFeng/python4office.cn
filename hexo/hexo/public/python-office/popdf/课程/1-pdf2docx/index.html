  • input_file 和 output_file：这组参数用来处理单个文件，适合零散的 PDF 转换。
  • input_path 和 output_path：这组参数才是今天的主角！input_path 是 PDF 文件夹路径，output_path 是输出 Word 文件夹路径。只要把 PDF 文件丢进输入文件夹，运行代码，Word 文件就自动出来了。
