本次发票识别项目，起初来自开源项目：poocr，这个项目接入了腾讯、华为、百度、阿里各个平台的api，实现ocr识别。

后来随着更多小伙伴的加入，我们把poocr项目，根据不同的来源拆分出了多个项目：

  • potx-cloud：基于腾讯云的ocr识别，https://atomgit.com/python4office/potx-cloud
  • pobd：基于百度的ocr识别，https://atomgit.com/python4office/pobd
  • pohw：基于华为的ocr识别，
  • poali：基于阿里的ocr识别。

目前potx-cloud，pobd也获得了atomgit的孵化项目认证，也是对团队小伙伴的认可。
