咖啡种类大全 | 拿铁/卡布奇诺/摩卡的区别与做法(详解)

从尼泊尔旅居归来，爱上了自制咖啡的乐趣。本文详细介绍各种咖啡的特点和配方，在家也能做出咖啡馆级别的美味。

前言

2019年夏天，在尼泊尔做义工期间，店长每天自制各种咖啡，我也体会到了喝咖啡的乐趣。

正好最近工作稳定下来了，上班没事我也自制各种咖啡来喝一喝，自娱自乐。

咖啡基础：浓缩咖啡 Espresso

所有咖啡饮料的基底，也叫意式浓缩。

特点

属性说明
浓度极高，约9-10个大气压
容量25-30ml
口感浓郁，带有焦糖香
特点纯粹的咖啡精华

制作方法

咖啡种类详解

1️⃣ 美式咖啡 Americano

配方：1份浓缩 + 2份水

属性说明
口感较清淡，保留咖啡香
特点适合日常饮用
咖啡因较高

做法：

  1. 萃取一份浓缩咖啡
  2. 加入2份热水
  3. 搅拌均匀即可

2️⃣ 拿铁咖啡 Latte

配方：1份浓缩 + 1.5份牛奶 + 0.5份奶泡

属性说明
口感丝滑，奶香浓郁
特点咖啡味较淡
拉花适合做拉花

做法：

  1. 萃取一份浓缩咖啡
  2. 加热牛奶至60-65°C
  3. 将牛奶倒入咖啡中
  4. 顶部铺上薄薄一层奶泡
  5. 可做拉花装饰

3️⃣ 卡布奇诺 Cappuccino

配方：1份浓缩 + 0.5份牛奶 + 1.5份奶泡

属性说明
口感绵密，奶泡丰富
特点咖啡味较浓
装饰可撒肉桂粉/可可粉

做法：

  1. 萃取一份浓缩咖啡
  2. 打发牛奶至绵密奶泡
  3. 1/3牛奶 + 2/3奶泡倒入咖啡
  4. 顶部堆叠厚奶泡
  5. 可撒肉桂粉或可可粉

4️⃣ 摩卡咖啡 Caffe Mocha

配方：浓缩 + 巧克力酱 + 牛奶 + 鲜奶油

属性说明
口感甜蜜，巧克力味浓
特点甜食爱好者首选
热量较高

做法：

  1. 杯底加入巧克力酱
  2. 萃取一份浓缩咖啡
  3. 加入热牛奶搅拌
  4. 顶部挤上打发鲜奶油
  5. 可淋巧克力酱装饰

5️⃣ 玛奇朵 Macchiato

配方：1份浓缩 + 0.5份奶泡

属性说明
口感浓郁，略带奶香
特点保留浓缩原味
份量小杯（60ml左右）

做法：

  1. 萃取一份浓缩咖啡
  2. 舀一小勺奶泡放在咖啡上
  3. "Macchiato"意为"染上印记"

6️⃣ 白咖啡 Flat White

配方：1份浓缩 + 1.5份热牛奶

属性说明
口感丝滑，咖啡味平衡
特点澳新特色
奶泡不加奶泡或极薄奶泡

做法：

  1. 萃取一份浓缩咖啡
  2. 加入1.5份热牛奶
  3. 不加或仅加薄薄奶泡
  4. 搅拌均匀

7️⃣ 康宝蓝 Espresso con Panna

配方：浓缩咖啡 + 打发鲜奶油

属性说明
口感香甜，奶油柔化
特点简单却美味
热量中等

做法：

  1. 萃取一份浓缩咖啡
  2. 顶部挤一团打发鲜奶油
  3. 直接饮用

新手入门建议

推荐学习顺序

  1. 美式 - 最简单，先学会萃取浓缩
  2. 拿铁 - 练习打奶泡
  3. 卡布 - 学习打发绵密奶泡
  4. 拉花 - 进阶技能
