项目介绍

用 Python 的 Pygame 库，开发一款完整的2D射击游戏——"外星人入侵"！第3版使用 VS Code 作为编辑器。

你将学会

  • ✅ 安装Pygame + 创建游戏窗口
  • ✅ 加载和绘制可移动的飞船
  • ✅ 发射子弹
  • ✅ 从屏幕上方涌来的外星人
  • ✅ 碰撞检测（子弹击中外星人）
  • ✅ 记分系统 + 游戏结束与重玩
