🎯 本讲总结

引用与身份：is 比较身份（同一对象），== 比较值。变量是标签不是盒子。

深浅拷贝：copy() 只复制第一层，deepcopy() 递归复制所有层。简单对象用切片，嵌套对象用深拷贝。

可变默认参数：永远不要用可变对象（list/dict/set）作为函数默认参数，用 None 代替。

可变 vs 不可变：理解哪些类型是可变的，避免在字典键、集合元素、默认参数中使用可变对象。
