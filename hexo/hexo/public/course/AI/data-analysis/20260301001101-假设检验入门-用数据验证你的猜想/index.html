大家好，我是正在实战各种AI项目的程序员晚枫。

今天学习假设检验，这是用数据验证猜想的科学方法。

A/B测试显示新界面转化率更高，是真的有效还是偶然？假设检验帮你做出判断。

什么是假设检验？

核心思想

通过样本数据，判断某个假设是否成立。

基本步骤

  1. 提出假设：原假设H₀（通常是无效果）vs 备择假设H₁
  2. 选择检验方法：根据数据类型选择t检验、卡方检验等
  3. 计算p值：在原假设成立时，观察到当前结果的概率
  4. 做出决策：p < 0.05则拒绝原假设，认为有显著差异

关键概念

  • p值：小于0.05通常认为"统计显著"
  • 置信度：1 - p，如95%置信度
  • 第一类错误：假阳性（实际没效果但认为有）
  • 第二类错误：假阴性（实际有效果但没发现）

卡方检验：分类变量关联性

卡方独立性检验

卡方拟合优度检验

检验观测分布是否符合期望分布。
