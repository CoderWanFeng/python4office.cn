描述性统计：我用这6个指标，3分钟看透一组数据的本质

发表于2026-03-01|更新于2026-04-30
|阅读量:

大家好，我是正在实战各种AI项目的程序员晚枫。

欢迎来到统计分析基础部分！

今天学习描述性统计，这是用数字概括数据特征的方法。不需要复杂的数学，掌握几个关键指标，你就能快速理解任何数据集。

为什么需要描述性统计？

想象你面前有一组客户的年龄数据：

直接看这些数字很费劲。但如果告诉你：

  • 平均年龄：47.3岁
  • 中位数：48岁
  • 标准差：14.2岁

是不是立刻有了概念？这就是描述性统计的力量。

6大核心指标

1. 集中趋势（数据集中在哪）

均值（Mean）

平均值，所有数据的总和除以个数。

中位数（Median）

排序后位于中间的值，不受异常值影响。

众数（Mode）

出现次数最多的值。

什么时候用什么？

  • 数据对称分布 → 用均值
  • 有异常值或偏态分布 → 用中位数
  • 类别数据 → 用众数

2. 离散程度（数据有多分散）

极差（Range）

最大值减最小值。

方差（Variance）

每个数据与均值差的平方的平均。

标准差（Standard Deviation）

方差的平方根，与原始数据同单位。

变异系数（CV）

标准差除以均值，用于比较不同量纲数据的离散程度。

3. 分布形态（数据长什么样）

百分位数（Percentile）

偏度（Skewness）

衡量分布的不对称性。

峰度（Kurtosis）

衡量分布的尖锐程度。
