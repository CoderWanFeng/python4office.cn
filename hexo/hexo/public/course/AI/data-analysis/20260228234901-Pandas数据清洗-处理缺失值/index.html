最佳实践建议

✅ 应该做的

  1. 先分析缺失模式：是随机的还是有规律的？
  2. 记录清洗过程：保留原始数据，新建清洗后的列
  3. 可视化检查：用图表看缺失值分布
  4. 对比验证：填充前后统计指标的变化

❌ 不应该做的

  1. 无脑删除：可能丢失重要信息
  2. 全部填0：会扭曲数据分布
  3. 不记录操作：无法复现和审计
  4. 忽视业务逻辑：要结合实际情况选择策略
