🎯 本讲总结

PyThreadState：每个 Python 线程的状态对象，包含栈帧、异常状态、GIL 持有状态。

线程调度：GIL 释放 -> 调度器选择 -> GIL 获取 -> 执行，协作式调度。

同步原语：Lock（互斥）、RLock（可重入）、Semaphore（计数信号量）。

线程通信：Queue（队列）、Event（事件）、Condition（条件变量）。
