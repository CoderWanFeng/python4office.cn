🎯 本讲总结

C 扩展基础：PyObject*、PyArg_ParseTuple、方法表、模块初始化函数。

参数解析：格式字符串（i、s、O 等）解析 Python 参数为 C 类型。

引用计数：借用引用 vs 新引用，DECREF 错误是最常见的 bug。

性能对比：C 扩展比纯 Python 快 10-100 倍，Cython 是更简单的替代方案。
