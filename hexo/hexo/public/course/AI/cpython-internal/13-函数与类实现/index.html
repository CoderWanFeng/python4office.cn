  1. 用 inspect 模块查看任意函数的 code 属性

🎯 本讲总结

PyFunctionObject：函数对象的 C 结构体，包含代码、全局命名空间、默认参数等。

PyTypeObject：类型对象的 C 结构体，定义了创建实例的所有行为。

描述符协议：get、set、delete 控制属性访问。

元类：「类型的类型」，type(name, bases, dict) 是元类的底层实现。
