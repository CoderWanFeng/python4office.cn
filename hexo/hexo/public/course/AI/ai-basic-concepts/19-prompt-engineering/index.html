什么是提示词工程？让AI听懂你的高级技巧


作者：程序员晚枫

为什么同样的AI，别人用得那么溜？程序员晚枫用大白话揭秘提示词工程，让你掌握和AI高效沟通的秘诀！

👋 先问个扎心的问题

你有没有遇到过这种情况：

  • 用AI问问题，它答非所问
  • 看别人用AI效率翻倍，自己用却很鸡肋
  • 不知道怎么让AI给出你想要的答案

别慌，今天咱们用大白话把提示词工程彻底讲清楚。

🎯 一句话先说清楚

::: tip 核心结论
提示词工程 = 和AI聊天的技巧

同样的AI，不同的问法，答案完全不一样。会提问的人，用AI效率翻倍；不会提问的人，觉得AI是"人工智障"。
:::

😱 为什么问法这么重要？

同一个问题，两种问法

❌ 问法1（很差）：

✅ 问法2（很好）：

(完全符合需求！)

❌ 差：你是个有用的助手
✅ 好：你是一位有10年经验的Python后端工程师，
擅长性能优化和代码重构

❌ 差：帮我看看这个代码
✅ 好：帮我检查这段Python代码，找出性能问题和安全漏洞

❌ 差：推荐个电影
✅ 好：推荐3部科幻电影，要近5年的，豆瓣8分以上，
类似《星际穿越》风格的

❌ 差：解释一下什么是Python
✅ 好：用3个要点解释Python，每个要点不超过20个字，
用Markdown格式输出

❌ 差：帮我写文案
✅ 好：帮我写产品文案，参考这个风格：
"限时特惠，立省50%！最后100件！"

✅ 完美的提示词 =
你是一位[身份]，帮我[任务]

背景：[相关背景]
要求：[具体要求]
格式：[输出格式]

如果有不确定的，请说"我不确定"，不要瞎编。

你是一位有50万粉丝的小红书美妆博主，擅长写种草文案。

帮我写一篇关于"防晒霜推荐"的小红书笔记。

背景：夏天来了，大家都在找防晒产品

要求：

  • 目标读者：20-30岁的女生
  • 语言风格：活泼带表情符号，像姐妹聊天
  • 突出产品的3个核心卖点，每个有具体对比效果
  • 开头要有吸引眼球的金句

格式：
【标题】（带上热门标签）

【开头金句】

【推荐理由】
卖点1：...
卖点2：...
卖点3：...

【使用建议】
具体的涂抹方法和注意事项

【结尾】
引导评论互动的话术

❌ 差：3+5*2=？

✅ 好：一步步计算这个数学题，
先乘除后加减，告诉我每一步的结果

❌ 差：判断情感：今天心情很差

✅ 好：判断这句话的情感（正面/负面/中性）：

示例：
"考满分" → 正面
"摔跤了" → 负面
"今天天气不错" → 正面

现在判断：
"失业了" → ？

第一版：得到初步答案
第二版：让它改进"更详细一点"
第三版：再优化"换个更活泼的语气"

❌ 直接问：
"1+2+3+...+100等于多少？"
→ AI直接给答案（可能算错）

✅ 加"一步步推理"：
"1+2+3+...+100等于多少？请一步步计算，
先告诉我用什么方法，再展示计算过程"
→ AI展示推理过程 → 更准确

适用场景：

  • 数学问题
  • 逻辑推理
  • 代码调试
  • 分析判断

❌ 泛泛要求：
"帮我写几句产品文案"
→ AI写的可能不符合你的风格

✅ 给例子：
"帮我写几句产品文案，参考以下风格：

示例：

  1. '限时特惠，立省50%！最后100件！'
  2. '用过都说好，10万人的选择！'
  3. '今天买，明天到，京东快递！'

请按照这个风格写3个文案：产品是XX'"

效果：准确率提升60%

❌ 不限制格式：
"帮我分析一下这篇报告"
→ AI写一大段话，你还得自己提炼

✅ 限制格式：
"帮我分析这篇报告，输出格式如下：

核心观点

（不超过3句话）

数据亮点

（用表格列出）

风险提示

（用列表列出3条）

建议行动

（用列表列出3条）"
→ AI直接给结构化答案 → 直接可用

❌ 普通提问：
"怎么写好一篇文章"

✅ 角色扮演：
"你是一位资深的新媒体编辑，10年经验，写过100篇10万+文章。
擅长用'扎心标题+真实故事+实用干货'的结构写作。

请告诉我，怎么写好一篇公众号文章：

  1. 标题怎么起
  2. 开头怎么写
  3. 正文结构怎么安排
  4. 结尾怎么收尾

请用我听得懂的话讲，不要用术语"

效果：答案专业度大幅提升

职位：提示词工程师（Prompt Engineer）

薪资：

  • 初级：15-25万/年
  • 中级：30-50万/年
  • 高级：50-100万/年

要求：

  • 会和AI"聊天"
  • 懂业务场景
  • 能写出高效提示词
  • 不需要会编程

以前：人写提示词
现在：AI帮写提示词

工具：

  • PromptPerfect：输入普通提示词，输出优化版
  • Jina AI：自动生成适合的提示词
  • Claude：内置提示词优化

效果：人人都能写出好的提示词

以前：只有文字提示词
现在：文字+图片+语音 = 多模态提示词

示例（多模态提示词）：

"[图片：一张混乱的办公桌照片]

这张桌子上有5个问题，请找出来并给出整理建议"

→ AI看图+理解文字 → 综合分析

效果：提示词更丰富，AI理解更准确
