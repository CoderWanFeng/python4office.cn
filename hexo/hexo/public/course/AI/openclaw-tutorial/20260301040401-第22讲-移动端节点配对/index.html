第22讲：移动端节点配对，在手机上也能够使用OpenClaw

发表于2026-03-01|更新于2026-04-28
|阅读量:

大家好，我是正在实战各种AI项目的程序员晚枫。

今天学习移动端节点（Node）配对，让你能够在iOS和Android设备上使用OpenClaw的全部功能。

什么是Node？

Node是连接到OpenClaw Gateway的移动设备。通过Node，你可以：

  • 在手机上与AI对话
  • 使用手机摄像头拍照分析
  • 语音控制AI助手
  • 接收推送通知

Node vs 通道

特性通道（Channel）Node
平台Telegram/WhatsApp等iOS/Android原生App
安装无需安装需要安装App
功能消息收发完整功能+硬件访问
离线依赖第三方服务直连Gateway

iOS节点配置

安装OpenClaw App

  1. 在App Store搜索 "OpenClaw"
  2. 或从TestFlight安装测试版
  3. 打开App，点击 "Pair with Gateway"

功能特性

  • Talk Mode: 语音对话模式
  • Canvas: 手机屏幕共享给AI
  • Camera: 拍照/录像发送给AI
  • Location: 位置信息共享

Android节点配置

安装APK

  1. 下载最新APK: https://github.com/openclaw/android/releases
  2. 允许安装未知来源应用
  3. 打开App进行配对

额外权限

Android支持更多系统级功能：

  • 后台运行
  • 自动启动
  • 通知监听
  • 辅助功能

Canvas功能详解

Canvas让AI能够看到你的手机屏幕。

使用场景

  • 截图分析错误信息
  • 指导操作步骤
  • 远程协助
  • UI设计评审
