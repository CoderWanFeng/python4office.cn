本讲小结

概念说明
for item in 列表:依次取出每个元素来处理
range(n)生成 0 到 n-1 的数字序列
while 条件:条件满足就一直做
break退出整个循环
continue跳过本次，进入下一次
.items()同时遍历字典的键和值
zip(a, b)同时遍历两个列表
enumerate()获取索引和值
列表推导式[x for x in 列表 if 条件]
