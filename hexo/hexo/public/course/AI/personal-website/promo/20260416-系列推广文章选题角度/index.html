AI搭建个人网站课程 · 系列推广文章选题

一、故事型（用真实经历打动人）

  1. 《35岁程序员被裁，网上搜不到自己》

    • 用朋友的真实故事引入：技术很好，但没有网上痕迹
    • 引出个人网站的重要性

  2. 《我是如何用个人网站实现睡后收入的》

    • 分享 python4office.cn 的真实数据
    • GEO流量、SEO流量、咨询转化

  3. 《从法学生到程序员，我的个人品牌是怎么建立的》

    • 程序员晚枫个人IP故事
    • 个人网站在其中扮演的角色

  4. 《我随手建了一个推广网站，上线3天破万访问》 ✅ 已写

    • 建站经历 + 数据结果 + 方法复盘

二、洞察型（输出观点，建立权威）

  1. 《2026年，GEO正在取代SEO》

    • 什么是GEO（Generative Engine Optimization）
    • 为什么AI搜索正在颠覆流量格局
    • 个人网站 + GEO = 新的流量红利

  2. 《你的命运，不应该掌握在算法手里》

    • 公众号限流、B站不投流就没播放
    • 算法随时可以封你账号
    • 为什么"完全属于自己的平台"才是答案

  3. 《为什么聪明人都在悄悄做个人网站》

    • 逆向思维：别人都在卷抖音，你做网站反而更稀缺
    • 个人网站是"信息主权"的体现

  4. 《一个零基础学员，两周上线网站，月入3000》

    • 学员故事放大版
    • 真实收入截图 + 方法论

三、教程型（展示干货，建立信任）

  1. 《我用AI工具，2小时建了一个推广落地页》

    • 实操全记录截图
    • 不写代码，用工具堆出来的
    • 文末推课

  2. 《个人网站 SEO 入门：让Google主动给你送流量》

    • 干货文章
    • 分享实用SEO技巧
    • 自然引到课程更系统的学习

  3. 《如何选择适合自己的建站工具？》

    • WordPress / Notion / GitHub Pages / Webflow 对比
    • 不同需求的推荐方案
    • 深度内容，推课给需要深度学习的

  4. 《网站上线后，我做了这5件事，流量翻倍》

    • 内容策略 + 外链 + 社交分享 + 数据分析 + 更新迭代

四、对比型（帮用户做选择）

  1. 《个人网站 vs 公众号 vs 抖音：到底选哪个？》

    • 三个平台各自优缺点
    • 我的建议：都要有，但个人网站是地基
    • 适合"还不知道该从哪里开始"的人

  2. 《免费 vs 付费建站：2026年哪个方案最划算？》

    • GitHub Pages / Vercel / Netlify（免费）
    • 付费主机的适用场景
    • 丰俭由人的方案选择

五、焦虑型（戳痛点，让人想改变）

  1. 《你在网上搜自己的名字，能找到什么？》

    • 反问式开头，引发焦虑
    • 然后给出解决方案（建个人网站）
    • 引导报名

  2. 《平台说关就关，你的作品还在吗？》

    • 某平台封号、某网站下架的新闻
    • 强调"自己掌控"的重要性
    • 引出课程

  3. 《副业做了3年，一分钱没赚到，问题出在哪？》

    • 没有"转化阵地"
    • 引流渠道 ≠ 转化阵地
    • 个人网站 = 转化主阵地

六、人群细分型（精准打击目标用户）

  1. 《程序员专属：怎么用个人网站接到外包单？》

    • 针对程序员群体
    • 展示作品集 + 接单转化

  2. 《自由职业者：让客户主动找上门的秘密》

    • 针对自由职业者
    • 个人网站 = 24小时在线名片

  3. 《职场人：为什么你需要一个网站当退路？》

    • 针对职场人
    • 给自己留一条后路

  4. 《内容创作者：除了平台，你还需要一个"家"》

    • 针对B站/抖音/小红书创作者
    • 防止平台封号后一无所有

  5. 《律师/医生/设计师：专业人怎么做个人网站？》

    • 针对有专业技能的人
    • 高净值用户的精准定位

七、课程营销型（直接卖课）

  1. 《【课程发布】AI搭建个人网站 · 15讲实战课》 ✅ 已写

    • 正式发布公告
    • 课程大纲 + 价格 + 报名方式

  2. 《早报名和晚报名，有什么区别？》

    • 限时优惠策略文章
    • 早学早受益、早上线早积累
    • 制造紧迫感

  3. 《学员故事：他们学完这套课，网站都上线了》

    • 学员好评 + 成果展示
    • 社会认同

  4. 《这套课和市面上其他课有什么区别？》

    • 课程差异化定位
    • 不只是教建站，而是教变现

发布节奏建议

周次文章类型目标
第1周故事型 × 2篇引发共鸣，建立信任
第2周洞察型 × 2篇输出观点，树立权威
第3周教程型 × 2篇展示干货，吸引关注
第4周人群细分 × 2篇精准打击，各个人群
第5周焦虑型 × 1篇 + 课程营销 × 2篇促成转化

核心逻辑：先给价值，再卖课程。用户信任你了，报名是自然的事。
