第 4 讲：域名与主机选择 - 给你的网站安个家

大家好，我是正在实战各种AI 项目的程序员晚枫。

🎯 域名选错，后悔三年

我有个朋友，2019年注册了 wanfeng.cn，一年后续费发现涨到了150元/年。更惨的是，他想注册 wanfeng.com 的时候，发现已经被域名投资人抢注了，开价5万。

域名就像门牌号，选好了跟着你一辈子，选错了换起来很痛苦。

主机就像房子，租错了还能搬，但搬家也很麻烦。

这一讲，我手把手教你注册域名、选择主机，一次选对。

第一部分：域名注册

什么是域名？

域名就是你网站的地址，比如：

  • baidu.com
  • google.com
  • python-office.com（我自己的）

没有域名的网站就像没有门牌号的房子——别人找不到你。

如何取一个好域名？

好域名的标准：

  • ✅ 简短好记（最好10个字母以内）
  • ✅ 容易拼写（避免生僻词）
  • ✅ 与个人品牌相关
  • ✅ 优先选.com后缀
  • ✅ 容易口头传播（"我的网站是xxx.com"）

取域名的技巧：

  • 用你的名字：wanfeng.com
  • 用你的名字+职业：designerwan.com
  • 用你的品牌名：pythonoffice.cn
  • 用动词+名词：learnai.com
  • 用行业+名字：aiwanfeng.com

避免的坑：

  • ❌ 太长（超过15个字母）
  • ❌ 连字符（wan-feng.com不好记，口头传播容易出错）
  • ❌ 数字（除非有特殊意义，wanfeng123.com很low）
  • ❌ 生僻单词（别人拼不出来）
  • ❌ 容易拼错的词

域名注册平台推荐

国际平台（推荐）：

  • Cloudflare：成本价，最便宜，推荐 ⭐
  • Namecheap：便宜，界面友好
  • GoDaddy：最大，但续费贵，不推荐

国内平台：

  • 阿里云：国内最大，.cn域名首选
  • 腾讯云：也不错
  • 西部数码：便宜

注册域名步骤（以Cloudflare为例）

  1. 打开 dash.cloudflare.com
  2. 搜索你想要的域名
  3. 加入购物车，注册账号
  4. 填写信息，付款
  5. 在DNS设置中配置解析

重要提醒：

  • ✅ 开启域名隐私保护（免费）
  • ✅ 开启自动续费（避免忘记续费导致域名丢失）
  • ✅ 设置到期提醒
  • ❌ 不要买域名保护服务（浪费钱）

第二部分：主机选择

什么是主机？

主机就是存放网站文件的地方，相当于网站的"房子"。

主机类型详解

共享主机（Shared Hosting）

  • 多人共享一台服务器
  • 便宜：100-300元/年
  • 适合：个人网站、小流量
  • 缺点：性能受限，邻居可能影响你

云服务器（VPS）

  • 独立虚拟服务器
  • 中等：300-1000元/年
  • 适合：中等流量、需要更多控制
  • 缺点：需要一定运维知识

托管主机（Managed Hosting）

  • 服务商帮你维护
  • 较贵：500-2000元/年
  • 适合：不想折腾的人
  • 缺点：贵，定制性差

主机推荐（按需求）

新手/零基础：

  • 用免主机方案（Notion/Framer自带）
  • 或 SiteGround（WordPress官方推荐，300元/年起）

WordPress用户：

  • 国际：SiteGround 或 Cloudways
  • 国内：阿里云轻量服务器

技术博客：

  • GitHub Pages（免费，静态网站）
  • Vercel（免费，支持Next.js）

国内业务：

  • 阿里云/腾讯云（速度快，需要备案）

第三部分：网站备案（国内）

什么是备案？

在中国大陆，网站需要向工信部备案才能访问。

需要备案吗？

需要备案：

  • 主机在国内（阿里云、腾讯云等）
  • 面向国内用户

不需要备案：

  • 主机在国外（推荐新手）
  • 用免主机方案（Notion/Framer等）
  • 用GitHub Pages/Vercel

备案流程（以阿里云为例）

  1. 购买国内主机
  2. 登录阿里云备案系统
  3. 填写个人信息
  4. 上传身份证照片
  5. 拍照核验（用阿里云APP）
  6. 等待审核（7-20天）
  7. 备案成功

备案注意事项

  • 备案免费
  • 需要身份证
  • 个人不能做商业网站（名义上）
  • 备案期间网站不能访问
  • 每年需要年审

我的建议：

  • 新手先用国外主机，免备案
  • 等网站稳定了再考虑备案
  • 如果主要面向国内用户，长期还是要备案

第四部分：域名解析

什么是域名解析？

把域名指向你的主机，让用户能通过域名访问网站。

解析步骤

  1. 登录域名注册商后台
  2. 找到DNS设置
  3. 添加A记录或CNAME记录
  4. 等待生效（几分钟到几小时）

常见记录类型

A记录：域名→IP地址

CNAME记录：域名→另一个域名

MX记录：邮件服务器

避坑指南：域名和主机的5个陷阱

陷阱1：在GoDaddy注册域名

❌ 首年便宜，但续费贵（.com续费$20+/年），而且WHOIS隐私要额外付费

✅ 用Cloudflare注册，成本价，WHOIS隐私免费

陷阱2：选最便宜的主机

❌ 一年50元的虚拟主机，随时可能宕机，数据丢失

✅ 选靠谱的服务商，多花100元/年省掉无数麻烦

陷阱3：不设自动续费

❌ 忘记续费，域名被别人抢注

✅ 开启自动续费+到期提醒

陷阱4：国内备案选错接入商

❌ 在A平台备案，主机却用B平台，备案会被注销

✅ 备案和主机用同一家

陷阱5：DNS配置后不验证

❌ 改了DNS就以为万事大吉

✅ 用 dig 或 nslookup 命令验证解析是否生效
