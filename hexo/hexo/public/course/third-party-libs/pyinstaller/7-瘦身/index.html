7.4 手术刀 3：按需动态加载（延迟加载）

思路：把可选算法包放在子进程或插件目录，主程序启动后再 importlib.import_module。
.spec 仅打包主链路，插件目录用 --add-data 带过去，启动时按需 pip install --no-deps --target plugins/。
