8.8 常见坑速查

坑症状解决
Windows 长路径构建失败git config --system core.longpaths true
UPX 被 Defender 误杀上传 exe 报毒CI 里 --upx-exclude 或关掉 UPX
macOS 签名失败“unsealed contents present”用 codesign --verify --deep --strict 先自检
