本讲 10 分钟带你一次性把 Python、虚拟环境、pyinstaller 统统装好，并跑通第一个 Hello World 可执行文件。

2.1 安装 Python 与虚拟环境（2 min）

平台推荐方式一键指令

2.4 第一次运行：常见报错速查表（5 min）

现象根因现场修复
pyinstaller 不是内部或外部命令虚拟环境未激活确认终端前缀有 (venv) 或 (pyi)
双击 exe 闪退没加 input() 看不到报错临时加 input() 或 pause
防病毒报毒pyinstaller bootloader 被误杀把 dist/ 加入白名单，或用 --upx-exclude 关闭 UPX
体积 50 MB+包含完整 Python 运行时正常，第 7 讲再瘦身
Linux 报 error while loading shared libraries: libpython3.11.so.1.0动态链接库缺失打包时加 --static-libpython

现场演示一次“闪退 → 加 pause → 看到正常输出”的全过程，给观众信心。
