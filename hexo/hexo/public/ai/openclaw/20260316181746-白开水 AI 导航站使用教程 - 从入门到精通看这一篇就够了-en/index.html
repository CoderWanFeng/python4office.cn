White Water AI Navigation Site Usage Tutorial: From Beginner to Master, This Article is Enough...

发表于2026-03-16|更新于2026-04-30|AI ProgrammingPractical Tutorial
|阅读量:

Hello everyone, this is programmer Wan Feng actively working on various AI projects.

White Water AI navigation site is released (https://www.python4office.cn/ai-nav/), but many people don't know how to use it.

Today's beginner-friendly tutorial takes you from scratch to learn how to use the navigation site.

📋 Preparation Work

1. Device Requirements

ItemRequirement
DeviceComputer/phone/tablet
BrowserAny modern browser
NetworkCan access internet (some tools need)

2. Account Preparation

Some tools require registration:

  • Email (recommend Gmail, Outlook)
  • Phone number (some domestic tools)
  • Payment method (paid tools)

🌐 Access Navigation Site

Step 1: Open Official Website

Browser access: https://www.python4office.cn/ai-nav/

Step 2: Browse Homepage

Homepage displays:

  • 8 major category entrances
  • Popular recommended tools
  • Newly added tools
  • Search box

Step 3: Use Search

Enter keywords in search box:

  • Tool name (e.g., "ChatGPT")
  • Function (e.g., "image generation")
  • Scenario (e.g., "programming")

📖 View Tool Details

Each tool contains following information:

1. Basic Information

  • Tool name
  • Official Logo
  • Brief introduction

2. Detailed Information

  • Complete feature description
  • Applicable scenarios
  • Technical features

3. Usage Information

  • Official website link
  • Price information
  • Usage difficulty
  • Recommendation index

4. Supplementary Information

  • Similar tools recommendations
  • Usage tips
  • Precautions

🔧 Use Navigation Site to Find Tools

Scenario 1: Want to find AI chat tool

Steps:

  1. Click "Large Models" category
  2. Browse list
  3. Compare ChatGPT, Claude, Wenxin Yiyan
  4. Choose suitable one and click official website link

Scenario 2: Want to find programming assistance tool

Steps:

  1. Click "AI Programming" category
  2. View OpenClaw, Copilot, Cursor
  3. Compare features and prices
  4. Choose and go to official website

Scenario 3: Want to do image generation

Steps:

  1. Click "Image Processing" category
  2. View Midjourney, SD, DALL-E 3
  3. Compare styles and prices
  4. Register and start using

Scenario 4: Want to improve office efficiency

Steps:

  1. Click "Office Efficiency" category
  2. View Notion AI, Grammarly, etc.
  3. Choose what fits your needs
  4. Start using

📚 Advanced Usage Tips

1. Use Filtering Wisely

Filter by following conditions:

  • Price (free/paid)
  • Difficulty (beginner/advanced)
  • Recommendation index (4 stars and above)
