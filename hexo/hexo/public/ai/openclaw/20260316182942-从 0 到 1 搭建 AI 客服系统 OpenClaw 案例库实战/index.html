踩坑记录

  1. API 限流

    • 问题：并发高时触发限流
    • 解决：添加请求队列

  2. 知识库匹配不准

    • 问题：简单匹配准确率低
    • 解决：添加同义词库

  3. 数据库锁表

    • 问题：高并发时锁表
    • 解决：改用连接池
