❓ 常见问题

Q1: API 密钥在哪里获取？

A: 各平台控制台 -> API 管理 -> 创建密钥

Q2: 图片格式有什么要求？

A: 支持 JPG/PNG，建议分辨率 800x600 以上

Q3: 准确率如何？

A: 实测约 85-94%，腾讯和华为最高

Q4: 可以离线部署吗？

A: 华为昇腾支持完全离线，其他需要联网调用 API

Q5: 如何优化准确率？

A:

  1. 使用更高质量的图片
  2. 增加训练数据
  3. 微调模型参数
