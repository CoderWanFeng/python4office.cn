📏 标准 3：扩展性（权重 20%）

扩展性检查清单

  • 用量翻倍时，价格是否线性增长？
  • 能否无缝升级到企业版？
  • 是否支持多区域部署？
  • 能否横向扩展（加机器）？
  • API 版本是否向后兼容？

血泪教训

我见过一个团队，初期选了小厂商，价格便宜。结果业务涨了 10 倍后：

  • 厂商说"我们最大就支持这么多"
  • 想迁移？API 不兼容，得重写
  • 最后花了 3 个月迁移，损失惨重

结论：选能陪你长大的，别选只能陪你起步的。
