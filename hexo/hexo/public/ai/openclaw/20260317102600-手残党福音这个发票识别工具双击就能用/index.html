⚠️ 常见问题（FAQ）

Q1：真的不用安装任何东西吗？

A：不用。

工具包已经打包好了所有依赖，下载解压就能用。

唯一需要：Windows 系统（Mac 版在开发中）。

Q2：安全吗？会不会有病毒？

A：安全。

  • 代码开源，可以审查
  • 只是调用腾讯云 API
  • 不收集任何数据

如果你担心：可以找懂技术的朋友帮忙看看代码。

Q3：密钥安全吗？

A：密钥存在本地 config.txt 里。

安全建议：

  • 不要发给别人
  • 不要上传到网上
  • 定期更换

Q4：识别错了怎么办？

A：有 3 种情况：

  1. 图片模糊：重新拍照，确保清晰
  2. 发票特殊：手动修正（Excel 里直接改）
  3. 系统问题：群里提问，我们帮你解决

准确率：98%+，2% 的错误可以手动修正。

Q5：一个月能识别多少张？

A：腾讯云免费额度 1000 次/月。

超出后：0.035 元/次。

大部分用户：免费额度够用。
