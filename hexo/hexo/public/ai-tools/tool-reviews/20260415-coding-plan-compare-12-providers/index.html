什么是 Coding Plan？

说人话：各家云厂商给开发者的"新人礼包"。

通常包括：

  • ✅ 免费 API 调用额度
  • ✅ 优惠算力资源
  • ✅ 专属开发者工具
  • ✅ 技术支持通道

目的：让你低成本试用，好用了再付费。

各家详细拆解

⭐⭐⭐⭐⭐ 第一梯队：免费额度大 + 中文好

1. DeepSeek（强烈推荐）

免费额度：目前在主流厂商中，免费额度最大的一家。

为什么推荐：

  • 开源模型 DeepSeek-V3 性能对标 GPT-4o
  • 价格是其他家的 1/10
  • API 调用简单，文档清晰
  • 适合：想省钱 + 想用开源的开发者

适合场景：代码生成、数学推理、长文本处理

2. 火山方舟（工具最丰富）

免费额度：有赠送额度

为什么推荐：

  • 支持模型多：Doubao、GLM、DeepSeek、Kimi、MiniMax
  • 工具不限用（这个很香）
  • 和 OpenClaw（龙虾）深度集成
  • 适合：想多模型对比、不想折腾配置的

适合场景：AI 编程、多模型测试、企业开发

3. 智谱 AI（GLM 系列）

免费额度：有免费调用量

为什么推荐：

  • 清华背景，GLM-4 模型中文理解一流
  • ChatGLM 开源版完全免费本地部署
  • 企业版稳定性好
  • 适合：需要稳定中文能力的生产环境

适合场景：中文对话、知识库、长文档处理

4. Kimi（Moonshot AI）

免费额度：月之暗面出品，有免费额度

为什么推荐：

  • 中文语境理解最强之一
  • 超长上下文（128K）
  • 联网搜索效果很好
  • 适合：内容创作、长文本分析

适合场景：写文章、做调研、处理长文档

⭐⭐⭐⭐ 第二梯队：各有特色

5. 阿里云百炼 / AI Model Studio

为什么推荐：

  • 阿里生态强大，钉钉、淘宝等天然集成
  • 通义系列模型覆盖全面
  • 适合已有阿里云资源的开发者

适合场景：企业级 AI 应用、电商场景

6. 腾讯云（Tencent Cloud）

为什么推荐：

  • 微信生态加持
  • 混元大模型接入了腾讯会议、企业微信
  • 适合需要在腾讯生态内做开发的

适合场景：社交 AI、游戏 AI、企业微信集成

7. 讯飞星火（iFLYTEK Spark）

为什么推荐：

  • 语音交互是传统强项
  • 语音合成、语音识别能力强
  • 适合有语音需求的场景

适合场景：客服机器人、语音助手、教育应用

8. MiniMax

为什么推荐：

  • 海螺 AI 视频生成效果不错
  • 语音合成能力突出
  • API 定价合理

适合场景：AI 视频、内容创作、数字人

⭐⭐⭐ 第三梯队：适合特定场景

9. 百度云（Baidu Cloud）

特点：文心一言 + 百度 AI 生态

适合：已有百度云资源、百度生态内开发者

10. 京东云（JD Cloud）

特点：电商场景强

适合：电商 AI、供应链优化

11. 移动云（China Mobile Cloud）

特点：运营商资源

适合：特定政企场景

12. 小米（MiMo）

特点：小米生态

适合：小米硬件生态内开发
