二、各平台部署流程

2.1 Coze 部署

步骤：

  1. 创建 Bot

    • 登录 Coze 平台
    • 点击 "Create Bot"
    • 填写名称和描述

  2. 配置 Prompt

  3. 添加插件

    • 搜索并添加所需插件
    • 配置插件参数

  4. 测试验证

    • 在测试窗口验证功能
    • 检查响应是否符合预期

  5. 发布上线

    • 点击 "Publish"
    • 选择发布渠道（Discord、Telegram 等）

    • 配置服务器地址
