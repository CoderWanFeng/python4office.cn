Coze Skill Store Complete Guide: Zero-Code Install Office Skills

Hello everyone, I'm Programmer Wanfeng,实战 various AI projects.

In January 2026, ByteDance's Coze Coze 2.0 officially launched, bringing the revolutionary Skill (Skill) feature. This means: people who can't code can also have AI help them with office automation.

This guide teaches you step by step how to use Coze Skill Store from scratch.

1. What is Coze Skill?

One Sentence Explanation

Skill is AI's "external plugin" - after installing it, AI can do things it couldn't do before.

Example

  • AI without Skill: "I can help you write text, but can't process Excel"
  • AI with Excel Skill: "Send me 100 Excel files, I'll merge them for you in 1 minute"

What can Skill do?

ScenarioSkill Capability
📊 ExcelBatch merge, data cleaning, report generation
📄 WordBatch generate documents, format conversion
📑 PDFMerge/split, add watermark, convert to images
📽 PPTAuto-generate, batch convert to PDF
🖼 ImageRemove watermark, format conversion, compress
📧 EmailBatch send, auto-reply

2. How to Access Skill Store

Step 1: Open Coze Official Website

Visit: https://www.coze.cn

Step 2: Enter Skill Store

  1. Log in (supports Douyin/Feishu/Phone login)
  2. Click "Skill Store" in the top navigation bar
  3. Enter Skill marketplace

Step 3: Browse Skill Categories

Skill Store currently has several categories:

  • Office Efficiency: Excel, PDF, document processing
  • Content Creation: Writing, PPT, image processing
  • Data Analysis: Reports, visualization
  • Communication Collaboration: Email, message processing

3. How to Install and Use Skill

Install Skill (Excel Merge as example)

Step 1: Find the Skill you want

  1. Search "Excel Merge" in Skill Store
  2. Browse results, check Skill's:
    • Function introduction
    • Usage count
    • User reviews
    • Developer information

Step 2: One-click install

  1. Click Skill card to enter detail page
  2. Click "Install" button
  3. Wait for installation to complete (usually a few seconds)

Step 3: Start using

  1. After installation, Skill automatically appears in your AI chat
  2. Directly tell AI: "Help me merge these Excel files"
  3. AI will automatically call Excel Merge Skill to process
