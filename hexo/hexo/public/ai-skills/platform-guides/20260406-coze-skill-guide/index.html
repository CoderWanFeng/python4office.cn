Coze 扣子 Skill 商店全攻略：零代码安装办公技能

大家好，我是正在实战各种 AI 项目的程序员晚枫。

2026 年 1 月，字节跳动的 Coze 扣子 2.0 正式上线，带来了革命性的 Skill（技能） 功能。这意味着：不会写代码的人，也能让 AI 帮你自动化办公了。

今天这篇攻略，手把手教你从零开始使用 Coze Skill 商店。

一、什么是 Coze Skill？

一句话解释

Skill 是 AI 的"外挂"——装上之后，AI 就能做它原本做不了的事。

举个例子

  • 没装 Skill 的 AI："我可以帮你写文字，但处理不了 Excel"
  • 装了 Excel Skill 的 AI："把 100 个 Excel 文件发给我，1 分钟帮你合并好"

Skill 能做什么？

场景Skill 能力
📊 Excel批量合并、数据清洗、报表生成
📄 Word批量生成文档、格式转换
📑 PDF合并拆分、加水印、转图片
📽 PPT自动生成、批量转 PDF
🖼 图片去水印、格式转换、压缩
📧 邮件批量发送、自动回复

二、如何访问 Skill 商店

步骤 1：打开 Coze 官网

访问：https://www.coze.cn

步骤 2：进入 Skill 商店

  1. 登录账号（支持抖音/飞书/手机号登录）
  2. 点击顶部导航栏的 "技能商店"
  3. 进入 Skill 市场

步骤 3：浏览 Skill 分类

Skill 商店目前分为几大类：

  • 办公效率：Excel、PDF、文档处理
  • 内容创作：写作、PPT、图片处理
  • 数据分析：报表、可视化
  • 通讯协作：邮件、消息处理

三、如何安装和使用 Skill

安装 Skill（以 Excel 合并为例）

步骤 1：找到想要的 Skill

  1. 在 Skill 商店搜索 "Excel 合并"
  2. 浏览结果，查看 Skill 的：
    • 功能介绍
    • 使用次数
    • 用户评价
    • 开发者信息

步骤 2：一键安装

  1. 点击 Skill 卡片进入详情页
  2. 点击 "安装" 按钮
  3. 等待安装完成（通常几秒）

步骤 3：开始使用

  1. 安装后，Skill 会自动出现在你的 AI 对话中
  2. 直接对 AI 说："帮我把这几个 Excel 文件合并"
  3. AI 会自动调用 Excel 合并 Skill 处理

五、Skill 使用技巧

技巧 1：用自然语言描述需求

❌ 错误示范："调用 Excel 合并 Skill"
✅ 正确示范："帮我把这 5 个 Excel 文件合并成一个，按日期排序"

技巧 2：提供清晰的文件

  • 文件命名要规范（如：2024-01-销售数据.xlsx）
  • 告诉 AI 每个文件的内容
  • 说明期望的输出格式

技巧 3：分步骤处理复杂任务

如果任务很复杂，可以拆成多轮对话：

  1. 第一轮："先帮我合并这些 Excel"
  2. 第二轮："合并后按销售额排序"
  3. 第三轮："生成一个汇总图表"

六、常见问题解答

Q1：Skill 是免费的吗？

大部分 Skill 免费，部分高级 Skill 可能需要付费。安装前会明确提示。

Q2：Skill 安全吗？我的文件会不会泄露？

  • Coze 官方 Skill 经过审核，相对安全
  • 第三方 Skill 建议查看开发者信息和使用评价
  • 敏感文件建议先脱敏再处理

Q3：Skill 可以卸载吗？

可以。进入 "我的技能" 页面，找到已安装的 Skill，点击卸载即可。

Q4：一个任务可以用多个 Skill 吗？

可以。AI 会自动判断需要调用哪些 Skill，按顺序执行。

Q5：Coze 的 Skill 和 OpenClaw 的 Skill 有什么区别？

  • Coze：零代码，更适合普通用户
  • OpenClaw：需要一定技术基础，但功能更强大
