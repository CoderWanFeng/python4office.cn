大家好，我是正在实战各种 AI 项目的程序员晚枫。

每天下班前最痛苦的事是什么？写日报——回忆今天做了什么、整理数据、写总结，至少 30 分钟。

今天教你配置 日报自动生成 Skill，让 AI 自动帮你汇总数据、生成日报。

一、Skill 简介

功能概述

日报自动生成 Skill 可以：

  • ✅ 自动汇总任务管理工具数据（飞书任务/钉钉/Notion 等）
  • ✅ 自动读取日历事件
  • ✅ 自动统计代码提交/Git 记录
  • ✅ 自动读取邮件/消息摘要
  • ✅ 生成结构化的日报文档
  • ✅ 定时自动发送

适用人群

人群收益
程序员自动汇总代码提交、Bug 修复
产品经理自动汇总需求进度、会议记录
销售人员自动汇总客户跟进、订单数据
运营人员自动汇总数据报表、活动进展
项目经理自动汇总项目进度、风险预警

二、安装 Skill
