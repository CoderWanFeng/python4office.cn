PythonеҹәзЎҖпјҡPythonе…Ҙй—Ёж•ҷзЁӢ В· 23и®ІпјҲе®Ңз»“пјү
Pythonиҝӣйҳ¶пјҡжҲ‘е·Із»ҸеӯҰе®ҢPythonеҹәзЎҖдәҶпјҢжғіеҺ»жүҫзЁӢеәҸе‘ҳе·ҘдҪңиҝҳйңҖиҰҒеӯҰд№ е“ӘдәӣзҹҘиҜҶпјҹ

Whatпјҹ

1. д»Җд№ҲжҳҜPythonпјҹ

PythonжҳҜдёҖз§Қзј–зЁӢиҜӯиЁҖпјҢе®ғжңүеҜ№иұЎгҖҒжЁЎеқ—гҖҒзәҝзЁӢгҖҒејӮеёёеӨ„зҗҶе’ҢиҮӘеҠЁеҶ…еӯҳз®ЎзҗҶгҖӮеҸҜд»ҘеҠ е…ҘдёҺе…¶д»–иҜӯиЁҖзҡ„еҜ№жҜ”гҖӮдёӢйқўжҳҜеӣһзӯ”иҝҷдёҖй—®йўҳзҡ„еҮ дёӘе…ій”®зӮ№пјҡ

a. PythonжҳҜдёҖз§Қи§ЈйҮҠеһӢиҜӯиЁҖпјҢpythonд»Јз ҒеңЁиҝҗиЎҢд№ӢеүҚдёҚйңҖиҰҒзј–иҜ‘гҖӮ

b. PythonжҳҜеҠЁжҖҒзұ»еһӢиҜӯиЁҖпјҢеңЁеЈ°жҳҺеҸҳйҮҸж—¶пјҢдёҚйңҖиҰҒиҜҙжҳҺеҸҳйҮҸзҡ„зұ»еһӢгҖӮ

c. PythonйҖӮеҗҲйқўеҗ‘еҜ№иұЎзҡ„зј–зЁӢпјҢеӣ дёәе®ғж”ҜжҢҒйҖҡиҝҮз»„еҗҲдёҺз»§жүҝзҡ„ж–№ејҸе®ҡд№үзұ»гҖӮ

d. еңЁPythonиҜӯиЁҖдёӯпјҢеҮҪж•°жҳҜз¬¬дёҖзұ»еҜ№иұЎгҖӮ

e. Pythonд»Јз Ғзј–еҶҷеҝ«пјҢдҪҶжҳҜиҝҗиЎҢйҖҹеәҰжҜ”зј–иҜ‘иҜӯиЁҖйҖҡеёёиҰҒж…ўгҖӮ

f. Pythonз”ЁйҖ”е№ҝжіӣпјҢеёёиў«з”ЁдҪңвҖңиғ¶ж°ҙиҜӯиЁҖвҖқпјҢеҸҜеё®еҠ©е…¶д»–иҜӯиЁҖе’Ңз»„д»¶ж”№е–„иҝҗиЎҢзҠ¶еҶөгҖӮ

g. дҪҝз”ЁPythonпјҢзЁӢеәҸе‘ҳеҸҜд»Ҙдё“жіЁдәҺз®—жі•е’Ңж•°жҚ®з»“жһ„зҡ„и®ҫи®ЎпјҢиҖҢдёҚз”ЁеӨ„зҗҶеә•еұӮзҡ„з»ҶиҠӮгҖӮ

2. д»Җд№ҲжҳҜPythonиҮӘзңҒпјҹ

pythonиҮӘзңҒжҳҜpythonе…·жңүзҡ„дёҖз§ҚиғҪеҠӣпјҢдҪҝзЁӢеәҸе‘ҳйқўеҗ‘еҜ№иұЎзҡ„иҜӯиЁҖжүҖеҶҷзҡ„зЁӢеәҸеңЁиҝҗиЎҢж—¶,иғҪеӨҹиҺ·еҫ—еҜ№иұЎзҡ„зұ»pythonеһӢгҖӮPythonжҳҜдёҖз§Қи§ЈйҮҠеһӢиҜӯиЁҖгҖӮдёәзЁӢеәҸе‘ҳжҸҗдҫӣдәҶжһҒеӨ§зҡ„зҒөжҙ»жҖ§е’ҢжҺ§еҲ¶еҠӣгҖӮ

3. д»Җд№ҲжҳҜPEP 8пјҹ

PEP8жҳҜдёҖз§Қзј–зЁӢи§„иҢғпјҢеҶ…е®№жҳҜдёҖдәӣе…ідәҺеҰӮдҪ•и®©дҪ зҡ„зЁӢеәҸжӣҙе…·еҸҜиҜ»жҖ§зҡ„е»әи®®гҖӮ

4. д»Җд№ҲжҳҜpicklingе’Ңunpicklingпјҹ

PickleжЁЎеқ—иҜ»е…Ҙд»»дҪ•PythonеҜ№иұЎпјҢе°Ҷе®ғд»¬иҪ¬жҚўжҲҗеӯ—з¬ҰдёІпјҢз„¶еҗҺдҪҝз”ЁdumpеҮҪж•°е°Ҷе…¶иҪ¬еӮЁеҲ°дёҖдёӘж–Үд»¶дёӯвҖ”вҖ”иҝҷдёӘиҝҮзЁӢеҸ«еҒҡpicklingгҖӮеҸҚд№Ӣд»ҺеӯҳеӮЁзҡ„еӯ—з¬ҰдёІж–Үд»¶дёӯжҸҗеҸ–еҺҹе§ӢPythonеҜ№иұЎзҡ„иҝҮзЁӢпјҢеҸ«еҒҡunpicklingгҖӮ

5. д»Җд№ҲжҳҜPythonиЈ…йҘ°еҷЁпјҹ

PythonиЈ…йҘ°еҷЁжҳҜPythonдёӯзҡ„зү№жңүеҸҳеҠЁпјҢеҸҜд»ҘдҪҝдҝ®ж”№еҮҪж•°еҸҳеҫ—жӣҙе®№жҳ“гҖӮ

6. д»Җд№ҲжҳҜPythonзҡ„е‘ҪеҗҚз©әй—ҙпјҹ

еңЁPythonдёӯпјҢжүҖжңүзҡ„еҗҚеӯ—йғҪеӯҳеңЁдәҺдёҖдёӘз©әй—ҙдёӯпјҢе®ғд»¬еңЁиҜҘз©әй—ҙдёӯеӯҳеңЁе’Ңиў«ж“ҚдҪңвҖ”вҖ”иҝҷе°ұжҳҜе‘ҪеҗҚз©әй—ҙгҖӮе®ғе°ұеҘҪеғҸдёҖдёӘзӣ’еӯҗпјҢжҜҸдёҖдёӘеҸҳйҮҸеҗҚеӯ—йғҪеҜ№еә”иЈ…зқҖдёҖдёӘеҜ№иұЎгҖӮеҪ“жҹҘиҜўеҸҳйҮҸзҡ„ж—¶еҖҷпјҢдјҡд»ҺиҜҘзӣ’еӯҗйҮҢйқўеҜ»жүҫзӣёеә”зҡ„еҜ№иұЎгҖӮ

7. д»Җд№ҲжҳҜеӯ—е…ёжҺЁеҜјејҸе’ҢеҲ—иЎЁжҺЁеҜјејҸпјҹ

е®ғд»¬жҳҜеҸҜд»ҘиҪ»жқҫеҲӣе»әеӯ—е…ёе’ҢеҲ—иЎЁзҡ„иҜӯжі•з»“жһ„гҖӮ

8. LambdaеҮҪж•°жҳҜд»Җд№Ҳпјҹ

иҝҷжҳҜдёҖдёӘеёёиў«з”ЁдәҺд»Јз Ғдёӯзҡ„еҚ•дёӘиЎЁиҫҫејҸзҡ„еҢҝеҗҚеҮҪж•°гҖӮ

9. *argsпјҢ**kwargsпјҹеҸӮж•°жҳҜд»Җд№Ҳпјҹ

еҰӮжһңжҲ‘д»¬дёҚзЎ®е®ҡиҰҒеҫҖеҮҪж•°дёӯдј е…ҘеӨҡе°‘дёӘеҸӮж•°пјҢжҲ–иҖ…жҲ‘д»¬жғіеҫҖеҮҪж•°дёӯд»ҘеҲ—иЎЁе’Ңе…ғз»„зҡ„еҪўејҸдј еҸӮж•°ж—¶пјҢйӮЈе°ұдҪҝиҰҒз”Ё*argsпјӣеҰӮжһңжҲ‘д»¬дёҚзҹҘйҒ“иҰҒеҫҖеҮҪж•°дёӯдј е…ҘеӨҡе°‘дёӘе…ій”®иҜҚеҸӮж•°пјҢжҲ–иҖ…жғідј е…Ҙеӯ—е…ёзҡ„еҖјдҪңдёәе…ій”®иҜҚеҸӮж•°ж—¶пјҢйӮЈе°ұиҰҒдҪҝз”ЁkwargsгҖӮ

10. д»Җд№ҲжҳҜPassиҜӯеҸҘпјҹ

PassжҳҜдёҖдёӘеңЁPythonдёӯдёҚдјҡиў«жү§иЎҢзҡ„иҜӯеҸҘгҖӮеңЁеӨҚжқӮиҜӯеҸҘдёӯпјҢеҰӮжһңдёҖдёӘең°ж–№йңҖиҰҒжҡӮж—¶иў«з•ҷзҷҪпјҢе®ғеёёеёёиў«з”ЁдәҺеҚ дҪҚз¬ҰгҖӮ

11. unittestжҳҜд»Җд№Ҳпјҹ

еңЁPythonдёӯпјҢunittestжҳҜPythonдёӯзҡ„еҚ•е…ғжөӢиҜ•жЎҶжһ¶гҖӮе®ғжӢҘжңүж”ҜжҢҒе…ұдә«жҗӯе»әгҖҒиҮӘеҠЁжөӢиҜ•гҖҒеңЁжөӢиҜ•дёӯжҡӮеҒңд»Јз ҒгҖҒе°ҶдёҚеҗҢжөӢиҜ•иҝӯд»ЈжҲҗдёҖз»„пјҢзӯүзӯүзҡ„еҠҹиғҪгҖӮ

11. жһ„йҖ еҷЁжҳҜд»Җд№Ҳпјҹ

жһ„йҖ еҷЁжҳҜе®һзҺ°иҝӯд»ЈеҷЁзҡ„дёҖз§ҚжңәеҲ¶гҖӮе®ғеҠҹиғҪзҡ„е®һзҺ°дҫқиө–дәҺyieldиЎЁиҫҫејҸпјҢйҷӨжӯӨд№ӢеӨ–е®ғи·ҹжҷ®йҖҡзҡ„еҮҪж•°жІЎжңүдёӨж ·гҖӮ

12. doc stringжҳҜд»Җд№Ҳпјҹ

Pythonдёӯж–ҮжЎЈеӯ—з¬ҰдёІиў«з§°дёәdocstringпјҢе®ғеңЁPythonдёӯзҡ„дҪңз”ЁжҳҜдёәеҮҪж•°гҖҒжЁЎеқ—е’Ңзұ»жіЁйҮҠз”ҹжҲҗж–ҮжЎЈгҖӮ

13. иҙҹзҙўеј•жҳҜд»Җд№Ҳпјҹ

Pythonдёӯзҡ„еәҸеҲ—зҙўеј•еҸҜд»ҘжҳҜжӯЈд№ҹеҸҜд»ҘжҳҜиҙҹгҖӮеҰӮжһңжҳҜжӯЈзҙўеј•пјҢ0жҳҜеәҸеҲ—дёӯзҡ„з¬¬дёҖдёӘзҙўеј•пјҢ1жҳҜз¬¬дәҢдёӘзҙўеј•гҖӮеҰӮжһңжҳҜиҙҹзҙўеј•пјҢпјҲ-1пјүжҳҜжңҖеҗҺдёҖдёӘзҙўеј•иҖҢпјҲ-2пјүжҳҜеҖ’ж•°з¬¬дәҢдёӘзҙўеј•гҖӮ

14. жЁЎеқ—е’ҢеҢ…жҳҜд»Җд№Ҳпјҹ

еңЁPythonдёӯпјҢжЁЎеқ—жҳҜжҗӯе»әзЁӢеәҸзҡ„дёҖз§Қж–№ејҸгҖӮжҜҸдёҖдёӘPythonд»Јз Ғж–Үд»¶йғҪжҳҜдёҖдёӘжЁЎеқ—пјҢе№¶еҸҜд»Ҙеј•з”Ёе…¶д»–зҡ„жЁЎеқ—пјҢжҜ”еҰӮеҜ№иұЎе’ҢеұһжҖ§гҖӮ

дёҖдёӘеҢ…еҗ«и®ёеӨҡPythonд»Јз Ғзҡ„ж–Үд»¶еӨ№жҳҜдёҖдёӘеҢ…гҖӮдёҖдёӘеҢ…еҸҜд»ҘеҢ…еҗ«жЁЎеқ—е’Ңеӯҗж–Үд»¶еӨ№гҖӮ

15. еһғеңҫеӣһж”¶жҳҜд»Җд№Ҳпјҹ

еңЁPythonдёӯпјҢдёәдәҶи§ЈеҶіеҶ…еӯҳжі„йңІй—®йўҳпјҢйҮҮз”ЁдәҶеҜ№иұЎеј•з”Ёи®Ўж•°пјҢе№¶еҹәдәҺеј•з”Ёи®Ўж•°е®һзҺ°иҮӘеҠЁеһғеңҫеӣһж”¶гҖӮ

16. CSRFжҳҜд»Җд№Ҳпјҹ

CSRFжҳҜдјӘйҖ е®ўжҲ·з«ҜиҜ·жұӮзҡ„дёҖз§Қж”»еҮ»пјҢCSRFзҡ„иӢұж–Үе…Ёз§°жҳҜCross Site Request ForgeryпјҢеӯ—йқўдёҠзҡ„ж„ҸжҖқжҳҜи·Ёз«ҷзӮ№дјӘйҖ иҜ·жұӮгҖӮ

Howпјҹ

1. еҰӮдҪ•и®©дҪ зҡ„зЁӢеәҸжӣҙе…·еҸҜиҜ»жҖ§пјҹ

йҖӮеҪ“ең°еҠ е…ҘйқһеүҚеҜјз©әж јпјҢйҖӮеҪ“зҡ„з©әиЎҢд»ҘеҸҠдёҖиҮҙзҡ„е‘ҪеҗҚгҖӮ

2. PythonжҳҜеҰӮдҪ•иў«и§ЈйҮҠзҡ„пјҹ

PythonжҳҜдёҖз§Қи§ЈйҮҠжҖ§иҜӯиЁҖпјҢе®ғзҡ„жәҗд»Јз ҒеҸҜд»ҘзӣҙжҺҘиҝҗиЎҢгҖӮPythonи§ЈйҮҠеҷЁдјҡе°Ҷжәҗд»Јз ҒиҪ¬жҚўжҲҗдёӯй—ҙиҜӯиЁҖпјҢд№ӢеҗҺеҶҚзҝ»иҜ‘жҲҗжңәеҷЁз ҒеҶҚжү§иЎҢгҖӮ

3. еҰӮдҪ•еңЁPythonдёӯжӢ·иҙқдёҖдёӘеҜ№иұЎпјҹ

еҰӮжһңиҰҒеңЁPythonдёӯжӢ·иҙқдёҖдёӘеҜ№иұЎпјҢеӨ§еӨҡж—¶еҖҷдҪ еҸҜд»Ҙз”Ёcopy.copy()жҲ–иҖ…copy.deepcopy()гҖӮдҪҶе№¶дёҚжҳҜжүҖжңүзҡ„еҜ№иұЎйғҪеҸҜд»Ҙиў«жӢ·иҙқгҖӮ

4. еҰӮдҪ•з”ЁPythonеҲ йҷӨдёҖдёӘж–Үд»¶пјҹ
