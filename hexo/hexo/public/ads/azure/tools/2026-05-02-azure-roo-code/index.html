Roo Code 是一个专注于代码审查和优化的 AI 工具，接入 Azure OpenAI 可以获得 GPT-4 的强大能力。

今天教大家怎么在 Roo Code 里接入 Azure OpenAI 大模型。

一、为什么要在 Roo Code 里用 Azure OpenAI Coding Plan？

Roo Code 是什么？

Roo Code 是一个专注于代码审查和优化的 AI 助手，帮助开发者提高代码质量。

接入 Azure OpenAI 有什么好处？

优势说明
🤖 GPT-4 加持OpenAI 最强模型，代码生成质量高
🏢 企业级安全Azure 安全合规，数据不外泄
🔧 配置简单OpenAI兼容接口，5分钟搞定
🌏 国内节点Azure 中国区，响应速度快
💰 按量计费用多少付多少，不浪费

二、配置步骤

Step 1：开通 Azure OpenAI 服务

👉 专属优惠通道：https://ai.azure.com

开通 Azure OpenAI 服务后，在 Azure 门户中创建一个 OpenAI 资源，获取你的 API Key 和端点地址。

Step 2：在 Roo Code 中配置 Azure OpenAI

  1. 打开 Roo Code 设置 → AI Providers
  2. 选择"Custom"或"OpenAI Compatible"
  3. 填写以下信息：
字段内容
API URLhttps://<你的资源名>.openai.azure.com/v1
API Key你的 Azure OpenAI API Key
模型名称gpt-4
  1. 保存并设置为默认模型

Step 3：开始使用

配置成功后，在 Roo Code 中直接使用 GPT-4 进行代码审查和优化。

三、常见问题

Q：需要付费吗？
