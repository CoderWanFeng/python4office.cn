在 Droid 里使用 Azure OpenAI Coding Plan

发表于2026-05-02|更新于2026-05-02|AI编程工具配置
|阅读量:

Droid 是一个新兴的 AI 编程工具，接入 Azure OpenAI 可以获得 GPT-4 的强大能力，同时享受 Azure 企业级安全保障。

今天教大家怎么在 Droid 里接入 Azure OpenAI 大模型。

一、为什么要在 Droid 里用 Azure OpenAI Coding Plan？

Droid 是什么？

Droid 是一个专注于轻量级代码生成的 AI 工具，帮助开发者快速完成编程任务。

接入 Azure OpenAI 有什么好处？

优势说明
🤖 GPT-4 加持OpenAI 最强模型，代码生成质量高
🏢 企业级安全Azure 安全合规，数据不外泄
🔧 配置简单OpenAI兼容接口，5分钟搞定
🌏 国内节点Azure 中国区，响应速度快
💰 按量计费用多少付多少，不浪费

二、配置步骤

Step 1：开通 Azure OpenAI 服务

👉 专属优惠通道：https://ai.azure.com

开通 Azure OpenAI 服务后，在 Azure 门户中创建一个 OpenAI 资源，获取你的 API Key 和端点地址。

Step 2：在 Droid 中配置 Azure OpenAI

  1. 打开 Droid 设置 → AI Providers
  2. 选择"Custom"或"OpenAI Compatible"
  3. 填写以下信息：
字段内容
API URLhttps://<你的资源名>.openai.azure.com/v1
API Key你的 Azure OpenAI API Key
模型名称gpt-4
  1. 保存并设置为默认模型

Step 3：开始使用

配置成功后，在 Droid 中直接使用 GPT-4 进行代码生成和优化。

三、常见问题

Q：需要付费吗？

A：需要。

Azure OpenAI 按量计费，用多少付多少。
