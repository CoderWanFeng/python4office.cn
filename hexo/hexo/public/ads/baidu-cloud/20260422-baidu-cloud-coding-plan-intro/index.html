说起国内 AI 大模型，百度绝对是老资格了。文心一言发布至今，已经有了相当的积累。现在，百度智能云也推出了 Coding Plan，给开发者带来了新的选择。

一、什么是百度智能云 Coding Plan？

百度的 Coding Plan 基于文心大模型（ERNIE）系列，提供 AI 编程辅助服务。

核心特点

特性说明
文心大模型百度自研，国产大模型元老
飞桨框架百度深度学习框架，生态完善
多场景覆盖代码生成、补全、审查等
企业级服务百度云的企业级支持

文心系列模型

  • ERNIE 4.0：百度最新旗舰大模型
  • ERNIE 3.5：成熟稳定的中端模型
  • ERNIE Speed：快速响应版

二、为什么选百度 Coding Plan？

1. 百度 AI 积累深

百度是国内最早布局 AI 的公司之一：

  • 2019 年就发布 ERNIE
  • 文心一言用户量国内领先
  • AI 技术积累深厚

2. 飞桨生态加持

如果你用飞桨（PaddlePaddle）框架，百度 Coding Plan 可以无缝衔接：

  • 框架层面的优化
  • 专属的开发工具
  • 完善的学习资源

3. 企业级支持

百度的企业级服务比较成熟：

  • SLA 保障
  • 专属技术支持
  • 发票、对公等服务完善

百度有多种套餐可选，从个人版到企业版。

第三步：接入开发工具

支持主流 IDE，具体看官方文档。
