Hello everyone, this is Programmer Wanfeng.

I've compiled the most common questions about Alibaba Cloud Coding Plan. If you're considering subscribing, this is a must-read.

I. Basic Questions

Q1: What is Alibaba Cloud Coding Plan?

An AI programming subscription service launched by Alibaba Cloud Bailian, providing:

  • Fixed monthly fee, no token anxiety
  • Multiple models freely switchable (Qianwen + Kimi + GLM, etc.)
  • Support for mainstream AI programming tools

Q2: What's the difference between Coding Plan and Tongyi Lingma?

Tongyi LingmaAlibaba Cloud Coding Plan
PositioningIntelligent coding assistantAI programming package
Usage methodIDE pluginAPI call
BillingFree/per person per monthFixed monthly fee
Suitable forIndividual developmentTool integration/teams

Simple understanding: Tongyi Lingma is a "partner", Coding Plan is "fuel".

Q3: Is the first month ¥7.9 real?

Yes, it's real. The Lite version has a special first-month price of ¥7.9, but only for the first month.

From the second month onwards, it returns to the original price, subject to the official website.

II. Packages and Billing

Q4: What's the difference between Lite and Pro versions?

Comparison itemLite versionPro version
First month price¥7.9¥39.9
Monthly request quota18,000 times90,000 times
Model supportAllAll
Suitable forIndividual/light useHeavy/team use

Q5: What if I use up my request quota?

  • Wait for next month's reset
  • Upgrade to Pro version
  • Purchase additional quota (if available)

Q6: Can I unsubscribe?

Unsubscription policy is subject to Alibaba Cloud's official website. Generally:

  • Monthly subscriptions can be cancelled at any time
  • After cancellation, you can still use it for the remaining days of the current month

Q7: Can I switch between Lite and Pro?

Yes. You can upgrade or downgrade the package in the console.

III. Model-related Questions

Q8: Which models are supported?

Qianwen series:

  • Qwen3.5-Plus
  • Qwen3-Max
  • Qwen3-Coder-Next
  • Qwen3-Coder-Plus

Third-party models:

  • Kimi-k2.5
  • GLM-5, GLM-4.7
  • MiniMax M2.5

Q9: Is the Qianwen code model strong?

Very strong. Qwen Coder performs excellently in multiple code evaluations:

  • Strong Chinese understanding ability
  • High code generation quality
  • Support for multiple programming languages

Q10: Can I manually switch models?

Yes. Manual model selection is supported, as well as Auto intelligent scheduling.

Q11: Will the models be updated?

Yes. Alibaba Cloud will update models不定期, and Coding Plan users automatically get updates.

IV. Tool Configuration Questions

Q12: Which programming tools are supported?

Supports mainstream AI programming tools:

  • Qwen Code (Alibaba official)
  • Claude Code
  • Cursor
  • Cline
  • Codex
  • Kilo CLI

Q13: How to configure Qwen Code?

  1. Download and install Qwen Code
  2. Run /auth and select Api-Key method
  3. Select Coding Plan
  4. Enter API Key

Q14: Can Cursor be used?

Yes. Configure Alibaba Cloud Coding Plan's API Key and Base URL in Cursor.

Q15: How to connect to VS Code?

Through the Cline plugin, configure Alibaba Cloud Coding Plan's API information.

V. Usage Experience Questions

Q16: How does it compare to Volcano Ark Coding Plan?

Comparison itemAlibaba CloudVolcano Ark
First month price¥7.9¥8.91
Core modelsQianwen seriesDoubao series
Chinese abilityStrongStronger
Ecosystem integrationAlibaba Cloud ecosystemByteDance ecosystem

Conclusion: Alibaba Cloud Qianwen code model is stronger, and the first month is cheaper.

Q17: Is the network stable?

Alibaba Cloud's domestic infrastructure ensures network stability and speed.

Q18: How is Chinese support?

Qianwen models have excellent Chinese support, providing a good experience for Chinese project development.

VI. Other Questions

Q19: Can enterprises use it?

Yes. Alibaba Cloud Coding Plan supports enterprise users, providing:

  • Team management
  • Usage statistics
  • Enterprise invoices
  • Technical support

Q20: Is there a free trial?

New users' first month at ¥7.9 is equivalent to an extremely low-cost trial. Specific free policies are subject to the official website.
