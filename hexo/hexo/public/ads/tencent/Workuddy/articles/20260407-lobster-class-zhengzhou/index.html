1. 实机操作，手把手教学

现场带电脑，专家1对1指导，有问题当场解决。

2. 认识同路人

来现场的，都是对AI、对一人公司感兴趣的人。交个朋友，说不定以后能合作。

3. 面对面交流

听完我的分享，有问题可以直接问我。这种互动，线上做不到。

4. 限定周边

腾讯限定龙虾周边，到场就送。 collector的福音。（笑）
