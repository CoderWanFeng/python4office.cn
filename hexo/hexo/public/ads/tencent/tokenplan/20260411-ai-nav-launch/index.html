🚀 AI导航网站地址：https://www.python-office.com/ai-nav/

这个网站是我昨天刚上线的，收录了我亲自测试过的 100+ 款优质AI工具，涵盖编程开发、图像设计、文案写作、办公效率、学习提升等十几个分类。

零广告、纯干货、亲自测评——每一款工具都是我用过觉得好用才放进去的。

为什么要做这个AI导航网站？

痛点：AI工具太多，找起来太费劲

相信很多朋友都有这种经历：

  • 刷抖音/B站看到某个AI工具很厉害，想用的时候却忘了叫啥名
  • 收藏夹里存了几十个AI网站，关键时刻一个都找不到
  • 想找个免费替代方案，结果搜出来全是广告和付费推广
  • 同样的功能，不知道哪个工具做得更好

我自己就是重度AI工具用户，平时写代码、做内容、处理文档都离不开各种AI助手。但工具一多，管理就成了大问题。

解决：做一个"我每天都在用"的导航站

与其依赖别人的导航站（要么广告太多，要么更新太慢），不如自己动手做一个：

  1. 亲测有效：每一款工具我都亲自用过，不好用的直接过滤
  2. 分类清晰：按使用场景分类，找工具一目了然
  3. 持续更新：我自己每天都在发现新工具，网站会同步更新
  4. 完全免费：没有付费墙，没有套路，打开就能用

技术实现：半天上线的秘密

很多朋友问我：做一个导航网站是不是要花很多时间？

答案是：半天就够了。

我的开发流程

第一步：明确需求（30分钟）

先想清楚网站要有什么：

  • 首页：分类卡片 + 热门推荐
  • 分类页：按类别展示工具
  • 详情页：工具介绍 + 直达链接 + 使用评价
  • 搜索功能：快速查找工具

第二步：选择技术栈（10分钟）

既然是导航站，不需要太复杂的功能：

  • 前端：Vue 3 + Vite（开发快、性能好）
  • 样式：Tailwind CSS（原子化CSS，不用写样式文件）
  • 部署：Vercel（免费、自动CI/CD）

第三步：准备数据（2小时）

这是最耗时的部分——整理工具列表：

第四步：开发页面（2小时）

用AI辅助写代码，主要做三个页面：

  1. 首页：分类网格 + 搜索框 + 热门推荐
  2. 分类页：侧边栏分类 + 工具卡片列表
  3. 详情页：工具信息 + 评价 + 相关推荐

第五步：部署上线（10分钟）

推送到GitHub，Vercel自动部署，绑定自定义域名，搞定！
