在 Droid 里使用 MiniMax M2.7

发表于2026-05-01|更新于2026-05-01|AI编程工具配置
|阅读量:

Droid 是一个新兴的AI编程工具，接入 MiniMax M2.7 可以获得更强的中文编程支持和多模态能力。

今天教大家怎么在 Droid 里配置 MiniMax。

一、配置步骤

Step 1：获取 API Key

👉 专属9折优惠通道：https://platform.minimaxi.com/subscribe/token-plan?code=8T7rWtR7CZ&source=link

Step 2：在 Droid 中配置

  1. 打开 Droid 设置 → AI Providers
  2. 选择"Custom"或"OpenAI Compatible"
  3. 填写以下信息：
字段内容
API URLhttps://api.minimaxi.com/v1
API Key你的 MiniMax API Key
ModelMiniMax-Text-01 或 m2.7-pro
  1. 保存并测试
