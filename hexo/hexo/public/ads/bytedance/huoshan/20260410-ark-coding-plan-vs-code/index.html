常用功能

1. 代码解释

选中代码 → 右键 → Explain

AI会解释这段代码的工作原理。

2. 代码生成

在侧边栏输入需求，AI生成代码。

3. 代码重构

选中代码 → 右键 → Edit → 输入修改要求

4. 自动补全

输入代码时，按Tab接受AI建议。

快捷键

快捷键功能

Cmd/Ctrl + Shift + L新对话
Tab接受建议
