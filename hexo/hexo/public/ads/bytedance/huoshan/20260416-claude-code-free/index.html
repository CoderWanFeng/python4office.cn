"Claude Code 太贵了，一个月好几百，学生党真的用不起。"

这句话我听过无数次。

但你知道吗？Claude Code 本身是免费开源的，你真正花钱的是它调用的 AI 模型。

如果能找到免费的模型 API，那 CC 就等于白嫖。

今天这篇文章，教你如何在国内零成本用上 Claude Code。


💡 先搞清一个概念
