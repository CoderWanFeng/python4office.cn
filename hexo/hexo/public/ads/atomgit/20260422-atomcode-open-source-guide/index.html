AtomCode开源项目托管指南！从0到1，看完就会


兄弟们！！！

想用AtomGit托管开源项目，但不知道怎么做？

看完这篇，10分钟上手。

我是程序员晚枫，全网30万+粉丝，python-office开源作者。

今天手把手教你。

🔥 为什么选AtomGit托管？

先说为什么推荐：

优势说明
国产国内访问，速度快
免费公开仓库完全免费
稳定大厂背书，服务靠谱
AI集成可以搭配AtomCode使用

📝 第一步：注册账号

3分钟搞定：

步骤：

  1. 打开AtomGit官网
  2. 点击注册
  3. 用手机号或邮箱注册
  4. 登录

🚀 第二步：创建仓库

登录后，创建你的第一个仓库：

步骤：

  1. 点击右上角的 + 号
  2. 点击 新建仓库
  3. 填写信息：
    • 仓库名称
    • 描述（可选）
    • 公开/私有
  4. 点击创建

💻 第三步：上传代码

仓库创建好了，上传你的代码：

方法1：Git命令行

方法2：用AtomCode
直接在AtomCode里操作，更简单！

👥 第四步：协作开发

如果是团队项目：

怎么做：

  1. 进入仓库 设置
  2. 点击 协作者管理
  3. 添加成员的AtomGit用户名
  4. 成员接受邀请后，就可以一起开发了

🎯 常用操作

操作说明
提交代码git commit + git push
创建分支git branch + git push
合并分支提交Pull Request
管理Issue在仓库的Issue页面
