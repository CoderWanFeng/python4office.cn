最近各大厂商都在推Coding Plan，很多朋友问我：小米、火山、智谱，这三家该怎么选？

今天我就给大家做一个详细的横向对比。

一、三家核心特点对比

先看一张核心对比表：

维度小米 MiMo火山方舟智谱 GLM
公司小米字节跳动智谱AI
核心模型MiMo-V2-ProDoubao/豆包GLM-5.1
价格档位4档5档4档
入门价39元19元149元
时间窗口无限制5小时5小时

1. 小米 MiMo

核心优势：

  • ✅ 无时间窗口限制！
  • ✅ 价格适中
  • ✅ 小米生态加持
  • ✅ TTS 免费

适合人群：

  • 喜欢自由度的开发者
  • 小米生态开发者
  • 对价格敏感的个人开发者

2. 火山方舟

核心优势：

  • ✅ 多模型支持！
  • ✅ 集成 Claude、GPT
  • ✅ 丰富的模型选择
  • ✅ 价格便宜（入门19元）

适合人群：

  • 需要尝试不同模型的开发者
  • 喜欢"尝鲜"的技术爱好者
  • 对比研究不同模型的用户

3. 智谱 GLM

核心优势：

  • ✅ GLM-5.1 性能强
  • ✅ 技术积累深厚
  • ✅ 国产化优势明显

适合人群：

  • 追求性能的专业开发者
  • 对国产化要求高的用户
  • 需要稳定服务的企业用户

四、怎么选？

1. 按使用习惯选

使用习惯推荐厂商
喜欢集中使用小米 MiMo（无时间窗口）
喜欢尝试不同模型火山方舟（多模型）
追求极致性能智谱 GLM

2. 按预算选

预算区间推荐厂商
20-50元火山方舟 Lite
50-100元小米 MiMo Standard
100-300元火山方舟/小米 Pro
300元以上智谱/小米 Max

3. 按场景选

场景推荐厂商
个人学习体验火山方舟（便宜）
日常开发使用小米 MiMo（无限制）
专业性能需求智谱 GLM（最强）
小米生态开发小米 MiMo（生态加持）

五、我的个人建议

入门阶段

选火山方舟（19元）

  • 便宜，试错成本低
  • 可以体验不同模型
  • 功能齐全

稳定使用阶段

选小米 MiMo（99元）

  • 无时间窗口限制
  • 性价比高
  • Agent能力强

专业需求阶段

按情况选择：

  • 追求性能 → 智谱 GLM
  • 追求生态 → 小米 MiMo
  • 追求灵活 → 火山方舟

总结

需求推荐厂商
最便宜入门火山方舟
无时间限制小米 MiMo
最强性能智谱 GLM
小米生态开发小米 MiMo
多模型体验火山方舟

没有最好的，只有最适合自己的！
