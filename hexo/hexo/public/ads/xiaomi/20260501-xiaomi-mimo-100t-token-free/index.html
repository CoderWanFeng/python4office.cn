太豪了！小米送100万亿Token，个人开发者白嫖全攻略

全网40万+粉丝，6年Python开发经验，开源项目python-office作者

⚠️ 活动时间：北京时间 2026年4月28日 - 5月28日，赠完即止。目前正在发放中，建议尽快申请。

好消息！小米的 MiMo 百万亿 Token 创造者激励计划 正式开始了！

这是一个完全免费的活动，小米的开放平台计划在 30天内发放总计100万亿 Token 权益，面向全球所有开发者，个人、团队、企业均可参与。

今天这篇攻略，帮你搞清楚：怎么领、能领多少、怎么用。

一、这是什么活动？

Xiaomi MiMo Orbit 百万亿 Token 创造者激励计划，是小米面向全球高质量、AI 驱动用户的限时 Token 定向发放活动。

关键信息：

项目内容
活动时间北京时间 2026年4月28日 00:00 至 5月28日 00:00
发放规模100万亿 Token
费用完全免费
截止条件赠完即止
面向对象全球所有用户（个人/团队/企业）

30天内发完、赠完即止——越早申请越好。

二、100万亿 Token 能做什么？

可能有同学还不太理解这个量的概念。

Token 是 AI 调用计费的基本单位，100万亿 Token 是什么概念呢？

按小米开放平台的定价估算：

  • 如果换算成 Lite 套餐（39元/月，含 6000万 Credits）
  • 100万亿 Token ≈ 约 16万个月的 Lite 套餐
  • 折算人民币约 630万元（仅估算参考，以实际到账为准）

所以这次活动的规模是相当大的。

三、什么是 MiMo？

Xiaomi MiMo 是小米自研的大模型，V2.5 是最新系列，包含：

  • 🧠 旗舰推理模型 - 复杂推理、代码生成
  • 🎨 多模态模型 - 图文理解、内容创作
  • 🔊 语音合成模型 - TTS 合成

覆盖文本、图像、语音等场景。API 已对接 Claude Code、Cursor、OpenClaw 等主流编程工具。

也就是说，你现在用 OpenClaw、Cursor 这些工具，可以直接调用小米的大模型了。

四、怎么申请？超详细步骤

第一步：填写申请表单

访问活动页面，点击「立即申请」按钮：
👉 https://100t.xiaomimimo.com

填写内容主要包括：

  • 基本信息（姓名、邮箱等）
  • 你使用的 AI 工具（Claude Code、Cursor、OpenClaw 等）
  • 底层模型（DeepSeek、OpenAI 等）
  • 项目描述（越详细越好）
  • 证明材料（可选，但建议提供）

第二步：等待评估

提交后等待约 3个工作日，系统会综合评估。

评估标准包括：

  • AI 工具使用情况
  • 底层模型
  • 项目描述详细程度
  • 证明材料

填写越详细、项目越具体，通过率和权益档位越高。

第三步：收到通知邮件

评估通过后会收到一封通知邮件，告知你的权益档位。

第四步：登录开放平台领取

收到邮件后：

  1. 使用申请时填写的邮箱登录/注册 MiMo API 开放平台
  2. 权益会在收到邮件后 24小时内 到账

五、注意事项（重要！）

官方特别提醒，以下情况会导致权益无法到账：

情况一：已有账号但邮箱不匹配

如果你的开放平台账号是由手机号注册的，请先绑定申请用的邮箱。

绑定地址：https://account.xiaomi.com

情况二：账号邮箱和申请邮箱不一致

解决方案：

  1. 方案A：用申请邮箱注册新的小米账号，登录开放平台
  2. 方案B：将原有小米账号的邮箱换绑为申请邮箱
    • 更换地址：https://id.mi.com
    • 换绑后 24 小时内权益自动到账

情况三：没收到邮件

检查：

  • 垃圾邮件箱
  • 填写的邮箱地址是否正确
  • 等待是否超过3个工作日

如仍未收到，通过开放平台「联系我们」反馈：https://platform.xiaomimimo.com/contact

六、权益怎么用？

收到权益后，你可以在开放平台：

  1. 调用 MiMo 系列模型（推理、多模态、语音）
  2. 配置到编程工具（Claude Code、Cursor、OpenClaw 等）
  3. 按量消费（赠送的 Token/Credits 直接充入账户）

权益有有效期，请及时使用。

七、申请建议

结合官方说明和经验，建议：

  1. 越早申请越好 - 100万亿 Token，赠完即止，不确定什么时候发完
  2. 表单填写详细 - 项目描述越具体，通过率和档位越高
  3. 准备好证明材料 - 比如 GitHub 链接、个人博客、作品截图等
  4. 确认邮箱可用 - 确保能收到评估结果邮件
  5. 留意权益有效期 - 收到后尽快使用

八、常见问题

Q：个人开发者可以申请吗？

可以。面向全球所有用户，个人、团队、企业均可参与。

Q：申请需要费用吗？

完全免费，不需要任何费用。

Q：每个账号能申请几次？

官方未明确限制次数，如3天内未收到结果可重新申请。

Q：Token 权益可以转让吗？

权益仅限本人使用，不可转让。

Q：国际版用户可以参加吗？

活动面向全球所有用户开放。

总结

这次小米 MiMo 的百万亿 Token 活动，对开发者来说是难得的机会：

  • ✅ 完全免费
  • ✅ 规模大（100万亿 Token）
  • ✅ 权益到账快（24小时内）
  • ✅ 可对接主流编程工具

建议尽快去申请，赠完即止。
