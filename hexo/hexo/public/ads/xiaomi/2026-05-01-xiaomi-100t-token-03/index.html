手把手教你申请小米100万亿Token权益（附避坑指南）

小米100万亿Token免费发放，但很多人申请的时候踩坑了。

这篇文章手把手教你申请，避免被拒。

一、申请入口

👉 官方地址：https://100t.xiaomimimo.com

直接点击「立即申请」即可。

二、填写表单的技巧

❌ 错误示范

问题错误回答
使用场景"写代码"
项目描述"我日常会用AI"
证明材料不填

✅ 正确示范

问题建议回答
使用场景"自动化代码审查工具开发"
项目描述"正在开发一个基于大模型的代码质量分析工具，需要调用API进行代码逻辑分析、潜在bug检测、代码风格建议等功能。预计每天调用API约500次"
证明材料附上GitHub链接、产品截图等

核心原则：越具体越好，越真实越好。

三、常见坑点

坑点1：邮箱不一致

问题：申请表填的邮箱 ≠ 小米账号绑定的邮箱

结果：权益发不到你的账户

解决：

  • 如果你已有小米账号，用申请时的邮箱去绑定
  • 如果你没有小米账号，收到通过邮件后，用申请时的邮箱注册

坑点2：描述太笼统

问题：填写"我想用AI"，"学习AI"，"写代码"

结果：被判定为低质量申请，拒绝或给最低档权益

解决：写清楚具体场景、项目名称、预期调用频率

坑点3：没有证明材料

问题：空着不填

结果：评估通过率降低

解决：附上GitHub主页、博客链接、产品截图等任意证明

坑点4：申请后不检查邮箱

问题：权益发放通过邮件通知，但你没看

结果：错过权益到账时间

解决：每天检查邮箱（包括垃圾箱）

四、权益到账后怎么用？

权益到账后，你可以在MiMo开放平台使用：

  • 配置API Key到Claude Code
  • 配置API Key到Cursor
  • 配置API Key到OpenClaw
  • 直接在网页端调用
