你敢信？小米直接甩出100万亿Token，免费送！

一、这是什么操作？

2026年4月28日，小米官方宣布启动「MiMo Orbit 百万亿 Token 创造者激励计划」。

简单说：100万亿Token，30天内全部免费发放。

不需要任何付费，只要你用AI做项目、搞创作、做开发，就可以申请。

这不是小米第一次搞大动作，但这次，是真的狠。

二、为什么要送？

你以为小米是在做慈善？

不，这是一场精准的用户争夺战。

2026年，AI大模型的竞争已经从"谁的模型更强"变成了"谁的用户更多"。

OpenAI有ChatGPT，Anthropic有Claude，国内有DeepSeek、豆包、文心一言...

小米的MiMo虽然技术不差，但用户认知度还不够。

怎么办？

用Token换用户，用免费换市场。

100万亿Token听起来吓人，但对小米来说，这只是获客成本的一部分。

三、谁能申请？

所有人都可以。

  • 个人开发者 ✅
  • 团队 ✅
  • 企业 ✅
  • 全球用户 ✅

只要你有一个真实的AI使用场景，就可以申请。

填写表单 → 等待评估 → 收到邮件 → 权益到账

整个过程大约3个工作日。

四、能拿到什么？

根据你的申请情况，可能获得：

  • Token Plan订阅套餐（可直接配置到Claude Code、Cursor等工具）
  • 账户赠金（用于API按量计费）

完全免费，不需要绑定信用卡，不需要后续付费。

五、怎么申请？

👉 官方申请入口：https://100t.xiaomimimo.com

流程很简单：

  1. 点击「立即申请」
  2. 填写表单（使用场景、项目描述）
  3. 等待评估邮件
  4. 登录MiMo开放平台查收权益

小贴士：填写越详细，通过率越高，权益档位也可能更高。
