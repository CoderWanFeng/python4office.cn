## 一、文章写作风格

### 1.1 标题风格

**核心原则：简洁有力、直击痛点、引发好奇**

#### 推荐标题模式

| 模式 | 示例 | 适用场景 |
|------|------|---------|
| **数字+痛点** | 花499元，我带你把全球畅销300万册的Python神书从头敲一遍 | 课程推广、实战教程 |
| **对比反差** | 700页太厚看不下去？跟着我3个项目全部实战，零基础也能学完 | 解决痛点型内容 |
| **承诺型** | Python入门第一书：我带你3个项目从入门到实战，不学完不罢休 | 自媒体文章、课程推荐 |
| **疑问型** | 为什么芯片被卡脖子，中国的AI反而越来越便宜？ | 深度分析、热点解读 |
| **故事型** | 30岁被裁员，多亏我提前学了AI | 成长故事、经验分享 |

#### 标题禁忌

- ❌ 太长（超过25字）
- ❌ 太官方（缺乏情感）
- ❌ 标题党但不切题
- ❌ 过于平淡（无法引起点击欲望）

### 1.2 文章开头风格

**核心原则：直击痛点、引发共鸣、快速切入主题**

#### 推荐开头模式

```
大家好，这里是[身份/角色]。

[直接提出问题或现象]

今天[做某事/分享某经验]...
```

**示例：**

> 大家好，我是程序员晚枫。
>
> 最近在AI编程交流群里，有个问题被反复问到："晚枫，Claude Code我想用，但是因为网络用不了怎么办？"
>
> 我每次都要解释一遍，干脆写篇文章，一次说清楚。

### 1.3 正文风格

**核心原则：大白话、分段落、善用列表、图文结合**

#### 写作特点

1. **口语化表达**
   - 使用"说实话"、"你知道吗"、"太他妈卷了"等口语化表达
   - 拉近与读者的距离

2. **分段清晰**
   - 每个章节不超过300字
   - 重要内容用**粗体**强调
   - 列表项控制在5-7个以内

3. **善用类比**
   - 用生活化的例子解释复杂概念
   - 例如："Claude Code就像一辆车的车身，Claude大模型只是其中一款发动机"

4. **情绪调动**
   - 适当使用感叹号和表情符号
   - 表达真实情感（焦虑、兴奋、无奈等）
   - 引起读者共鸣

### 1.4 文章结尾风格

**核心原则：总结要点、提供行动指引、自然过渡到推荐**

#### 推荐结尾模式

```
## 总结

- [要点1]
- [要点2]
- [要点3]

## 相关阅读

- [相关链接1]
- [相关链接2]

---

[作者介绍/公众号信息]
```

### 1.5 标签(Tags)使用规范

**原则：精准、不堆砌、方便检索**

| 文章类型 | 推荐Tags |
|---------|---------|
| AI编程 | AI编程, OpenClaw, Claude Code, DeepSeek |
| Python教程 | Python, 自动化办公, Pandas |
| 成长故事 | 博主, 自媒体, 个人成长, 转行程序员 |
| 课程推广 | 课程, 付费课程, Python入门, AI编程 |
| 热点分析 | AI热点, 深度分析, 行业观察 |

## 二、图片使用规范

### 2.1 图片来源

| 来源 | 网址 | 适用场景 |
|------|------|---------|
| **Unsplash** | https://unsplash.com/ | 通用免费图片（推荐） |
| **Pexels** | https://www.pexels.com/zh-cn/ | 通用免费图片 |
| **AtomGit** | https://atomgit.com/ | 项目图片、截图 |

### 2.2 图片类型与使用场景

#### 封面图片 (cover)

- **位置：** 文章顶部 YAML 配置中
- **尺寸：** 1200px 宽度
- **主题：** 与文章核心主题相关

**示例：**
```yaml
cover: https://images.unsplash.com/photo-1518770660439-4636190af475?q=80&w=1200&auto=format&fit=crop
```

#### 章节插图

- **位置：** 每个章节关键段落之后
- **尺寸：** 800px 宽度
- **原则：** 图片内容与段落主题紧密相关

**示例：**
```markdown
## 中国人太能卷了，直接把价格打到地板

[段落内容...]

![](https://images.unsplash.com/photo-1507676270988-22c95378f4b3?q=80&w=800&auto=format&fit=crop)
```

#### 产品/课程宣传图

- **位置：** 具体产品介绍处
- **来源：** 通常是官方提供或自行制作

### 2.3 图片主题对照表

| 文章主题 | 封面图片关键词 | 插图关键词 |
|---------|--------------|-----------|
| AI/大模型 | chip, circuit, AI, technology | data center, server, code |
| Python编程 | python, code, programming | computer, keyboard, code screen |
| 职场/成长 | office, meeting, teamwork | career, business, professional |
| 游戏开发 | game, gaming, pixel | game controller, arcade |
| 数据分析 | chart, graph, analytics | data visualization, dashboard |
| 创业/商业 | startup, business, money | business meeting, money, growth |

### 2.4 图片URL获取方法

#### Unsplash 图片获取

1. 访问 https://unsplash.com/
2. 搜索关键词（如：AI, python, data center）
3. 点击图片进入详情页
4. 复制图片URL，通常格式为：
   ```
   https://images.unsplash.com/photo-[图片ID]?q=80&w=1200&auto=format&fit=crop
   ```

#### 图片尺寸参数说明

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| `q` | 图片质量 | 80 |
| `w` | 宽度 | 800（插图）/ 1200（封面） |
| `auto=format` | 自动格式优化 | - |
| `fit=crop` | 裁剪适应尺寸 | - |

## 三、文章结构模板

### 3.1 教程类文章结构

```markdown
---
title: [教程标题]
date: [日期]
tags: [相关标签]
categories: [分类]
description: [简短描述]
cover: [封面图片URL]
---

<!-- more -->

[开头：自我介绍 + 痛点引入]

## 一、[核心概念/问题]
[详细说明]

![相关插图](图片URL)

## 二、[操作步骤/解决方案]
[步骤1]
[步骤2]
[步骤3]

## 三、[进阶技巧/注意事项]
[补充说明]

## 总结
[要点回顾]

## 相关阅读
- [链接1]
- [链接2]

---

[作者信息/公众号引导]
```
