## 🔁 怎么追加新广告记录

1. **改 `promo.js`**：把 `PROMO` 对象里对应字段改成新版本（这是真正生效的配置）
2. **同步改本文件**：
   - 把"当前在用"表格更新为新版本（写上**启用日期 = 今天**）
   - 把旧版本移到"历史版本"区域，标上**停用日期 = 今天**
3. **保留旧版本**：永远不要删除历史，只追加
