# 公众号文章创作指南

> 本文件是「程序员晚枫」公众号长文专属创作规范。
> 每次创作新文章前，必须先阅读本文件。
> 每次发布后根据阅读数据反馈，更新优化本文件。
> **本文件不涉及抖音口播稿创作，任何视频脚本相关规范请查看「口播稿创作指南.md」。**

## 一、账号定位与内容主线

**账号名称**：程序员晚枫  
**内容定位**：AI编程搞钱大白话  
**核心slogan**：科技不高冷，AI很好用

**目标受众**：对AI/科技感兴趣的普通大众，覆盖70后/80后职场人、宝妈、想做轻副业的普通人

## 二、不可动摇的核心创作原则

| 原则 | 说明 |
|------|------|
| **内容价值优先** | 所有内容必须提供真实可落地的价值 |
| **绝对零门槛** | 全程不用代码、编程术语、行业黑话 |
| **信任优先** | 绝不制造焦虑，不割韭菜，帮用户避坑 |
| **聚焦不发散** | 严格围绕AI/科技主线 |
| **阅读友好** | 适配长文阅读习惯，段落清晰，重点突出 |

## 三、每次创作前的热点搜索

| 搜索类别 | 搜索关键词 | 目的 |
|---------|-----------|------|
| ① 当下节日/营销节点 | `2026年X月热点节日营销` | 借势节日情绪 |
| ② AI/科技近期热点 | `最新AI新闻 7天内` / `AI热点 近期` | 抓住时效性热点 |
| ③ 账号自身动态 | `程序员晚枫 AI编程课程` / `python-office` | 结合产品/活动 |

**操作流程**：
```
1. 搜索「当下节日/营销节点」→ 判断是否值得借势
2. 搜索「AI科技近期热点」→ 找3-5个可切入角度
3. 搜索「账号自身动态」→ 确认有无产品/活动需要推广
4. 综合3类信息，确定选题角度和切入角度
```

## 四、标题公式

```
【具体数字/时间锚点】+【事件/产品】+【悬念/预测】= 点击欲望
```

**三种标题类型**：

1. **数字预测型**（最佳）：具体时间 + 预测性结论
   - 30天后，机器人就要进你家了
   - 整整一个亿，单纯为了去找更有创意的作品和人？

2. **事件切入型**：具体事件 + 深度解读
   - 芯片被卡脖子 + AI反而更便宜

3. **认知冲突型**：已知热点 + 反常识现象
   - AI这么火 + 为什么一堆公司倒闭

**写作规范**：
1. 长度控制在15-30字之间（比口播稿标题可略长，公众号标题展示空间更大）
2. 必须包含具体数字/时间锚点
3. 要有预测性或悬念感
4. 从具体产品/事件切入，避免空泛

## 五、口播稿同步规则

> **核心原则**：每次创作新的口播稿时，必须同步生成1篇对应的公众号文章。两者为内容同源的不同平台版本，核心观点、案例、数据保持100%一致，仅做平台适配优化。

### 5.1 同步逻辑

| 对比项 | 口播稿 | 公众号文章 |
|--------|--------|-----------|
| 核心论点 | 必须完全一致 | 完全一致 |
| 案例/数据 | 必须完全一致 | 完全一致 |
| 语言风格 | 口语化短句，朗读友好 | 段落式长文，阅读友好 |
| 结构 | 悬念驱动，节奏紧凑 | 逻辑递进，层次分明 |
| 结尾引流 | 标准口播引流格式 | 底部课程板块 |

### 5.2 改编规则

1. **保留100%核心信息**：论点、案例、数据不能因改编而走样
2. **排版适配**：加入二级/三级小标题，每段不超过3行，提升阅读体验
3. **重点突出**：核心观点、数据、金句加粗显示
4. **保留引流元素**：进群banner、课程链接、固定slogan、底部引流板块全部保留

## 七、语言风格

**必须做到**：

1. **生活化比喻**：用70后熟悉的事物（粮票、供销社、大锅饭、下海、万元户）
2. **段落节奏**：每段不超过3行，留足呼吸空间
3. **小标题体系**：每300-500字一个大章节，加二级小标题
4. **加粗强调**：核心数据、金句加粗，方便快速浏览
5. **情绪价值**：有共情、有温度、有共鸣
6. **70后友好**：避免过于新潮的网感词汇，用大白话讲明白就行

## 八、黑色幽默与嘲讽公式

> 公众号文章节奏比口播稿缓，黑色幽默可适当展开，但同样要遵循点到为止原则。

**核心公式**：
```
具体嘲讽对象 + 反差感 + 一句话点到为止
```

**三种用法**：

① **嘲讽不行动的旁观者**
> Excel用了二十年，每天加班到晚上十点，数据处理还是手动复制粘贴。
> 嗯，那祝你习惯愉快。

② **嘲讽假装努力的假象**
> AI工具买了，钱花了，账号注册了，然后呢？放在那里落灰。
> 这就像你办了健身卡，然后天天看着它过期。

③ **嘲讽自以为占便宜的心态**
> 你以为你在白嫖，其实你才是那个被嫖的。

**禁用**：
- ❌ 不要嘲讽具体的人，只嘲讽行为和现象
- ❌ 不要人身攻击，保持「朋友善意调侃」的尺度

## 十、内容合集与选题方向

### 三大内容合集

| 合集 | 定位 | 结构 |
|------|------|------|
| AI热点大白话 | 把AI/科技热点翻译成大白话 | 热点事件→核心解读→3个影响→行动指引 |
| 普通人的AI工具库 | 实测能直接用的AI工具 | 痛点引入→工具价值→实操步骤→避坑提醒 |
| AI入门避坑指南 | 帮零基础避坑、破解焦虑 | 坑/焦虑点→拆解逻辑→正确认知→避坑清单 |

### 高潜力选题特征

1. **数字预测类**（最佳）：具体时间/数字 + 预测性结论
2. **事件切入类**：从具体发布会/产品/事件切入
3. **认知冲突类**：已知热点 + 反常识现象
4. **揭秘避坑类**：揭示行业套路
5. **技术解读类**：用大白话解读前沿技术

## 十二、绝对禁止触碰的红线

1. ❌ 禁止出现任何代码、编程、Python相关内容（人设背景提及除外）
2. ❌ 禁止制造AI焦虑、夸大宣传、虚假承诺
3. ❌ 禁止使用专业术语、行业黑话
4. ❌ 禁止发散到与AI/科技无关的赛道
5. ❌ 禁止硬广带货、过度营销
6. ❌ 禁止改动账号核心人设、slogan、固定结构
