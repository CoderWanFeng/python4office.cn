# Python Office Tools - 项目总结

## 项目概述

本项目基于python-office库，精选了20个核心办公自动化功能，使用PySide6实现图形界面，并打包为可执行文件。所有工具都具有统一的科技感UI设计风格，操作流畅且美观。

## 功能模块详解

### Excel相关工具

1. **Excel表格合并 (01_excel_merge.py)**
   - 功能：合并多个Excel文件到一个工作簿中
   - 特点：支持合并到单个工作表或不同工作表，可选择添加来源信息

2. **Excel数据筛选 (02_excel_filter.py)**
   - 功能：根据指定条件筛选Excel数据
   - 特点：支持多种筛选条件，可视化数据预览

3. **Excel数据透视表生成 (03_excel_pivot.py)**
   - 功能：根据Excel数据自动生成数据透视表
   - 特点：拖拽式界面设计，支持自定义行列和值字段

4. **Excel图表生成 (04_excel_chart.py)**
   - 功能：根据Excel数据生成各类图表
   - 特点：支持多种图表类型，可自定义样式和导出

5. **Excel公式批量应用 (05_excel_formula.py)**
   - 功能：批量应用公式到Excel文件的指定单元格
   - 特点：支持常见数学、统计和文本公式

### Word相关工具

6. **Word文档批量替换 (06_word_replace.py)**
   - 功能：批量替换Word文档中的文本
   - 特点：支持普通文本替换、格式保留和批量处理

7. **Word文档合并 (07_word_merge.py)**
   - 功能：合并多个Word文档到一个文档中
   - 特点：支持多种合并模式，可自定义格式和样式

8. **Word文档转PDF (08_word_to_pdf.py)**
   - 功能：将Word文档转换为PDF格式
   - 特点：支持多种转换方法，包括LibreOffice、Win32 COM和docx2pdf

9. **Word报告生成 (09_word_report.py)**
   - 功能：基于模板和数据生成Word报告
   - 特点：支持自定义模板，可添加页眉页脚

10. **Word表格处理 (10_word_table.py)**
    - 功能：批量处理Word文档中的表格
    - 特点：支持数据提取、样式修改和表格操作

### PDF相关工具

11. **PDF合并 (11_pdf_merge.py)**
    - 功能：合并多个PDF文件到一个PDF中
    - 特点：支持自定义合并顺序和合并方式

12. **PDF拆分 (12_pdf_split.py)**
    - 功能：将PDF文件按指定方式拆分为多个文件
    - 特点：支持按页数、页面范围等多种拆分模式

13. **PDF水印添加 (13_pdf_watermark.py)**
    - 功能：为PDF文件添加文本或图片水印
    - 特点：支持自定义水印样式和位置

14. **PDF转图片 (14_pdf_to_image.py)**
    - 功能：将PDF页面转换为图片格式
    - 特点：支持多种图片格式和分辨率设置

15. **PDF加密解密 (15_pdf_encrypt_decrypt.py)**
    - 功能：为PDF文件添加或移除密码保护
    - 特点：支持多种权限设置

### 其他办公自动化工具

16. **文件批量重命名 (16_file_rename.py)**
    - 功能：根据规则批量重命名文件
    - 特点：支持多种命名模式和预览功能

17. **邮件批量发送 (17_email_send.py)**
    - 功能：批量发送个性化邮件
    - 特点：支持附件和HTML格式

18. **图片批量处理 (18_image_process.py)**
    - 功能：批量调整图片大小、格式等
    - 特点：支持多种图片处理操作

19. **数据清洗 (19_data_clean.py)**
    - 功能：清洗和标准化数据
    - 特点：支持多种数据源和清洗规则

20. **定时任务自动化 (20_auto_task.py)**
    - 功能：创建定时执行的任务
    - 特点：支持多种触发条件和操作类型

## 技术栈

- **核心UI库**: PySide6 (Qt6的Python绑定)
- **办公自动化**: python-office, pandas, openpyxl, python-docx, PyPDF2
- **图像处理**: Pillow, matplotlib
- **PDF处理**: reportlab, pdf2image, PyMuPDF
- **打包工具**: PyInstaller

## UI设计

所有工具采用统一的科技感UI设计，具有以下特点：
- 深蓝色主题背景，营造专业感
- 渐变效果和半透明元素，增强视觉层次
- 统一的组件风格和交互方式
- 响应式布局，适应不同窗口大小

## 使用方法

### 运行单个工具
```bash
cd code/excel_tools
python 01_excel_merge.py
```

### 打包为可执行文件
```bash
cd code
python build_all.py
```

### 运行所有已打包的工具
- Windows: 双击 run_all.bat
- macOS/Linux: 执行 run_all.sh

## 安装依赖

```bash
pip install -r requirements.txt
```

## 开发说明

每个工具都基于统一的UI基类开发，具有以下结构：
- 继承自BaseMainWindow的主窗口类
- 文件选择组件用于输入
- 操作组件用于配置参数
- 预览组件用于显示结果
- 进度显示和状态反馈

## 后续扩展

项目采用模块化设计，便于后续扩展新功能：
1. 在对应工具目录下创建新的Python文件
2. 继承BaseMainWindow类实现UI
3. 实现具体功能逻辑
4. 更新build_all.py脚本以包含新工具

## 注意事项

1. 某些工具需要额外的依赖库，请参考requirements.txt
2. PDF转图片功能需要安装pdf2image或PyMuPDF
3. Word转PDF功能需要安装LibreOffice或Microsoft Word
4. 打包过程可能需要较长时间，取决于脚本数量和复杂度
