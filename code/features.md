# Python-Office 核心功能精选

## Excel相关工具 (5个)
1. **Excel表格合并** - 合并多个Excel文件到一个工作簿
2. **Excel数据筛选** - 根据条件筛选Excel数据
3. **Excel数据透视表生成** - 自动生成数据透视表
4. **Excel图表生成** - 根据数据生成各类图表
5. **Excel公式批量应用** - 批量应用公式到指定单元格

## Word相关工具 (5个)
6. **Word文档批量替换** - 批量替换文档中的文本
7. **Word文档合并** - 合并多个Word文档
8. **Word文档转PDF** - 将Word文档转换为PDF格式
9. **Word报告生成** - 基于模板生成Word报告
10. **Word表格处理** - 批量处理Word中的表格

## PDF相关工具 (5个)
11. **PDF合并** - 合并多个PDF文件
12. **PDF拆分** - 将PDF文件拆分为多个文件
13. **PDF水印添加** - 为PDF添加文本或图片水印
14. **PDF转图片** - 将PDF页面转换为图片
15. **PDF加密解密** - 为PDF添加或移除密码保护

## 其他办公自动化工具 (5个)
16. **文件批量重命名** - 根据规则批量重命名文件
17. **邮件批量发送** - 批量发送个性化邮件
18. **图片批量处理** - 批量调整图片大小、格式等
19. **数据清洗** - 清洗和标准化数据
20. **定时任务自动化** - 创建定时执行的任务
